# 你说我画

> **一句话描述**：一人描述，一人画图，不能说出关键词——看懂「表达」和「倾听」之间的鸿沟。

---

## 1. 概览

| 项目 | 内容 |
|------|------|
| **目的标签** | 💬 沟通表达 / 团队协作 |
| **形式** | 室内 · 智力+趣味 |
| **人数** | 2人/组，多组同时进行 |
| **时长** | 20–30 min |
| **场景标签** | 培训 / 团建 / 工作坊 |
| **难度** | ⭐ |
| **道具需求** | 有（纸 + 笔 + 图片） |

**核心目标**：让参与者体验「我以为我说清楚了，但对方理解完全不同」的沟通困境，学习精准表达和主动确认。

---

## 2. 核心规则

### 准备阶段（3 min）
1. 将参与者两两分组（一人为**描述者**，一人为**画图者**）。
2. 每组领取：A4 纸 1 张、笔 1 支。
3. 教练准备 **5–8 张神秘图片**（见下方题库），背面朝上。

### 第一轮：背对背（8 min）
1. **描述者**看到图片，**画图者背对**，不能看。
2. 描述者用语言描述图片内容，但**不能说出图片中的关键词**（如图片是「猫坐在月亮上」，不能说「猫」「月亮」）。
3. 画图者根据描述画图，**不能提问**（第一轮规则）。
4. 时间到，展示原图 vs 画图，全场爆笑。

### 第二轮：面对面（8 min）
1. 交换角色（上轮的描述者 ↔ 画图者）。
2. **允许画图者提问**，描述者可以回答，但**仍不能说关键词**。
3. 时间到，再次对比。

### 复盘（5–10 min）
- 哪组两轮都画得最像？
- 哪组第一轮和第二轮差别最大？为什么？

---

## 3. 引导要点

### 讲解时
> "这个游戏的名字叫『你说我画』，但真正的名字叫**『我以为我说清楚了』**。"

- 强调**不能说关键词**的规则（违者扣分或搞笑惩罚）
- 提醒画图者：**主动确认**比盲目画更重要

### 过程中
- 观察并记录：
  - 哪组描述者**最有逻辑**（先说整体，再说细节）？
  - 哪组画图者**最能问对问题**？
  - 有没有人用**比喻/类比**来描述？（这是高级沟通技巧）

### 展示时
- 把「原图」和「画图」并列展示，用手机拍照投屏效果更佳
- 引导大家发现：**最搞笑的画图，往往是最好的复盘素材**

---

## 4. 复盘问题（4F 模型）

### Facts（事实）
- 你们组第一轮画出了什么？第二轮有变化吗？
- 描述者最想说但不能说的关键词是什么？

### Feelings（感受）
- 当你画得完全不对，但描述者还在努力解释时，你是什么感受？
- 看到其他组的「神作」和「鬼画」时，你笑的同时有没有一丝共鸣？

### Findings（发现）
- 你们组描述者用的最有效的表达技巧是什么？（比喻？分步？确认？）
- 如果重来，你们会在**第一步**做什么调整？
- 这个游戏和你们工作中「布置任务」有什么相似之处？

### Future（未来）
- 回到工作岗位，当你需要向同事/下属解释一个复杂概念时，你会怎么做来确保对方听懂了？
- 你们团队可以建立一个什么样的「确认机制」？（如：「请复述一遍你理解的需求」）

## 5. 图片题库（教练专用）

| 难度 | 图片描述 | 关键词（禁止说出） |
|------|---------|-------------------|
| ⭐ 简单 | 太阳下有一朵小花 | 太阳、花、小 |
| ⭐ 简单 | 鱼在鱼缸里游泳 | 鱼、鱼缸、游泳 |
| ⭐⭐ 中等 | 猫坐在弯月亮上，旁边有星星 | 猫、月亮、星星、坐 |
| ⭐⭐ 中等 | 戴帽子的人在雨中撑伞 | 帽子、雨、伞、人 |
| ⭐⭐⭐ 困难 | 大象用鼻子卷着一棵树，树上有三只鸟 | 大象、鼻子、树、鸟 |
| ⭐⭐⭐ 困难 | 海底有两只章鱼在打乒乓球 | 章鱼、海底、乒乓球 |

> 💡 **教练提示**：
> - 简单图片适合第一轮热身
> - 困难图片留到第二轮（面对面+可提问）
> - 可以自己画简笔画，也可以打印谷歌图片

## 6. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **时间紧（15 min）** | 只做一轮，去掉「不能提问」限制 |
| **大型团建（40+人）** | 改为「你画我猜」——画图者画，描述者猜，更像综艺节目 |
| **培训深度版** | 增加「需求变更」环节——画到一半，教练宣布：「客户要加一个元素」 |
| **线上场景** | 用在线白板（Miro/Figma），描述者用语音，画图者用鼠标 |
| **管理培训版** | 增加「压力测试」——描述者边说，教练边播放噪音/干扰音乐 |

## 教练笔记

> 💡 **这个游戏的精髓**：让参与者在笑声中体验「表达者以为自己说清楚了，倾听者以为自己听懂了，但双方完全不在一个频道」的沟通困境。

**常见卡点**：
- 描述者直接说出关键词 → 教练介入：「你刚才是不是说了关键词？认罚还是重来？」（用幽默方式）
- 画图者完全不动笔 → 教练介入：「你是不是在等一个完美的开始？工作中你也这样吗？」

**安全提示**：纯智力游戏，无安全风险。但注意不要让画图能力差的人感到尴尬——强调「画得好不好不重要，沟通得好不好才重要」。
