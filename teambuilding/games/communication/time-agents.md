# 时间特工（团队推理版）

> 基于桌游《时间特工》改造，适合团建/培训的沟通推理游戏。  
> 原游戏：`boardgames/category/party/time-agents/`

---

## 1. 概览 + 标签

| 属性 | 说明 |
| :--- | :--- |
| **游戏名称** | 时间特工（团队推理版） |
| **目的标签** | 💬 沟通表达 / 🧊 破冰热场 |
| **形式标签** | 室内 · 智力+推理 |
| **人数** | 6–12人（最佳8人） |
| **时长** | 30–45 min |
| **难度** | 中等 |
| **场景标签** | 团建/培训/破冰 |
| **核心价值** | 锻炼信息分享、逻辑推理、信任判断 |

**游戏简介**：玩家分为"守护者"和"破坏者"两个阵营。守护者需要合作修复混乱的时间线，同时找出隐藏在队伍中的破坏者；破坏者则要伪装成守护者，暗中破坏时间线。通过信息交流、投票决策和身份推理，体验紧张刺激的心理博弈。

---

## 2. 核心规则（团建简化版）

### 准备阶段

1. **布置场地**：将玩家分成两组（A组/B组），每组3–6人
2. **身份分发**：
   - 每组有1–2名"破坏者"（教练提前私下告知）
   - 其余玩家都是"守护者"
   - **重要**：破坏者身份保密
3. **设置时间线**：在白板或大纸上画出5个时代（古代/中世纪/工业革命/现代/未来）
4. **初始状态**：所有时代标记为"混乱"状态

### 游戏流程（每轮）

| 阶段 | 时长 | 内容 |
| :--- | :--- | :--- |
| 信息交流 | 90秒 | 玩家自由讨论，分享"信息碎片"（教练准备） |
| 投票决策 | 30秒 | 投票选择要调整的时代 |
| 事件调整 | — | 根据投票结果调整时代状态 |
| 身份讨论 | 60秒 | 讨论谁可能是破坏者 |

### 胜利条件

- **守护者胜利**：所有5个时代都变为"稳定"状态，且找出所有破坏者
- **破坏者胜利**：3个或以上时代变为"崩溃"状态

### 简化调整（适合团建）

- 讨论时间延长到90秒（降低时间压力）
- 破坏者比例降低（6人游戏只有1名破坏者）
- 可以增加"线索卡"（教练每两轮公布一条线索）

## 3. 引导要点

### 教练讲解要点

1. **讲清核心冲突**：
   > "这个游戏的核心不是比谁聪明，而是——你能否在有限信息下，判断谁在说真话，谁在误导你。"

2. **强调心理博弈**：
   > "破坏者的目标是伪装成守护者。所以，看起来最积极修复时间线的人，不一定真的是守护者。"

3. **说明讨论规则**：
   > "90秒内，任何人都可以发言。你可以说真话，也可以说谎——但记住，破坏者一定会说谎。"

### 教练观察点

| 观察维度 | 关注什么 |
| :--- | :--- |
| **信息分享** | 谁主动分享信息？谁保持沉默？ |
| **投票行为** | 谁总是投给"让事件崩溃"的选项？ |
| **推理逻辑** | 谁的逻辑最清晰？谁容易被误导？ |
| **团队动态** | 有没有人被打标签？有没有人主导讨论？ |

### 关键把控点

- **时间控制**：严格按90秒/30秒执行，不要延长
- **身份保密**：确保破坏者身份不被意外暴露
- **引导讨论**：如果讨论冷场，教练可以提问："大家觉得古代事件应该先修复吗？"

## 4. 复盘问题（4F模型）

### Facts（事实层面）

- "刚才哪一刻，你觉得自己掌握了关键信息？"
- "破坏者是怎么暴露的？（或者为什么没暴露？）"
- "信息交流阶段，你们组分享了多少真实信息？"

### Feelings（感受层面）

- "作为守护者，当你怀疑某人是破坏者时，你是什么感受？"
- "作为破坏者，伪装成功时你是什么感受？"
- "时间压力下，你感到焦虑还是兴奋？"

### Findings（发现层面）

- "这个游戏和我们的团队协作有什么相似之处？"
- "在有限信息下做决策，我们平时工作中是否也面临这种情况？"
- "信任是怎么建立的？又是因为什么被破坏的？"

### Future（未来层面）

- "如果再来一次，你会改变什么策略？"
- "回到工作中，当你需要判断同事是否在说真话时，你会注意什么信号？"
- "今天我们学到了什么关于沟通和信任的经验？"

## 5. 变体适配

### 变体A：纯团队合作模式（去除破坏者）

**适用场景**：新团队破冰，不想制造对抗情绪

**规则调整**：
- 所有玩家都是守护者
- 目标：合作修复所有5个时代
- 增加难度：每轮只能调整1个时代，且信息碎片不完整

**复盘重点**：团队合作、信息共享、集体决策

---

### 变体B：跨部门对抗赛

**适用场景**：大型团建，多个部门参与

**规则调整**：
- 每个部门派出3–4人组成一队
- 各队同时进行游戏，计算"修复速度"
- 最快修复时间线的队伍获胜

**复盘重点**：跨部门协作、资源竞争、团队策略

---

### 变体C：无道具简化版（适合场地受限）

**适用场景**：没有桌游道具，或者时间有限

**规则调整**：
- 教练口头描述5个时代的状态
- 玩家通过举手投票决定调整哪个时代
- 教练暗中记录破坏者的破坏行为

**所需道具**：
- 白板笔 + 白板（画时间线）
- 计时器
- 身份卡（可以用扑克牌代替：红心=守护者，黑桃=破坏者）

## 7. 教练工具箱（补充）

### 引导话术示例

**游戏开始前**：
> "这个游戏没有标准答案。你可能是守护者，也可能是破坏者——但记住，破坏者的目标是要让你们相信他是守护者。所以，保持怀疑，但也要保持开放。"

**游戏中**：
> "还有30秒，大家加快讨论。如果你有关键信息，现在不说，可能就来不及了。"

**复盘开始时**：
> "我们先不揭晓身份。大家先分享一下，你是怎么判断谁是破坏者的？"

### 安全提示

- 如果团队氛围比较紧张，建议去掉"破坏者"设定，改为纯合作模式
- 如果有人玩得太投入（情绪激动），可以适当调侃缓解气氛
- 确保所有人都有发言机会，不要让1–2人主导讨论
