# 盲人方阵（Blind Polygon）



| **游戏名称** | 盲人方阵（Blind Polygon） |
| **目的标签** | 沟通表达 · 团队协作 · 信任建立 |
| **形式标签** | 室内/户外 · 10–30人 · 混合 |
| **人数** | 8–24 人（最佳 12–16 人） |
| **时长** | 中篇 30–45 min（含复盘） |
| **难度** | ⭐⭐ 中等 |
| **场景标签** | 团建 · 培训 · 通用 |
| **核心目标** | 体验精准沟通、角色自发分工、在信息不对称下协作 |

---

## 2. 核心规则

### 准备

- 一根长绳（长度 = 人数 × 1.2 米，例如 12 人需要约 15 米绳子）
- 眼罩（每人一个，或每组备 70% 的量）
- 空旷场地（移除障碍物）

### 流程

```
1. 所有人蒙上眼罩，教练将绳子放在场地中央（绳子是打结的圈）
2. 教练宣布挑战目标（例如："用这根绳子在地上摆成一个最大的正方形"）
3. 参与者只能靠语言沟通，不能通过手拉手来"感知"形状
4. 摆好后，所有人摘下眼罩，看结果
5. 讨论：和预期相差多少？哪里出了问题？
```

### 挑战目标示例

| 难度 | 目标 |
|------|------|
| 简单 | 摆成一个圆形 |
| 中等 | 摆成一个正方形（四边等长、四角直角） |
| 困难 | 摆成两个 concentric circles（同心圆）|
| 困难+ | 摆成数字"2026" 或团队 logo 形状 |

### 胜负判定

- **合作模式**（推荐）：全组一起挑战，可以多次尝试，最后讨论"哪次最好"
- **竞赛模式**：多个小组同时挑战同一目标，裁判测量误差，误差最小者获胜
