# 传话游戏（电话游戏）



| **时长** | 15–25 min |
| **场景标签** | 培训 / 团建 / 破冰 |
| **难度** | ⭐ |
| **道具需求** | 无（或纸条） |

**核心目标**：用幽默的方式让参与者亲身体验「信息在传递过程中的失真」，理解精准沟通的重要性。

---

## 2. 核心规则

### 准备阶段（2 min）
1. 将参与者分成 2–4 组，每组 5–10 人。
2. 每组排成一列，**不许回头**、**不许交流**。
3. 教练准备 **3–5 句「难传的话」**（见下方题库）。

### 游戏进行（10–15 min）
1. 教练把第一句话**悄悄**告诉每组的**队尾**那个人（只能说一遍！）。
2. 队尾的人**悄悄**传给前一个人（耳语，不能重复）。
3. 依次传递，直到队首。
4. 队首大声说出自己听到的内容。
5. 教练公布**原句**，全场爆笑。

### 进阶轮次（可选，5–10 min）
- 第二轮：增加干扰（背景音乐 / 教练在旁边故意说废话）
- 第三轮：允许传话人问一个问题（但不能重复说整句）

---

## 3. 引导要点

### 讲解时


### Feelings（感受）
- 当你发现听到的话和原句完全不一样时，你是什么感受？
- 看到其他组的「神变形」时，你笑的同时有没有一丝担忧？

### Findings（发现）
- 你们组失真的原因是什么？（听力问题？表达问题？理解问题？）
- 如果重来，你们会在哪个环节做调整？
- 这个游戏和你们团队日常沟通（邮件、微信、会议）有什么相似之处？

### Future（未来）
- 回到工作岗位，当你需要传递重要信息时，你会怎么做来确保准确性？
- 你们团队可以建立一个什么样的「沟通校验机制」？

## 5. 题库（教练专用）

| 难度 | 原句示例 |
|------|---------|
| ⭐ 简单 | 今天下班后一起去吃火锅 |
| ⭐ 简单 | 明天的会议改到下午三点 |
| ⭐⭐ 中等 | 这个项目需要在周五之前完成第一版方案，重点是用户体验和数据安全 |
| ⭐⭐ 中等 | 请转告小王，客户希望把交付时间提前一周，但预算不变 |
| ⭐⭐⭐ 困难 | 关于Q3的营销策略，CEO强调要聚焦存量用户的深度运营，而非盲目拉新，核心KPI是留存率而非转化率 |
| ⭐⭐⭐ 困难 | 请通知技术部，新功能的上线时间推迟到下周三，但需要在本周五前完成内部测试，重点是兼容iOS 12以上的版本 |



| **时间紧（10 min）** | 每组只传 1 句话，省去多轮 |
| **大型团建（30+人）** | 改为「传话接力」——全场合为一个大组，从第一个人传到最后一个人 |
| **培训深度版** | 增加「书写环节」——传话结束后，每人把自己听到的写在纸上，对比「听说」vs「读写」的失真差异 |
| **线上场景** | 用私信/微信一对一传递，最后一个人把结果发到群里 |
| **趣味加强版** | 教练故意给每组**不同的原句**，但内容相互矛盾，看最后各组如何「脑补」统一 |
