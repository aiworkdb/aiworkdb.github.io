# 人体拼字



| **时长** | 20–30 min |
| **场景标签** | 团建 / 培训 / 破冰 |
| **难度** | ⭐ |
| **道具需求** | 无（或题卡） |

**核心目标**：让团队在爆笑中体验「快速达成共识 + 分工协作」的快感。

---

## 2. 核心规则

### 准备阶段（2 min）
1. 将参与者分成 6–20 人小组（人数越多越难）。
2. 教练准备**题卡**（见下方题库），或现场出题。

### 比赛进行（15–20 min）
1. 教练宣布**题目**（一个字或词）。
2. 全组用**身体**（手、脚、身体姿态）拼出这个字/词。
3. **规则**：
   - 不能借助任何道具（纯人肉）
   - 每个人都必须参与（不能有人站着不动）
   - 拼好后，教练拍照确认

4. 每组轮流进行，**最快且最像**的组得分。

### 升级挑战（可选，5–10 min）
- **盲拼**：拼的时候，其他组不能看（用屏风/围圈）
- **观众猜**：拼好后，其他组猜是什么字（猜对也得的分）

---

## 3. 引导要点

### 讲解时


### Feelings（感受）
- 拼的时候，你是什么感受？（兴奋？混乱？好笑？）
- 看到其他组的「神作」时，你笑的同时有没有一丝佩服？

### Findings（发现）
- 你们组最快达成共识的是哪个字？当时发生了什么？
- 如果重来，你们会在**第一步**做什么不同的事？
- 这个游戏和你们团队「快速达成共识」有什么相似之处？

### Future（未来）
- 回到工作岗位，当你们团队需要快速共识时，你会怎么做？
- 你们团队可以建立一个什么样的**「快速共识协议」**？

---

## 5. 题库（教练专用）

| 难度 | 题目 | 说明 |
|------|------|------|
| ⭐ 简单 | 一、二、三、十、人、口 | 笔画少，容易拼 |
| ⭐⭐ 中等 | 团队、合力、信任、赢 | 2–4 个字，需要配合 |
| ⭐⭐⭐ 困难 | 全力以赴、共创未来 | 成语/长词，挑战极大 |
| ⭐⭐⭐⭐ 地狱 | 公司口号/品牌名 | 定制化，最有意义 |



| **时间紧（15 min）** | 只拼 3–5 个简单字，去掉投票环节 |
| **大型团建（50+人）** | 改为「拼图接龙」——每组拼一个字，连起来是一句话 |
| **培训深度版** | 增加「客户评审」环节——拼完后，教练扮演「客户」，给出「修改意见」，要求 2 分钟内调整 |
| **线上场景** | 用摄像头+身体拼字，截图后发到群里投票 |

## 7. 物资清单

| 物资 | 数量 | 备注 |
|------|------|------|
| 题卡（打印） | 教练 1 份 | 提前准备，防止现场想不出来 |
| 计时器 | 1 个 | 手机即可，投屏更佳 |
| 相机/手机 | 1 个 | 拍照记录「神作」 |
| 投票贴纸 | 1 本 | 用于投票环节（可选） |
