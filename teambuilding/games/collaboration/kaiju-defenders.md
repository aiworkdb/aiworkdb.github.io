# 怪兽守护者 (Kaiju Defenders)

## 🎮 游戏概述

《怪兽守护者》改编自同名合作桌游，是一款**团队协作模拟游戏**。参与者扮演守护城市的巨型怪兽团队，在面对外星入侵时，通过角色分工、资源调配和战术配合，共同保护城市重要设施。

这款游戏强调**跨部门协作、危机决策、资源优化**，适合团队协作训练、领导力发展、危机管理演练。

## 🎯 核心学习目标

1. **角色认知**：理解不同角色的价值和贡献
2. **资源优化**：在限制条件下做出最优决策
3. **危机沟通**：高压环境下的信息传递和决策
4. **团队协作**：跨越"部门"壁垒，为共同目标努力

## 🦖 角色分配

参与者分为 **2-4个怪兽小队**，每队 **3-5人**。每个小队包含以下角色：

### 小队角色（每队必备）

| 角色 | 职责 | 技能 |
| :--- | :--- | :--- |
| **队长** | 统筹决策 | 可查看事件卡堆顶；关键时刻可重新决策 |
| **通讯员** | 跨队沟通 | 可与其他小队传递信息；协调联合行动 |
| **资源官** | 管理能量核心 | 优化资源分配；交易资源时获得优势 |
| **战术员** | 战斗决策 | 攻击判定+1；可发动合作攻击 |
| **工程师** | 修复建筑 | 修复速度×2；可预防性加固 |

*注：3人小队可一人兼任多职；5人小队可增加"侦察员"（预判威胁）*

## 🎲 游戏流程

### 阶段一：威胁评估（10分钟）

1. **背景介绍**：外星舰队已抵达城市外围，需要在10个回合内摧毁母舰或坚持防御
2. **小队组建**：分组并分配角色
3. **初始资源**：每队获得3个能量核心（随机颜色）
4. **地图熟悉**：各队研究地图，制定初步策略

### 阶段二：防御作战（40分钟）

游戏按**回合制**进行，每回合包含以下步骤：

#### 小队回合（每队5-7分钟）

1. **情报共享**（1分钟）
   - 各队报告当前位置、资源、威胁
   - 通讯员协调跨队行动

2. **行动决策**（2分钟）
   - 选择移动目标（用骰子或抽签决定步数）
   - 决定资源使用策略
   - 规划技能释放时机

3. **执行行动**（2分钟）
   - 移动棋子（用便签标记位置）
   - 触发位置事件
   - 发动角色技能（如适用）

4. **外星回合**（1分钟）
   - 裁判翻开事件卡
   - 更新威胁状态
   - 检查失败条件

#### 回合示例

**第3回合（蓝队）**：
- 当前位置：工业区🏭
- 决策：使用移动技能，移动+2步
- 行动：到达外星基地👽，触发战斗
- 战斗：战术员掷骰得到4点，成功摧毁基地
- 奖励：获得1个能量核心
- 外星回合：翻开"能量武器"事件，摧毁1个住宅区

### 阶段三：总结复盘（15-20分钟）

见"复盘问题清单"部分

## ⚡ 核心机制

### 行动点系统

每回合每队有 **5个行动点**，可用于：
- 移动（1点/格）
- 攻击（2点）
- 修复（1点）
- 传递资源（1点）
- 发动技能（消耗能量核心）

### 能量核心系统

能量核心是发动技能的必需资源，分为三种：

| 类型 | 颜色 | 获取方式 | 用途 |
| :--- | :--- | :--- | :--- |
| 攻击核心 | 🔴 | 摧毁外星基地 | 用于攻击型技能 |
| 防御核心 | 🔵 | 修复建筑 | 用于防御型技能 |
| 移动核心 | 🟢 | 停留在特殊地形 | 用于移动型技能 |

### 合作攻击规则

多个小队可以同时攻击同一个外星基地：
- 每队独立进行攻击判定
- 任意一队成功即摧毁基地
- 鼓励跨队协作，共享胜利

---

## 🏆 胜利条件

**合作胜利**（所有小队共同获胜）：

1. **摧毁母舰**（进攻路线）
   - 至少一队到达母舰港口🚀
   - 连续成功3次攻击判定
   - 可以使用能量核心增加成功率

2. **坚持防御**（防守路线）
   - 坚持10个回合
   - 住宅区剩余≥3个

**失败条件**：
- 住宅区全部被摧毁（城市健康=0）
- 超过时间限制（10回合）未完成目标

## 🎓 教练引导要点

### 游戏前（设置阶段）

1. **营造氛围**
   - 用讲故事的方式介绍背景："外星舰队已抵达，城市危在旦夕..."
   - 强调这是**合作游戏**，不是竞争

2. **明确角色价值**
   - 解释每个角色的独特贡献
   - 鼓励玩家尝试不同角色

3. **制定沟通规则**
   - 建议：每回合开始时各队报告状态
   - 提醒：通讯员是关键桥梁

### 游戏中（进行阶段）

1. **观察互动**
   - 谁在主导决策？
   - 是否有人被忽视？
   - 跨队沟通是否顺畅？

2. **适时介入**
   - 如果某队落后太多，可以给予提示
   - 如果出现争执，引导回到合作目标

3. **控制节奏**
   - 保持游戏的紧张感
   - 避免某个环节过长

### 游戏后（复盘阶段）

使用"复盘引导框架"（见 teambuilding/toolbox/debrief.md）

## 🔄 变体规则

### 简化版（适合新手团队）

- 移除能量核心系统，所有技能免费使用
- 外星威胁降低（减少负面事件）
- 时间延长至12回合

### 挑战版（适合成熟团队）

- 增加"时间压力"：限制为8回合
- 资源稀缺：初始资源减半
- 信息不完全：某些事件卡不公开

### 无道具版（临时团建）

- 不使用任何打印素材
- 用口述代替卡牌
- 用手机计时器代替回合标记
- 适合临时决定

## 💡 设计思路

### 为什么选择这个游戏？

1. **高度合作**：所有玩家目标一致，天然促进团队感
2. **角色互补**：不同技能卡模拟真实工作中的角色分工
3. **危机感**：倒计时机制创造适度压力，激发潜能
4. **可扩展性**：通过调节难度适应不同团队

### 与工作场景的映射

| 游戏要素 | 工作场景 |
| :--- | :--- |
| 外星威胁 | 市场竞争、项目截止日期 |
| 能量核心 | 预算、人力、时间等资源 |
| 角色技能 | 团队成员的专长 |
| 跨队协作 | 跨部门合作 |
| 城市健康 | 客户满意度、项目健康度 |
