# 岛屿求生（团建改造版）

> **原游戏**：[岛屿求生](../../boardgames/category/cooperative/island-survival/README.md)
> **改造目标**：将合作生存桌游转化为团队协作 + 问题解决的高强度团建游戏

---

## 1. 概览

| 项目 | 内容 |
|------|------|
| **目的标签** | 🤝 团队协作 / 🔧 问题解决 / 沟通表达 |
| **形式** | 室内 · 策略+协作 |
| **人数** | 4–16人（分 1–4 组） |
| **时长** | 45–75 min |
| **场景标签** | 团建 / 培训 / 工作坊 |
| **难度** | ⭐⭐⭐ |
| **道具需求** | 有（打印地图 + 卡牌 + 标记物） |

**核心目标**：在资源有限、环境不确定的压力下，体验「团队协作决策」和「风险管理」——正是高效团队每天面对的挑战。

---

## 2. 团建改造要点

### 与原版的区别
| 原版规则 | 团建改造版 |
|---------|------------|
| 1–4 人游戏 | **4–8 人/组，每组有角色分工** |
| 自由分配行动 | **每轮有「角色讨论时间」（5 min）** |
| 胜利即结束 | **增加全程复盘节点**（每 3 天一次） |
| 单纯合作 | **增加「资源争夺」环节**（模拟部门墙） |

### 角色分工（每组 4–8 人）
| 角色 | 职责 | 领导力指标 |
|------|------|------------|
| **队长** | 最终决策（听取所有人意见后） | 决策力 |
| **资源官** | 管理食物/淡水/材料库存 | 精细化管理 |
| **探索员** | 探索新区域，汇报风险 | 信息收集 |
| **建造师** | 规划建造顺序和优先级 | 系统思维 |
| **天气预报员** | 分析天气卡，预警风险 | 风险管理 |

> 💡 每组可以自荐角色，也可以由教练指定（模拟组织任命）。

---

## 3. 改造后流程（60 min 标准版）

### 第一阶段：角色分配 + 规则讲解（10 min）
- 每组抽取/自荐角色
- 教练讲解核心规则（参考原游戏 README）
- **特别强调**：「每轮讨论 5 分钟，队长最终决策」

### 第二阶段：生存挑战（35 min）
- 每「天」= 5–7 分钟现实时间
- 第 3 天和第 6 天各暂停一次，做**迷你复盘**（2 min）
- 教练巡回观察，记录关键时刻

### 第三阶段：深度复盘（15 min）
- 按 4F 模型完整复盘
- 重点讨论：「你们组最危险的时刻是什么？怎么度过的？」

## 4. 引导要点

### 讲解时
> "这个游戏不是比谁先获救，是看你们**在压力下怎么一起做决定**。"

- 强调**角色分工**但不限制跨界帮助
- 提醒：**天气系统**是不可控变量，要学会适应而不是抱怨

### 过程中
- 观察并记录：
  - 谁最先发现「资源即将耗尽」的危机？
  - 队长做决策时，有没有**征求所有人意见**？
  - 遇到暴风雨（危机）时，团队是**更团结**还是**互相埋怨**？

### 迷你复盘时（第 3、6 天）
- 只问一个问题：「**现在资源够吗？如果不，谁注意到了？**」
- 这个瞬间往往能发现团队里的「**预警者**」

---

## 5. 复盘问题（4F 模型）

### Facts（事实）
- 你们在第几天遇到资源危机？怎么解决的？
- 谁的提议最终救了全组？

### Feelings（感受）
- 当队长做出一个你不赞同的决定时，你是什么感受？
- 暴风雨来的时候，你是对队长有信心，还是想指责？

### Findings（发现）
- 你们组**最会未雨绸缪**的是谁？TA 做了什么？
- 如果重来，你们会在**第几天**做什么不同的事？
- 这个游戏和你们团队面对「项目危机」时有什么相似？

### Future（未来）
- 回到工作岗位，当你发现「资源（时间/预算）快不够了」，你会怎么做？
- 你们团队可以建立一个什么样的**「预警机制」**？

## 6. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **人数少（4–6人）** | 去掉角色分工，每人自由行动，但每轮必须**全员同意**才能执行行动 |
| **时间紧（30 min）** | 改为「快节奏模式」——每天只有 3 分钟现实时间 |
| **培训深度版** | 增加「**部门墙**」环节——前 3 天每组独立，第 4 天宣布「现在可以交易资源了」 |
| **高管版** | 增加「**董事会压力**」——每 3 天教练宣布一个「总部指令」（额外限制或目标） |
| **线上场景** | 用 Tabletopia 或类似工具，视频会议讨论 |

## 8. 教练笔记

> 💡 **这个改造版的精髓**：把「合作生存游戏」变成「团队协作诊断工具」——你能从一组人玩 60 分钟里，看出他们所有的协作优点和痛点。

**观察要点清单**（教练随身记录）：
- [ ] 谁最先提出「我们应该先建 XX」？
- [ ] 队长是「命令型」还是「共识型」？
- [ ] 遇到危机时，是谁先提出解决方案？
- [ ] 有没有人一直没发言？
- [ ] 资源分配时，有没有人主动「让」给别人？

**常见卡点**：
- 队长**独断专行**，不听其他人 → 复盘时问：「如果你们的领导不征求意见就做决定，你会怎么反应？」
- **没人愿意当队长** → 教练介入：「领导力不是职位，是**在需要的时候站出来**。」
- 遇到危机时**互相埋怨** → 这正是好时机：「项目中出问题时，你们也是这样吗？」

**金句收藏**（用于复盘）：
- 「资源永远不够，关键是**怎么分配**。」
- 「危机来的时候，团队要么**更团结**，要么**更分裂**——你们是哪一种？」
- 「预警者是最被忽视的角色。你们团队里，有没有人总是说『这样会有问题』，但没人听？」
- 「生存不是一个人的事，是**每个人都在自己的角色里做好了**。」
