# 星际矿工（团队危机管理版）

> 基于桌游《星际矿工》改造，适合团建的团队合作与领导力观察游戏。  
> 原游戏：`boardgames/category/cooperative/space-miners/`

---

## 1. 概览 + 标签

| 属性 | 说明 |
| :--- | :--- |
| **游戏名称** | 星际矿工（团队危机管理版） |
| **目的标签** | 🤝 团队协作 / 👑 领导力 |
| **形式标签** | 室内 · 合作+危机管理 |
| **人数** | 5–8人（最佳6人） |
| **时长** | 45–60 min |
| **难度** | 中等 |
| **场景标签** | 团建/培训/工作坊 |
| **核心价值** | 锻炼团队协作、危机应对、资源分配、领导力观察 |

**游戏简介**：玩家扮演一支太空矿工团队的成员，在小行星带矿洞中执行采矿任务。团队必须合作应对陨石撞击、致命辐射、氧气泄漏等灾害，在矿洞坍塌前收集足够矿石并安全撤离。每位玩家有独特职业角色（工程师、医生、导航员、矿工、船长），角色分工和团队协作是胜利的关键。

## 2. 核心规则（团建扩展版）

### 人数扩展方案

原版支持1–5人，团建场景扩展至6–8人。提供两种方案：

#### 方案A：标准团队模式（5–6人）

- 使用原版5个职业角色（工程师、医生、导航员、矿工、船长）
- 6人时，增加1个"副船长"角色（协助船长，船长缺席时可以替代）
- 严格按角色能力分工

#### 方案B：大团队模式（7–8人）

- 增加新角色：
  - **科学家**：分析灾害卡，提前预警
  - **通讯员**：可以与"太空站"通讯，获得额外资源（教练扮演太空站）
- 或者分成2个小组（每组3–4人），小组间竞赛：哪组先采集到指定数量的矿石？

### 准备阶段

1. **布置地图**：将矿洞地图（A3纸打印）放在桌面中央
2. **选择角色**：每位玩家选择一个职业角色（确保角色能力互补）
3. **设置危机进度条**：从0开始
4. **初始化资源**：
   - 每位玩家获得起始装备（根据角色）
   - 矿石目标：团队总共需要采集15个矿石（可根据难度调整）
5. **制定策略**（5分钟）：
   - 船长统筹：谁负责采集？谁负责应对灾害？谁负责治疗？
   - 资源分配：装备卡如何分配？

### 游戏流程（每轮）

| 阶段 | 时长 | 内容 |
| :--- | :--- | :--- |
| 行动计划 | 60秒 | 团队讨论：本轮谁去哪里？做什么？ |
| 角色行动 | 每人1次 | 按角色能力执行行动（采集、修复、治疗、移动） |
| 灾害触发 | — | 翻开1张灾害卡，团队共同应对 |
| 危机进度 | — | 危机进度条+1（或根据灾害卡调整） |
| 撤离判定 | — | 如果采集够矿石且到达撤离点，可以撤离 |

### 胜利条件（合作胜利）

达成以下所有条件即获胜：

1. **采集目标**：团队采集到指定数量的矿石（如15个）
2. **安全撤离**：至少1名玩家到达撤离点（太空站）
3. **危机控制**：危机进度条 ≤ 8（不超过10）

### 失败条件

- 危机进度条到达10（矿洞坍塌）
- 所有玩家生命值耗尽
- 氧气耗尽（如果启用氧气机制）

## 3. 引导要点

### 教练讲解要点

1. **强调角色依赖**：
   > "你们每个人都有自己的特长，但也都有自己的弱点。工程师修不了伤病，医生挖不了矿石。你们必须互相依赖才能生存。"

2. **说明危机压力**：
   > "危机进度条是倒计时。每轮它都会上升，灾害卡会加速这个过程。你们必须平衡'采集'和'应对灾害'。"

3. **解释领导方式**：
   > "船长是统筹者，但不是独裁者。你可以听取大家的意见，也可以独断专行——但记住，如果决策错误，所有人一起承担后果。"

### 教练观察点

| 观察维度 | 关注什么 | 复盘提问方向 |
| :--- | :--- | :--- |
| **角色分工** | 角色分工是明确的还是模糊的？谁在做额外的工作？ | "你们是怎么分工的？有没有人做了角色之外的工作？" |
| **资源分配** | 装备卡和矿石如何分配？是按需分配还是谁抢到归谁？ | "当有多余的装备时，你们是怎么决定的？" |
| **危机应对** | 面对灾害卡，团队是冷静应对还是慌乱失措？ | "当翻开'氧气泄漏'这张灾害卡时，你们团队内部发生了什么？" |
| **领导方式** | 船长是民主协商还是独断专行？团队是否接受领导？ | "作为船长，你是怎么做决策的？作为队员，你接受船长的决策吗？" |
| **沟通效率** | 60秒行动计划时间，团队能达成共识吗？ | "每次行动计划时间，你们是快速达成共识还是争论不休？" |

### 关键把控点

- **角色平衡**：确保每组有工程师（修复）、医生（治疗）、导航员（路线）
- **时间控制**：严格按60秒行动计划时间执行
- **危机反馈**：当危机进度条上升时，提醒团队："现在危机进度是X，你们需要加快行动了。"
- **失败处理**：如果团队失败，不要直接重新开始，先引导复盘"为什么失败"

## 4. 复盘问题（4F模型）

### Facts（事实层面）

- "谁能描述一下，游戏中你们团队是怎么分工的？"
- "最危险的时刻是什么？你们是怎么应对的？"
- "船长是怎么做出关键决策的？"

### Feelings（感受层面）

- "当你角色的能力不足以应对当前危机时，你是什么感受？"
- "作为船长，当你做出错误决策导致团队受损时，你是什么感受？"
- "游戏结束时，你是感到成就感还是遗憾？为什么？"

### Findings（发现层面）

- "这个游戏和我们的团队协作有什么相似之处？"
- "在危机压力下做决策，这和我们工作中的什么场景类似？"
- "你们团队中有没有'自然领导者'？其他人是如何配合的？"

### Future（未来层面）

- "如果再来一次，你们会调整什么策略？"
- "回到工作中，当你需要在危机下快速决策时，你会注意什么？"
- "今天我们学到了什么关于团队合作和领导力的经验？"

## 5. 变体适配

### 变体A：竞争模式（资源争夺）

**适用场景**：观察团队在资源竞争下的行为

**规则调整**：
- 将玩家分成2个小组（每组3–4人）
- 每组有自己的危机进度条和矿石目标
- 增加"资源点"：地图上某些位置有额外装备卡，两组可以争夺
- 先完成目标的小组获胜

**复盘重点**：竞争 vs 合作、资源争夺、零和思维

---

### 变体B：无声危机模式（沟通观察）

**适用场景**：观察非语言沟通和默契

**规则调整**：
- 前3轮禁止语言交流（只能用手势或写字）
- 第4轮开始恢复正常交流
- 观察：无声环境下团队如何协作？

**复盘重点**：非语言沟通、默契程度、沟通方式多样性

---

### 变体C：角色轮换模式（领导力观察）

**适用场景**：观察不同的人担任领导角色时的团队动态

**规则调整**：
- 每3轮轮换一次船长角色
- 其他角色也可以轮换（可选）
- 观察：不同的船长领导方式有何不同？团队反应有何变化？

**复盘重点**：领导风格、团队适应性、权力交接

---

### 变体D：简化版（适合时间有限）

**规则调整**：
- 危机进度条缩短（从10格改为6格）
- 矿石目标减少（从15个改为10个）
- 游戏时长缩短至30分钟

**适用场景**：作为培训中的一个小环节

## 8. 竞争 vs 合作讨论（复盘延伸）

### 讨论主题

1. **资源分配哲学**：
   - 在资源有限的情况下，你们是优先考虑"团队整体"还是"个人生存"？
   - 这和你们工作中的资源分配有什么相似之处？

2. **领导方式反思**：
   - 船长的作用是"做决策"还是"协调共识"？
   - 你们平时工作中，领导者是怎么做决策的？

3. **危机下的团队动态**：
   - 危机压力下，团队是更团结还是更分裂？
   - 这和我们面对工作危机时的反应有什么相似之处？
