# 怪物厨师 (Monster Chef)

## 🎯 游戏概述（团建版）

《怪物厨师》是一款合作类桌游，灵感来源于热门动漫《迷宫饭》(Dungeon Meshi)。

玩家扮演一群误入地下城的冒险者，为了生存并深入地下城，必须——**狩猎怪物、采集食材、现场烹饪、大快朵颐！**

### 团建改造重点

- **合作而非竞争**：所有玩家共同对抗地下城，强调团队协作
- **角色分工**：每位玩家承担不同职责（战斗、烹饪、采集、指挥）
- **资源管理**：有限的体力和食材需要团队共同决策
- **复盘框架**：每轮结束后引导团队反思协作过程

## 🎲 游戏规则（团建版）

### 准备阶段

1. **布置地图**：将第1层地图放在桌面中央
2. **选择棋子**：每位玩家选择一个颜色的棋子，放在第1层起点🟢
3. **设置体力**：将共享体力标记为 **25点**（比原版多5点，降低难度）
4. **分配角色**（重要！）：每位玩家选择一个角色，详见下方"角色分工"
5. **分发起始物资**：
   - 每位玩家获得 **2张食材卡**（随机）
   - 团队共享 **2张厨具卡**（菜刀🔪×2）
   - 洗混所有卡牌堆
6. **决定先手**：掷骰子，点数最大者先手

### 角色分工（团建核心改造）

为加强团队协作，每位玩家需承担一个角色：

| 角色 | 职责 | 特殊能力 |
| :--- | :--- | :--- |
| **队长** | 协调团队决策，尤其在体力分配和风险管理上 | 可在任何玩家回合提供1次额外骰子（每局限3次） |
| **战斗专家** | 负责战斗决策，可协助其他玩家战斗 | 战斗时骰子上限值+1 |
| **厨师** | 负责烹饪决策，优化食材使用 | 烹饪成功率判定降低1点（即点数≥3即成功） |
| **侦察员** | 记录地图信息，提醒陷阱位置 | 可提前查看下一个节点的类型（不触发效果） |
| **医务员** | 关注团队体力，提醒及时恢复 | 在起点🟢休息时，体力恢复+1 |
| **物资管理员** | 管理食材卡和料理卡，提醒资源使用 | 起始多获得1张食材卡 |

> 💡 **团建提示**：第一轮可以让玩家自由选择角色，第二轮轮换角色，体验不同职责。

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 移动阶段

- 当前玩家掷1颗骰子，点数=可移动的节点数
- 沿着本层路径向前移动（不能后退）
- 停在目标节点后，执行该节点效果

#### 2. 节点事件执行

根据棋子停留的节点，执行相应动作：

| 节点类型 | 操作 |
| :--- | :--- |
| **起点** 🟢 | 可休息：恢复1点体力（每局限1次/人） |
| **食材点** 🟡 | 抽取1张随机食材卡加入手牌 |
| **怪物巢穴** 🔴 | 进行战斗（详见下方战斗规则） |
| **烹饪站** 🟠 | 消耗手牌食材，烹饪1道料理（详见烹饪规则） |
| **陷阱** 🟣 | 抽取1张陷阱卡，执行效果 |
| **楼层出口** 🔵 | 全体玩家到达出口后，进入下一层 |
| **Boss房间** ⭐ | 触发Boss战（详见Boss战规则） |

#### 3. 战斗规则

当玩家停留在怪物巢穴🔴时，触发战斗：

1. 翻开1张怪物卡，查看怪物战斗力（如：史莱姆战斗力=3）
2. 玩家掷2颗骰子，点数之和 ≥ 怪物战斗力 = **战斗胜利**
3. **合作规则**：玩家可邀请其他玩家协助战斗！协助者掷1颗骰子，点数加入主攻玩家的骰子点数中
4. **胜利**：获得怪物卡上标注的食材卡，加入手牌
5. **失败**：失去2点体力，留在当前节点（下回合可重试）

> 💡 **团建提示**：鼓励玩家在战斗前进行简短讨论："我们谁去战斗？谁协助？"

#### 4. 烹饪规则

当玩家停留在烹饪站🟠时，可烹饪料理：

1. 展示手牌中的食材卡，选择要烹饪的料理
2. 将所需食材卡放入弃牌堆
3. 掷1颗骰子判定烹饪成功率：
   - 点数 ≥ 4：**烹饪成功**，获得对应料理卡
   - 点数 ≤ 3：**烹饪失败**，食材已消耗，但未获得料理卡（可下次再试）
4. 若玩家拥有厨具卡，可提升成功率
5. 烹饪成功后，料理卡可立即食用或保留至需要时

#### 5. 食用料理

玩家可在**自己回合的任何时候**食用料理卡：

- 将料理卡放入弃牌堆
- 执行料理卡上的效果（恢复体力/获得Buff等）
- **注意**：攻击/防御Buff仅对下一场战斗有效

#### 6. 回合结束

- 检查共享体力：若体力 ≤ 0，游戏失败
- 检查是否到达楼层出口：若全体玩家均到达出口，进入下一层
- 将回合交给下一位玩家

### Boss战规则（第5层）

当任意玩家到达第5层Boss房间⭐时，触发Boss战：

1. **红龙战斗力** = 10（基础值）
2. **全员参与**：所有玩家围攻Boss，每回合大家依次攻击
3. **攻击方式**：每回合玩家掷2颗骰子，总和与红龙战斗力比较
   - 总和 ≥ 10：对红龙造成1点伤害
   - 若有攻击Buff：额外伤害
4. **红龙反击**：每回合结束后，红龙攻击随机一名玩家，失去2点体力
5. **胜利条件**：红龙受到5点伤害后倒下 → **全员胜利！**
6. **失败条件**：共享体力降至0 → **全员失败...**

## 💡 团建引导提示

### 游戏前（5分钟）

1. **介绍主题**："今天我们要一起深入地下城，团队合作狩猎怪物、烹饪美食、击败红龙！
2. **强调合作**："这不是竞争游戏，你们所有人是同一个团队，共享体力、共享胜利、共享失败。"
3. **角色选择**：让玩家自由选择角色，或教练根据玩家特点分配角色
4. **鼓励沟通**："游戏过程中，请大声讨论你们的策略！"

### 游戏中（观察要点）

教练或观察员应注意记录：

1. **谁在主导决策？** 是队长，还是某位强势玩家？
2. **谁很少发言？** 是否有玩家被边缘化？
3. **资源分配是否公平？** 食材和料理是否优先给了最需要的人？
4. **面对失败（战斗失败/烹饪失败）时，团队反应如何？** 是互相指责，还是共同寻找解决方案？
5. **是否在适当的时候使用料理？** 还是舍不得用，导致体力过低？

### 游戏后（复盘框架）

使用以下框架引导团队复盘（15分钟）：

#### 1. 直觉反应（3分钟）

> "游戏结束后，你现在的直觉感受是什么？用1个词或1句话描述。"

可能的回答：紧张、兴奋、遗憾、团结、混乱...

#### 2. 角色体验（5分钟）

> "请分享你在游戏中担任[角色]的体验：
> - 你的职责是什么？
> - 你觉得自己履行得好吗？
> - 你是否需要其他人的支持？他们是否提供了支持？"

#### 3. 关键决策反思（5分钟）

> "游戏中有一个关键决策点：[举例：是否在Boss战前使用龙血酒]
> - 我们是如何做出这个决策的？
> - 是否有不同意见？我们如何处理分歧？
> - 如果重新来一次，我们会怎么做？"

#### 4. 迁移到工作（2分钟）

> "这个游戏让你联想到工作中的哪个场景？
> - 我们刚才体验到的[协作/沟通/决策]模式，在真实工作中是否也存在？
> - 我们可以做哪些改变？"

## 🔄 变体玩法（团建专用）

### 变体1：观察员模式（适合大型团队）

如果团队人数超过6人，可以多出的玩家担任**观察员**：

**观察员职责**：
1. 记录团队协作过程（可使用观察记录表）
2. 不打断游戏，但在复盘时提供第三方视角
3. 关注：谁在主导、谁被忽略、决策过程、冲突处理

**观察记录表**（可在白板上绘制）：

| 观察维度 | 记录 |
| :--- | :--- |
| 主导决策者 | |
| 沉默者 | |
| 冲突时刻 | |
| 协作亮点 | |
| 改进建议 | |

### 变体2：限时挑战（增加紧张感）

- 设置沙漏计时 **50分钟**
- 在规定时间内击败红龙即为胜利
- 超时则游戏失败
- **适合**：时间有限的团建活动

### 变体3：角色轮换（深度体验）

- 每通过2层，轮换一次角色
- 确保每位玩家体验不同职责
- **适合**：强调"换位思考"的团建主题

### 变体4：无声协作（沟通挑战）

- 在第3-4层，玩家**不能说话**（只能用肢体语言和写字）
- 增加沟通难度，凸显非语言沟通的重要性
- **适合**：沟通技巧培训

## ⚙️ 难度调整

### 简单模式（新手团队/第一次团建）

- 共享体力起始值提升至 **35点**
- 红龙战斗力降低至 **8点**
- 烹饪成功率判定降低（点数 ≥ 3 即成功）
- 每位玩家起始多获得1张食材卡
- **适合**：刚组建的团队、非游戏爱好者

### 标准模式（推荐）

- 共享体力起始值 **25点**
- 红龙战斗力 **10点**
- 烹饪成功率判定（点数 ≥ 4 即成功）
- **适合**：大多数团建场景

### 困难模式（成熟团队/挑战主题）

- 共享体力起始值降低至 **20点**
- 红龙战斗力提升至 **12点**，且每回合反击伤害提升至 **3点**
- 增加"饥饿机制"：每通过一层，全体失去1点体力
- 怪物巢穴出现频率增加（地图增加🔴节点）
- **适合**：高层管理团队、强调抗压能力的团建

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：食材卡手牌上限是多少？**
A：每位玩家手牌上限为 **5张食材卡**。超过上限时，需选择丢弃至5张。料理卡和厨具卡不计入上限。

**Q：可以跳过战斗吗？**
A：不可以。停在怪物巢穴🔴必须进行战斗，但其他玩家可以协助你战斗。

**Q：烹饪失败后食材会返还吗？**
A：不返还。烹饪是消耗性操作，失败后食材进入弃牌堆。

**Q：如果团队体力降到0怎么办？**
A：共享体力=0时游戏立即失败。教练可以决定是否"复活"团队（恢复10点体力），但会作为失败记录。

**Q：观察员可以中途加入游戏吗？**
A：可以。如果观察员想体验游戏，可以在某一层结束后替换当前玩家。

## 📚 教练工具箱

### 引导话术

**游戏前**：
- "记住，你们是一个团队。一个人的失败就是所有人的失败。"
- "不要舍不得吃料理！及时的体力恢复是团队责任。"
- "我鼓励你们大声讨论策略，但请记住时间。"

**游戏中**（适时提醒）：
- "你们现在体力还剩多少？是否需要优先恢复？"
- "有没有人想尝试烹饪？我们现在有食材。"
- "队长，你在协调团队决策吗？"

**复盘时**：
- "让我们先安静30秒，回想一下游戏过程。"
- "我想听听[沉默的玩家]的想法，你当时在想什么？"
- "这个游戏让我们学到了什么关于团队协作的教训？"

### 安全与边界

1. **心理安全**：如果玩家对失败感到沮丧，教练应强调"这是游戏，失败是学习的机会"
2. **包容性原则**：如果玩家不擅长游戏，教练可提供额外指导，而非让其感到羞耻
3. **时间管理**：严格控制游戏时间和复盘时间，避免拖延
