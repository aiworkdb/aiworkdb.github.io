# 齐眉棍（手指传棍）

> **一句话描述**：全组人用食指托住一根棍子，从眉心高度降到地面——手指不能离开棍子。

---

## 1. 概览

| 项目 | 内容 |
|------|------|
| **目的标签** | 🤝 团队协作 / 沟通表达 |
| **形式** | 室内/户外 · 体力+协作 |
| **人数** | 8–20人（全员参与） |
| **时长** | 20–30 min |
| **场景标签** | 团建 / 培训 / 工作坊 |
| **难度** | ⭐⭐ |
| **道具需求** | 有（PVC管或竹竿 1–2 根） |

**核心目标**：让团队体验「每个人都对，但结果错了」的经典协作困境，理解系统思维和互相配合。

---

## 2. 核心规则

### 准备阶段（2 min）
1. 全员站成**面对面两排**（像夹道欢迎的阵型）。
2. 将一根轻质棍子（PVC管/竹竿/铝合金管）横放在大家**食指高度**。
3. 所有人的**食指伸直**，托住棍子底部（不能勾、不能抓）。

### 挑战目标
> 全组协作，让棍子从**眉心高度**缓慢降到**地面**。
> **规则**：所有人的食指必须始终接触棍子，任何人的手指离开 = 重新开始。

### 进行阶段（15–20 min）
1. 教练喊「开始」，全员用食指托住棍子。
2. 棍子会**诡异地往上升**（因为每个人都不自觉地用指尖顶了棍子）。
3. 全组需要**统一呼吸、统一下放**，让棍子平稳下降。
4. 成功落地 = 挑战完成 ✅
5. 如果有人手指离开棍子 = 从头再来 ❌

### 升级挑战（可选，5–10 min）
- **闭眼版**：所有人**闭上眼睛**，只用语言沟通来协调
- **无声版**：全程**不准说话**，只能用非语言信号

---

## 3. 引导要点

### 讲解时
> "这个游戏的神奇之处在于：**每个人都觉得自己做对了，但棍子就是往上升**。想知道为什么吗？来试试。"

- 强调**手指不能离开棍子**（这是最难的部分）
- 提醒：**不要用力顶**，轻轻托住即可

### 过程中
- 观察并记录：
  - 棍子第一次往上升时，谁最先说「等等，它在往上升」？
  - 有人开始**发号施令**了吗？效果如何？
  - 失败 3 次以上，团队情绪是**更团结**还是**开始互相埋怨**？

### 关键干预时机
| 情况 | 教练干预话术 |
|------|--------------|
| 棍子一直往上升，团队沮丧 | 「你们在对抗的不是棍子，是**每个人的本能反应**。停下来，想想怎么让所有人的动作同步。」 |
| 有人开始指责他人 | 「指责能让棍子下降吗？如果不能，那我们在做什么？」 |
| 成功一次后欢呼 | 「做得好！现在，**再做一次**，证明这不是运气。」 |

---

## 4. 复盘问题（4F 模型）

### Facts（事实）
- 你们试了几次才成功？最多的时候失败了几次？
- 棍子最反常（一直往上升）的时候，大家在做什么？

### Feelings（感受）
- 手指离开棍子被叫「重来」的时候，你是什么感受？
- 看到有人被指责时，你是什么感受？

### Findings（发现）
- 你们最终成功的关键是什么？（有人统一指挥？找到了节奏？）
- 这个游戏和你们团队工作中「每个人都觉得自己做对了，但结果不对」的情况有什么相似？
- 有人在这次游戏里展现了平时工作中没看到的特质吗？

### Future（未来）
- 回到工作岗位，当你们团队遇到「集体努力但结果不对」的情况时，你会怎么调整？
- 你们可以建立一个什么样的**「同步检查机制」**，让每个人的动作真正对齐？

---

## 5. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **人数多（20+人）** | 分成 2 组比赛，看哪组先成功 |
| **时间紧（15 min）** | 从膝盖高度开始（比眉心高度容易），降低难度 |
| **培训深度版** | 增加「角色分配」——指定 1 人指挥，其余人只能执行；然后轮换指挥 |
| **户外场景** | 用更长更重的棍子（如扫把柄），增加挑战 |
| **线上场景** | 改为「同步鼓掌」挑战——全组通过视频会议，实现完全同步的鼓掌 |

## 6. 物资清单

| 物资 | 数量 | 备注 |
|------|------|------|
| 轻质棍子（PVC管/竹竿） | 1–2 根 | 长度约 1–1.5 米，重量适中 |
| 备用棍子 | 1 根 | 防止折断 |
| 平整地面 | 1 块 | 最好在室内地板或平整草地 |
| 计时器 | 1 个 | 手机即可，记录最快纪录 |

> 💡 **道具替代方案**：没有 PVC 管？用**扫把柄**、**羽毛球拍**（去掉拍头）、**长尺子**（多人拿短棍需要更多配合）都可以。

## 教练笔记

> 💡 **这个游戏的精髓**：让团队亲身体验「**系统惯性**」——每个人都做对了（用食指托住），但集体行为却产生了意想不到的结果（棍子上升）。

**常见卡点**：
- 有人用手指**顶**棍子（而不是托）→ 教练介入：「你们每个人都在用力，但方向对吗？」
- 失败后有人**互相埋怨** → 这正是好时机：「埋怨能让棍子下降吗？如果不能，那我们现在应该做什么？」

**安全提示**：
1. 棍子掉落时可能砸到脚 → 提醒「松开手指，不要试图接住」
2. 人数多时，两端的人容易被忽略 → 教练特别关照两端参与者

**金句收藏**（用于复盘）：
- 「每个人都对，但结果错了——这不是某个人的问题，是**系统**的问题。」
- 「你们花了多少时间在『指责』上，又花了多少时间在『找方法』上？」
- 「真正的协作，不是每个人都用力，而是每个人都**用对力**。」
