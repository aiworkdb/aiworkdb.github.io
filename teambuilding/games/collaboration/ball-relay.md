# 珠行万里（乒乓球传递）



| **时长** | 20–30 min |
| **场景标签** | 团建 / 培训 / 工作坊 |
| **难度** | ⭐⭐ |
| **道具需求** | 有（PVC 半管 + 乒乓球 + 杯子） |

**核心目标**：让团队体验「每个人都很重要，任何一个环节失误都会导致全员重来」的协作困境。

---

## 2. 核心规则

### 准备阶段（3 min）
1. 全员站成一列或几列（根据人数）。
2. 每人领取 **1 根 PVC 半管**（剖开的 PVC 管，槽朝上）。
3. 教练在起点放乒乓球，终点放一个**纸杯/水杯**。

### 挑战目标


### 进行阶段（15–20 min）
1. 教练喊「开始」，起点的人把球放在自己的管子上。
2. 球滚到尽头时，那人**立刻跑到队尾**，把自己的管子接在最后。
3. 如此循环，直到球滚进终点杯。
4. **球掉地 = 全体重来**（这是最残酷也最练团队凝聚力的规则）。

### 升级挑战（可选，5–10 min）
- **计时赛**：多组比赛，最快完成的获胜
- **障碍版**：中间放一个椅子/障碍物，管子必须绕过去
- **无声版**：传递过程中**不准说话**

---

## 3. 引导要点

### 讲解时


### Feelings（感受）
- 球掉的那一刻，你是什么感受？有人埋怨你时呢？
- 成功把球送进杯子的那一刻，你是什么感受？

### Findings（发现）
- 你们最终成功的关键是什么？（节奏？站位？鼓励？）
- 这个游戏和你们团队项目中「一个环节出错，全组加班」有什么相似？
- 谁是你们团队里的「凝聚者」？TA 做了什么？

### Future（未来）
- 回到工作岗位，当你们团队有一个「掉球就全体重来」的项目时，你会怎么调整自己的态度？
- 你们团队可以建立一个什么样的**「不掉球文化」**？

---

## 5. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **人数多（30+人）** | 分成 2–3 组比赛，增加竞争性 |
| **时间紧（15 min）** | 去掉「掉地重来」，改为「掉地扣分，总分最少者胜」 |
| **户外场景** | 增加距离（10 米以上），用更长的管子 |
| **培训深度版** | 增加「角色分配」——有人专门负责鼓励、有人负责策略 |
| **线上场景** | 改为「在线接力」——用在线白板，每人画一段轨道，球（图标）依次经过 |

## 6. 物资清单

| 物资 | 数量 | 备注 |
|------|------|------|
| PVC 半管（剖开） | 每人 1 根 | 约 30–50 cm 长，五金店可定制剖开 |
| 乒乓球 | 3–5 个 | 备用，防止丢失 |
| 纸杯/水杯 | 1–2 个 | 终点容器 |
| 平整地面 | 1 块 | 室内地板或平整草地 |
| 计时器 | 1 个 | 手机即可，比赛时用 |
