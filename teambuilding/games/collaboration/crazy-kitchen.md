# 疯狂厨房（团队抗压协作版）

> 基于桌游《疯狂厨房》改造，适合团建的时间压力与团队协作游戏。  
> 原游戏：`boardgames/category/party/crazy-kitchen/`

---

## 1. 概览 + 标签

| 属性 | 说明 |
| :--- | :--- |
| **游戏名称** | 疯狂厨房（团队抗压协作版） |
| **目的标签** | 🤝 团队协作 / 💬 沟通表达 |
| **形式标签** | 室内 · 合作+时间压力 |
| **人数** | 4–8人（最佳6人） |
| **时长** | 30–45 min |
| **难度** | 简单 |
| **场景标签** | 团建/培训/破冰 |
| **核心价值** | 锻炼团队协作、时间管理、压力应对、角色分工 |

**游戏简介**：玩家扮演一支忙碌的厨师团队，在时间压力下快速完成各种订单。游戏融合了实时协作、手牌速配和角色扮演机制，营造出既紧张刺激又欢乐搞笑的厨房氛围。通过角色分工、快速决策和应对突发事故，体验团队合作在压力下的真实状态。

---

## 2. 核心规则（团建版）

### 角色分工（4个核心工位）

| 工位 | 职责 | 关键能力 |
| :--- | :--- | :--- |
| **灶台区** | 完成热菜类订单 | 专注、多任务处理 |
| **备菜台** | 处理冷食和简单菜品 | 预处理、效率 |
| **洗碗区** | 清理"打翻"事故 | 救火、耐心 |
| **出餐口** | 将完成的菜品交给顾客 | 统筹、传递 |

**扩展角色**（6–8人时）：
- **厨师长**：统筹全局，可以重新分配订单
- **采购员**：去食材仓库补充手牌

### 准备阶段

1. **布置厨房地图**：将厨房布局图（A3纸打印）放在桌面中央
2. **分配角色**：每位玩家选择一个工位角色
3. **设置计时器**：30秒沙漏（或手机计时器）
4. **初始化订单**：翻开3张订单卡作为当前任务
5. **分发起始手牌**：每位玩家获得5张食材卡

### 游戏流程（每轮）

| 阶段 | 时长 | 内容 |
| :--- | :--- | :--- |
| 订单展示 | — | 展示当前3张订单卡（所需食材） |
| 协作完成 | 30秒 | 玩家合作：出食材卡、完成订单 |
| 事故触发 | — | 随机翻开1张事故卡，团队应对 |
| 新订单 | — | 补充1张新订单卡 |
| 压力上升 | — | 厨房压力等级+1 |

### 胜利条件（合作胜利）

- **目标**：在压力等级到达10之前，完成 **12个订单**
- **失败条件**：压力等级到达10（厨房爆炸！）

### 简化调整（适合团建）

- 时间压力降低：每轮45秒（原计划30秒）
- 事故卡减少：每2轮才触发1次事故
- 可以增加"团队讨论时间"：每完成3个订单，有2分钟团队策略调整

## 3. 引导要点

### 教练讲解要点

1. **强调时间压力**：
   > "这个游戏的核心不是比谁聪明，而是——你们能不能在时间压力下，还能保持有效沟通？"

2. **说明角色依赖**：
   > "没有人能独自完成所有订单。灶台区需要备菜台提供食材，出餐口需要其他人把菜品送过来。你们是一个流水线。"

3. **解释事故机制**：
   > "事故卡是随机的，可能让你们失去手牌、增加压力、或者打翻已完成的菜品。关键不是避免事故，而是——事故发生后，你们如何快速恢复？"

### 教练观察点

| 观察维度 | 关注什么 | 复盘提问方向 |
| :--- | :--- | :--- |
| **沟通效率** | 30秒内，团队能完成多少次有效沟通？ | "刚才那轮，你们是怎么在30秒内分配任务的？" |
| **角色分工** | 角色分工是明确的还是混乱的？有没有人越权？ | "作为备菜台，当你发现灶台区缺食材时，你是怎么做的？" |
| **压力应对** | 压力等级上升时，团队是更团结还是更慌乱？ | "当压力等级到达7的时候，你们团队内部发生了什么？" |
| **错误恢复** | 发生事故（打翻菜品）后，团队是快速恢复还是崩溃？ | "刚才'火灾'事故让你们失去了2个完成的订单，你们是怎么调整的？" |
| **领导力** | 有没有人自然承担了领导角色？效果如何？ | "你们团队中有没有人主动协调大家？如果没有，为什么？" |

### 关键把控点

- **时间控制**：严格按30秒（或45秒）执行，不要延长
- **事故执行**：事故卡效果要严格执行，不要因为"太难"就忽略
- **压力反馈**：当压力等级上升时，提醒团队："现在压力是X，你们需要加快速度了。"
- **欢乐氛围**：这款游戏设计是"欢乐搞笑"的，不要因为失败而沮丧，强调"再来一次"的乐趣

## 4. 复盘问题（4F模型）

### Facts（事实层面）

- "谁能描述一下，游戏中你们团队是怎么分工的？"
- "最紧张的时刻是什么？当时压力等级是多少？"
- "你们完成了几个订单？是在什么时候开始跟不上的？"

### Feelings（感受层面）

- "当30秒倒计时开始，你感到兴奋还是焦虑？"
- "当订单卡需要的食材你正好有，但你在出餐口够不到时，你是什么感受？"
- "游戏结束时，你是感到'意犹未尽'还是'终于结束了'？"

### Findings（发现层面）

- "这个游戏和我们的团队协作有什么相似之处？"
- "在时间压力下做决策，这和我们工作中的什么场景类似？"
- "你们团队中有没有'瓶颈工位'？就是那个总是跟不上的环节？这和我们工作中的什么类似？"

### Future（未来层面）

- "如果再来一次，你们会调整什么策略？"
- "回到工作中，当你需要在时间压力下快速决策时，你会注意什么？"
- "今天我们学到了什么关于团队合作和压力管理的经验？"

## 5. 变体适配

### 变体A：无声厨房模式（沟通观察）

**适用场景**：观察非语言沟通和默契

**规则调整**：
- 前3轮禁止语言交流（只能用手势或写字）
- 第4轮开始恢复正常交流
- 观察：无声环境下团队如何协作？

**复盘重点**：非语言沟通、默契程度、沟通方式多样性

---

### 变体B：角色轮换模式（抗压观察）

**适用场景**：观察不同的人在不同角色下的表现

**规则调整**：
- 每完成3个订单，轮换一次角色（顺时针）
- 每个人都要体验所有4个核心工位
- 观察：不同的人在相同工位下，效率有何不同？

**复盘重点**：角色适应、多技能培养、团队灵活性

---

### 变体C：竞争模式（跨部门对抗）

**适用场景**：大型团建，多个部门参与

**规则调整**：
- 每个部门派出3–4人组成一队
- 各队同时进行游戏，计算"完成订单数"
- 完成订单最多的队伍获胜
- 增加"干扰环节"：每轮可以选一张事故卡"送给"对方队伍

**复盘重点**：跨部门协作、竞争压力、团队策略

---

### 变体D：无道具简化版（适合场地受限）

**适用场景**：没有桌游道具，或者时间有限

**规则调整**：
- 教练口头描述订单（如"需要：牛肉+西红柿+米饭"）
- 玩家通过举手示意"我有这个食材"
- 团队合作：在10秒内，举起正确数量的手（表示凑齐食材）
- 教练计数：凑齐次数多的队伍获胜

**所需道具**：
- 白板笔 + 白板（写订单）
- 计时器

**准备时间**：5分钟以内

## 7. 教练工具箱（补充）

### 引导话术示例

**游戏开始前（角色分配）**：
> "你们有3分钟时间选择工位。记住，没有'最好的工位'，只有'最适合你们团队当前状态的分工'。灶台区和出餐口沟通最频繁，备菜台需要效率，洗碗区需要耐心。"

**游戏中（时间压力）**：
> "还有10秒！谁能凑齐'牛肉+西红柿+米饭'？快快快！"

**事故触发时**：
> "哦不！'火灾'事故！灶台区失去所有手牌。你们现在需要快速调整策略——备菜台，你能把食材直接送到出餐口吗？"

**复盘开始时**：
> "我们先不讨论胜负。我想知道，当压力等级到达8的时候，你们团队是加快了速度，还是开始互相指责？"

### 抗压复盘框架（专门针对这款游戏）

| 压力等级 | 团队状态 | 复盘提问 |
| :--- | :--- | :--- |
| 1–3（低压力） | 轻松、欢乐 | "刚开始的时候，你们是怎么分配任务的？有没有人偷懒？" |
| 4–6（中压力） | 开始紧张 | "当压力等级到达5的时候，你们团队内部的沟通方式变了吗？" |
| 7–9（高压力） | 紧张、可能慌乱 | "压力等级7的时候，你们有没有人开始慌张？这时候团队领导出来了没？" |
| 10（崩溃） | 失败 | "厨房爆炸了。现在，你们是互相指责，还是一起分析'为什么爆炸'？" |
