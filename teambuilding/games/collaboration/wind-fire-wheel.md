# 风火轮（报纸履带）

> **一句话描述**：全组站进报纸履带里，协力走到终点——走着走着，报纸就破了。

---

## 1. 概览

| 项目 | 内容 |
|------|------|
| **目的标签** | 🤝 团队协作 / 沟通表达 |
| **形式** | 户外/室内 · 体力+协作 |
| **人数** | 8–15人/组，多组竞赛 |
| **时长** | 30–40 min |
| **场景标签** | 团建 / 培训 / 工作坊 |
| **难度** | ⭐⭐ |
| **道具需求** | 有（大量报纸 + 胶带） |

**核心目标**：让团队在「资源会耗尽」的压力下，体验协作、节奏统一和面对失败的态度。

---

## 2. 核心规则

### 准备阶段（5 min）
1. 将参与者分成 8–15 人小组。
2. 每组领取：报纸 20–30 张、透明胶带 2–3 卷。
3. 宣布挑战：**用报纸和胶带做一个「履带」，全组站进去，走到终点再走回来**。

### 制作阶段（10–15 min）
1. 各组用报纸和胶带制作「履带」——把报纸连接成一个大圆圈，能容纳全组站立。
2. 教练提醒：**接头要牢固**，否则走一半会断。

### 比赛阶段（10–15 min）
1. 各组站进自己的「履带」里。
2. 听到「开始」，全组**同步迈步**，履带向前滚动。
3. **规则**：
   - 报纸破了可以**现场修补**（提前预留修补材料）
   - 有人踩出履带 = 暂停，所有人合力把 TA 拉回来
   - 最快往返的组获胜

### 复盘（5–10 min）
- 哪组最快？他们的履带有什么特点？
- 报纸破了的时候，团队是怎么反应的？

---

## 3. 引导要点

### 讲解时
> "你们做的履带，走着走着**一定会破**。破的时候，你们的反应决定了输赢——还有团队的凝聚力。"

- 强调**同步迈步**的重要性（节奏统一）
- 提醒：预留一些报纸和胶带用于**中途修补**

### 过程中
- 观察并记录：
  - 哪组**最先统一节奏**？（协作指标）
  - 报纸第一次破时，团队是**一起修**还是**互相埋怨**？
  - 有没有人主动**指挥节奏**？（领导指标）

### 比赛时
- 报纸破的那一刻，是全场最高能的瞬间
- 观察：破的次数少的组，是不是**走得更慢更稳**？（慢就是快）

---

## 4. 复盘问题（4F 模型）

### Facts（事实）
- 你们的履带破了几次？每次是怎么修补的？
- 最快的组用了多少时间？

### Feelings（感受）
- 报纸第一次破的时候，你是什么感受？
- 看到其他组走得很稳，你们在着急吗？

### Findings（发现）
- 你们组最快迈出第一步之前，发生了什么？
- 如果重来，你们会在**制作阶段**做什么不同的事？
- 这个游戏和你们团队做项目（边做边修 bug）有什么相似之处？

### Future（未来）
- 回到工作岗位，当你们的项目「破了」（出问题）时，你会怎么反应？
- 你们团队可以建立一个什么样的**「修补文化」**？

---

## 5. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **人数少（< 8人）** | 改为「两人三脚报纸版」——两人共用一小段报纸履带 |
| **时间紧（20 min）** | 去掉制作环节，教练提前做好履带（用厚纸/布代替报纸） |
| **户外场景** | 增加距离（20 米以上），用更牢固的材料（编织袋+胶带） |
| **培训深度版** | 增加「客户变更」环节——比赛中途教练宣布：「终点改到那边！」 |
| **线上场景** | 改为「线上接力」——每人用自己家的报纸做一段，视频会议拼接 |

---

## 6. 物资清单

| 物资 | 数量 | 备注 |
|------|------|------|
| 报纸（大张） | 每组 30–40 张 | 提前收集，越多越好 |
| 透明胶带 | 每组 2–3 卷 | 宽胶带更牢固 |
| 剪刀 | 每组 1–2 把 | 注意安全 |
| 卷尺/标志物 | 1–2 个 | 标记起点和终点 |
| 计时器 | 1 个 | 手机即可 |
| 备用报纸 | 若干 | 用于中途修补 |

> 💡 **道具替代方案**：没有足够报纸？用**编织袋**、**大垃圾袋**剪开代替，更牢固。

## 教练笔记

> 💡 **这个游戏的精髓**：让团队在「一定会坏」的事情里，体验**一起修**比**互相骂**有用得多。

**常见卡点**：
- 制作时有人**不参与**，站在一边看 → 教练介入：「你是在等别人把履带做好，然后你来享受成果吗？工作中你也这样吗？」
- 报纸破了，团队**崩溃** → 这正是好时机：「项目上线后出 bug 了，你们是这样反应的吗？」

**安全提示**：
1. 走路时注意不要摔倒（报纸滑）
2. 剪刀使用需注意安全
3. 户外进行时，注意地面是否平整（防止绊倒）

**金句收藏**（用于复盘）：
- 「你们花了多少时间在『埋怨履带不结实』，又花了多少时间在『一起修』？」
- 「最好的履带不是最结实的，是**破了之后最快修好的**。」
- 「慢就是快。那些走得很稳的组，往往比急着冲的组更快到终点。」
