# 台词之王（团队赛版）

> 基于桌游《台词之王》改造，适合团建/培训的表演沟通游戏。  
> 原游戏：`boardgames/category/party/line-master/README.md`

---

## 1. 概览 + 标签

| 属性 | 说明 |
| :--- | :--- |
| **游戏名称** | 台词之王（团队赛版） |
| **目的标签** | 🧊 破冰热场 / 💬 沟通表达 |
| **形式标签** | 室内 · 表演+即兴 |
| **人数** | 6–20人（最佳8–12人） |
| **时长** | 30–50 min |
| **难度** | 简单 |
| **场景标签** | 团建/培训/破冰/年会暖场 |
| **核心价值** | 降低社交压力、练习即兴表达、团队欢笑连接 |

**游戏简介**：每轮一位"演员"抽取场景卡+角色卡+情绪卡，即兴表演一段台词。观众用掌声/贴纸投票。团建版去掉了竞争性的"导演卡"个人对抗，改为**团队积分制**，并增加"团队接力表演"等变体，让全员参与、笑声最大化。

## 2. 核心规则（团建团队赛版）

### 准备阶段

1. **分组**：将玩家分为2–4队，每队3–5人
2. **准备卡牌**（三叠）：
   - 场景卡：写在便利贴或卡片上
   - 角色卡：同上
   - 情绪卡：同上
3. **每队初始分数**：0分
4. **确定顺序**：各队轮流派出演员，顺序自定

### 游戏流程（每轮）

| 阶段 | 时长 | 内容 |
| :--- | :--- | :--- |
| 抽卡 | 30秒 | 演员从三叠卡中各抽1张，展示给观众 |
| 准备 | 45秒 | 演员思考台词（团队可以帮忙出主意） |
| 表演 | 60秒 | 演员表演，团队可以当"背景板"辅助 |
| 投票 | 30秒 | 其他队用掌声/贴纸投票 |
| 计分 | — | 获得最多投票的演员为其队伍+3分 |

### 胜利条件

- 所有玩家各表演1轮后，总分最高的队伍获胜
- 如果平局：各队派1名代表进行"终极对决"（同一张卡，同时表演，全体投票）

## 3. 引导要点

### 教练引导话术

**游戏开始前**（降低压力）：
> "这个游戏没有'演得好'的标准答案。真正加分的是——观众笑了。所以，别追求演技，追求笑点！"

**抽卡后**（帮助思考）：
> "抽到'严肃领导'和'惊慌失措'？想想你们领导最怕什么事——然后把那个场景演出来！"

**表演结束后**（强化团队感）：
> "让我们把掌声送给XX队的所有人——因为表演虽然是XX一个人，但笑声是大家一起创造的！"

### 教练观察点

| 观察维度 | 关注什么 |
| :--- | :--- |
| **参与度** | 有没有人始终拒绝表演？是否需要调整为"团队出一个人辅助"模式？ |
| **团队支持** | 表演者的队友是在帮忙，还是在笑场帮倒忙？ |
| **欢笑质量** | 是所有人都在笑，还是只有个别人笑？ |
| **创意亮点** | 哪个表演最有创意？可以在复盘时提及 |

### 关键把控点

- **自愿原则**：如果有人极度不愿意表演，可以改为"团队代言人"（团队帮他出主意，他只需要念台词）
- **时间控制**：每轮严格控制在3分钟以内，保持节奏感
- **不评判表演质量**：只投票"最有趣"，不投票"最差"——避免负面情绪

## 4. 复盘问题（4F模型）

### Facts（事实层面）

- "今天哪个表演让全场笑得最大声？为什么？"
- "有没有哪个表演，大家笑的点和你不一样？"
- "哪一组的团队合作最默契（比如有人当背景板）？"

### Feelings（感受层面）

- "轮到你表演的时候，你是什么感受？紧张？兴奋？想逃跑？"
- "看到队友在表演，你在台下是什么感受？"
- "有没有哪个瞬间，你觉得自己'居然可以这么放得开'？"

### Findings（发现层面）

- "这个游戏和工作中的'即兴应对'有什么相似之处？"
- "我们在台上'不顾形象'地表演，和平时工作中的'保持专业形象'，哪一种让团队更亲近？"
- "今天有没有发现某个同事'意想不到的一面'？"

### Future（未来层面）

- "回到工作中，如果我们能像今天一样'不怕出丑'，团队会有什么变化？"
- "今天学到的'快速构思'能力，能怎么用在工作中？"
- "如果下次团建再玩这个游戏，你会改变什么策略？"

## 5. 变体适配

### 变体A：团队接力表演（推荐8人以上）

**适用场景**：希望全员深度参与，不只是1人表演

**规则调整**：
- 每队同时派2–3人上台
- 抽到同一套卡（场景+角色+情绪）
- 3人轮流每人说1句台词，连成一个完整"场景"
- 投票投给"最默契的团队表演"

**复盘重点**：团队协作、默契、接力创意

---

### 变体B：主题限定版（适合有明确团建主题）

**适用场景**：团建主题明确（如"创新""客户服务""团队协作"）

**规则调整**：
- 所有场景卡围绕主题设计
- 示例（创新主题）：
  - S01：你向老板提议"周五下午全体去公园开会"，老板说…
  - S02：你发明了一款"可以吃的PPT翻页笔"…
- 表演后增加"主题连接"环节：这个表演和"创新"有什么关系？

**复盘重点**：主题深化、价值观对齐、创意思维

---

### 变体C：零道具版（适合场地受限/时间紧急）

**适用场景**：没有准备卡片，或者临时起意要玩

**规则调整**：
- 教练口述场景、角色、情绪（现场编）
- 示例："现在请你演——在电梯里遇到老板，你刚发的朋友圈被他刷到了，角色是'心虚的你'，情绪是'强装镇定'！"
- 用手机计时，用举手投票代替贴纸

**所需道具**：
- 手机计时器
- 教练的想象力和急智

**准备时间**：0分钟（随口即来）

---

### 变体D：职场专属版（最适合企业团建）

**适用场景**：企业内部团建，希望结合工作场景

**规则调整**：
- 场景卡全部改为职场相关：
  - "需求又改了，你说'好的'的时候内心在想什么"
  - "周报写不出，你决定把上周的复制粘贴改个日期"
  - "团建你去不了，理由是'加班'——但其实你要去撸猫"
- 角色卡改为职场角色：
  - 你的直属领导 / 你的下属 / 合作部门的同事 / 客户 / HR
- 情绪卡不变

**复盘重点**：工作压力释放、职场共鸣、团队共情

## 7. 教练工具箱（补充）

### 引导话术示例

**帮助紧张者**：
> "记住——台下的人不是在评委，他们是在等着笑。你越紧张，他们越期待，因为紧张本身就是笑点！"

**处理"我不知道怎么演"**：
> "不知道怎么演？那就把你想说的话，用最不像你的方式说出来——比如你平时说话很大声，现在就用最小的声音说。"

**提升团队感**：
> "请XX队的队友们注意——你们不只是观众，你们是'气氛组'！适当的时候可以'咦——'、'哇——'来配合表演者！"

### 常见问题处理

| 问题 | 处理方式 |
| :--- | :--- |
| 有人拒绝表演 | 改为"团队代言人"模式：团队帮他出主意，他念稿即可 |
| 有人表演时其他人笑场到无法进行 | 温和干预："请大家给表演者一点空间，等他演完再笑！" |
| 某队一直得0票，士气低落 | 教练可以"颁发特别奖"："XX队虽然票数不多，但创意满分！" |
| 时间超时 | 温和打断："精彩！我们留一点悬念，下轮继续！" |

### 安全提示

- **心理安全**：如果团队信任度较低，第一次建议用"变体C（零道具版）"，教练先示范
- **不强迫**：永远给"Skip"选项，不要点名强迫某人表演
- **避免职场敏感内容**：场景卡不要涉及真实同事/领导的具体事件，保持"泛化"
- **注意文化差异**：如果团队有跨文化成员，避免使用需要特定文化背景才能理解的笑点
