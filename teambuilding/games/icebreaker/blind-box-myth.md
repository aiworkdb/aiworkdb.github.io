# 盲盒神话（团建改造版）


> **原游戏**：[盲盒神话](https://aiworkdb.github.io/teambuilding/boardgames/category/party/blind-myth/README.html)


| **时长** | 30–40 min |
| **场景标签** | 团建 / 培训 / 破冰 |
| **难度** | ⭐ |
| **道具需求** | 有（盲盒人物卡 + 行动卡 + 金币代币） |

**核心目标**：通过交易谈判，体验「沟通」和「共赢思维」——不是零和游戏。

---

## 2. 团建改造要点

### 与原版的区别
| 原版规则 | 团建改造版 |
|---------|------------|
| 集齐一套即获胜（竞争） | **15 分钟内，全组收集套数最多**（合作竞争） |
| 个人计分 | **小组计分**，鼓励内部交易 |
| 稀有卡强力 | 保留，增加「分享稀有卡」的加分项 |

### 改造后流程（30–40 min）

**准备（5 min）**
- 4–8 人一组，每组一套完整卡牌
- 每人发 10 枚金币

**交易环节（20 min）**
- 自由交易，教练播放轻快背景音乐
- 每 5 分钟教练宣布「市场行情更新」（虚构价格波动，增加趣味性）
- **关键规则**：每送出去 1 张卡给别人，得 1 分（鼓励分享）

**展示 + 复盘（10–15 min）**
- 每组展示收集到的套数
- 教练问：「你们组最会交易的是谁？最愿意分享的呢？」

## 3. 引导要点

### 讲解时


### Feelings（感受）
- 有人拒绝和你交易时，你是什么感受？
- 成功完成一笔双赢交易时，你是什么感受？

### Findings（发现）
- 你们组最成功的交易，是怎么达成的？
- 这个游戏和你们工作中「跨部门协作」有什么相似？
- 有没有人囤积卡不交易？工作中也有这样的人吗？

### Future（未来）
- 回到工作岗位，当你需要「谈判」而不是「命令」时，你会怎么做？
- 你们团队可以建立一个什么样的**分享文化**？

## 5. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **人数多（8+人/组）** | 分 2 组，增加组间交易环节 |
| **时间紧（20 min）** | 去掉「市场行情」播报，直接交易 |
| **培训深度版** | 增加「角色卡」——每人抽一个谈判风格（强硬型/合作型/分析型），必须按角色谈判 |
| **线上场景** | 用在线卡牌工具（如 Tabletopia）或微信群发卡牌图片交易 |
| **信任加强版** | 增加「盲交易」——不看对方卡面，纯靠描述谈判 |
