# 探险家日记（分享引导版）

> 基于桌游《探险家日记》改造，适合团建/培训的破冰分享游戏。  
> 原游戏：`boardgames/category/party/explorer-journal/`

---

## 1. 概览 + 标签

| 属性 | 说明 |
| :--- | :--- |
| **游戏名称** | 探险家日记（分享引导版） |
| **目的标签** | 🧊 破冰热场 / 💬 沟通表达 |
| **形式标签** | 室内 · 分享+故事 |
| **人数** | 6–16人（最佳8–12人） |
| **时长** | 30–45 min |
| **难度** | 简单 |
| **场景标签** | 团建/培训/破冰/新员工融入 |
| **核心价值** | 促进成员相互了解、练习倾听与表达、建立心理安全 |

**游戏简介**：每位玩家抽取"地点卡"和"事件卡"，结合自己的真实经历，分享一段与卡片主题相关的个人故事。其他玩家通过"共鸣贴纸"投票，选出最打动人心的故事。游戏重点是**真实分享**而非虚构故事，通过故事让大家看到彼此鲜为人知的一面。

---

## 2. 核心规则（团建分享版）

### 准备阶段

1. **布置场地**：椅子围成圆圈或U型，确保每个人都能看到其他人
2. **准备卡片**：
   - 地点卡：写出团队所在城市/省份/国家（让成员抽到的都是大家熟悉的地点）
   - 事件卡：改为"话题卡"，如「一次尴尬的经历」「最自豪的瞬间」「第一次…」
3. **发放共鸣贴纸**：每人5枚（可用便利贴撕小代替）
4. **确定分享顺序**：按顺时针，或从最年长/最年轻的人开始

### 话题卡参考（直接替换原事件卡）

| 编号 | 话题 | 分享难度 |
| :--- | :--- | :--- |
| T01 | 分享一个你小时候的梦想 | ⭐ |
| T02 | 说一件工作上让你最自豪的事 | ⭐⭐ |
| T03 | 你遇到过最尴尬的场面 | ⭐ |
| T04 | 如果可以重新选择，你还会做现在的工作吗？ | ⭐⭐⭐ |
| T05 | 分享一个你从失败中学到的教训 | ⭐⭐ |
| T06 | 你最喜欢的团队经历 | ⭐⭐ |
| T07 | 如果明天是世界末日，你最想和谁在一起？ | ⭐⭐ |
| T08 | 说一个你克服困难的真实故事 | ⭐⭐ |
| T09 | 你人生中最关键的转折点 | ⭐⭐⭐ |
| T10 | 分享一个你的小癖好或有趣习惯 | ⭐ |

### 游戏流程（每轮）

| 阶段 | 时长 | 内容 |
| :--- | :--- | :--- |
| 抽卡 | 30秒 | 分享者抽1张地点卡+1张话题卡 |
| 思考 | 60秒 | 结合两张卡，准备一段真实故事（2–3分钟） |
| 分享 | 3分钟 | 讲述故事，其他人认真倾听 |
| 共鸣投票 | 30秒 | 听众用贴纸投票（最打动你的故事） |
| 追问环节 | 2分钟 | 其他人可以提问或分享类似经历 |

### 分享引导规则

- **真实优先**：不要求故事精彩，要求真实
- **不评判**：听完再投票，过程中不评价
- **自愿原则**：如果某张卡片让分享者不适，可以"跳过"并重新抽卡（每轮限1次）
- **时间礼貌**：分享者注意控制时长，留给他人足够时间

### 胜利条件（弱化竞争，强化分享）

- 游戏不进行分数排名
- 最终统计"共鸣贴纸"最多的3位，获得"最佳分享者"称号
- **重点**：每轮结束后，教练引导大家说出"我从XX的故事中学到了…"

## 3. 分享引导要点

### 教练引导话术

**游戏开始前**（建立心理安全）：
> "今天我们要玩一个特别的游戏。规则只有一条：**真诚分享，用心倾听**。你可以分享任何事，不用顾虑是否正确、是否精彩。我们在这里，是为了更好地了解彼此。"

**抽卡后**（帮助思考）：
> "抽到这张卡不知道说什么？没关系，就从'我记得有一次…'开始。不需要开头就惊天动地，真实的小事往往最打动人。"

**分享结束后**（强化连接）：
> "谢谢XX的分享。我想问问大家，听完这个故事，你对XX有什么新的认识？"

### 教练观察点

| 观察维度 | 关注什么 |
| :--- | :--- |
| **开放度** | 谁愿意分享私密/脆弱的故事？谁始终停留在表面？ |
| **倾听质量** | 有人分享时，其他人是在看手机，还是专注倾听？ |
| **群体反应** | 哪些故事引发了共鸣（多人点头/相似经历）？ |
| **沉默者** | 有没有人一直不说话？是否需要邀请（但不要强迫）？ |

### 关键把控点

- **时间管理**：严格按3分钟分享时间，避免个别人占用过多时间
- **情绪安全**：如果有人分享了过于沉重的内容，教练要及时"接住"，引导转向积极方向
- **防止评判**：如果有人讲完后，其他人开始"给建议"，要温和制止："现在是分享时间，不是建议时间"

## 4. 复盘问题（4F模型）

### Facts（事实层面）

- "今天哪个故事最让你意外？为什么？"
- "有没有哪个分享让你对自己团队伙伴有了新的认识？"
- "分享过程中，你注意到大家的注意力是怎么变化的？"

### Feelings（感受层面）

- "轮到你分享的时候，你是什么感受？紧张？期待？还是…？"
- "听到别人的故事时，你有什么情绪变化？"
- "有没有哪个故事让你产生了'我也是！'的共鸣感？"

### Findings（发现层面）

- "这个游戏和平时的工作沟通有什么相似之处？"
- "我们平时有多少机会，像今天这样真正了解彼此？"
- "分享和倾听，哪个更难？为什么？"

### Future（未来层面）

- "回到工作中，我们可以怎样创造更多'真实交流'的机会？"
- "今天之后，你会用不同的方式看待某位同事吗？"
- "如果我们的团队是一个'探险家日记'，下一页会写什么？"

## 5. 变体适配

### 变体A：主题式分享（适合有明确团建主题）

**适用场景**：团建有明确主题（如"信任""创新""抗压"）

**规则调整**：
- 所有话题卡围绕一个主题设计
- 示例（信任主题）：「一次你选择信任别人的经历」「一次别人信任你的时刻」
- 分享后增加"主题连接"环节：这个故事和"信任"有什么关系？

**复盘重点**：主题深化、价值观对齐

---

### 变体B：小组接力故事（适合大团队）

**适用场景**：16人以上，需要分组

**规则调整**：
- 分成3–4人小组
- 每组抽取1张地点卡+1张话题卡
- 小组内每人分享1段与话题相关的故事（1分钟/人）
- 然后全组合作，将4段故事"串联"成一段完整故事
- 各组派代表讲述"串联版故事"，全体投票

**复盘重点**：团队协作、故事串联、共同创造

---

### 变体C：无声共鸣版（适合内向型团队）

**适用场景**：团队成员偏内向，或减少社交压力

**规则调整**：
- 分享者讲述故事
- 听众不鼓掌、不提问、不评论
- 只用"共鸣贴纸"投票
- 所有分享结束后，教练读出获得贴纸最多的3个故事的主题
- 然后让大家用便利贴写下"我被哪个故事触动，因为…"

**复盘重点**：深度倾听、非评判态度、内在连接

## 7. 教练工具箱（补充）

### 分享引导话术示例

**帮助沉默者开口**：
> "不着急，你可以想想再说。或者，你可以先说说——你为什么不想分享这个故事？"

**处理过于简短的分享**：
> "谢谢你的分享。我想多了解一点——当时你是什么感受？"

**处理过于冗长的分享**（温和打断）：
> "你的故事很精彩，不过我们需要把时间留给其他人。你可以用一句话总结一下核心吗？"

### 常见问题处理

| 问题 | 处理方式 |
| :--- | :--- |
| 有人分享时其他人在笑/私下说话 | 温和提醒："请大家专注倾听，这是对分享者的尊重" |
| 有人分享了不合适的内容（过于私人/负面） | 感谢分享，然后引导："谢谢你信任我们。我们换个角度——从这件事你学到了什么？" |
| 有人拒绝分享 | 尊重："没问题，你可以选择跳过。我们请下一位。" |
| 时间不够 | 提前5分钟通知："还有3位分享者，请每位控制在2分钟内" |

### 安全提示

- **心理安全优先**：如果团队信任度较低，第一次玩建议用"变体C（无声共鸣版）"
- **不强迫**：永远给参与者"跳过"的权利
- **教练以身作则**：第一轮，教练可以先分享一个自己的小故事，作为示范
