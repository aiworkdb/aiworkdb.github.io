# 人类宾果（Human Bingo）

> 经典的"寻找同类"破冰游戏。让参与者在轻松氛围中快速认识彼此，发现共同点。

---

## 1. 概览与标签

| 项目 | 内容 |
|------|------|
| **游戏名称** | 人类宾果（Human Bingo） |
| **目的标签** | 破冰热场 · 沟通表达 |
| **形式标签** | 室内 · 任意人数 · 智力 |
| **人数** | 任意人数（建议 8–50 人） |
| **时长** | 热身 5–15 min |
| **难度** | ⭐ 简单 |
| **场景标签** | 团建 · 培训 · 通用 |
| **核心目标** | 快速破冰、让参与者记住彼此名字、发现共同点 |

---

## 2. 核心规则

### 准备

- 提前制作 **宾果卡**（Bingo Card）：一张纸分成 5×5 的格子
- 每个格子里写一句描述，例如：
  - "去过 3 个以上国家"
  - "会玩一种乐器"
  - "养过宠物"
  - "小时候梦想的职业是警察/医生/老师"
  - "能说三种以上方言"
  - "参加过马拉松"
  - "有纹身"
  - "会做饭"
  - "害怕蜘蛛"
  - "出生在夏天"
  - _（中间格子可写"自由格"或"我是______"）_

### 流程

```
1. 每人发一张宾果卡（或用PPT投影统一卡片）
2. 参与者需要在房间里走动，找到符合格子里描述的人
3. 找到后，请对方在格子里签名字（每人只能帮你签一个格）
4. 第一个填满一行/一列/全卡的人喊"Bingo！"，游戏结束
5. 请"Bingo"的人简单介绍：你找到了哪些有趣的人？
```

### 胜负判定

- **个人赛**：第一个喊 Bingo 的人获胜
- **团队赛**：分组竞赛，全组填满最多格子的组获胜
- **无胜负**：纯粹破冰，不设获胜者，所有人完成即结束

---

## 3. 引导要点

### 如何讲解（1 分钟讲完）

> "大家手里有一张宾果卡，每个格子里都有一句描述。
> 你们的任务是：在房间里找到符合条件的人，请他在格子里签上名字。
> **注意：一个人只能帮你签一个格。** 所以你需要去找不同的人。
> 第一个填满一整行的人喊 Bingo，然后我们请他介绍一下找到了哪些有趣的人。
> 现在，开始！"

### 示范要点

- 教练自己先示范一次：走到某人面前问"你去过 3 个以上国家吗？"→ 对方点头 → 签名
- 强调**签名字**，不是打勾——这样大家会记住彼此的名字

### 关键把控

| 情况 | 处理方式 |
|------|---------|
| 有人害羞不敢开口 | 教练主动走近，示范如何开口问问题 |
| 有人很快完成 | 请他/她去帮助还没完成的人 |
| 时间到了没人完成 | 改为"填最多格子的人获胜"，或集体分享有趣发现 |
| 人数太少（<8人） | 改为"互相采访"模式，每人采访 3 人后介绍给对方 |

## 4. 复盘问题（4F 模型）

> 复盘时间：3–5 分钟（这款游戏复盘可以很短，重点是过渡到下一个活动）

### Facts（事实）

> "刚才大家找到了哪些有趣的人？有没有哪个格子所有人都符合？"

### Feelings（感受）

> "在去找陌生人的过程中，一开始是什么感觉？紧张？还是兴奋？"

### Findings（发现）

> "我们的团队里，原来藏了这么多有趣的人。平时工作中，我们有没有忽略彼此的这些面向？"

### Future（行动）

> "今天认识了新同事，明天上班见面的时候，你会主动打个招呼吗？"

## 5. 变体适配

### 变体一：主题宾果（适合培训场景）

将格子内容替换为与培训主题相关的描述，例如：
- 领导力培训："曾经带领过 5 人以上团队"
- 创新工作坊："有过把'不可能'变成'可能'的经历"
- 销售培训："有过被客户拒绝 10 次以上还在坚持的经历"

### 变体二：照片宾果（适合有手机的场合）

格子内容改为"拍一张你和______的合影"，增加互动感和纪念价值。

### 变体三：线上版（适合远程团队）

使用在线白板工具（Miro / 腾讯文档），每人一个虚拟棋子，滚动骰子走到格子上，全组一起完成。

### 人数调整

| 人数 | 调整方式 |
|------|---------|
| **< 8人** | 改为"互相采访"模式，每人采访 3 人后向大家介绍受访者 |
| **> 50人** | 分组进行，每组 15–20 人，各组独立比赛 |
