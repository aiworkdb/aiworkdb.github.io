| **游戏名称** | 人类宾果（Human Bingo） |
| **目的标签** | 破冰热场 · 沟通表达 |
| **形式标签** | 室内 · 任意人数 · 智力 |

## 2. 核心规则

### 准备

- 提前制作 **宾果卡**（Bingo Card）：一张纸分成 5×5 的格子
- 每个格子里写一句描述，例如：
  - "去过 3 个以上国家"
  - "会玩一种乐器"
  - "养过宠物"
  - "小时候梦想的职业是警察/医生/老师"
  - "能说三种以上方言"
  - "参加过马拉松"
  - "有纹身"
  - "会做饭"
  - "害怕蜘蛛"
  - "出生在夏天"
  - _（中间格子可写"自由格"或"我是______"）_

### 流程

```
1. 每人发一张宾果卡（或用PPT投影统一卡片）
2. 参与者需要在房间里走动，找到符合格子里描述的人
3. 找到后，请对方在格子里签名字（每人只能帮你签一个格）
4. 第一个填满一行/一列/全卡的人喊"Bingo！"，游戏结束
5. 请"Bingo"的人简单介绍：你找到了哪些有趣的人？
```

### 胜负判定

- **个人赛**：第一个喊 Bingo 的人获胜
- **团队赛**：分组竞赛，全组填满最多格子的组获胜
- **无胜负**：纯粹破冰，不设获胜者，所有人完成即结束
