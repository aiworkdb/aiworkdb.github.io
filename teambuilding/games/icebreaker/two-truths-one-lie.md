# 两真一假（Two Truths One Lie）



| **游戏名称** | 两真一假（Two Truths One Lie） |
| **目的标签** | 破冰热场 · 沟通表达 |
| **形式标签** | 室内 · 任意人数 · 智力 |
| **人数** | 5–30 人（最佳 8–20 人） |
| **时长** | 短篇 10–20 min |
| **难度** | ⭐ 简单 |
| **场景标签** | 团建 · 培训 · 通用 |
| **核心目标** | 快速破冰、让参与者记住彼此、发现共同点 |

---

## 2. 核心规则

### 流程

```
1. 每人准备三个关于自己的陈述句：
   - 两句是真的
   - 一句是假的（编的）

2. 轮流站在大家面前，读出三句话（不说是真是假）

3. 其他人投票：哪一句是假的？
   - 举手投票，或写在纸上

4. 公布答案：哪句是假的？

5. 可以请当事人展开讲一下那句"真的"背后的故事
```

### 陈述句示例

```
好的陈述句（具体、有画面感）：
  ✅ "我小时候在家养过 6 只猫"
  ✅ "我曾经一个人背包旅行过 3 个国家"
  ✅ "我大学学的是兽医专业"

不好的陈述句（太模糊，无法验证）：
  ❌ "我喜欢旅行"（太普通，没人会猜是假的）
  ❌ "我是个好人"（无法验证，不好玩）
```

### 胜负判定

- **无胜负**（推荐）：纯粹破冰，重点是分享和了解
- **竞赛模式**：猜对最多的人获胜（适合竞争氛围强的团队）

## 3. 引导要点

### 如何讲解（1 分钟讲完）

```
"大家每人想三个关于自己的句子：两句是真的，一句是编的。
然后读给大家听，大家投票猜哪句是假的。
最后你公布答案，还可以讲讲那两句'真的'背后的故事。

举个例子，我说三句：
  1. 我小时候在家养过 6 只猫
  2. 我大学学的是兽医专业
  3. 我曾经一个人背包旅行过 3 个国家
你们猜，哪句是假的？

（等大家猜）
答案是：第 2 句是假的——我没学过兽医😄

好，现在给大家 2 分钟，想一想自己的三句话。"
```
