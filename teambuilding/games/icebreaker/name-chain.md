## 2. 核心规则

### 流程

```
1. 所有人站成一个圆圈，面向中央

2. 从任意一人开始（通常教练指定或自愿）：
   - 大声说自己的名字
   - 同时做一个动作（比心、跳跃、摆 pose 均可）
   - 例如："我叫张三！"（同时比心）

3. 左边的人跟着做：
   - 先复述前一个人的名字+动作
   - 再说自己的名字+动作
   - 例如："他是张三！"（重复张三的动作）"我叫李四！"（同时摆 pose）

4. 依次顺时针传递，每个人都要：
   - 复述前一个人的名字+动作
   - 再说自己的

5. 转完一整圈后，最后一个人要复述所有人的名字+动作
   （或改为：教练随机点名，被点的人要说出左边和右边人的名字）
```
