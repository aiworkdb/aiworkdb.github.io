# 沉默领袖



| **时长** | 30–40 min |
| **场景标签** | 培训 / 工作坊 |
| **难度** | ⭐⭐⭐ |
| **道具需求** | 有（眼罩 + 任务卡） |

**核心目标**：让准管理者体验「无法用权力（说话/命令）时，如何真正领导团队」——这是领导力最核心的考验。

---

## 2. 核心规则

### 准备阶段（5 min）
1. 将参与者分成 2–3 组，每组 4–8 人。
2. 每组**秘密选出 1 人**当「领袖」（其他人不知道是谁）。
3. 教练给每组领袖**单独下达任务**（其他人不知道任务内容）。
4. **关键规则**：领袖**全程不能说话、不能写字**，只能通过**动作/手势/表情**引导团队完成任务。

### 任务进行（15–20 min）
- 各组同时开始，领袖用非语言方式引导团队
- 团队成员需要**观察、猜测、配合**领袖的意图
- 教练巡回观察，记录精彩瞬间

### 揭晓阶段（5–10 min）
- 时间到，各组展示任务完成情况
- 教练问：「你们组谁是关键领袖？」——团队成员投票
- 揭晓真正的领袖，对比「大家认为的领袖」vs「真正的领袖」

---

## 3. 任务题库（教练专用）

| 难度 | 任务内容 | 说明 |
|------|---------|------|
| ⭐ 简单 | 全员手拉手围成一个圆圈，然后一起坐下再站起来 | 需要协调和节奏 |
| ⭐ 简单 | 按生日大小（不说话）排成一列 | 需要观察和手势 |
| ⭐⭐ 中等 | 用身体拼出「TEAM」四个字母 | 需要创意和指挥 |
| ⭐⭐ 中等 | 全员按顺序（不说话）依次触摸房间里的 5 个指定物品 | 需要规划和引导 |
| ⭐⭐⭐ 困难 | 在 10 分钟内，让团队每个人都「被看见」（每个人都得到一次赞美） | 需要深度观察和情商 |
| ⭐⭐⭐ 困难 | 团队在没有语言的情况下，完成一幅集体画作（用纸笔） | 需要创意和协调 |



### Feelings（感受）
- 当领袖不能说话时，你是什么感受？
- 发现「真正的领袖」不是你以为的那个人时，你是什么感受？

### Findings（发现）
- 真正的领袖（非语言）和「以为的领袖」（最活跃的人）往往是不同的人。为什么？
- 这个游戏和你们团队中「谁在真正领导」有什么相似之处？
- 有人在任务中「默默做事但没人注意到」——工作中也是这样吗？

### Future（未来）
- 回到工作岗位，你会怎么调整自己的「领导方式」？
- 你有没有忽略过某个「沉默的领袖」？以后会怎么改变？

## 6. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **人数多（20+人）** | 分成 3–4 组，增加「领袖互换」环节（中途换一个领袖） |
| **时间紧（20 min）** | 只做简单任务，去掉「猜领袖」环节 |
| **培训深度版** | 增加「双重领袖」——两个领袖，一个知道任务，一个不知道，观察团队更听谁的 |
| **线上场景** | 用视频会议 + 聊天框（领袖不能发消息，只能用摄像头手势） |
| **高管版** | 任务改为「在不确定中做决策」——教练只给模糊指令，看领袖如何应对 |
