# 信任背摔（Trust Fall）

> 团建经典游戏，也是信任建立最有效的方式之一。从"敢不敢把后背交给队友"开始，体验信任的本质。

---

## 1. 概览与标签

| 项目 | 内容 |
|------|------|
| **游戏名称** | 信任背摔（Trust Fall） |
| **目的标签** | 信任建立 · 团队协作 |
| **形式标签** | 室内/户外 · 8–20人 · 体力 |
| **人数** | 8–20 人（最佳 10–15 人） |
| **时长** | 中篇 20–40 min（含复盘） |
| **难度** | ⭐⭐ 中等（对安全意识要求高） |
| **场景标签** | 团建 · 培训 · 通用 |
| **核心目标** | 体验"交出控制权"的感受，建立团队信任基础 |

---

## 2. 核心规则

### 安全前提（必须宣读）

```
"这个游戏需要大家高度专注和负责。
背摔的人完全信任接住他的人。
接的人肩上扛着别人的安全。
所以：不参与 = 可以，开玩笑 = 不可以。
我会在整个过程监控安全。"
```

### 流程

```
1. 所有人站成一个"接人阵型"：
   - 两排人面对面站，伸出双臂，掌心朝上
   - 两排间距约 30–40 cm（背摔者刚好能站进去）
   - 第一排人肩膀顶住第二排人的肩膀（增加稳定性）

2. 背摔者站到阵型一端的高处（椅子/台阶，约 50–80 cm）
   （初次体验建议从地面开始，不用高处）

3. 背摔者动作：
   - 双脚并拢，身体直立
   - 双手交叉放在胸前（或让队友用丝带轻绑手腕）
   - 大声说："我准备好了，你们准备好了吗？"
   - 团队齐声回答："准备好了！"
   - 身体笔直向后倒（不要跳、不要弯腰）

4. 接人阵型的人：
   - 用双臂接住背摔者，平稳地传递到大腿高度
   - 确认安全后，大家一起说："我们接住你了！"

5. 每个人轮流完成一次
```

### 胜负判定

- **合作模式**（推荐）：不设胜负，每个人的安全完成就是全组成功
- **可以加入"无声背摔"变体**：接人阵型不能说话，只能靠默契

## 3. 引导要点

### 如何讲解（3 分钟讲完）

```
第一步：说意义（1 分钟）
  "信任这个词，我们说了无数次。
  但真正把后背交给别人，是另一回事。
  今天这个游戏，就是让你体验：'我相信你'到底是什么感觉。"

第二步：讲安全规则（1.5 分钟）
  "接人的人：你们的动作是——双臂伸直，掌心朝上，
   接到人后，先托住肩膀和腰部，再慢慢放下。
   不要只接手，要接整个身体。
   背摔的人：身体笔直，像一块木板一样倒下来。
   不要跳，不要弯腰，不然接的人会受伤。"

第三步：示范（30 秒）
  "我找一个志愿者，我们演示一次——"
```

### 关键把控

| 情况 | 处理方式 |
|------|---------|
| 有人不敢尝试 | **不要强迫**。让他/她先当接人的人，观察别人，再决定是否尝试 |
| 接的人阵型不稳 | 立即喊停，重新整理阵型，教练站在旁边监控 |
| 有人开玩笑/不认真 | 立即喊停，严肃说明："这不是游戏，这是安全训练" |
| 背摔者身体弯曲 | 喊停，让团队给他/她反馈，重新尝试 |

### 教练站位

- 站在背摔者正后方，随时准备保护性接住（双重保险）
- 眼睛全程盯着背摔者的身体姿态

## 4. 复盘问题（4F 模型）

### Facts（事实）

> "刚才背摔的时候，倒数'3、2、1'的那几秒，大家是什么状态？"

### Feelings（感受）

> "身体倒出去的那一瞬间，你是什么感觉？紧张？放松？还是别的？"
>
> "接人的人，在接住队友的那一刻，又是什么感觉？"

### Findings（发现）

> "'我相信你'这几个字，和今天真正把后背交出去，有什么不一样？"
>
> "在工作中，我们有没有过'不敢把后背交给队友'的时候？是什么阻止了我们？"

### Future（行动）

> "回到工作中，如果你需要'信任队友'才能完成任务，今天这个体验能给你什么？"

## 5. 变体适配

### 变体一：无声背摔

**规则**：接人阵型不能说话，背摔者也不知道自己会被怎么接。

**适用场景**：强化"信任 = 即使没有确认，也相信对方会做好"。

---

### 变体二：盲眼背摔

**规则**：背摔者蒙上眼罩再倒。

**适用场景**：强化"在不确定中信任"的体验。

**安全提示**：必须有教练在后方双重保护。

---

### 变体三：团队背摔（大型团队）

**规则**：20 人以上，所有人围成一个大圈，背摔者站在中间倒向任意方向。

**适用场景**：大组团建，建立"整个团队都在接住你"的感受。

---

### 人数调整

| 人数 | 调整方式 |
|------|---------|
| **8–12 人** | 标准阵型，1 组完成 |
| **13–20 人** | 分成两组，同时或轮流进行 |
| **> 20 人** | 必须分组，或改为"团队背摔"变体 |

## 6. 物资清单

### 必备

- [ ] 稳固的椅子或台阶（高度 50–80 cm，初次可从地面开始）
- [ ] 空旷场地（周围无硬物）
- [ ] 教练 1 人（必须全程监控安全）

### 可选

- [ ] 丝带/软绳（轻绑背摔者手腕，防止本能抓人）
- [ ] 瑜伽垫/软垫（铺在地面，增加安全感）
- [ ] 口哨（紧急喊停用）

### 安全红线

- ⚠️ **绝对禁止**：有人在背摔过程中开玩笑、推搡
- ⚠️ **绝对禁止**：背摔者跳下来（不是倒下来）
- ⚠️ **绝对禁止**：接人阵型的人没有站稳就开始
- ⚠️ 有严重腰伤/颈椎问题的人，**不参与背摔**，改为接人角色

### 准备时间

- 10 分钟（检查场地、摆放椅子、整理阵型）
