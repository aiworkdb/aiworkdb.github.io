| **形式标签** | 室内/户外 · 两两配对 · 混合 |

## 2. 核心规则

### 准备

- 眼罩（每两人备 1 个，总量 = 人数的一半）
- 选择一条有轻微变化的路线（平地 → 轻微上坡 → 转弯 → 终点）
- 确保路线上无危险障碍

### 流程

```
第一轮：
1. 两人一组，抽签决定谁先蒙眼（A = 蒙眼者，B = 引导者）
2. A 蒙上眼罩，B 用任何非语言方式（轻触手臂/手掌）引导 A
3. 路线约 100–200 米，耗时 5–8 分钟
4. 到达终点后，A 摘下眼罩，两人不说话，交换角色

第二轮：
5. B 蒙眼，A 引导，走同一条或不同路线
6. 两轮结束后，集中复盘
```

### 引导者规则（关键）

- **不能说话**，只能用轻触信号沟通
- 常用信号：轻拍左肩 = 左转，轻拍右肩 = 右转，轻推后背 = 前进，轻拉手臂 = 停下
- **引导者必须走在蒙眼者的侧前方**，不能让他/她撞到任何东西
- 遇到台阶/坡道，引导者要用力信号明确提示

## 3. 引导要点

### 如何讲解（2 分钟讲完）

```
"两人一组。一人蒙上眼睛，另一人负责引导他走一段路。
规则只有三条：
  1. 引导者不能说话，只能用轻触信号
  2. 蒙眼者完全信任引导者，不能偷看
  3. 引导者走在蒙眼者的侧前方，保护他不被撞到

走完一轮后，两人交换角色，再走一轮。
准备好了吗？先蒙眼的人举一下手——"
```

### 路线设计建议

| 路线元素 | 目的 | 注意事项 |
|---------|------|---------|
| 平地行走 | 建立基础信任 | 路线不能太简单，要有轻微变化 |
| 轻微上坡/下坡 | 体验"引导者的责任" | 提前走一遍，确认安全 |
| 转弯 | 体验"沟通信号是否清晰" | 转弯处不能有障碍物 |
| 终点设置 | 给"被引导"一个仪式感 | 到达后引导者可以说一句话（或轻握一下手） |

### 关键把控

| 情况 | 处理方式 |
|------|---------|
| 蒙眼者偷看 | 温和提醒："相信你的引导者，不用眼睛也能走" |
| 引导者不合适（走太快/太激进） | 教练悄悄走近，低声提醒："慢一点，感受对方的速度" |
| 有人走错路线 | 没关系，让引导者自己纠正，不要喊停 |
| 到达终点后情绪激动 | 趁热打铁进入复盘，不要拖延 |
