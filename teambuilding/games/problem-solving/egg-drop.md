# 鸡蛋坠落挑战



| **时长** | 45–60 min |
| **场景标签** | 团建 / 培训 / 工作坊 |
| **难度** | ⭐⭐⭐ |
| **道具需求** | 有（丰富） |

**核心目标**：在 constrained 条件下，通过快速试错和团队协作，完成工程挑战，体验「fail fast, learn fast」。

---

## 2. 核心规则

### 准备阶段（5 min）
1. 将参与者分成 4–6 人小组。
2. 每组领取**材料包**（见物资清单）。
3. 宣布挑战：**用提供的材料，设计一个鸡蛋保护装置，从 3 米高度自由落下，鸡蛋不碎**。

### 设计阶段（10 min）
- 各组讨论方案，画草图
- **不允许**提前测试（材料有限）
- 教练提醒：考虑**缓冲、分散冲击力、结构稳定**三大原理

### 制作阶段（20 min）
- 各组制作保护装置
- 中途可以**修改方案**，但材料不能增加
- 教练巡回观察，记录亮点

### 测试阶段（10 min）
- 各组依次到指定高度（3 米）释放装置
- **同一颗鸡蛋**必须经历 3 次掉落（难度递增：3m → 5m → 随意高度）
- 鸡蛋碎的组淘汰，完整者进入下一轮

### 复盘（10–15 min）
- 存活组分享设计原理
- 失败组分享：哪里出了问题？如果重来会改什么？

---

## 3. 引导要点

### 讲解时


### Feelings（感受）
- 看到鸡蛋碎的那一刻，你的第一反应是什么？
- 看到其他组成功时，你是什么感受？

### Findings（发现）
- 如果重新来一次，你们会**第一步**做什么不同的事？
- 你们组在设计过程中，有没有出现过意见分歧？是如何解决的？
- 这个游戏和你们日常工作中「在不确定中做决策」有什么相似之处？

### Future（未来）
- 回到工作岗位，当你面对一个「不知道能不能成功」的项目时，你会怎么调整心态？
- 你们团队可以建立一个什么样的「快速试错」文化？

## 5. 变体适配

| 场景 | 调整方案 |
|------|---------|
| **时间紧（30 min）** | 去掉设计阶段，直接制作；高度降为 2 米 |
| **材料受限版** | 每组只给 10 根吸管 + 5 根牙签 + 1 张 A4 纸 + 20 cm 胶带 |
| **大型团建（50+人）** | 改为「鸡蛋赛车」——装置不仅要保护鸡蛋，还要能滑行最远距离 |
| **培训深度版** | 增加「客户需求变更」环节——制作中途教练宣布：「客户要求装置必须能打开重复使用」 |
| **户外场景** | 从 5 米高的二楼掉落；增加风力干扰（用电风扇） |
| **线上场景** | 每人用自己家里的材料（保鲜膜、纸巾、橡皮筋等），视频会议展示 |
