鸡蛋坠落挑战

**一句话描述**：用有限材料设计一个保护装置，让生鸡蛋从 3 米高处落下不碎。

1. 概览

项目 内容
**目的标签** 🔧 问题解决 / 创新思维 / 团队协作
**形式** 室内/户外 · 动手+策略
**人数** 4–6人/组，多组竞赛
**时长** 45–60 min
**场景标签** 团建 / 培训 / 工作坊
**难度** ⭐⭐⭐
**道具需求** 有（丰富）

**核心目标**：在 constrained 条件下，通过快速试错和团队协作，完成工程挑战，体验「fail fast, learn fast」。

2. 核心规则

准备阶段（5 min）

  1. 将参与者分成 4–6 人小组。
  2. 每组领取**材料包**（见物资清单）。
  3. 宣布挑战：**用提供的材料，设计一个鸡蛋保护装置，从 3 米高度自由落下，鸡蛋不碎**。

设计阶段（10 min）

  • 各组讨论方案，画草图
  • **不允许**提前测试（材料有限）
  • 教练提醒：考虑**缓冲、分散冲击力、结构稳定**三大原理

制作阶段（20 min）

  • 各组制作保护装置
  • 中途可以**修改方案**，但材料不能增加
  • 教练巡回观察，记录亮点

测试阶段（10 min）

  • 各组依次到指定高度（3 米）释放装置
  • **同一颗鸡蛋**必须经历 3 次掉落（难度递增：3m → 5m → 随意高度）
  • 鸡蛋碎的组淘汰，完整者进入下一轮

复盘（10–15 min）

  • 存活组分享设计原理
  • 失败组分享：哪里出了问题？如果重来会改什么？

3. 引导要点

讲解时

"你们的目标是：让这颗鸡蛋活下来。但真正的目标其实是：让你们的**团队协作能力**活下来。"
  • 强调**快速原型**理念：先做一个「丑版本」，测试后改进
  • 提醒：**材料有限**，每一个决定都有机会成本

制作过程中

  • 观察并记录：
  • 谁提出了关键设计思路？
  • 团队如何分配任务（有人设计、有人制作、有人测试）？
  • 有没有组进行了「模拟测试」（用想象代替实际）？

测试时

  • **仪式感**：每组释放时，全体倒数「3—2—1—释放！」
  • 鸡蛋碎的瞬间，观察团队反应：有人笑、有人沮丧、有人立刻分析失败原因
  • 这正是引导**抗挫力**的好时机

复盘时

  • 引导大家关注：**最成功的组，设计原理是什么？**
  • 引导大家关注：**失败最快的组，学到了什么？**

4. 复盘问题（4F 模型）

Facts（事实）

  • 你们组的设计原理是什么？用了哪些材料？
  • 鸡蛋在第几次掉落时碎了？当时发生了什么？

Feelings（感受）

  • 看到鸡蛋碎的那一刻，你的第一反应是什么？
  • 看到其他组成功时，你是什么感受？

Findings（发现）

  • 如果重新来一次，你们会**第一步**做什么不同的事？
  • 你们组在设计过程中，有没有出现过意见分歧？是如何解决的？
  • 这个游戏和你们日常工作中「在不确定中做决策」有什么相似之处？

Future（未来）

  • 回到工作岗位，当你面对一个「不知道能不能成功」的项目时，你会怎么调整心态？
  • 你们团队可以建立一个什么样的「快速试错」文化？

5. 变体适配

场景 调整方案
**时间紧（30 min）** 去掉设计阶段，直接制作；高度降为 2 米
**材料受限版** 每组只给 10 根吸管 + 5 根牙签 + 1 张 A4 纸 + 20 cm 胶带
**大型团建（50+人）** 改为「鸡蛋赛车」——装置不仅要保护鸡蛋，还要能滑行最远距离
**培训深度版** 增加「客户需求变更」环节——制作中途教练宣布：「客户要求装置必须能打开重复使用」
**户外场景** 从 5 米高的二楼掉落；增加风力干扰（用电风扇）
**线上场景** 每人用自己家里的材料（保鲜膜、纸巾、橡皮筋等），视频会议展示

教练笔记

💡 **这个游戏的精髓**：让参与者在「一定会失败几次」的环境中，体验「快速试错」的快乐，而不是恐惧失败。

**常见卡点**：

  • 有组陷入「分析瘫痪」，一直讨论不开始制作 → 教练介入：「你们还有 10 分钟，现在不开始就来不及了。」
  • 有组鸡蛋碎了之后情绪崩溃 → 这正是好时机：「在真实项目中，你是不是也遇到过『鸡蛋碎了』的时刻？那时你是怎么恢复的？」

**安全提示**：

  1. 鸡蛋碎后要立即清理，防止滑倒
  2. 剪刀使用需注意安全
  3. 高处掉落测试时，下方不得站人

**进阶玩法**：

  • 成功存活的组，可以尝试「创意掉落」——从更奇怪的角度/方式掉落（如旋转着掉落）
  • 增加「成本计算」环节：每种材料有「价格」，总分 = (100 - 成本) × 存活轮数
