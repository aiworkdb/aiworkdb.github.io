# 教练 / 培训师使用指南

> 5 分钟读懂这套资料怎么用。无论你是团建教练还是企业培训师，都能快速上手。

---

## 一、如何选游戏

接到需求后，先回答四个问题，锁定游戏范围：

```
1. 这场活动想达成什么？
   → 破冰？沟通？协作？信任？领导力？创新？问题解决？

2. 场地和条件是什么？
   → 室内会议室 / 户外空地 / 线上Zoom？有道具吗？

3. 多少人参加？
   → <10人（小组深度） / 10-30人（中组互动） / >30人（大组氛围）

4. 有多少时间？
   → 5-15min（热身） / 15-30min（课堂插入） / 30-60min（主体活动） / >60min（沙盘/全天）
```

回答完后，打开 [`README.md`](../teambuilding/README.md) 的索引表，按四个维度交叉筛选，找到匹配的游戏。

**选游戏的三条原则：**
- 时间宁短勿长：留 20% 缓冲，别把流程排满
- 人数宁少勿多：大组建议拆分成多个小组并行
- 难度宁浅勿深：规则能在 2 分钟内讲清楚最好

## 二、如何引导游戏

### 开场三步法

```
第一步：说目的（30秒）
  "今天我们玩这个游戏，是想让大家体验一下______"
  → 先说价值，再讲规则，参与者更有投入感

第二步：讲规则（2分钟）
  → 只讲核心规则，变体和细节等第一轮结束后补充
  → 配合示范（找两个参与者演示一轮）
  → 规则讲完后问："大家都听懂了吗？"  → 别只等点头，让一个人复述

第三步：定分组（1分钟）
  → 提前准备好分组方式（见工具箱：分组方法）
  → 宣布开始后，教练退后观察，不要一直说话
```

### 游戏进行中的教练角色

| 阶段 | 教练在做什么 |
|------|-------------|
| **开始前** | 检查场地安全、道具就位、分组完成 |
| **进行中** | 观察 > 干预。记录精彩瞬间和典型行为 |
| **失控时** | 吹停 → 简短干预 → 重新开始（不要长篇说教） |
| **结束时** | 准时喊停，不要因为"还没玩完"而拖延 |

**关键原则：教练是引导者，不是表演者。** 你的目标是让参与者彼此互动，而不是看你表演。

## 三、如何做复盘

复盘是团建/培训最有价值的部分。**没有复盘的游戏，只是玩。**

### 4F 复盘模型

```
Facts（事实）   →  "刚才发生了什么？大家记得哪些关键时刻？"
Feelings（感受） →  "当时你是什么感受？紧张？兴奋？沮丧？"
Findings（发现） →  "这和工作/团队有什么相似的地方？"
Future（行动）   →  "回到工作中，你会做一件什么不同的事？"
```

### 复盘引导话术模板

**开场：**
> "好，游戏结束。给大家 1 分钟，先安静回想一下刚才的过程。"

**Facts（事实）：**
> "谁愿意分享一下，刚才最印象深刻的一个瞬间是什么？"

**Feelings（感受）：**
> "当 ______ 发生的时候，大家是什么感受？"

**Findings（发现）：**
> "如果把这个游戏看成我们团队工作的缩影，大家看到了什么？"

**Future（行动）：**
> "带着这个体验，回到工作中，你明天会做一件什么不同的事情？"

### 复盘注意事项

- 复盘时间 = 游戏时间的 **1/3 到 1/2**（30分钟游戏 → 10-15分钟复盘）
- 先让参与者说，教练最后总结。**不要一开始就给答案。**
- 记录关键洞察，活动结束后整理成反馈发给客户

## 四、常见场景快速方案

| 场景 | 推荐组合 |
|------|---------|
| **新员工入职培训 · 破冰** | 破冰游戏（15min）→ 团队协作游戏（30min）→ 复盘（10min） |
| **管理层领导力工作坊** | 领导力游戏（45min）→ 深度复盘（30min）|
| **销售团队激励团建** | 热身（10min）→ 合作游戏（40min）→ 减压游戏（20min）→ 复盘（15min）|
| **线上远程团队凝聚** | 线上破冰（15min）→ 线上协作游戏（30min）→ 线上复盘（10min）|
| **全天团建活动** | 破冰（30min）→ 主体活动×2（各60min）→ 午餐 → 大型合作挑战（90min）→ 复盘总结（30min）|

## 五、给新手教练的建议

1. **第一次带某个游戏，自己先玩一遍**（找朋友或同事试跑）
2. **准备 Plan B**：如果某个游戏冷场，马上切换到备用游戏
3. **控制自己的表达欲**：说得多 ≠ 效果好
4. **记录每次带游戏的反思**：什么效果好？什么需要调整？积累成自己的经验库
5. **和客户提前确认**：人数、场地、特殊需求（如有人身体不适不能参与体力活动）
