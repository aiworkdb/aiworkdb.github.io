```
**选游戏的三条原则：**
- 时间宁短勿长：留 20% 缓冲，别把流程排满
- 人数宁少勿多：大组建议拆分成多个小组并行
- 难度宁浅勿深：规则能在 2 分钟内讲清楚最好

## 二、如何引导游戏

### 开场三步法

```
第一步：说目的（30秒）
  "今天我们玩这个游戏，是想让大家体验一下______"
  → 先说价值，再讲规则，参与者更有投入感

第二步：讲规则（2分钟）
  → 只讲核心规则，变体和细节等第一轮结束后补充
  → 配合示范（找两个参与者演示一轮）
  → 规则讲完后问："大家都听懂了吗？"  → 别只等点头，让一个人复述

第三步：定分组（1分钟）
  → 提前准备好分组方式（见工具箱：分组方法）
  → 宣布开始后，教练退后观察，不要一直说话
```

### 游戏进行中的教练角色

| 阶段 | 教练在做什么 |
|------|-------------|
| **开始前** | 检查场地安全、道具就位、分组完成 |

Feelings（感受） →  "当时你是什么感受？紧张？兴奋？沮丧？"
Findings（发现） →  "这和工作/团队有什么相似的地方？"
Future（行动）   →  "回到工作中，你会做一件什么不同的事？"
```

## 五、给新手教练的建议

1. **第一次带某个游戏，自己先玩一遍**（找朋友或同事试跑）
2. **准备 Plan B**：如果某个游戏冷场，马上切换到备用游戏
3. **控制自己的表达欲**：说得多 ≠ 效果好
4. **记录每次带游戏的反思**：什么效果好？什么需要调整？积累成自己的经验库
5. **和客户提前确认**：人数、场地、特殊需求（如有人身体不适不能参与体力活动）
