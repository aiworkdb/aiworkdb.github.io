# 复盘引导框架

> 复盘是团建/培训最有价值的部分。**没有复盘的游戏，只是玩。**
> 本文档提供 4F 模型、提问清单和典型场景复盘话术。

---

## 一、4F 复盘模型

| 阶段 | 英文 | 中文 | 核心问题 |
|------|------|------|---------|
| **1** | Facts | **事实** | 刚才发生了什么？ |
| **2** | Feelings | **感受** | 你当时是什么感觉？ |
| **3** | Findings | **发现** | 这和工作/团队有什么相似？ |
| **4** | Future | **行动** | 回到工作中你会改变什么？ |

### 时间分配建议

| 游戏时长 | 建议复盘时长 | 每个 F 分配 |
|---------|-------------|-------------|
| 5–15 min | 3–5 min | 各 30–60 秒 |
| 15–30 min | 8–12 min | Facts 2min / Feelings 2min / Findings 4min / Future 2min |
| 30–60 min | 15–20 min | Findings 和 Future 可以深度展开 |
| > 60 min | 20–30 min | 可以分组讨论再集中分享 |

## 二、提问清单（按阶段）

### Facts（事实）—— 帮大家回忆过程

```
基础问法：
- "刚才发生了什么？大家记得哪些关键时刻？"
- "游戏分为哪几个阶段？每个阶段我们在做什么？"
- "有没有哪个瞬间，让大家都笑了/沉默了？"

如果没人说话：
- "我们从头回忆一下——最开始大家在做什么？"
- "谁愿意分享一个具体的画面？"

如果回忆太散：
- "我观察到了三件事：______、______、______。大家觉得呢？"
```

### Feelings（感受）—— 打开情绪层面

```
基础问法：
- "当 ______ 发生的时候，你是什么感受？"
- "紧张？兴奋？沮丧？还是别的？"
- "有没有人感受和其他人不一样？"

如果大家只说"挺好的"：
- "'挺好的'有很多种，能具体说说吗？是松了一口气，还是很有成就感？"

如果情绪比较负面：
- "我听到有人感到沮丧。这种感受是怎么产生的？"
- （不要急于"解决"负面情绪，先让情绪被看见）
```

### Findings（发现）—— 连接到工作和团队

```
基础问法：
- "如果把这个过程看成我们团队工作的缩影，大家看到了什么？"
- "今天体验到的 ______，和我们在办公室里遇到的 ______ 像吗？"
- "我们团队里，谁扮演了什么角色？这个角色在工作是一样的吗？"

引导深度讨论：
- "刚才有人说 ______。能不能展开说说？"
- "有没有人想回应 ______ 说的那句话？"

如果大家看不出联系：
- "我分享一个我的观察：______。大家觉得呢？"
- （教练可以适当自我披露，但不要用"说教"语气）
```

### Future（行动）—— 转化为实际行动

```
基础问法：
- "带着这个体验，回到工作中，你会做一件什么不同的事？"
- "不需要大改变，一个小的行动就好。有谁愿意分享？"
- "如果只能改变一件事，你会改变什么？"

帮助具体化：
- "你说'多沟通'——具体会怎么做？明天会发一封邮件？还是约一次 1-on-1？"
- （帮助参与者把模糊的"感悟"变成具体的"行动"）

收尾：
- "我今天印象最深的一句话是：'______'。
  希望大家带着这个体验，回到工作中。"
```

## 三、典型场景复盘话术

### 场景一：破冰游戏后的复盘（简短版）

```
"好，游戏结束。给大家 30 秒回想一下——

谁愿意分享：你今天认识了哪些新同事？
（等 2-3 人分享）

好。带着这个开始，明天上班见面的时候，
你会主动打个招呼吗？
（不需要回答，自己在心里回答就好。）

好，我们进入下一个活动——"
```

### 场景二：团队协作游戏后的复盘（标准版）

```
Facts：
"刚才解结的过程，大概经历了哪几个阶段？"

Feelings：
"当结越来越紧、感觉解不开的时候，大家是什么感受？"

Findings：
"我们这个团队，是'先商量再行动'还是'先行动再调整'？
这和我们平时的工作方式像吗？"

Future：
"回到工作中，当我们团队遇到一个'打不开的结'的时候，
今天这个体验能给我们什么启发？"
```

### 场景三：沟通主题游戏后的复盘（深度版）

```
Facts：
"我们摆出来的正方形，和理想中的差了多少？"

Feelings：
"蒙上眼睛的那一刻，你是什么感觉？
当听到别人说话却不确定方向的时候呢？"

Findings：
"当有人说'往左一点'，每个人对'左'的理解是一样的吗？
这和我们工作中的沟通有什么相似？"

Future：
"回到工作中，当我们布置任务时，
有没有可能对方理解的'正方形'和我们的不一样？
今天我们学到了什么关于'精准沟通'的东西？"
```

### 场景四：培训课程中的复盘（连接学习内容）

```
"刚才的游戏，和我们今天上午讲的 ______ 理论，
有没有什么联系？
（引导参与者自己建立联系，不要直接告诉答案）

我听到有人说 ______，这正好印证了 ______ 模型里的 ______。"
```

## 四、复盘引导技巧

### 让沉默的人说话

```
不要点名（会造成压力），而是：
- "我注意到有人还没说话。我们等一下，看看还有谁想分享？"
- "如果没有人主动说，我随机点一个——______（指向身边最近的人），你呢？"
```

### 处理"只有少数人说话"

```
"我发现 ______ 和 ______ 说了很多（正面肯定），
我想听听还没说话的人的视角——
你们观察到的，和刚才分享的有没有不一样的地方？"
```

### 处理"复盘跑题"

```
（温和地拉回来）
"这是个很有意思的话题，不过和我们今天的主题______
可能有点远。我们先记下来，活动结束后可以继续讨论。
现在回到 ______ 这个问题——"
```

### 处理"复盘时间不够"

```
"我今天听到了很多有价值的分享。
如果只能选一个最打动你的点，会是什么？
（每人一句话，快速收尾）"
```
