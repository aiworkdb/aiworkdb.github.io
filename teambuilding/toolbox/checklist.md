## 二、分组方法

### 方法一：报数分组（最快速）

```
"大家站成一排。从 1 开始报数，1、2、3、4、1、2、3、4……
报 1 的站这边，报 2 的站这边，以此类推。"
```
**适用**：任意人数，1 分钟内完成分组。
**缺点**：熟人容易分到同一组，需要人工微调。

---

### 方法二：卡片分组（可控性强）

```
提前准备卡片，每张卡片写一个提示（如：动物名称、电影名称、颜色）。
每人抽一张，找到和自己卡片"同类"的人，组成一组。
```
**示例**：
- 动物卡片：猫、狗、兔 → 抽到同一种动物的人一组
- 电影卡片：抽到同一部电影的人一组

**适用**：希望控制分组结果时使用。
**优点**：可以提前安排哪些人分到一组。

---

### 方法三：生日排序分组（破冰+分组二合一）

```
"大家按生日（月-日）从左到右排成一排，
然后每 ______ 个人一组。"
```
**适用**：破冰阶段，同时完成分组。
**优点**：自然打乱熟人关系。

---

### 方法四：颜色/物品分组（趣味性强）

```
"穿白色衣服的站这边，穿黑色衣服的站这边……"
或："口袋里有纸巾的站这边，没有的站那边……"
```
**适用**：轻松氛围，快速分组。
**注意**：可能导致人数不均，需要微调。

---

### 方法五：教练指定（精准控制）

```
教练提前根据参与者背景，手动安排分组，
活动开始时直接宣布："A 组是 ______、______、______……"
```
**适用**：培训场景，需要特定组合（如：跨部门混合、新老搭配）。
**缺点**：耗时，需要提前了解参与者信息。

## 四、效率工具

### 工具一：活动流程卡片（打印备用）

> 把常用活动流程打印成卡片，现场快速参考。

建议制作：
- A4 纸打印，过塑（防水防皱）
- 每个游戏一张卡，正面游戏规则，背面引导要点
- 现场不用翻手机/电脑，抬头就能看

---

### 工具二：倒计时提示卡（大型活动用）

> 当教练需要同时关注多个小组时，用视觉提示代替口头提醒。

制作方式：
- A4 纸，用马克笔写大字："剩余 10 分钟"、"剩余 5 分钟"、"剩余 1 分钟"
- 举牌提示，不打断活动节奏

---

### 工具三：分组名单模板

```
【A 组】（______ 人）
1. ______
2. ______
3. ______

【B 组】（______ 人）
1. ______
2. ______
3. ______
```
> 提前用这个模板安排好分组，活动当天直接宣布，节省时间。

## 五、物资管理建议

1. **准备"团建道具箱"** —— 把所有常用道具装在一个大整理箱里，每次出门直接拎走
2. **定期检查** —— 每月检查一次：眼罩是否干净、绳子是否磨损、马克笔还有没有墨
3. **多备 20%** —— 消耗品（姓名贴、卡片）多备一些，现场不够很尴尬
4. **准备"Plan B 道具"** —— 万一主要道具出问题，有备用方案
