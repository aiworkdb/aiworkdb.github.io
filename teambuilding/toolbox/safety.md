# 安全与管理

> 团建/培训活动的安全底线和突发情况应对预案。教练必须提前阅读，活动现场放在手边。

---

## 一、户外活动安全清单

### 场地检查（活动前 30 分钟完成）

- [ ] 检查场地周围是否有：尖锐物品、坑洞、湿滑地面、高空坠物风险
- [ ] 确认最近医院/诊所的位置和联系方式
- [ ] 确认手机信号良好（户外偏僻地点提前测试）
- [ ] 检查天气：雨天/高温/大风 是否有备用方案

### 参与者健康确认（活动开始前）

```
教练开场前必须问：
"有没有人今天身体不舒服，或者有以下情况，请举手告诉我：
 - 心脏病 / 高血压
 - 严重腰伤 / 膝盖伤
 - 怀孕
 - 近期做过手术

有的话请悄悄告诉我，我会给你安排合适的角色（观察员/记录员）。"
```

### 活动中监控要点

| 风险 | 监控方式 | 应对 |
|------|---------|------|
| **摔倒/扭伤** | 教练站在高处全局观察 | 立即停止活动，评估伤情，必要时送医 |
| **中暑** | 注意参与者面色和出汗量 | 转移到阴凉处，补充水分，严重时送医 |
| **情绪激动** | 注意言语冲突和身体对抗 | 立即喊停，单独沟通，必要时结束活动 |
| **过敏** | 提前询问过敏史（食物/花粉/昆虫） | 备抗过敏药，严重时立即送医 |

## 二、室内活动安全清单

### 场地检查

- [ ] 移开桌椅等硬物（体力活动前必须清空）
- [ ] 检查地面：湿滑？有地毯边缘会绊脚？
- [ ] 确认消防通道位置
- [ ] 确认最近洗手间位置（告诉参与者）

### 特殊道具安全

| 道具 | 风险 | 预防措施 |
|------|------|---------|
| 绳子 | 缠绕勒伤 | 活动前演示正确抓握方式，教练全程监控 |
| 眼罩 | 碰撞/摔倒 | 蒙眼前确认所有人站定，教练站中央监控 |
| 小球/沙包 | 砸伤 | 规定"只能轻抛"，禁止用力投掷 |
| 卡片/纸张 | 滑倒 | 活动结束后立即收拾，不让纸张留在地面 |

## 三、突发情况预案

### 预案一：有人受伤

```
第一步：停止活动，确认伤者状况
第二步：让其他人安静坐下/站定（避免混乱）
第三步：评估伤情
  - 轻伤（擦伤/轻微扭伤）→ 教练处理，活动继续
  - 中伤（明显疼痛/不能动）→ 送医，活动暂停
  - 重伤（骨折/昏迷）→ 立即拨打 120，专人陪同
第四步：向客户（甲方）汇报情况
```

### 预案二：天气突变（户外活动）

```
雨天：
- 提前准备室内备用方案（或取消/延期）
- 活动中遇雨：立即转移到避雨处，评估是否继续

高温（> 32°C）：
- 每 30 分钟安排一次休息和补水
- 准备遮阳棚
- 有人出现中暑症状：立即转移到阴凉处

大风：
- 户外有大型道具的活动，风速 > 5 级建议取消
```

### 预案三：参与者冲突

```
第一步：立即喊停，分开双方
第二步：不现场评判对错，先让双方冷静
第三步：单独沟通，了解原因
第四步：根据情况决定：
  - 轻微摩擦 → 重新分组，继续活动
  - 严重冲突 → 结束活动，与客户沟通后续处理
```

### 预案四：客户（甲方）临时改需求

```
"我理解您的需求变化。不过我需要 5 分钟，
和活动团队说明一下调整内容，确保大家理解一致。
可以吗？"

（不要当场抱怨，也不要直接拒绝——
用"需要和大家确认"来争取缓冲时间）
```

### 预案五：活动效果不好，冷场

```
（不要掩饰，把"失败"变成学习内容）
"嗯，这个活动今天好像没有预期的效果。
你们觉得是哪里出了问题？"

（引导参与者一起分析，把冷场变成复盘素材）
```

## 四、免责声明模板

> 以下模板供参考，根据实际情况调整。建议打印并在活动开始前请参与者签字（或口头确认）。

### 简易版（适用于低风险室内活动）

```
本人参与本次团建/培训活动，确认：
1. 身体状况良好，适合参与本次活动
2. 如有身体不适，会立即告知教练
3. 活动中遵守教练指引，不做危险动作
4. 如个人原因导致受伤，责任自负

参与者签字：______    日期：______
```

### 标准版（适用于户外活动或有一定体力消耗的活动）

```
本人自愿参与本次团建/培训活动，并确认：
1. 本人身体健康，无不适合参与本次活动的疾病
  （包括但不限于心脏病、高血压、严重腰伤/膝盖伤、怀孕等）
2. 活动中将遵守教练指引，不做危险动作
3. 如身体不适，会立即告知教练
4. 活动中如因个人原因导致受伤，责任自负
5. 活动组织方已提前告知活动风险和安全注意事项

参与者签字：______    日期：______

（未成年人参与者需监护人签字：______）
```

## 五、教练安全自律准则

1. **不接受超出能力范围的活动** —— 没把握的游戏不带
2. **活动前必须做场地检查** —— 无论去过多少次，每次都要检查
3. **高风险活动必须买保险** —— 户外拓展建议客户购买团体意外险
4. **活动中不喝含酒精饮料** —— 教练必须保持清醒
5. **活动结束后确认所有人安全离开** —— 不要人还在就先走
6. **保留活动记录** —— 日期、地点、参与人数、特殊情况，至少保留 6 个月
