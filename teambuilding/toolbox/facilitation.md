# 引导话术库



### 从 Facts 过渡到 Feelings

```
"刚才我们说了发生了什么。现在我想问——
当 ______ 发生的时候，大家是什么感受？"
```

### 从 Feelings 过渡到 Findings

```
"感谢大家的分享。我注意到 ______（总结一个关键观察）。
如果把这个过程看成我们团队工作的缩影，
大家看到了什么相似的地方？"
```

### 从 Findings 过渡到 Future

```
"今天体验到的这些东西，回到工作中，
有什么是你明天可以做一些不同的事情的？
不需要大改变，一个小的行动就好。"
```

### 复盘收尾

```
"谢谢大家的分享。我今天印象最深的一句话是：'______'。
希望大家带着这个体验，回到工作中。
我们今天的团建/培训，到这里就结束了。谢谢大家！"
```

## 五、特殊情况应对话术

### 有人质疑游戏的意义

```
"我理解你的疑问。这样吧，我们先玩完，
然后复盘的时候你告诉我，你觉得这个游戏和我们的工作有没有关系。
可以吗？"
```

### 有人身体不适，不能参与

```
"没问题，你今天担任'观察员'角色。
你的任务是：观察团队在游戏中的互动方式，
复盘的时候告诉我们你看到了什么。
这个角色其实很重要。"
```

### 活动效果不好，冷场了

```
（不要掩饰，直接承认）
"嗯，这个活动今天好像没有预期的效果。
你们觉得是哪里出了问题？
（引导参与者一起分析，把'失败'本身变成学习内容）
其实你们刚才已经在体验'如何应对失败'了。"
```

### 客户（甲方）在场，参与者不敢说话

```
"我知道今天有 ______ 总/老师在场，
但今天的规则是：没有人会因为说错话而被评判。
我们这里只有体验，没有对错。
______ 总/老师，您今天也和大家一样，是参与者，可以吗？"
```

---

## 六、话术使用原则

1. **用"我们"不用"你们"** —— 教练是团队的一部分，不是评判者
2. **问题比答案更有力量** —— 引导参与者自己思考，而不是直接给结论
3. **沉默是工具** —— 问完一个问题，等 5 秒。有人会说话的
