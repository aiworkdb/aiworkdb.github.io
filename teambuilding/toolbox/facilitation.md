# 引导话术库

> 教练即拿即用的话术模板。覆盖开场、规则讲解、过程干预、复盘引导四个场景。

---

## 一、开场白模板

### 场景：第一次见面（新员工 / 陌生团队）

```
"大家好，我是今天的教练 ______。
今天我们会通过几个小活动，让大家互相认识、体验一下团队协作的感觉。
不需要任何基础，所有活动大家都能参与。
规则很简单，我会讲完再示范一次，有任何问题随时问我。
好，我们先从第一个活动开始——"
```

### 场景：熟团队但第一次团建

```
"大家好，平时大家在会议室里开会很多，今天我们把椅子挪开，
用身体的方式感受一下——我们团队是怎么一起解决问题的。
今天没有 KPI，没有老板打分，只有参与和体验。
准备好了吗？"
```

### 场景：培训课程中的插入游戏

```
"刚才我们讲了很多理论，现在让我们用 15 分钟，
体验一下这些理论在真实互动中是什么感觉。
大家站起来，我们玩一个 ______。"
```

## 二、规则讲解话术

### 标准讲解结构（适用于任何游戏）

```
第一步：说目的（30秒）
  "这个游戏我们想体验的是 ______"

第二步：讲核心规则（1-2分钟）
  "规则很简单，只有三条：
   1. ______
   2. ______
   3. ______"

第三步：示范（30秒）
  "我找两位志愿者，我们演示一下——"

第四步：确认理解
  "大家都听懂了吗？有没有不清楚的地方？"
  （不要只等点头，让一个人复述关键规则）
```

### 当参与者问"有没有更简单的方法？"

```
"规则设计是有原因的，我们先按规则玩一轮，
如果觉得需要调整，第一轮结束后我们一起改。
先试一次？"
```

### 当有人不认真听规则讲解

```
（不要点名批评，用幽默化解）
"我看到有人已经开始在脑子里玩了，先等等，
我把规则讲完，你们就能真的玩了——"
```

## 三、过程干预话术

### 情况：活动进行中，场面混乱

```
（吹口哨或拍手，等安静）
"好，停一下。我观察了一下，大家现在的问题是 ______。
我们调整一下：______。
然后继续，从 ______ 开始。
准备好了吗？3、2、1——"
```

### 情况：有人主导一切，其他人没机会参与

```
（走近那个主导者，低声说）
"你组织得很好。我想给你一个更大的挑战——
这轮你不说话，看看团队没有你会怎么解决。
可以吗？"
```

### 情况：有人完全不参与，站在旁边

```
（走近，低声说）
"你怎么看刚才的过程？愿意加入吗？
或者你可以先当观察员，结束后告诉我们你看到了什么。"
```

### 情况：团队卡住了，不知道怎么办

```
（不要直接给答案）
"你们现在卡在哪里了？"
（等他们回答）
"好，那你们觉得可以怎么解决？"
（引导他们自己提出方案，而不是直接指挥）
```

### 情况：时间快到了但游戏没结束

```
"好，时间到了。我们停下来看一下——
如果给你们再多 5 分钟，能完成吗？
（根据情况决定：能 → 继续；不能 → 提前进入复盘）
今天我们先到这里，这个'未完成'本身就是一个值得讨论的话题。"
```

## 四、复盘引导话术

> 详见 [`debrief.md`](debrief.md) 获取完整的 4F 模型提问清单。
> 以下是开场和过渡的关键话术。

### 复盘开场

```
"好，游戏结束。给大家 1 分钟，先安静回想一下刚才的过程。
（等 1 分钟）
好，谁愿意分享一下，刚才最印象深刻的一个瞬间是什么？"
```

### 从 Facts 过渡到 Feelings

```
"刚才我们说了发生了什么。现在我想问——
当 ______ 发生的时候，大家是什么感受？"
```

### 从 Feelings 过渡到 Findings

```
"感谢大家的分享。我注意到 ______（总结一个关键观察）。
如果把这个过程看成我们团队工作的缩影，
大家看到了什么相似的地方？"
```

### 从 Findings 过渡到 Future

```
"今天体验到的这些东西，回到工作中，
有什么是你明天可以做一些不同的事情的？
不需要大改变，一个小的行动就好。"
```

### 复盘收尾

```
"谢谢大家的分享。我今天印象最深的一句话是：'______'。
希望大家带着这个体验，回到工作中。
我们今天的团建/培训，到这里就结束了。谢谢大家！"
```

## 五、特殊情况应对话术

### 有人质疑游戏的意义

```
"我理解你的疑问。这样吧，我们先玩完，
然后复盘的时候你告诉我，你觉得这个游戏和我们的工作有没有关系。
可以吗？"
```

### 有人身体不适，不能参与

```
"没问题，你今天担任'观察员'角色。
你的任务是：观察团队在游戏中的互动方式，
复盘的时候告诉我们你看到了什么。
这个角色其实很重要。"
```

### 活动效果不好，冷场了

```
（不要掩饰，直接承认）
"嗯，这个活动今天好像没有预期的效果。
你们觉得是哪里出了问题？
（引导参与者一起分析，把'失败'本身变成学习内容）
其实你们刚才已经在体验'如何应对失败'了。"
```

### 客户（甲方）在场，参与者不敢说话

```
"我知道今天有 ______ 总/老师在场，
但今天的规则是：没有人会因为说错话而被评判。
我们这里只有体验，没有对错。
______ 总/老师，您今天也和大家一样，是参与者，可以吗？"
```

---

## 六、话术使用原则

1. **用"我们"不用"你们"** —— 教练是团队的一部分，不是评判者
2. **问题比答案更有力量** —— 引导参与者自己思考，而不是直接给结论
3. **沉默是工具** —— 问完一个问题，等 5 秒。有人会说话的
4. **真诚 > 技巧** —— 话术是工具，但真诚的态度无法伪装
5. **根据现场调整** —— 话术模板是起点，不是照本宣科
