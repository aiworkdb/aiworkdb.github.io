# 病毒危机 (Virus Outbreak)

> **团队协作 · 危机应对 · 沟通决策**  
> 适合：团队协作 / 问题解决 / 领导力训练

---

## 🎯 游戏目标

模拟突发公共卫生事件的应急响应的过程，让团队在高压环境下体验**协作决策、资源分配、信息共享**的重要性。

**团建核心价值**：
- 🤝 团队协作：必须通力合作才能获胜
- 💬 沟通表达：信息分享和决策讨论
- 🎯 问题解决：在有限资源下做出最优决策
- 👑 领导力：危机中的指挥和分工

每位玩家选择一个角色，每个角色有独特能力：

| 角色 | 职责 | 特殊能力 | 团建观察点 |
| :--- | :--- | :--- | :--- |
| 🩺 **流行病学专家** | 信息收集 | 每回合可查看下一张病毒卡 | 信息分享和决策依据 |
| 🔬 **疫苗研究员** | 技术研发 | 研究行动效率+1 | 专注目标，长期坚持 |
| 🚔 **隔离指挥官** | 区域控制 | 可隔离区域，停止扩散 | 风险控制，果断决策 |
| 📦 **后勤协调员** | 资源管理 | 每回合额外获得1物资 | 资源调配，支持团队 |

## 🎮 游戏流程

### 准备阶段（5分钟）

1. **讲解背景**：突发未知病毒疫情，团队需要合作应对
2. **选择角色**：每位玩家选择1个角色
3. **布置地图**：放置地图，设置初始感染区域（住宅区、交通区）
4. **分发物资**：每人获得对应数量的行动卡

### 游戏回合（每回合10-15分钟）

每回合分为3个阶段：

#### 阶段1：团队讨论（3分钟）

**关键决策**：
- 本回合优先处理哪个区域？
- 谁去哪个区域执行行动？
- 资源如何分配？
- 信息如何共享？

**教练观察点**：
- 谁在主导讨论？
- 是否听取所有人意见？
- 决策过程是否高效？
- 有没有明确分工？

#### 阶段2：执行行动（5-7分钟）

每位玩家依次执行行动：
- 移动棋子到目标区域
- 打出行动卡，执行对应行动
- 消耗行动点（每人每回合3点）

**可选行动**：
| 行动 | 消耗 | 效果 |
| :--- | :--- | :--- |
| 🚧 隔离区域 | 2点 | 选中区域停止感染扩散1回合 |
| 💊 治疗感染 | 1点 | 所在区域感染-1 |
| 📦 物资调配 | 1点 | 获得2点物资 |
| 🚀 移动转移 | 1点 | 移动到相邻区域 |
| 🧪 疫苗研究 | 2点 | 推进疫苗进度1格 |
| 🔍 大规模检测 | 2点 | 揭露隐藏感染 |
| 🏥 建立临时医院 | 3点 | 所在区域治疗+1 |
| 📢 发布政策 | 2点 | 全城感染-1（仅1次） |

#### 阶段3：危机升级（2-3分钟）

1. **病毒扩散**：翻开1张病毒卡，执行扩散效果
   - 可能单个区域感染+1
   - 可能相邻区域同时感染
   - 可能全城范围感染

2. **随机事件**：翻开1张事件卡
   - ✅ 好消息：物资+2 或 感染-1
   - ❌ 坏消息：感染+1 或 物资-2
   - ⚡ 突发危机：立即触发额外病毒扩散
   - 🌈 转机：疫苗进度+1

3. **恐慌检查**：若有区域感染≥4，放置恐慌标记
   - 6个恐慌标记全部放完 → **游戏失败！**

#### 阶段4：简短复盘（1-2分钟）

每回合结束后，教练可提问：
- 本回合决策是否最优？
- 有没有遗漏的信息？
- 团队协作是否顺畅？

## 🏆 胜利与失败

### 胜利条件

✅ **团队胜利**：收集5个研究线索，疫苗研发成功！

**研究线索获取方式**：
1. 🧬 病毒基因序列 - 在医院区研究
2. 📊 传播路径分析 - 调查2个以上区域
3. 💉 抗体样本 - 治疗3名感染者
4. 📋 临床数据 - 完成大规模检测
5. 🧪 疫苗配方 - 累计3次疫苗研究

### 失败条件

❌ **团队失败**（任一发生）：
- 6个恐慌标记全部放完（城市崩溃）
- 感染标记用完30个（病毒失控）
- 病毒卡堆翻完但疫苗未完成（时间耗尽）

- 恐慌标记改为4个
- 进行2回合即揭晓结果

## 💡 教练引导指南

### 游戏前（设定期望）

**开场话术**：
> "接下来我们要面对一场突发疫情危机。你们是一个公共卫生团队的精英成员，必须在病毒全面扩散前研发出疫苗。这个游戏没有个人胜负，只有团队共同的胜利或失败。关键在于——你们能否高效协作、合理分工、果断决策。"

**强调要点**：
- 这是合作游戏，不是竞争
- 信息共享非常重要
- 每个人的角色都很关键
- 遇到困难时，团队要一起想办法

### 游戏中（观察与引导）

**观察要点**：

1. **沟通模式**：
   - 谁在说话？谁沉默？
   - 信息是否充分共享？
   - 讨论是否高效？有没有争执？

2. **决策过程**：
   - 决策是如何做出的？
   - 是否征求所有人意见？
   - 有没有人主导决策？

3. **角色执行**：
   - 每个人是否理解自己的角色？
   - 是否充分发挥角色能力？
   - 有没有越权或推诿？

4. **压力应对**：
   - 面对危机，团队情绪如何？
   - 有人焦虑/慌乱吗？
   - 有人保持冷静并稳定军心吗？

**适时介入**：
- 若团队陷入争执：提醒"时间紧迫，需要快速决策"
- 若有人被忽视：提问"XX，你怎么看？"
- 若信息传递不畅：建议"把你们的计划告诉大家"

### 游戏后（深度复盘）

**复盘框架**（使用 toolbox/debrief.md）：

#### 1. 事实回顾（5分钟）

**提问**：
- 游戏结果是什么？（胜利/失败）
- 哪个时刻最紧张？
- 你们一共完成了几个研究线索？

#### 2. 感受分享（5分钟）

**轮流发言**：
- 你在游戏中最强烈的感受是什么？
- 哪个时刻让你感到压力/兴奋/沮丧？
- 你对团队的表现满意吗？

#### 3. 学习洞察（10分钟）

**讨论问题**：

**关于沟通**：
- 你们的信息共享是否充分？
- 有没有重要信息被遗漏或误解？
- 如果重新玩一次，你们会如何改进沟通？

**关于协作**：
- 团队的分工是否合理？
- 有没有出现"各自为战"的情况？
- 角色之间如何配合？

**关于决策**：
- 你们做决策的过程是怎样的？
- 有没有后悔的决策？为什么？
- 在压力下，决策质量如何？

**关于领导力**：
- 团队中有没有明确领导者？
- 领导方式是怎样的？（指令型/协商型/服务型）
- 这种领导方式效果如何？

#### 4. 行动迁移（5分钟）

**联系实际工作**：
- 这个游戏让你想到工作中的哪些场景？
- 游戏中的经验如何应用到实际工作中？
- 你们团队可以立即改进的一件事是什么？

## 🔄 变体玩法

### 变体1：沉默模式（沟通挑战）

**规则**：
- 游戏前半段（前2回合）禁止说话
- 只能通过写字或手势沟通
- 后半段恢复正常沟通

**目的**：
- 体验沟通障碍
- 锻炼非语言沟通
- 珍惜正常沟通的机会

**复盘重点**：
- 没有语言沟通时，你们如何协调？
- 误解是如何产生的？
- 恢复正常沟通后，有什么感受？

### 变体2：角色轮换（同理心训练）

**规则**：
- 每回合结束后，角色顺时针轮换
- 必须适应新的能力和责任

**目的**：
- 理解不同角色的挑战
- 培养同理心
- 提升适应能力

### 变体3：记者模式（信息透明）

**规则**：
- 每回合指定1位"记者"
- 记者可以查看所有卡牌和标记
- 但必须通过口头报告分享信息

**目的**：
- 锻炼信息整合和表达
- 训练倾听和理解能力
- 观察信息传递的效率

### 变体4：限时挑战（压力测试）

**规则**：
- 每个决策只有30秒讨论时间
- 用计时器严格限制
- 强制快速决策

**目的**：
- 体验高压决策
- 训练快速共识达成
- 观察压力下的团队动态

## 📝 常见问题

**Q：如果团队一直失败怎么办？**
A：可以降低难度（增加行动点、减少病毒卡），重点是体验过程，不是结果。失败后的复盘往往比胜利更有价值。

**Q：有人过于强势，主导所有决策怎么办？**
A：适时介入，提醒"听听其他人的意见"。复盘时讨论"领导风格"和"团队参与"。

**Q：有人完全不说话怎么办？**
A：主动提问"XX，你觉得呢？"。复盘时讨论"如何让每个人参与"。

**Q：游戏时间拖得太长怎么办？**
A：设置严格的时间限制，用计时器。或简化为"快速流程"。

**Q：能不能多人扮演同一个角色？**
A：可以，但建议4-6人最佳。超过6人会有等待时间，影响体验。

## 🌟 成功案例

### 案例1：某互联网公司产品团队

**背景**：跨部门协作项目，沟通不畅

**游戏过程**：
- 团队在游戏中体验到信息孤岛的危害
- 疫苗研究员没有及时分享研究进展，导致决策失误
- 复盘时联想到工作中类似问题

**收获**：
- 建立了定期同步机制
- 明确了信息共享的责任
- 改进了跨部门沟通流程

### 案例2：某制造企业中层管理培训

**背景**：危机管理能力提升

**游戏过程**：
- 在"限时挑战"变体中，团队体验到高压决策
- 领导者快速做出决策，但后续发现信息不足
- 复盘讨论"快速决策"与"充分信息"的平衡

**收获**：
- 认识到信息收集的重要性
- 练习了在压力下保持冷静
- 建立了危机应对的沟通协议
