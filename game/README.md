## 🎮 如何玩

**操作只有一件事**：点击屏幕，翻转重力方向。

```
手机向右倾斜  →  重力向下，小球落向底部
手机向左倾斜  →  重力向上，小球飞向顶部
轻点屏幕      →  立即翻转重力（紧急逃脱用）
```

## ✨ 特色系统

### 连击加分
连续收集能量球可触发倍数加成：
- **combo × 2** = combo ≥ 3 时，基础分翻倍
- **combo × N** = combo 越高，加分越多

### 翻转无敌帧
每次翻转后有 **0.15 秒无敌时间**，防止手滑误触障碍物。

### 危险边缘预警
障碍物靠近时，通道**对应方向的边缘**会亮起红色光晕，提示你提前准备。不闪屏，不晃眼。

### 重力感应
支持手机陀螺仪控制方向，沉浸感更强。两种操控方式并存，灵活切换。
