# 团建文章公众号发布 - 本地运行指南

## 📋 概述

本指南将帮助您在本地电脑上批量发布团建文章到微信公众号。

**已准备的文件：**
- ✅ `publish_local.py` - 批量发布脚本
- ✅ 5 篇核心文章已格式化并添加了图片

---

## 🚀 快速开始（3 步）

### 步骤 1：安装 wenyan-mcp

在本地电脑打开终端（命令行），运行：

```bash
npm install -g @wenyan-md/mcp
```

**验证安装：**
```bash
wenyan-mcp --version
```

如果显示版本号（如 `2.0.3`），说明安装成功。

---

### 步骤 2：下载代码

如果您还没有下载代码，运行：

```bash
git clone https://github.com/aiworkdb/aiworkdb.github.io.git
cd aiworkdb.github.io
```

如果已下载，更新到最新版本：

```bash
git pull
```

---

### 步骤 3：运行发布脚本

```bash
python publish_local.py
```

脚本将自动发布 5 篇核心文章到公众号草稿箱。

**预计时间：** 约 1-2 分钟（每篇间隔 3 秒）

## ❓ 常见问题

### Q1：脚本报错"找不到 wenyan-mcp"

**解决方案：**

**Windows:**
```bash
# 使用完整路径
set WENYAN_PATH=C:\Users\您的用户名\AppData\Roaming\npm\wenyan-mcp.cmd
```

**Mac/Linux:**
```bash
# 使用完整路径
export WENYAN_PATH=/usr/local/bin/wenyan-mcp
```

然后编辑 `publish_local.py`，修改：
```python
WENYAN_PATH = "完整路径"
```

---

### Q2：文章发布失败（图片下载失败）

**原因：** 图片 URL 可能无法访问

**解决方案：**

1. 手动下载图片到本地
2. 将图片放到 `teambuilding/images/` 目录
3. 修改文章的 `cover:` 字段为本地路径

示例：
```yaml
---
title: "文章标题"
author: "AIWorkDB"
cover: "images/cover.png"
---
```

---

### Q3：微信公众号 API 报错（IP 白名单）

**错误信息：** `40164: invalid ip, not in whitelist`

**解决方案：**

1. 登录微信公众号后台
2. 进入：**开发 → 基本配置 → IP 白名单**
3. 添加您本地电脑的公网 IP
4. 保存后重新运行脚本

**查看您的公网 IP：**
访问：https://ip.sb 或 https://ifconfig.me

---

### Q4：发布成功但公众号后台看不到

**可能原因：**

1. 文章在"草稿箱"，不在"素材库"
2. 刷新页面
3. 检查是否登录了正确的公众号

## 🔧 高级用法

### 发布单篇文章

如果您只想发布某篇文章，可以直接对 AI 说：

```
使用 wenyan-mcp 发布 teambuilding/README.md 到公众号，主题选择 pie
```

### 修改主题

编辑 `publish_local.py`，修改：
```python
THEME_ID = "classic"  # 改为其他主题
```

**可用主题：**
- `pie` - 科技风（推荐）
- `classic` - 经典风格
- `news` - 新闻风格
- `gallery` - 画廊风格

查看所有主题：
```bash
wenyan-mcp list_themes
```

---

### 发布更多文章

编辑 `publish_local.py`，在 `ARTICLES` 列表中添加更多文章：

```python
ARTICLES = [
    {
        "file": "teambuilding/README.md",
        "name": "团建游戏库介绍",
        "priority": 1
    },
    # 添加更多文章...
]
```
