# 桌游目录

本目录用于存放桌游相关的资料和信息。

## 目录结构

```
boardgames/
├── README.md          # 本文件，桌游索引目录
├── category/          # 按类别分类的桌游
│   ├── strategy/      # 策略类桌游
│   ├── party/         # 派对类桌游
│   ├── card/          # 卡牌类桌游
│   └── cooperative/   # 合作类桌游
└── resources/         # 资源文件
    ├── images/        # 桌游图片
    └── rules/         # 规则说明
```

## 桌游索引

| 游戏名称 | 类别 | 玩家人数 | 游戏时长 | 难度 |
| :--- | :--- | :--- | :--- | :--- |
| 示例游戏 | 策略 | 2-4人 | 30-60分钟 | 中等 |
| [宝藏猎人](category/strategy/treasure-hunter/README.md) | 策略 | 2-4人 | 45-60分钟 | 中等 |

## 添加新桌游

如需添加新桌游，请按照以下步骤操作：

1. 在相应的类别目录下创建游戏文件夹
2. 添加游戏介绍、规则说明和相关图片
3. 在本README的索引表格中更新游戏信息
