# 2026-05-01

## 卡牌设计工具全面升级

将 `tools/card-maker.html` 从基础卡牌生成器（663行）升级为专业级桌游卡牌设计工具。核心改进：

### 视觉升级
- 经典桌游卡牌风格：双层金色装饰边框 + 四角花纹 SVG 装饰
- 四级稀有度系统：普通(灰) / 精良(蓝光) / 稀有(金光) / 传说(彩虹流光动画)
- 四种背景纹理：羊皮纸 / 暗夜魔法 / 战场铁甲 / 自然森林
- 十二种颜色主题 + 宝石风格数值标签（费用/攻击/防御/生命值）
- 三种卡牌模板：经典 / 对战 / 资源

### 功能升级
- 三栏布局：左编辑 / 中预览 / 右列表
- 48+ Emoji 图标 + 自定义图片上传
- 多卡组管理（创建/切换/删除）
- localStorage 持久化保存
- JSON 导入导出
- html2canvas PNG 导出（单张/批量）
- 带裁切线的 A4 打印
- 撤销/重做（Ctrl+Z / Ctrl+Y）
- 卡牌列表支持编辑/复制/删除

### 技术要点
- 用 wrapper div（card-outer + card-inner）实现渐变边框，避免 mask 兼容性问题
- CSS 变量系统控制主题色
- 稀有度通过 CSS 类切换 + box-shadow/animation 实现
- 主题映射表 THM 将主题 ID 映射到 CSS 类后缀
- 同时更新了 `tools/README.md` 文档
