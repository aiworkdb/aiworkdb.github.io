# 基因编辑实验室 Gene Editing Lab

> 扮演基因编辑科学家团队，合作治愈罕见病患者！研究基因序列、设计编辑方案、应对突发变异，在病毒扩散前完成基因疗法！

## 🎮 游戏概述

**基因编辑实验室**是一款 1-4 人的合作策略桌游。2026年，基因编辑技术已取得突破，罕见病患者等待救命疗法。玩家组成科研团队，通过**收集基因序列**、**设计编辑工具**、**测试安全性**来研发基因疗法。

游戏的核心在于**团队合作与时间的赛跑**：患者体内的病毒正在扩散，你们必须在有限回合内完成疗法研发。同时，基因编辑有脱靶风险，需要平衡进度与安全。

游戏适合喜欢合作解谜、科学主题的玩家，规则中等复杂度，紧张刺激。

## 🎮 游戏规则

### 准备阶段

1. **洗牌**：将基因序列卡、编辑工具卡、病毒卡、事件卡分别洗匀。
2. **发放起始卡**：每位玩家抽取 **3 张基因序列卡**和 **2 张编辑工具卡**。
3. **摆放面板**：每人取一张实验室面板放在自己面前。
4. **放置病毒标记**：在病毒扩散 tracker 上放置 **0 标记**。
5. **确定先手**：最近关注生物科技新闻的玩家先手。

### 游戏流程

每轮按以下步骤进行：

#### 第一步：玩家行动 🔬
每位玩家依次进行 **2 个行动**（可相同）：
- **收集基因**：抽取 1 张基因序列卡
- **设计编辑**：支付成本，使用编辑工具卡修复突变基因
- **测试安全**：对修复基因进行安全性测试
- **应对事件**：处理当前事件卡

#### 第二步：病毒扩散 🦠
- 翻一张**病毒卡**，推进病毒扩散进度
- 病毒进度达到 **10** 时，游戏失败！

#### 第三步：检查胜利条件 🏆
- 检查是否**修复了所有突变基因**
- 完成者立即获胜
- 若未完成，进入下一轮

### 游戏结束

**修复所有突变基因**或**病毒扩散进度达到 10**时，游戏结束。

| 结果 | 条件 |
| :--- | :--- |
| **胜利** | 在病毒扩散前修复所有突变基因 |
| **失败** | 病毒扩散进度达到 10 |

## 💡 新手策略提示

1. **优先修复关键靶点**：关键靶点修复后，其他突变基因修复难度降低！
2. **平衡进度与安全**：不要只追求速度，安全性测试很重要！
3. **团队合作**：分享手牌信息，协同设计编辑方案！
4. **管理资源**：编辑工具卡有限，合理使用！
5. **应对事件**：提前准备应对事件的资源！
