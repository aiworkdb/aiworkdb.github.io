# 大圣西游 (Wukong's Journey)

## 🎮 游戏概述

《大圣西游》是一款以中国古典名著《西游记》和热门游戏《黑神话：悟空》为背景的合作类桌游。玩家将扮演唐僧的四个弟子——孙悟空、猪八戒、沙悟净、白龙马，携手踏上西天取经的万里征程。

在旅途中，你们将共同面对九九八十一难：凶猛的妖魔、险恶的地形、诡异的幻术，以及内心的考验。只有团结协作、合理运用每位角色的独特能力，才能战胜一切困难，取得真经，修成正果！

这是一个适合家庭聚会和朋友合作的策略游戏，规则清晰易懂，强调团队配合，充满中国神话色彩和冒险乐趣。

## 🎲 游戏规则

### 准备阶段

1. **选择角色**：每位玩家选择1个角色（1人局可选择2-3个角色轮流操作）
2. **放置棋子**：所有棋子放在起点「长安 🏯」
3. **初始化状态**：
   - 每位玩家生命值设为角色卡上的初始值
   - 每人获得1张随机法宝卡
   - 将妖魔卡、法宝卡、劫难卡分别洗混，面朝下放好
4. **设置回合数**：将回合标记放在第1回合位置（最大25回合）
5. **决定先手**：掷骰子，点数最高者先手

### 游戏流程（每个回合）

所有玩家**共同完成1个回合**（合作游戏，轮流行动但共享胜利/失败）。

每回合分为以下阶段：

#### 阶段1：移动阶段
- 当前玩家掷2颗骰子，取点数之和作为移动步数
- 沿地图路线移动棋子（可自主选择分支路线）
- 部分角色有移动加成（如白龙马+2步）

#### 阶段2：地形事件
根据停留的站点类型，执行对应操作：

| 站点类型 | 操作 |
| :--- | :--- |
| **安全营地** | 可恢复1-2生命，选择是否使用法宝 |
| **佛经宝地** | 抽取1张佛经卡，加入团队收藏 |
| **妖魔巢穴** | 翻转1张妖魔卡，进入**战斗阶段** |
| **劫难事件点** | 抽取1张劫难卡，立即执行效果 |
| **Boss战点** | 进入**Boss战阶段**（全员参与） |

#### 阶段3：战斗阶段（如遇妖魔）

战斗采用骰子机制：
1. 玩家掷2颗骰子，加上角色攻击加成和装备加成
2. 若总点数 ≥ 妖魔生命值，则击败妖魔
3. 若总点数 < 妖魔生命值，则玩家受到妖魔攻击力伤害
4. 可使用法宝卡增强战斗或保护自己
5. 其他玩家可在自己回合协助战斗（见"团队协作"）

**战斗示例**：
> 孙悟空（攻击+2，金箍棒+2）攻击小妖（生命2）。
> 掷骰得到3+4=7，加上攻击加成2+2=4，总攻击力11 ≥ 2，击败！

#### 阶段4：Boss战阶段（如遇Boss）

Boss战需要**全员协作**：
1. 所有玩家依次对Boss进行攻击
2. Boss在每个玩家攻击后反击（对全员造成伤害）
3. Boss有特殊能力，需团队策略应对
4. 击败Boss后，全员获得Boss奖励包（法宝+佛经）

#### 阶段5：行动结束
- 玩家可使用法宝卡（消耗类）
- 检查佛经收集进度
- 将回合交给下一位玩家

### 胜利条件

**团队胜利**（满足以下任一条件）：
1. 到达终点「灵山 ✨」并通过最终考验（连续击败2场战斗）
2. 集齐7张佛经卡并在任意Boss战中获胜

**团队失败**（满足以下任一条件）：
1. 所有玩家生命值归零（阵亡）
2. 回合数达到25回合仍未到达终点
3. 在劫难卡"唐僧被抓"效果下未能及时击败妖魔

## 📖 完整游戏流程示例

### 示例1：正常流程（2人局）

**第1回合（玩家A-孙悟空）**：
1. 掷骰子得到5点，移动5步，到达「流沙河 🌊」（普通路段）
2. 无战斗，选择使用"筋斗云"法宝，再移动5步
3. 到达「五庄观 🌳」（佛经宝地），抽取《金刚经》，团队防御+1
4. 回合结束，生命值：悟空5/5，八戒6/6

**第2回合（玩家B-猪八戒）**：
1. 掷骰子得到3点，移动3步，到达「白骨洞 ⚰️」（妖魔巢穴）
2. 翻转妖魔卡：白骨夫人（生命7，攻击3）
3. 八戒攻击：掷骰4+2=6，攻击加成+1（钉耙）=7，总攻击7
4. 白骨夫人生命7，恰好击败！获得Boss奖励包（随机2法宝卡）
5. 回合结束

**第3回合（玩家A-孙悟空）**：
1. 掷骰子得到6点，移动6步，到达「西凉女国 🏰」（安全营地）
2. 选择休息：恢复1生命（悟空5→5，已满）
3. 使用法宝"蟠桃"，恢复2生命（无变化，已满）
4. 回合结束

**第4回合（玩家B-猪八戒）**：
1. 掷骰子得到4点，移动4步
2. 到达「火焰山 🔥」（劫难事件点）
3. 抽取劫难卡：「妖雾弥漫」——下1个妖魔战斗力+1
4. 继续移动2步（火焰山效果：需击败牛魔王才能通过，触发Boss战）
5. **Boss战：牛魔王（生命9，攻击4）**
   - 八戒先攻：掷骰5+3=8，攻击+1=9，总攻击9 = 牛魔王生命9，击败！
   - 获得Boss奖励包（紫金红葫芦+《法华经》）
6. 回合结束，团队攻击+1（法华经效果）

**第7回合（两队汇合）**：
1. 悟空和八戒同时到达「小雷音寺 🛕」
2. 触发双Boss战：黄眉老佛（生命6，攻击3）+ 二十八星宿（生命8，攻击3）
3. 两人协作：悟空攻击黄眉（击败），八戒攻击星宿（剩余3生命，下回合继续）
4. 回合结束

### 示例2：Boss战协作（4人局）

**场景**：4位玩家同时到达第21站「铜台府 🏯」，触发Boss战「六耳猕猴」

1. **回合开始**：六耳猕猴（生命8，攻击3，特殊：模仿）
2. **玩家1（悟空）**：攻击掷骰6+1=7，攻击+2（金箍棒）=9，总攻击11 ≥ 8，Boss剩余0，但六耳猕猴触发"分身"——重生为4生命
3. **玩家2（八戒）**：攻击重生后的六耳猕猴（生命4），掷骰3+2=5，攻击+1=6，总攻击6 ≥ 4，击败分身！但真身仍在
4. **玩家3（悟净）**：使用法宝"紫金红葫芦"，直接吸收真身（剩余1次使用）
5. **玩家4（白龙马）**：使用"筋斗云"快速移动到下一站
6. **结果**：Boss被击败，全员获得《般若经》+ 2随机法宝

## 💡 新手策略提示

1. **优先收集佛经**：佛经卡的团队增益非常强大，尽早收集可大幅降低后期难度
2. **合理分配角色**：悟空适合进攻，八戒均衡，悟净防守，白龙马辅助——根据团队需要选择
3. **节约消耗法宝**：蟠桃、人参果等恢复类法宝留到Boss战前使用
4. **利用安全营地**：经过营地时务必休息恢复，不要着急赶路
5. **关注劫难卡**：部分劫难卡有正面效果（如"佛祖显灵"），不要害怕触发
6. **团队配合是关键**：合作游戏的核心是沟通，商量好谁主攻、谁辅助
7. **白龙马的移动优势**：早期让白龙马快速探索地图，标记安全路线
8. **集齐7张佛经是强力后盾**：即使未到终点，集齐佛经也能触发"真经之力"大幅提升战力

## 🔄 变体规则

### 单人修行模式
- 1位玩家操作所有4个角色
- 每个角色轮流行动，共享回合
- 角色阵亡后可用其他角色继续
- 适合熟悉游戏机制

### 竞速模式
- 将合作改为竞争：每位玩家独立操作角色
- 先到达灵山并通过最终考验的玩家获胜
- 可互相使用劫难卡干扰对方

### 黑暗取经（高难度）
- 妖魔卡和劫难卡翻开后不告知其他玩家
- 玩家需依靠角色能力（如火眼金睛）来提前获取信息
- 适合喜欢挑战和推理的玩家

### 真·八十一难（专家模式）
- 地图扩展至81站（对应八十一难）
- 增加更多妖魔种类和Boss战
- 回合数限制提升至40回合
- 推荐有经验的玩家尝试

### 黑神话联动模式
- 引入《黑神话：悟空》中的Boss（如赤尻马猴、黄风大圣）
- 增加"天命人"角色（特殊能力：可复活1次）
- 添加"奇术"卡牌系统（参考游戏中的法术）

## ❓ 常见问题

**Q：如果我没有足够的卡纸打印卡牌怎么办？**
A：可以用普通A4纸打印后贴在硬纸板上，或者用现成的扑克牌背面贴打印的卡面。

**Q：骰子可以用什么替代？**
A：手机上有很多免费的骰子应用，或者自己制作1-6的数字抽签条。

**Q：4人游戏时角色分配不完怎么办？**
A：可以让部分玩家操作多个角色，或者使用"分身术"规则（见角色卡说明）。

**Q：佛经卡被拿完了怎么办？**
A：将弃牌堆（已使用的消耗类佛经效果）洗混后重新使用。

**Q：Boss战太难了，打不过怎么办？**
A：确保收集了足够的佛经卡，合理使用法宝卡，优先升级团队攻击和防御。

**Q：劫难卡的效果可以拒绝吗？**
A：不可以，劫难卡代表取经路上的无常，必须接受。

**Q：可以在自己的回合使用其他玩家的法宝吗？**
A：可以，合作游戏鼓励资源共享，但需经过持有者同意。

**Q：游戏可以暂停保存吗？**
A：可以，用便签纸记录当前回合数、各角色生命值和位置，下次继续。
