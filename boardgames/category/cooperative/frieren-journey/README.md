# 芙莉莲的旅程

> *"去了解更多关于这个世界的魔法吧。"* —— 芙莉莲

---

## 1. 游戏概述

**芙莉莲的旅程**是一款合作类桌游，灵感来源于热门奇幻动漫《葬送的芙莉莲》。

玩家将扮演大魔法师**芙莉莲**、勇者弟子**费伦**、以及年轻战士**修塔尔克**，组成冒险小队，在广阔的艾兰大陆上展开一段缓慢而深刻的旅途。

在旅途中，你们需要：
- 🧙 收集并学习古老的魔法
- ⚔️ 与魔物战斗，保护彼此
- 📜 完成村庄委托，获得报酬与声望
- 🏔️ 穿越危险地形，前往传说中的「天国之地」

**适合人群**：适合2-4人合作游玩，推荐喜爱奇幻冒险、合作推理、慢节奏策略的玩家。游戏充满《芙莉莲》特有的「旅途感」——不是为了打败最终BOSS，而是享受沿途的风景与成长。

## 4. 游戏地图详解

### 地图布局概览

游戏地图为一条从**起点「奥伊伦贝尔」**到**终点「天国之地」**的线性旅途路线，途中经过多个地点。

```
🏰 奥伊伦贝尔（起点）
    ↓
🌲 艾兰大森林
    ↓
🏔️ 维根山脉
    ↓
🏘️ 各各马大公国
    ↓
🏜️ 南部荒漠
    ↓
🌊 西部海岸
    ↓
🏛️ 王都
    ↓
⛰️ 终焉山脉
    ↓
🏔️ 天国之地（终点）✨
```

### 地形类型说明

| 地形 | Emoji | 效果 | 出现频率 |
| :--- | :--- | :--- | :--- |
| 森林 | 🌲 | 可采集草药（回复HP） | 常见 |
| 山脉 | 🏔️ | 移动消耗+1，但可获得魔法石 | 常见 |
| 村庄 | 🏘️ | 可休息（完全回复HP/MP），接取委托 | 中等 |
| 沼泽 | 🐊 | 移动需掷骰>3，否则停留原地 | 少见 |
| 遗迹 | 🏛️ | 可学习新魔法（抽取魔法卡） | 中等 |
| 洞穴 | 🕳️ | 可能遭遇魔物，但宝藏多 | 少见 |
| 城镇 | 🏰 | 可交易资源，补充物资 | 中等 |
| 终点区域 | ✨ | 到达即触发最终章节 | 唯一 |

### 地图特点与资源统计

- 地图共 **8个主要区域**，每个区域有独特事件与资源
- 完整旅途需要 **15-20回合**（时间沙漏token用尽前）
- 每个区域到达时，需抽取1张**地形卡**决定具体情况
- 地图上共有 **6个村庄**，可接取委托和休息
- **魔法遗迹**（3处）是学习高级魔法的唯一途径

## 6. 游戏规则

### 6.1 准备阶段

1. **选择角色**：每位玩家选择1张角色卡，领取对应的HP/MP标记。
2. **洗牌**：将魔法卡、魔物卡、事件卡、委托卡、地形卡分别洗匀，面朝下放好。
3. **放置棋子**：将所有玩家棋子放在起点「奥伊伦贝尔」。
4. **初始资源**：每位玩家获得魔法石×2，金币×1，草药×1。
5. **时间沙漏**：放置6个时间token（代表旅途的时间限制）。
6. **决定旅途节奏**：选择「慢旅途」（20回合，简单）或「快旅途」（15回合，困难）。

### 6.2 游戏流程（每回合）

每回合按以下顺序进行：

#### 阶段1：移动阶段
- 当前玩家掷骰子（1d6）
- 棋子沿路线前进对应步数
- 经过村庄可短暂停留（接取委托/休息）

#### 阶段2：地形揭示
- 到达新区域时，抽取1张**地形卡**
- 根据地形类型，触发对应效果

#### 阶段3：事件阶段
- 抽取1张**事件卡**
- 友好事件：立即获得好处
- 危险事件：立即承受后果
- 抉择事件：团队讨论后共同决定（需全员同意，或多数表决）

#### 阶段4：委托阶段（可选）
- 如果当前在村庄，可以接取1张**委托卡**
- 完成委托后获得奖励

#### 阶段5：遭遇阶段（如触发战斗）
- 如果事件或地形触发了战斗，抽取**魔物卡**
- 进入战斗流程（见下方）

#### 阶段6：学习阶段（如在遗迹/魔法塔）
- 可以消耗魔法石学习1张新魔法
- 从魔法卡堆顶抽取等价数量的卡牌，选择1张学习

#### 阶段7：回合结束
- 检查时间沙漏：消耗1个时间token（每到达1个新区消耗1个）
- 如果时间token用尽，触发「时间耗尽」——必须在3回合内到达终点，否则**团队失败**

### 6.3 战斗流程

当遭遇魔物时，进入战斗：

1. **魔物行动顺序**：魔物在玩家行动**之后**攻击（类似回合制）
2. **玩家回合**：玩家按顺序行动，每人每回合可以：
   - 使用1张魔法卡（需消耗MP）
   - 进行1次物理攻击（消耗攻击力数值）
   - 使用1个物品（草药/魔法石）
   - 防御（本回合受到伤害减半）
3. **魔物攻击**：所有魔物按顺序攻击，计算伤害
4. **战斗结束条件**：
   - 所有魔物HP归零 → **胜利**，获得奖励
   - 所有玩家HP归零 → **失败**，失去1个时间token，回到最近村庄

> 💡 **合作提示**：魔物的攻击目标由玩家共同决定（保护HP低的角色！）

### 6.4 胜利条件

团队**共同胜利**需要满足以下条件之一：

| 胜利方式 | 条件 |
| :--- | :--- |
| 🏆 完美结局 | 到达「天国之地」且完成至少3个B级委托 |
| 🎯 标准结局 | 到达「天国之地」 |
| 🌟 隐藏结局 | 收集全部8张II级以上魔法卡，且全员存活到达终点 |

### 6.5 失败条件

团队**共同失败**如果满足以下任一条件：

- ⏳ 时间沙漏token全部用尽，且3回合内未到达终点
- 💀 所有玩家HP归零（且无人持有「复活魔法」）
- 💀 累计失败5次委托

> **场景**：芙莉莲（玩家A）、费伦（玩家B）、修塔尔克（玩家C）从「奥伊伦贝尔」出发。

## 9. 变体规则

### 9.1 单人模式（Solo Mode）

- 控制全部3名角色，每回合可以操作所有角色
- 时间token增加至8个（单人节奏慢一些）
- 敌人HP降低20%（平衡单人难度）

### 9.2 困难模式（Heroic Mode）

- 时间token减少至4个
- 所有魔物HP+30%，攻击力+1
- 事件卡中有50%概率出现「双重事件」（同时触发2张事件卡）

### 9.3 快速模式（Quick Mode）

- 时间token固定为10个（无压力模式）
- 地图简化为5个区域（去掉荒漠和海岸）
- 适合新手学习规则，时长约30-45分钟

### 9.4 剧情模式（Story Mode）

- 按照动漫剧情顺序解锁区域
- 每个区域有固定事件（不再是随机抽取）
- 加入「情感片段」卡：触发后朗读一段动漫经典台词，全员回复1HP（叙事治愈效果）

### 9.5 扩展角色：海塔

- 4人游戏时，加入海塔（僧侣/治疗者）
- 海塔的特殊能力「治愈术」每回合可以回复2HP
- 加入海塔后，时间token+1（平衡4人难度）

## 10. 常见问题

### Q1：如果所有玩家都倒下了怎么办？
**A**：如果持有「复活魔法」卡，可以立即使用，复活1个角色至3HP。如果没有，则团队失败，游戏结束。建议在新游戏时，芙莉莲优先学习「复活魔法」。

### Q2：时间token用尽了怎么办？
**A**：时间token用尽后，触发「时间耗尽」状态。团队必须在**3回合内**到达终点「天国之地」，否则团队失败。在「时间耗尽」状态下，所有角色每回合HP-1（旅途疲劳）。

### Q3：可以跳过战斗吗？
**A**：部分魔法可以跳过战斗（如「闪光魔法」使魔物本回合无法攻击，然后可以逃离），或者使用「隐身魔法」直接通过。但BOSS战斗无法跳过。

### Q4：魔法卡用完了怎么办？
**A**：魔法卡在用完后（施展魔法后会放入「魔法墓地理牌堆」），可以在遗迹地区重新抽取新的魔法卡。另外，部分事件卡也会奖励魔法卡。

### Q5：多人游戏时，怎么决定行动顺序？
**A**：建议按「芙莉莲→费伦→修塔尔克」的顺序（动漫中的角色顺序）。也可以每回合轮换起始玩家。

### Q6：禁忌魔法真的要用吗？
**A**：禁忌魔法非常强大，但副作用也很大。建议只在BOSS战或危急时刻使用。记住：累计使用3次会触发「魔法反噬」——请谨慎！

### Q7：打印出来的卡牌太小怎么办？
**A**：printables.html中的卡牌设计为标准尺寸（63mm×88mm，与市面卡牌尺寸一致）。可以用剪刀裁剪，或者送到打印店塑封后使用。

### Q8：游戏太难了怎么办？
**A**：可以先玩「快速模式」（见变体规则），熟悉规则后再挑战标准模式。也可以增加时间token数量（比如用8个而不是6个）。

### Q9：可以和朋友在线远程玩吗？
**A**：可以！使用printables.html打印道具后，通过视频通话分享屏幕，或者各自打印一套道具，通过语音通话协同游戏。

### Q10：「天国之地」到底是什么？
**A**：在动漫中，「天国之地」是大陆最南端的地方，据说去了那里就能见到已故的勇者辛美尔。在游戏中，它是旅途的终点，也是团队共同目标的实现地。
