# 芙莉莲的旅程 (Frieren's Journey)

## 🎮 游戏概述

《芙莉莲的旅程》是一款合作类桌游，改编自热门动漫《葬送的芙莉莲》。玩家将扮演大魔法使芙莉莲、弟子费伦、战士修塔尔克和祭司海塔，组成冒险小队，接受委托，探索魔法，击败魔族，体验旅途中的喜怒哀乐。

这是一款**合作游戏**，所有玩家共同对抗游戏系统。只有通过团队合作、合理分配资源、善用每个角色的独特能力，才能顺利完成旅途中的所有章节！

**适合人群**：喜欢合作游戏、动漫粉丝、策略思考者、2-4人聚会

1. **线性路线**：从出发村到终点，玩家沿着路线前进
2. **分支路径**：某些位置有多个方向可选
3. **资源分布**：治愈点和魔法点分布在路线上
4. **战斗分布**：怪物卡在关键位置等待玩家

### 移动规则

- 玩家轮流进行，每回合掷1颗骰子
- 按照地图上的道路移动棋子
- 只能上下左右移动，不能斜向移动
- 某些地形（山脉）需要额外步数

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择角色**：每位玩家选择一张角色卡，获得对应棋子
3. **放置棋子**：所有棋子放在出发村🏘️
4. **初始状态**：
   - 生命值：根据角色卡设定
   - 魔力值：初始3点
   - 手牌：每位玩家抽取3张卡牌（根据角色类型）
5. **洗混卡牌**：将怪物卡、事件卡、任务卡分别洗混，放在地图旁
6. **放置标记**：将生命值标记和魔力标记放在旁边

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 当前玩家掷1颗骰子，点数决定移动步数
- 按照地图上的道路移动棋子
- 注意：某些地形（山脉🏔️）需要额外步数

#### 2. 执行所在位置的事件

根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **出发村** | 🏘️ | 可以休息，恢复1点生命值 |
| **道路** | ⬛ | 无特殊效果，可以结束回合 |
| **森林** | 🌲 | 可以采集草药，恢复2点生命值 |
| **任务点** | 📜 | 抽取1张任务卡，接受委托 |
| **战斗点** | ⚔️/👹 | 抽取1张怪物卡，进入战斗 |
| **宝藏点** | 💎 | 抽取1张宝藏卡 |
| **魔法点** | 🔮 | 学习1张新魔法卡 |
| **治愈点** | ✨ | 恢复3点生命值 |
| **终点** | 🏯 | 如果完成所有任务，游戏胜利！ |

#### 3. 战斗系统（如果停在战斗点）

当玩家停在战斗点时，触发战斗：

1. **抽取怪物卡**：从怪物卡堆顶抽取1张怪物卡
2. **战斗回合**：
   - 玩家回合：使用卡牌攻击怪物（魔法卡/装备卡/祈祷卡/知识卡）
   - 怪物回合：怪物攻击玩家，造成对应伤害
3. **伤害计算**：
   - 玩家攻击力 = 角色基础攻击 + 卡牌加成
   - 怪物攻击力 = 怪物卡上标注的攻击力
4. **战斗结束**：
   - 怪物生命值 ≤ 0：玩家胜利，获得奖励
   - 玩家生命值 ≤ 0：玩家阵亡，需要海塔使用复活祈祷

#### 4. 使用卡牌（可选）

在回合中任意时刻可以使用手牌：

- **魔法卡**：芙莉莲使用，消耗魔力值
- **装备卡**：修塔尔克使用，装备后持续生效
- **祈祷卡**：海塔使用，消耗魔力值
- **知识卡**：费伦使用，即时效果

#### 5. 抽取新卡牌

回合结束前，玩家从对应卡牌堆抽取1张卡牌补充手牌。

#### 6. 回合结束

将回合交给下一位玩家。

### 胜利条件

**合作胜利**：所有玩家共同完成以下目标即获胜：

1. 完成所有任务卡（通常5-7张，取决于游戏难度）
2. 到达终点🏯

**失败条件**：

- 所有玩家阵亡（生命值 ≤ 0 且无法复活）
- 回合数超过30回合（可选限时模式）

## 📖 完整游戏流程示例

### 示例1：正常探索回合

**玩家A（芙莉莲）的回合：**

1. **掷骰子**：得到4点，移动4步
2. **停留位置**：森林🌲
3. **执行事件**：采集草药，恢复2点生命值（从6恢复到8）
4. **使用卡牌**：使用"火焰魔法🔥"（消耗2魔力），但因为不在战斗，所以保留
5. **补充手牌**：从魔法卡堆抽取1张"冰结魔法❄️"
6. **回合结束**：将回合交给玩家B

**玩家B（修塔尔克）的回合：**

1. **掷骰子**：得到3点，移动3步
2. **停留位置**：战斗点⚔️
3. **触发战斗**：抽取1张怪物卡"哥布林👺"（生命值5，攻击力2）
4. **战斗回合1**：
   - 玩家B攻击：使用"勇者之剑⚔️"（伤害+3），造成3点伤害，哥布林生命值剩余2
   - 哥布林攻击：造成2点伤害，玩家B生命值从10降到8
5. **战斗回合2**：
   - 玩家B攻击：使用普通攻击（伤害3），造成3点伤害，哥布林阵亡
   - 获得奖励：1张宝藏卡"魔法水晶💎"
6. **战斗结束**：返回主流程
7. **补充手牌**：从装备卡堆抽取1张"精灵铠甲🛡️"
8. **回合结束**：将回合交给玩家C

### 示例2：合作战斗回合

**玩家C（海塔）的回合：**

1. **掷骰子**：得到5点，移动5步
2. **停留位置**：战斗点👹
3. **触发战斗**：抽取1张怪物卡"巨魔👹"（生命值8，攻击力3）
4. **战斗回合1**：
   - 玩家C攻击：使用"治愈祈祷💚"（消耗2魔力），恢复3点生命值（从7恢复到10），但没有造成伤害
   - 巨魔攻击：造成3点伤害，玩家C生命值降到7
5. **玩家协作**：玩家A（芙莉莲）使用"火焰魔法🔥"协助攻击，造成3点伤害，巨魔生命值剩余5
6. **战斗回合2**：
   - 玩家C攻击：使用"守护祈祷🛡️"（消耗2魔力），本回合所有玩家受到的伤害-1
   - 巨魔攻击：造成2点伤害（3-1=2），玩家C生命值降到5
   - 玩家A协助攻击：使用"冰结魔法❄️"，造成2点伤害并冻结巨魔，巨魔生命值剩余3
7. **战斗回合3**：
   - 玩家C攻击：普通攻击（伤害1），造成1点伤害，巨魔生命值剩余2
   - 巨魔被冻结，无法攻击
8. **战斗回合4**：
   - 玩家B（修塔尔克）协助攻击：使用"勇者之剑⚔️"，造成3点伤害，巨魔阵亡
   - 获得奖励：2张宝藏卡"魔法水晶💎"和"古代魔法书📕"
9. **战斗结束**：返回主流程
10. **补充手牌**：从祈祷卡堆抽取1张"净化祈祷✨"
11. **回合结束**：将回合交给玩家D

## 💡 新手策略提示

1. **团队合作最重要**：这是合作游戏，不要只关注自己的角色，要互相帮助
2. **合理分配资源**：魔力值有限，不要把所有魔力都用在攻击上，留一些用于治愈和防御
3. **善用角色能力**：
   - 芙莉莲：高级魔法伤害高，但消耗魔力多，合理使用
   - 费伦：可以预知危险，在战斗前查看怪物卡，制定策略
   - 修塔尔克：近战肉盾，吸引怪物攻击，保护队友
   - 海塔：治愈专家，优先保证队友的生命值
4. **优先完成简单任务**：初期完成简单任务，积累宝藏卡和装备
5. **注意生命值管理**：生命值降到低时要及时治愈，避免阵亡
6. **合理使用道具**：宝藏卡和装备卡要在关键时刻使用
7. **观察地图**：提前规划路线，避免连续遭遇强敌

## 🔄 变体规则

### 简单模式（新手推荐）

- 怪物生命值减半
- 初始生命值+3
- 初始魔力值+2
- 可以跳过部分任务

### 困难模式（高手挑战）

- 怪物生命值+50%
- 怪物攻击力+1
- 初始手牌减少1张
- 所有任务必须完成

### 限时模式

- 设置60分钟时间限制
- 在时间结束前完成所有任务并到达终点即获胜
- 时间结束后未完成则失败

### 单人模式

- 1名玩家控制所有4个角色
- 每回合可以操作所有角色
- 其他规则不变

### 剧情模式

- 按照动漫剧情设置章节
- 每个章节有特定任务和目标
- 完成所有章节解锁隐藏结局

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**

A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：卡牌用完了怎么办？**

A：可以将弃牌堆洗混后重新使用。

**Q：可以传递卡牌给其他玩家吗？**

A：可以！这是合作游戏，鼓励玩家之间互相帮助。你可以在你的回合将手牌传递给相邻位置的玩家。

**Q：如果所有玩家都阵亡了怎么办？**

A：游戏失败，重新开始新游戏。

**Q：怪物太强打不过怎么办？**

A：这是合作游戏，多个玩家可以协作攻击同一个怪物。也可以先避开强敌，提升自己的等级和装备后再来挑战。

**Q：任务卡必须按顺序完成吗？**

A：不需要，可以按照任意顺序完成。但某些任务有位置要求，需要到达指定地点才能完成。

**Q：可以放弃任务吗？**

A：可以，但需要消耗1点生命值作为惩罚。
