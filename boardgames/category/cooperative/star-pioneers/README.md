# 星际拓荒者 (Star Pioneers)

## 🎮 游戏概述

《星际拓荒者》是一款合作生存类桌游。玩家扮演星际探险队的成员，共同乘坐"开拓者号"飞船降落在未知的星球上。你们需要：

- 🌍 **探索星球**：揭开未知区域的地图
- ⛏️ **收集资源**：采集外星矿物和能源
- 🏗️ **建造基地**：建立防御工事和研究中心
- 👾 **抵御威胁**：对抗外星生物和宇宙灾害
- 🚀 **完成目标**：在氧气耗尽前完成星球勘测任务

所有玩家**共同对抗游戏系统**，只有所有人都存活并完成目标，才能赢得胜利！

这款游戏适合喜欢合作挑战、科幻题材的玩家，规则简单易学，但策略深度丰富。

## 🗺️ 游戏地图详解

游戏地图是一个 **8x8 的六边形网格星球表面**，包含多种地形和探索点。

### 地图布局概览

```
        ⬢   ⬢   ⬢   ⬢
      ⬢   🚀   ⬢   ⬢   ⬢
    ⬢   ⬢   ⬢   ⚡   ⬢   ⬢
  ⬢   🪨   ⬢   ⬢   🧬   ⬢   ⬢
    ⬢   ⬢   👾   ⬢   ⬢   ⬢
      ⬢   🏗️   ⬢   ⬢   ⬢
        ⬢   ⬢   ⬢   ⬢
```

### 地形类型说明

| 地形 | 图示 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- |
| **降落舱** | 🚀 | 金色 | 起始位置，所有玩家从此出发 |
| **平原** | ⬢ | 浅灰 | 安全区域，无特殊效果 |
| **矿物区** | 🪨 | 蓝色 | 停留可获得1张矿物资源卡 |
| **能源区** | ⚡ | 黄色 | 停留可获得1张能源资源卡 |
| **生物区** | 🧬 | 绿色 | 停留可获得1张生物样本卡 |
| **外星生物** | 👾 | 红色 | 停留需进行战斗 |
| **建造点** | 🏗️ | 紫色 | 可建造基地设施 |
| **目标点** | 🎯 | 橙色 | 完成探索目标的地点 |

### 地图特点

1. **六边形网格**：玩家可以向六个方向移动（上下左右+左上右下/右上左下）
2. **初始探索**：游戏开始时只有降落舱区域可见，其他区域被探索卡覆盖
3. **资源分布**：矿物、能源、生物样本区域均匀分布
4. **威胁位置**：外星生物位于关键路径上，需要谨慎应对

### 移动规则

- 玩家每回合有**3点行动力**，移动消耗1点/格
- 只能移动到相邻六边形格子
- 首次进入某格子时，翻开探索卡揭示该区域内容

### 地图资源统计

| 资源类型 | 数量 | 位置分布 |
| :--- | :--- | :--- |
| 矿物区 | 6个 | 分布在地图边缘和中部的蓝色区域 |
| 能源区 | 6个 | 分布在地图右上和左下区域 |

## 🃏 道具详解

### 资源卡（基础材料）

资源卡是游戏中最重要的道具，所有建造和研究都需要消耗资源。

| 资源 | 图示 | 获得方式 | 用途 |

| **能源** | ⚡ | 停在黄色能源区 | 激活设备、使用武器 |
| **生物样本** | 🧬 | 停在绿色生物区 | 完成研究目标、解锁技能 |

**资源卡使用规则**：
- 每位玩家手牌上限为**5张**，超过需丢弃
- 玩家之间可以**自由交换**资源卡
- 资源卡可以**储存**在基地（如有建造仓库）

### 装备卡（探险工具）

| **探测器** | 🔍 | 能源×1 + 矿物×1 | 揭开相邻区域的探索卡 |
| **通讯器** | 📻 | 矿物×1 | 所有玩家额外获得1点行动力 |

### 武器卡（战斗装备）

武器卡用于对抗外星生物。

| 武器 | 图示 | 所需资源 | 攻击力 | 特殊效果 |

| **激光枪** | 🔫 | 能源×2 | 2点 | 无 |

**战斗规则**：
- 外星生物有**生命值**（2-5点不等）
- 玩家使用武器卡造成攻击
- 战斗结束后武器卡**丢弃**
- 如果玩家无法击败生物，需**撤退**并受到伤害（失去1点氧气）

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央，所有区域用空白卡覆盖（除降落舱）
2. **选择角色**：每位玩家随机选择一张角色卡，获得角色技能
3. **初始资源**：每位玩家获得**2张矿物卡、1张能源卡**
4. **设置氧气**：将氧气标记设为**20点**（1-2人）或**30点**（3-5人）
5. **洗混卡牌**：将探索卡、事件卡、目标卡分别洗混，背面朝上放置
6. **抽取目标**：翻开3张目标卡作为本局胜利条件

### 角色介绍

| 角色 | 技能 | 起始装备 |
| :--- | :--- | :--- |
| **队长** | 所有玩家+1行动力 | 通讯器 |
| **工程师** | 建造费用-1资源 | 扫描仪 |
| **生物学家** | 生物样本额外+1 | 采集器 |
| **战士** | 武器攻击力+1 | 激光枪 |
| **医生** | 氧气消耗-1 | 氧气罐 |

### 游戏流程（每个回合）

所有玩家**轮流行动**，每个回合包含以下阶段：

#### 阶段1：事件阶段
- 抽取1张**事件卡**，执行其效果
- 如果事件卡为"即时生效"，立即执行
- 如果事件卡为"持续效果"，放置在地图旁，下回合再消失

#### 阶段2：玩家行动阶段
每位玩家依次进行行动，每回合有**3点行动力**：

| 行动 | 消耗 | 说明 |
| :--- | :--- | :--- |
| **移动** | 1点/格 | 移动到相邻格子 |
| **探索** | 1点 | 揭开当前格子的探索卡 |
| **收集资源** | 1点 | 在当前格子收集资源（如果是资源区） |
| **建造** | 2点 | 在建造点建造设施 |
| **战斗** | 2点 | 攻击当前格子的外星生物 |
| **使用装备** | 1点 | 激活装备卡效果 |
| **交换资源** | 0点 | 与同格玩家交换资源（免费） |
| **休息** | 0点 | 不行动，跳过回合 |

#### 阶段3：检查阶段
- 检查是否完成任意**目标卡**条件
- 检查**氧气剩余量**，如果≤0，所有玩家失败
- 检查所有玩家是否都完成了行动

#### 阶段4：回合结束
- 将回合标记前进1
- 如果玩家手牌超过5张，需丢弃至5张
- 第一位玩家将行动权交给下一位玩家

### 胜利条件

**玩家获胜**：完成**3张目标卡**上的所有条件！

**游戏失败**：
- 团队氧气降至**0或以下**
- 任意玩家**阵亡**（生命值≤0）
- 达到**15回合**仍未完成目标（可选限时模式）

## 📖 完整游戏流程示例

### 示例1：探索与收集（玩家A的回合）

**回合3，玩家A（生物学家）行动：**

1. **移动**（消耗1点）：从降落舱移动到相邻的矿物区
2. **收集资源**（消耗1点）：获得1张矿物卡🪨
3. **使用装备**（消耗1点）：激活"采集器"，额外获得1张矿物卡
4. **回合结束**：玩家A现在有3张矿物卡，手牌未超限

**结果**：玩家A成功收集资源，为后续建造做准备。

---

### 示例2：战斗与撤退（玩家B的回合）

**回合4，玩家B（战士）行动：**

1. **移动**（消耗1点）：移动到外星生物区域👾
2. **战斗**（消耗2点）：使用激光枪（攻击力2）攻击外星生物（生命值3）
3. **战斗结果**：外星生物剩余1点生命值，玩家B无法继续攻击（武器已使用）
4. **撤退**：玩家B选择撤退到相邻安全区域
5. **回合结束**

**下回合**：玩家C（工程师）前来支援，使用等离子刀（攻击力3）击败外星生物！

---

### 示例3：合作建造（玩家C和D的回合）

**回合5，玩家C（工程师）行动：**

1. **移动**（消耗1点）：到达建造点🏗️
2. **建造**（消耗2点）：使用矿物×2建造"研究站"
3. **研究站效果**：所有玩家在研究站相邻格子收集资源时+1

**回合6，玩家D（队长）行动：**

1. **移动**（消耗1点）：移动到玩家C所在的研究站旁边
2. **收集资源**（消耗1点）：获得1张能源卡⚡（因研究站效果，实际获得2张）
3. **使用通讯器**（消耗1点）：所有玩家额外获得1点行动力
4. **回合结束**

**结果**：团队合作建造设施，提高效率！

## 💡 新手策略提示

1. **角色搭配很重要**：
   - 队长+工程师组合：高行动力+低建造成本
   - 生物学家+医生组合：快速完成研究目标
   - 战士前排抗伤害，其他角色后排输出

2. **资源管理**：
   - 优先收集**能源卡**，很多装备和武器都需要
   - 不要囤积资源，及时使用或建造
   - 玩家之间多**交换资源**，互补不足

3. **探索策略**：
   - 优先探索**资源区**附近的区域
   - 使用扫描仪或探测器提前查看危险
   - 不要单独深入未知区域，组队行动更安全

4. **氧气管理**：
   - 氧气是团队共享资源，省着用！
   - 医生角色的"氧气消耗-1"技能非常关键
   - 及时使用氧气罐补充

5. **目标优先**：
   - 开局先查看3个目标，制定策略
   - "收集样本"和"勘测星球"通常比较容易完成
   - "击败威胁"需要战士角色配合

## 🔄 变体规则

### 简单模式（新手推荐）
- 氧气初始值+10
- 外星生物生命值-1
- 事件卡数量减半
- 只需完成**2个目标**即可获胜

### 困难模式（老手挑战）
- 氧气初始值-5
- 外星生物数量+2，生命值+1
- 每回合抽取**2张事件卡**
- 需完成**4个目标**才能获胜
- 增加"黑夜阶段"：每3回合后，所有已探索区域变为未知，需重新探索

### 竞速模式
- 设置**60分钟**时间限制
- 在时间结束前完成目标的队伍获胜
- 如果时间到未完成，统计完成的目标数，多的队伍获胜

### 角色技能扩展
- 每位角色增加**第二个技能**，需要在研究站研究后才能解锁
- 解锁条件：生物学家在 research station 花费3个生物样本

### 多人团队模式（4-5人）
- 分成2个小组，每组2-3人
- 小组之间可以交易资源，但目标是各自完成3个目标
- 先完成目标的团队获胜（增加竞争性）

## ❓ 常见问题

**Q1：如果没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。行动点数也可以直接用卡牌决定（抽1张探索卡，数字为行动点数）。

**Q2：资源卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用。或者，玩家可以选择"采集"行动，从资源堆抽取1张。

**Q3：可以和同格的玩家交换资源吗？**
A：可以！交换资源不消耗行动力，鼓励玩家之间合作。

**Q4：如果所有玩家都阵亡了怎么办？**
A：游戏立即失败。阵亡条件通常是生命值≤0或氧气≤0。

**Q5：外星生物会移动吗？**
A：基础规则中外星生物**不会移动**，但困难模式可以增加"外星生物每回合移动1格"规则。

**Q6：可以同时完成多个目标吗？**
A：可以！例如，探索地图时同时收集生物样本，可以同时满足"勘测星球"和"收集样本"两个目标。

**Q7：建造设施有什么用？**
A：不同设施有不同的效果：
- 研究站：收集资源+1
- 防御塔：相邻格子外星生物生命值-1
- 仓库：增加手牌上限至7张
- 医疗站：回合结束时所有相邻玩家恢复1点生命值

**Q8：打印时卡牌颜色不明显怎么办？**
A：打印时务必**勾选"背景图形"**选项。建议使用**卡纸**打印，颜色更鲜艳。也可以在卡牌背面做标记区分类型。
