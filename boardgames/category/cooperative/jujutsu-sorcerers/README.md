# 咒术师团 (Jujutsu Sorcerers)

## 🎮 游戏概述

《咒术师团》是一款基于人气动漫《咒术回战》主题的合作类桌游。玩家们将扮演咒术师，组成一支精英小队，共同对抗突然暴走的诅咒，保护无辜平民。

在游戏中，诅咒会不断从裂缝中涌出，向平民区逼近。各位咒术师必须合理分配任务、善用自身术式、互相配合支援，在平民受害之前将所有诅咒祓除！

这是一款强调团队配合与策略规划的桌游，适合喜欢合作游戏和《咒术回战》的玩家。游戏规则清晰易懂，但想要精通需要反复磨练战术。

### 地图特点

1. **平民区分布**：4个平民区（学校、医院、公园、商店、神社中的4个）随机分布，诅咒会优先向最近的平民区移动
2. **裂缝位置**：地图上有3个诅咒裂缝，游戏开始后每回合从裂缝处生成新诅咒
3. **咒力资源**：5个咒力点分布在地图各处，咒术师需要占领这些点位来获取咒力
4. **最短路径**：任意两个区域之间的最短路径约3-5步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 咒术师移动步数由骰子决定（1颗骰子）
- 诅咒每回合自动移动1-2格，向最近的平民区逼近
- 当诅咒进入平民区时，该区域平民受害计数+1

## 🎯 游戏目标

**所有玩家共同合作**，达成以下胜利条件之一：

1. ✅ **祓除所有诅咒**（包括裂缝中涌出的所有诅咒）
2. ✅ **保护平民**，使受害平民数不超过3人

**失败条件**（满足任一即全体失败）：

1. ❌ 受害平民数达到 **5人**
2. ❌ 所有咒术师均 **失去全部生命值**
3. ❌ 达到 **第15回合** 仍未祓除所有诅咒（困难模式）

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择角色**：每位玩家选择1名咒术师角色（虎杖悠仁/伏黑惠/钉崎野蔷薇/五条悟），将角色卡放在面前
3. **放置棋子**：将所有玩家的棋子放在地图上的「学校」位置（起始点）
4. **放置诅咒**：将3个三级诅咒棋子放在地图上的诅咒裂缝位置
5. **发放初始卡牌**：每位玩家获得 **3张咒力卡** 和 **2张随机术式卡**
6. **设置平民**：在5个平民区各放置1个平民棋子
7. **洗混卡牌**：将诅咒卡堆、事件卡堆、支援卡堆分别洗混，背面朝上放置

### 游戏流程（每回合）

游戏按回合进行，每回合分为以下 **5个阶段**：

#### 阶段1：诅咒行动阶段

- 所有诅咒按照以下顺序行动：
  1. **移动**：每个诅咒向最近的平民区移动1-2格（掷骰子决定）
  2. **攻击**：如果诅咒与咒术师相邻，对咒术师造成对应伤害
  3. **侵入判定**：如果诅咒进入平民区，该区域平民受害计数+1

#### 阶段2：事件阶段

- 翻开1张事件卡，执行其效果
- 如果事件卡效果为"抽取事件卡"，则额外再翻开1张

#### 阶段3：玩家行动阶段

玩家按顺时针方向轮流行动，每位玩家的回合可以进行 **2个行动**：

| 行动 | 说明 |
| :--- | :--- |
| **移动** | 掷1颗骰子，移动对应步数（上下左右方向） |
| **攻击** | 消耗术式卡，对相邻诅咒造成伤害 |
| **收集** | 停留在咒力点，抽取1张咒力卡 |
| **休息** | 不移动，恢复1张咒力卡到手牌 |
| **支援** | 使用支援卡，帮助队友 |
| **开启领域** | 消耗5张咒力卡，使用1张领域卡 |

**重要**：玩家之间可以随时交流策略，但行动时必须按照回合顺序执行。

#### 阶段4：诅咒生成阶段

- 在每个诅咒裂缝位置生成1个新的三级诅咒
- 如果裂缝位置已有诅咒，则在相邻空格生成
- 如果周围无空格，诅咒能量累积，下回合生成二级诅咒

#### 阶段5：回合结束阶段

- 检查游戏是否结束（胜利/失败条件）
- 如果未结束，将回合标记移动到下一回合
- 清理本回合的临时效果

### 战斗规则

#### 攻击判定

1. 玩家声明使用哪张术式卡进行攻击
2. 消耗对应数量的咒力卡
3. 对目标诅咒造成术式卡上标注的伤害值
4. 如果诅咒生命值降为0，将其移出游戏，玩家抽取对应奖励卡

#### 受到伤害

1. 当咒术师受到伤害时，将其生命值标记减少对应数值
2. 如果生命值降为0，该玩家**暂时退场**
3. 退场的玩家在3回合后可用1张支援卡复活（生命值恢复为1）

## 📖 完整游戏流程示例

### 示例1：第一回合（游戏初期）

**阶段1 - 诅咒行动**：
- 裂缝A处的三级诅咒（2 HP）向最近的学校移动1格
- 裂缝B处的三级诅咒向医院移动2格
- 裂缝C处的三级诅咒向公园移动1格
- 无诅咒与咒术师相邻，无攻击发生

**阶段2 - 事件阶段**：
- 翻开事件卡「咒力涌现」：所有玩家各抽1张咒力卡

**阶段3 - 玩家行动**（玩家A先行动）：
- **玩家A（虎杖）**：掷骰子得4，移动到学校旁的咒力点，收集1张咒力卡
- **玩家B（伏黑）**：掷骰子得3，移动到与裂缝A诅咒相邻的位置，使用「强化拳」术式攻击该诅咒，造成1点伤害（诅咒剩余1 HP）
- **玩家C（钉崎）**：掷骰子得5，移动到事件点，抽取事件卡「平民疏散」
- **玩家D（五条）**：休息，恢复1张咒力卡

**阶段4 - 诅咒生成**：
- 3个裂缝各生成1个新的三级诅咒

**阶段5 - 回合结束**：
- 检查：平民受害数=0，无玩家退场，继续游戏
- 当前回合：第1回合结束，进入第2回合

---

### 示例2：第五回合（游戏中段）

**当前状态**：
- 已祓除6个诅咒，场上还有4个诅咒
- 平民受害数=2（医院1人、公园1人）
- 玩家B（伏黑）生命值=1（受伤状态）

**阶段1 - 诅咒行动**：
- 1个二级诅咒（4 HP）进入医院，平民受害数变为3
- 1个三级诅咒攻击玩家B，玩家B生命值降为0，**暂时退场**

**阶段2 - 事件阶段**：
- 翻开事件卡「支援到来」：所有在场玩家恢复1点生命值

**阶段3 - 玩家行动**：
- **玩家A（虎杖）**：使用「共同攻击」支援卡，与玩家D一起攻击同一个二级诅咒
  - 玩家A使用「强化拳」造成1点伤害
  - 玩家D使用「苍/赫」造成3点伤害
  - 二级诅咒（4 HP）被祓除！玩家A抽取2张咒力卡
- **玩家C（钉崎）**：移动到玩家B的位置，使用「紧急治疗」支援卡，但玩家B已退场，改为恢复自身生命值
- **玩家D（五条）**：开启领域「无量空处」，所有诅咒本回合无法行动

**阶段4 - 诅咒生成**：
- 本回合因「无量空处」效果，裂缝不生成新诅咒

**阶段5 - 回合结束**：
- 玩家B可使用1张支援卡复活，生命值恢复为1
- 平民受害数=3（危险！再失去2名平民则游戏失败）

## 💡 新手策略提示

1. **优先保护平民区**：诅咒会优先攻击平民区，尽量在平民区周围部署咒术师
2. **合理分配角色**：五条悟适合进攻，伏黑惠适合控制，钉崎适合远程支援，虎杖适合近战坦克
3. **节约咒力**：咒力卡是有限资源，不要随意使用高消耗术式
4. **善用支援卡**：合作游戏的核心是互相帮助，及时使用支援卡可以挽救战局
5. **优先祓除特级诅咒**：特级诅咒会不断召唤小诅咒，必须优先处理
6. **占领咒力点**：控制地图上的咒力点，确保稳定的咒力供应
7. **开启领域需谨慎**：领域卡消耗5张咒力，确保在关键时刻使用
8. **注意回合限制**：困难模式下只有15回合，不要拖延时间过长

## 🔄 变体规则

### 简单模式（适合新手）

- 平民受害上限提高到 **7人**
- 所有诅咒生命值 **-1**
- 玩家初始手牌增加 **1张咒力卡** 和 **1张支援卡**
- 取消回合限制

### 困难模式（适合老手）

- 平民受害上限降低到 **3人**
- 所有诅咒攻击力 **+1**
- 每回合生成诅咒数量 **+1**（即每回合生成4个诅咒）
- 第 **12回合** 后出现「特级诅咒·漏瑚」

### 单人模式

- 1名玩家控制所有4名咒术师角色
- 每回合每个角色可以行动2次
- 初始手牌翻倍

### 剧情模式

按照《咒术回战》动漫剧情设置特殊任务：

| 剧情 | 特殊规则 |
| :--- | :--- |
| **咒胎戴天** | 玩家从1级开始，逐步升级术式 |
| **幼鱼逆罚** | 场地内有水域地形，某些诅咒可以潜水 |
| **起了执念** | 钉崎专属关卡，远程攻击伤害加倍 |
| **涩谷事变** | 大量平民区，时间限制变为10回合 |
| **死灭回游** | PvP模式，玩家之间也有竞争关系 |

## ❓ 常见问题

**Q：如果卡牌用完了怎么办？**
A：将弃牌堆洗混后重新使用。

**Q：可以查看诅咒卡堆的内容吗？**
A：可以，但需要使用「战术指导」支援卡，或者掷骰子得到6点时允许查看1张。

**Q：玩家退场后怎么办？**
A：退场玩家在3回合后可用1张支援卡复活（生命值恢复为1）。如果无人使用支援卡，也可在下一轮开始时自动复活（生命值=1）。

**Q：多个诅咒同时攻击同一名玩家怎么办？**
A：伤害叠加计算。例如2个三级诅咒同时攻击，共造成2点伤害。

**Q：领域卡可以叠加使用吗？**
A：不可以。同一时刻只能有1个领域生效，新的领域会覆盖旧的领域。

**Q：如果所有咒力点都被占据怎么办？**
A：其他玩家可以继续移动，但无法收集咒力。也可以击败诅咒后获得咒力卡作为替代。

**Q：诅咒可以攻击其他诅咒吗？**
A：不可以。诅咒只攻击咒术师和侵入平民区。

**Q：游戏最多可以进行多少回合？**
A：简单模式无限制，困难模式限制为15回合。
