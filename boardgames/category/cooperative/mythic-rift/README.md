# 神话裂隙 (Mythic Rift)

## 1. 游戏概述

**神话裂隙**是一款合作类桌游，灵感来源于《黑神话：悟空》等热门神话题材作品。玩家将扮演来自不同神话世界的"神话猎人"，探索连接各神话世界的神秘裂隙，收集远古神器，合作封印企图入侵的远古邪神"混沌之主"。

游戏结合了探险、资源管理、战术战斗和团队合作元素。每位玩家选择独特的神话角色（中国、希腊、北欧、埃及等），利用角色技能和神器组合，共同面对越来越强大的挑战。

**适合人群**：
- 喜欢神话题材和团队合作玩家
- 2-5人聚会或家庭游戏夜
- 喜欢有一定策略深度的合作游戏
- 《黑神话：悟空》、《咒术回战》等作品的粉丝

## 4. 游戏地图详解

### 4.1 地图布局概览

神话裂隙的地图由24块六边形拼板随机组成，每次游戏都会生成不同的裂隙空间。

```
        ⬡ ⬡ ⬡
      ⬡ ⬡ ⬡ ⬡
    ⬡ ⬡ ⬡ ⬡ ⬡
      ⬡ ⬡ ⬡ ⬡
        ⬡ ⬡ ⬡
```

### 4.2 地形类型说明

| 地形 | 图示 | 效果 | 特殊规则 |
| :--- | :--- | :--- | :--- |
| 普通空间 | ⬜ | 无特殊效果 | 可自由移动 |
| 能量节点 | 💎 | 获得1个裂隙能量 | 每位玩家只能采集一次 |
| 神器遗迹 | ⭐ | 抽取1张神器卡 | 需要消耗2个行动点 |
| 混沌裂隙 | 🌀 | 触发事件卡 | 可能有利也可能有害 |
| Boss房间 | 👑 | Boss战场所 | 需要集齐3个神器才能进入 |
| 安全营地 | 🏕️ | 恢复2点理智值 | 可休息跳过回合 |

### 4.3 地图特点

- **随机生成**：每局游戏的地图布局都不同
- **路径连接**：所有拼板都有路径连接
- **资源分布**：能量节点和神器遗迹均匀分布
- **策略性**：玩家需要规划最优路径

### 4.4 资源统计（标准配置）

| 资源类型 | 数量 | 分布 |
| :--- | :--- | :--- |
| 能量节点 | 6个 | 随机分布 |
| 神器遗迹 | 4个 | 远离起始点 |
| 混沌裂隙 | 6个 | 分散在地图中 |
| Boss房间 | 1个 | 地图中心或深处 |
| 安全营地 | 3个 | 路径关键位置 |

## 6. 游戏规则

### 6.1 准备阶段

1. **选择角色**：每位玩家选择一张角色卡，获取对应的棋子和中立标记。
2. **布置地图**：随机放置24块地图拼板，确保路径连接。
3. **放置标记**：在对应位置放置能量节点、神器遗迹、混沌裂隙等标记。
4. **抽取起始手牌**：每位玩家抽取3张技能卡。
5. **放置Boss**：将三张Boss卡放在Boss房间，根据章节翻面。
6. **决定先手**：掷骰子，点数最大者先手。

### 6.2 游戏流程

游戏分为三个阶段（章节），每个章节有独立的目标和挑战。

#### 回合结构

每位玩家的回合分为以下几个阶段：

1. **恢复阶段**：
   - 恢复所有行动点
   - 抽取1张技能卡

2. **行动阶段**（按行动点顺序）：
   - 移动（消耗1行动点可移动1格）
   - 探索（停留在某地可触发效果）
   - 战斗（遇到敌人或Boss）
   - 使用技能卡
   - 装备神器

3. **事件阶段**：
   - 如果停留在混沌裂隙，抽取1张事件卡
   - 执行事件效果

4. **结束阶段**：
   - 检查理智值（降为0则阵亡）
   - 检查胜利条件

#### 战斗规则

**攻击**：
- 玩家 declaration 攻击
- 掷骰子（1D6）：
  - 1-2：未命中
  - 3-5：命中，造成（攻击力）点伤害
  - 6：会心一击，造成（攻击力×2）点伤害
- 使用攻击技能卡可改变骰子结果或增加伤害

**防御**：
- 受到攻击时可以选择防御
- 掷骰子（1D6）：
  - 1-2：防御失败，受到全额伤害
  - 3-5：防御成功，减少（防御力）点伤害
  - 6：完美防御，不受伤害
- 使用防御技能卡可增加防御成功率

**Boss战**：
- Boss每回合有2个行动点
- Boss的行动在玩家回合结束后执行
- Boss卡描述了Boss的攻击模式和特殊能力
- 所有玩家可以集火攻击Boss

### 6.3 胜利条件

**章节目标**：

| 章节 | 目标 | 失败条件 |
| :--- | :--- | :--- |
| 第一章 | 击败混沌守护者 | 任意2名玩家阵亡 |
| 第二章 | 击败虚空领主 | 累计3名玩家阵亡 |
| 第三章 | 击败混沌之主 | 所有玩家阵亡 |

**最终胜利**：
- 所有玩家合作完成三个章节
- 击败混沌之主
- 至少1名玩家存活

**失败条件**：
- 所有玩家阵亡
- 理智值全为0
- 超过时间限制（可选）

## 7. 完整游戏流程示例

### 示例1：第一章节开局（3人游戏）

**玩家角色**：
- 玩家A：孙悟空（攻击型）
- 玩家B：拉（辅助型）
- 玩家C：赫拉克勒斯（防御型）

**第1回合**：

**玩家A回合**：
1. 恢复阶段：恢复3行动点，抽取1张技能卡（猛击）
2. 行动阶段：
   - 消耗1行动点，移动1格，到达能量节点
   - 消耗1行动点，采集能量（获得1裂隙能量）
   - 保留1行动点
3. 事件阶段：无事件
4. 结束阶段：理智值5，状态正常

**玩家B回合**：
1. 恢复阶段：恢复4行动点，抽取1张技能卡（治愈之光）
2. 行动阶段：
   - 消耗1行动点，移动1格，跟随玩家A
   - 消耗2行动点，使用"太阳之泪"技能，为所有队友恢复2理智（但大家都满理智，所以保留）
3. 事件阶段：无事件
4. 结束阶段：理智值5，状态正常

**玩家C回合**：
1. 恢复阶段：恢复3行动点，抽取1张技能卡（格挡）
2. 行动阶段：
   - 消耗1行动点，移动1格，到达安全营地
   - 消耗2行动点，休息，恢复2理智（但满理智，所以保留）
3. 事件阶段：无事件
4. 结束阶段：理智值6，状态正常

**Boss阶段**：混沌守护者召唤1个小怪

---

**第2回合**：

**玩家A回合**：
1. 恢复阶段：恢复3行动点，抽取1张技能卡（连击）
2. 行动阶段：
   - 消耗1行动点，移动1格，到达神器遗迹
   - 消耗2行动点，探索神器遗迹，抽取1张神器卡（破魔之箭，攻击+2）
   - 装备破魔之箭
3. 事件阶段：无事件
4. 结束阶段：攻击力变为6，理智值5

**玩家B回合**：
1. 恢复阶段：恢复4行动点，抽取1张技能卡（疾风步）
2. 行动阶段：
   - 消耗1行动点，移动1格
   - 消耗2行动点，使用"光明审判"技能（需要3行动点，但只有4，所以保留）
   - 改为消耗1行动点，使用"治愈之光"技能卡，恢复2理智（但满理智）
3. 事件阶段：无事件
4. 结束阶段：意识到行动点管理很重要

**玩家C回合**：
1. 恢复阶段：恢复3行动点，抽取1张技能卡（铁壁）
2. 行动阶段：
   - 消耗1行动点，移动1格，与玩家B会合
   - 保留2行动点以应对突发情况
3. 事件阶段：无事件
4. 结束阶段：防御力5，理智值6

**Boss阶段**：混沌守护者攻击小怪附近的玩家C，造成1点伤害（玩家C防御力5，格挡成功，受到0伤害）

---

**第5回合**（遇到Boss）：

**玩家A回合**：
1. 恢复阶段：恢复3行动点，抽取1张技能卡（重击）
2. 行动阶段：
   - 消耗1行动点，进入Boss房间
   - 消耗2行动点，使用"猛击"技能卡，攻击Boss（掷骰子4，命中，造成6点伤害）
3. 事件阶段：无事件
4. 结束阶段：Boss生命值剩余9

**玩家B回合**：
1. 恢复阶段：恢复4行动点，抽取1张技能卡（预知）
2. 行动阶段：
   - 已经在Boss房间
   - 消耗3行动点，使用"光明审判"技能，揭露Boss弱点（下回合攻击+2）
   - 保留1行动点
3. 事件阶段：无事件
4. 结束阶段：玩家A下回合攻击变为8

**玩家C回合**：
1. 恢复阶段：恢复3行动点，抽取1张技能卡（守护）
2. 行动阶段：
   - 已经在Boss房间
   - 消耗2行动点，使用"神力护盾"技能，为所有队友抵挡3点伤害
   - 保留1行动点
3. 事件阶段：无事件
4. 结束阶段：所有队友有护盾

**Boss阶段**：混沌守护者攻击玩家A，但被护盾完全抵挡

### 示例2：理智值管理（关键时刻）

**情况**：玩家B（拉）理智值只剩2点，即将面对混沌裂隙事件。

**玩家B回合**：
1. 恢复阶段：恢复4行动点，抽取1张技能卡（太阳之泪"）
2. 行动阶段：
   - 意识到理智值危险，优先恢复
   - 消耗2行动点，使用"太阳之泪"技能，为所有队友恢复2理智
   - 自身理智值恢复至4
   - 保留2行动点
3. 事件阶段：无事件（未停留在裂隙）
4. 结束阶段：理智值4，安全

**教训**：理智值管理非常重要，辅助角色要优先保证自身和队友的理智值。

## 8. 新手策略提示

### 8.1 角色搭配建议

| 玩家人数 | 推荐角色组合 | 说明 |
| :--- | :--- | :--- |
| 2人 | 孙悟空+拉 | 高攻击+高治疗，适合新手 |
| 3人 | 孙悟空+赫拉克勒斯+拉 | 攻防辅平衡，推荐组合 |
| 4人 | 孙悟空+托尔+赫拉克勒斯+拉 | 多角色配合，战术丰富 |
| 5人 | 全角色 | 完整阵容，需要良好配合 |

### 8.2 资源管理

1. **行动点管理**：
   - 优先完成目标（探索、战斗）
   - 不要浪费行动点在低价值行动上
   - 辅助角色保留行动点用于关键时刻

2. **理智值管理**：
   - 保持理智值在3点以上
   - 优先使用安全营地恢复
   - 辅助角色优先保证治疗能力

3. **裂隙能量管理**：
   - 优先用于购买关键神器
   - 不要浪费在低级神器上
   - 保留能量用于紧急情况

### 8.3 战斗策略

1. **优先击杀顺序**：
   - 小怪（召唤型）> 小怪（高攻击）> Boss
   - 避免被小怪淹没

2. **集火策略**：
   - 所有玩家集中攻击同一个目标
   - 快速减少敌人数量

3. **防御时机**：
   - Boss攻击前使用防御技能
   - 理智值低时优先防御

### 8.4 神器选择

| 角色类型 | 优先神器 | 说明 |
| :--- | :--- | :--- |
| 攻击型 | 攻击神器、特殊神器 | 最大化输出 |
| 防御型 | 防御神器、辅助神器 | 保护队友 |
| 辅助型 | 辅助神器、特殊神器 | 提供增益和治疗 |

### 8.5 常见错误

❌ **错误1：浪费行动点**
- 正确做法：每回合规划好行动，优先完成目标

❌ **错误2：忽视理智值**
- 正确做法：理智值低于3时优先恢复

❌ **错误3：单独行动**
- 正确做法：团队协作，集火攻击

❌ **错误4：过早使用大招**
- 正确做法：保留大招用于Boss战

❌ **错误5：不考虑角色搭配**
- 正确做法：根据角色特点分配任务

## 9. 变体规则

### 9.1 简单模式（适合新手）

**调整**：
- 玩家起始理智值+2
- Boss生命值-20%
- 增加1个安全营地
- 事件卡负面效果减半

### 9.2 困难模式（适合老手）

**调整**：
- 玩家起始理智值-1
- Boss生命值+30%
- Boss特殊能力增强
- 事件卡负面效果翻倍
- 减少1个安全营地

### 9.3 极限模式（硬核挑战）

**调整**：
- 只有1条命（理智值降为0立即阵亡）
- Boss生命值+50%
- 所有小怪强化
- 禁止重复使用同一技能卡
- 时间限制（60分钟）

### 9.4 单人模式

**规则调整**：
- 控制2个角色
- 起始手牌6张
- 每回合可切换激活角色
- 胜利条件不变

### 9.5 团队对抗模式（非合作）

**规则调整**：
- 2-3队，每队2人合作
- 各队竞争击败Boss
- 可以先攻击其他队伍的玩家
- 先击败Boss的队伍获胜

## 10. 常见问题

### Q1：地图拼板必须随机放置吗？

**A**：推荐随机放置以增加重玩价值。但新手可以要求预先设计好地图布局，熟悉游戏后再随机生成。

### Q2：可以同时装备多个神器吗？

**A**：可以！神器没有装备数量限制，但手牌上限为5张神器卡。建议根据战术选择合适的神器组合。

### Q3：理智值为0怎么办？

**A**：理智值降为0时，该玩家阵亡。其他玩家可以使用"复活"技能卡或赫拉克勒斯的"十二试炼"技能复活。如果所有玩家都阵亡，游戏失败。

### Q4：Boss战可以同时多人攻击吗？

**A**：可以！所有玩家可以在自己的回合攻击Boss。这是合作游戏的核心策略——集火攻击。

### Q5：事件卡的效果可以拒绝吗？

**A**：大部分事件卡效果必须执行。但某些神器（如"灵慧披风"）可以减少或免疫负面效果。某些技能卡（如"净化"）可以移除负面状态。

### Q6：技能卡用完了怎么办？

**A**：每回合恢复阶段会抽取1张技能卡。如果手牌已满（5张），必须先使用或丢弃一些技能卡才能抽取新卡。

### Q7：可以查看Boss的技能和属性吗？

**A**：可以！Boss卡是公开信息，所有玩家都可以查看Boss的当前状态、特殊能力和攻击模式。

### Q8：游戏时间太长怎么办？

**A**：可以设置时间限制（建议90分钟），超时则游戏失败。或者可以简化某些规则，如减少Boss生命值、增加起始资源等。

### Q9：打印材料用什么纸最好？

**A**：推荐使用200g以上的卡纸打印卡牌和地图，增加耐用性。普通A4纸也可以，但容易损坏。打印后建议塑封或贴上透明胶带。

### Q10：可以自定义角色和神器吗？

**A**：可以！鼓励玩家发挥创意，设计自己的角色和神器。但需要注意平衡性，避免过于强大或弱小。
