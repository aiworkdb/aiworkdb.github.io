# 🌊 深海探险 Abyss Dive

---

## 1. 游戏概述

在未来的2247年，人类发现了太平洋深处一座远古海底文明的遗迹。卫星扫描显示，在马里亚纳海沟下方数千米处，有一座散发着神秘蓝光的巨型建筑群——**深海之心**。你将率领一支精英深海探险队，驾驶「蛟龙号」深潜器，穿越层层深海区域，寻找这传说中的失落文明核心。

然而，深海远比想象中危险。随着深度增加，水压急剧上升，氧气供应不断消耗，未知的深海生物潜伏在黑暗中等待猎物，而海洋洋流随时可能将你卷入深渊。探险队必须紧密合作——分配氧气、修理设备、对抗深海巨兽、破解古代谜题，才能抵达深海之心并安全返回水面。

**适合人群**：喜欢合作策略、紧张感十足的玩家群体。适合家庭聚会和桌游俱乐部活动，尤其推荐给喜欢《瘟疫危机》《禁闭岛》等合作桌游的玩家。

| 层级 | 名称 | 深度范围 | 氧气消耗倍率 | 生物威胁等级 | 遗物数量 | 特殊规则 |

### 4.3 地图区域说明

每层包含4个**探索区域**（第5层除外，仅有1个核心区域）。玩家在同一回合中可选择探索同一层的不同区域，每个区域有以下属性：

- **危险类型**：生物威胁、环境陷阱、遗物线索、补给点
- **事件池**：每个区域有专属的事件卡子池
- **连接关系**：部分相邻区域可互通，玩家可在回合内移动到相连区域

| 角色名称 | 图示 | 生命值 | 被动技能 | 主动技能（每局限N次） |
| :--- | :--- | :--- | :--- | :--- |
| 潜航员 | 🧑‍🚀 | 3 | 每回合移动时可额外前进1个区域 | 「紧急下潜」：无视氧气消耗移动2层（3次） |
| 工程师 | 🔧 | 3 | 修理装备时额外恢复1点效果 | 「紧急修复」：不消耗行动修复所有损坏装备（2次） |
| 生物学家 | 🔬 | 4 | 战斗检定+1；生物遭遇时可选择「观察」代替战斗 | 「生物安抚」：使当前生物卡退回牌堆顶部（3次） |
| 导航员 | 🧭 | 3 | 暗流事件中可选择方向而非随机 | 「精确导航」：指定全队移动到任意已知区域（2次） |
| 医官 | 💊 | 3 | 每回合开始时可恢复1名队友1点生命 |「紧急抢救」：恢复全队各1点生命，消耗2氧气（2次） |

> 💡 **生命值说明**：每位玩家初始3-4点生命（根据角色），生命归零后进入「濒死」状态，需要队友使用医疗急救包或医官技能施救。若无人施救，该玩家「阵亡」，全队失败条件之一。

## 6. 游戏规则

### 6.1 准备阶段

**步骤1：布置地图**
将深海地图展开，确保所有玩家可以看到5个深度层级。将深度指示器放在第1层（浅海区），将威胁指示器设为1。

**步骤2：洗牌与发牌**
- 将所有装备卡洗混，形成**装备牌堆**，放在地图旁
- 将所有生物卡洗混，形成**生物牌堆**，放在装备牌堆旁
- 将所有遗物卡洗混，形成**遗物牌堆**
- 将所有环境卡洗混，形成**环境牌堆**

**步骤3：分配角色**
每位玩家从5张角色卡中选择1张（2-3人游戏时推荐选择互补角色组合）。将角色卡正面朝上放在自己面前。

**步骤4：初始装备**
每位玩家获得以下初始物品：
- 🤿 潜水头盔（装备卡）x1
- 🔧 修理工具箱（装备卡）x1
- 👤 角色卡 x1
- 🗺️ 快速参考卡 x1

**步骤5：设置团队资源**
- 将氧气标记设为**30枚**（团队共享氧气池）
- 将深海之心标记放置在第5层核心区域
- 团队面板初始状态：氧气30、修理点数0、遗物进度0/3

**步骤6：确定起始玩家**
掷骰，点数最高的玩家为起始玩家（潜航队长）。从潜航队长开始，顺时针轮流进行回合。

### 6.2 游戏流程

每个回合按以下顺序执行：

#### 阶段A：回合开始（回合起始）
1. 检查氧气：若团队氧气为0，立即进入**紧急上浮**流程
2. 恢复阶段：所有装备和角色的「每回合」被动效果生效
3. 检查水压：根据当前深度层级的水压倍率，标记水压伤害（不影响氧气，但影响角色生命）

#### 阶段B：玩家行动（核心阶段）
当前回合玩家可以执行以下行动，**共3个行动点**：

| 行动类型 | 消耗行动点 | 描述 |
| :--- | :--- | :--- |
| 🚶 移动 | 1点 | 移动到相邻区域（同层或相邻层） |
| 🔍 探索 | 1点 | 在当前区域进行探索，抽取1张生物卡或遗物卡 |
| 🛠️ 修理 | 1点 | 修复1件损坏的装备，恢复其效果 |
| 🔄 交换 | 1点 | 与同一区域的队友交换1张装备卡 |
| 🫧 补给 | 1点 | 在补给点区域恢复2点团队氧气 |
| ⚔️ 战斗 | 1点 | 与遭遇的深海生物进行战斗（掷2d6+加成） |

> 💡 **行动组合示例**：移动(1) + 探索(1) + 战斗(1) = 3行动点；或 移动(1) + 移动(1) + 补给(1) = 3行动点

#### 阶段C：氧气消耗
回合结束时，根据当前深度消耗氧气：

| 深度层级 | 每回合氧气消耗 | 每人额外消耗 |
| :--- | :--- | :--- |
| 第1层 | 1点 | 0点 |
| 第2层 | 1点 | 0点 |
| 第3层 | 2点 | 1点（共x2） |
| 第4层 | 3点 | 2点（共x3） |
| 第5层 | 4点 | 3点（共x4） |

> 💡 **氧气管理**：氧气是团队共享资源。氧气归零不等于立即失败，但每回合氧气为0时，全队各损失1点生命，并强制进入紧急上浮。

#### 阶段D：环境事件
回合结束时，从环境牌堆抽1张，执行效果后放入弃牌堆。环境牌堆耗尽后，将弃牌堆洗混重新形成环境牌堆（威胁等级+1）。

#### 阶段E：回合结束
将回合传递给下一位玩家（顺时针方向）。

### 6.3 氧气与水压系统

**氧气系统**：
- 团队共享氧气池，初始30点
- 每回合根据深度消耗氧气
- 可通过补给点、氧气瓶装备、环境卡恢复氧气
- 氧气为0时进入紧急状态

**水压系统**：
- 水压不直接消耗氧气，而是作为隐藏压力值
- 每层有水压上限：第1-2层无限制，第3层上限3，第4层上限2，第5层上限1
- 超过水压上限时，全队每回合额外-1点生命
- 使用「抗压外壳」装备可增加水压上限

**紧急上浮**：
- 当氧气为0或全队决定撤退时触发
- 每回合自动上升1层
- 上升过程中不消耗行动，但每层仍消耗氧气（若还有剩余）
- 安全返回水面后，游戏以**撤退失败**结束

### 6.4 胜利条件

**主胜利条件**：团队收集到任意**3件遗物**，携带遗物抵达**第5层海底之心区域**，激活深海之心，然后**安全返回水面**。

> ⚠️ **关键**：找到深海之心不等于胜利！团队必须在获得深海之心后，带着它安全返回第1层以上才算最终胜利。返回途中仍需消耗氧气、应对生物遭遇和环境事件。

**副胜利条件**：若团队在第5层击败「深海守护者」（战力7的BOSS），也可直接获得深海之心，跳过遗物收集要求（但仍需返回水面）。

### 6.5 失败条件

满足以下**任一条件**，全队游戏失败：

| 失败条件 | 描述 |
| :--- | :--- |
| 💀 全员阵亡 | 所有玩家生命值归零且无法施救 |
| 🫧 氧气耗尽超过3回合 | 连续3回合氧气为0 |
| 🌊 威胁等级达到6 | 环境牌堆重洗次数过多，威胁指示器达到6 |
| ⏰ 时间耗尽 | 游戏回合数达到40回合仍未获胜 |

## 7. 完整游戏流程示例

### 示例：4人游戏第1-3回合

**玩家阵容**：小明（潜航员🧑‍🚀）、小红（工程师🔧）、小刚（生物学家🔬）、小李（导航员🧭）

---

#### 📋 第1回合 — 小明（潜航员）的回合

**阶段A - 回合开始**：
团队氧气30/30。深度：第1层（阳光带）。水压：无影响。威胁等级：1。

**阶段B - 玩家行动**（3行动点）：

> **行动1 - 移动（1点）**：小明从起始位置移动到「珊瑚礁」区域。

> **行动2 - 探索（1点）**：小明在珊瑚礁探索，从生物牌堆抽1张——翻出「热带水母群」🪼（战力1）。小红提醒小明，他有鱼叉枪以外的装备可用。小明决定战斗：掷2d6 = 4+2 = 6，加上潜航员无战斗加成，结果6 vs 战力1，轻松击败！获得1点修理点数。

> **行动3 - 补给（1点）**：珊瑚礁区域有浅海补给标记，小明消耗1行动点进行补给，团队氧气+2。

**阶段C - 氧气消耗**：第1层消耗1点，团队氧气 30-1 = 29。

**阶段D - 环境事件**：抽取环境卡——「温暖洋流」🌊，团队氧气+2。氧气变为 29+2 = 31（超过上限30，仍记为30）。

**回合结束**：轮到小红。

---

#### 📋 第2回合 — 小红（工程师）的回合

**阶段A - 回合开始**：团队氧气30/30。无异常。

**阶段B - 玩家行动**（3行动点）：

> **行动1 - 移动（1点）**：小红移动到「鱼群区」区域，与小明相邻。

> **行动2 - 探索（1点）**：小红在鱼群区探索——抽到遗物卡「光之水晶」💎！小红将遗物卡放在团队面板上，遗物进度 1/3。

> **行动3 - 交换（1点）**：小红与小明在同一区域，将自己手上的「医疗急救包」💊交给小明（小明之前的战斗中轻微受伤）。

**阶段C - 氧气消耗**：1点。氧气 30-1 = 29。

**阶段D - 环境事件**：抽取「水压波动」📉，团队氧气-2。氧气 29-2 = 27。

**回合结束**：轮到小刚。

---

#### 📋 第3回合 — 小刚（生物学家）的回合

**阶段A - 回合开始**：团队氧气27/30。小刚的生物学家被动：战斗+1骰值。

**阶段B - 玩家行动**（3行动点）：

> **行动1 - 移动（1点）**：小刚移动到「海藻林」。

> **行动2 - 移动（1点）**：利用行动点继续移动到第1层边界，准备下潜。

> **行动3 - 探索（1点）**：小刚在海藻林边缘探索——遭遇「珊瑚礁鲨」🦈（战力2）。小刚选择使用被动技能「观察」代替战斗，观察后得知该区域鲨鱼无害。获得1点修理点数，生物卡退回牌堆。

**阶段C - 氧气消耗**：1点。氧气 27-1 = 26。

**阶段D - 环境事件**：抽取「鱼群经过」🐟——利好事件！全队可在补给点额外获得1件装备。小红立即从装备牌堆抽取1张——获得「声呐探测仪」📡。

**回合结束**：轮到小李。团队开始准备向第2层暮光带进发……

---

> 💡 **3回合小结**：经过3回合，团队氧气从30降至26，收集了1件遗物（光之水晶💎），击败了1只生物，获得1件新装备。团队状态良好，准备向更深处进发。

## 8. 新手策略提示

**🫧 氧气管理优先**：氧气是团队的生命线。在浅海区多利用补给点囤积氧气，不要急于下潜。推荐在氧气低于15点时考虑返回浅海补给。

**🤝 角色互补搭配**：
- 2人游戏：推荐 潜航员 + 工程师（移动+修复兼顾）
- 3人游戏：推荐 + 医官（增加生存力）
- 4-5人游戏：所有角色均衡搭配效果最佳

**⚓ 利用装备组合**：「深海探照灯」+「鱼叉枪」= 第3层战斗无忧；「水声探测仪」+「导航员」= 环境事件几乎无威胁。

**🐢 稳扎稳打**：不要在第1-2层浪费太多回合，但也不要跳过探索。每层至少收集1件补给或装备再继续下潜。

**🛡️ 保护生物学家**：生物学家是团队的核心战斗角色，确保其存活可以大幅降低生物遭遇的风险。

**📦 遗物选择策略**：优先收集「能源类」和「护符类」遗物，它们提供持续效果，在下潜越深时价值越高。「一次性」遗物留到关键时刻使用。

**🔄 知道何时撤退**：如果氧气低于10且不在补给点附近，果断启动紧急上浮。活着回来比完成任务更重要——下次尝试时你对深海会有更多了解。

## 9. 变体规则

### 9.1 🌀 「深海迷雾」模式（简单变体）

- 环境牌堆中移除所有★★★★难度卡牌
- 团队初始氧气提升至40点
- 深海守护者战力降至5
- 适合10-12岁玩家或首次体验

### 9.2 🦑 「深渊噩梦」模式（困难变体）

- 环境牌堆中额外加入5张「生物潮涌」卡
- 团队初始氧气降至20点
- 第4层及以上氧气消耗倍率+1
- 深海守护者战力提升至9
- 每3回合增加1只随机生物巡逻
- 适合资深玩家寻求极限挑战

### 9.3 🏆 「竞速模式」

- 目标不变，但增加回合限制：20回合内完成
- 每层必须至少探索2个区域才能继续下潜
- 完成时间越短，评价等级越高（S/A/B/C）
- 适合多次游玩后追求效率的玩家

### 9.4 🎭 「角色成长」模式（战役变体）

- 使用多局游戏，角色在每局结束后保留1件装备
- 每完成1局获得1点角色技能强化（掷骰+1或额外1次主动技能）
- 共3局战役，难度逐局递增
- 适合长期体验和深度策略玩家

### 9.5 👤 「独行探险」模式（单人变体）

- 1人控制2个角色
- 行动点提升至4点/回合
- 氧气初始25点
- 移除所有「全队」效果的环境卡
- 生物战力-1（单人难度补偿）

## 10. 常见问题

**Q1：氧气耗尽后游戏立即结束吗？**
A：不是。氧气为0后，团队有3回合的缓冲期。每回合氧气为0时全队各损失1点生命，且触发紧急上浮（每回合自动上升1层）。在3回合内返回水面即可避免全灭。缓冲期过后氧气仍为0则全队阵亡。

**Q2：遗物可以被丢弃或交换吗？**
A：已收集的遗物不能丢弃，也不能在玩家间交换。遗物属于团队共有资源，放置在团队面板上。但若遭遇「吞噬水母」事件，可能会丢失1件已收集的遗物（放回遗物牌堆重洗）。

**Q3：如果全队都在不同区域，如何进行战斗？**
A：只有遭遇生物的玩家（或与生物在同一区域的玩家）参与战斗。其他玩家可以消耗1个行动点「支援」（远程支援，掷1d6加战斗值），但支援骰值只有一半效果（向上取整）。

**Q4：环境牌堆洗混后威胁等级增加有什么影响？**
A：威胁等级主要影响两个方面：一是某些高难度环境卡只在威胁等级达到一定值后才会生效；二是威胁等级达到6即为失败条件之一，相当于给游戏设定了时间压力。玩家应尽量高效推进，避免过多回合的拖延。

**Q5：深海之心获得后，返回途中还需要探索吗？**
A：不需要。获得深海之心后，团队可以直接上浮，每回合自动上升1层（类似紧急上浮但不会扣生命）。但环境事件仍会正常触发，且生物遭遇可能发生。建议保留一些氧气瓶和医疗包用于返程。

**Q6：角色阵亡后还能复活吗？**
A：阵亡的玩家在下回合开始时可以消耗3点团队氧气和另一名队友1点生命来复活，复活后生命值为1。若团队无法支付复活代价，则该玩家永久阵亡。

**Q7：多个玩家可以在同一区域吗？**
A：可以。同一区域可以容纳任意数量的玩家。在同一区域时，玩家之间可以交换装备（消耗1行动点），且战斗时同一区域的玩家可以协助。

**Q8：如何处理生物遭遇中的骰子冲突？**
A：使用2枚六面骰（2d6），将掷出的点数相加，再加上装备加成和角色技能加成。若结果大于等于生物战力，则击败生物。若小于生物战力，全队受到（生物战力 - 掷骰结果）的伤害，分配到各玩家生命值上。
