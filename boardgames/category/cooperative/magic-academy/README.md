# 魔法学院 (Magic Academy)

## 🎮 游戏概述

《魔法学院》是一款合作类桌游，所有玩家共同对抗黑暗势力。每位玩家扮演一名魔法学徒，在古老的魔法学院中学习法术、收集魔法材料、解开远古谜题，最终封印觉醒的黑暗之神。

这款游戏强调团队合作，所有玩家共享信息、资源和策略，只有团结协作才能赢得胜利。适合家庭聚会、朋友游戏夜，培养团队精神和沟通能力。

### 地图特点

1. **楼层设计**：三层塔楼，每层有独特主题和挑战
2. **资源分布**：法术和材料点均匀分布，鼓励探索
3. **黑暗蔓延**：黑暗点会随时间增加，增加紧迫感
4. **通关路径**：从底层到塔顶需要经历约20-25步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 只能在有道路（⬛）或特殊点的位置停留
- 移动步数由掷骰子决定（2颗骰子之和）
- 到达楼层入口（🌲/🌟/🏰）可进入上一层

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择角色**：每位玩家选择一个魔法学徒角色（火/水/土/风）
3. **放置棋子**：所有玩家棋子放在起点🚪
4. **初始化资源**：
   - 每位玩家获得3张材料卡（随机）
   - 将法术卡洗混作为公共牌库
   - 将黑暗卡放在地图旁
   - 设置黑暗计量表为0
5. **确定先手**：掷骰子，点数最高者先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的路径移动棋子
- **重要**：玩家可以商量移动策略，但决定权在自己

#### 2. 执行所在位置的事件

根据棋子停留的位置，执行相应动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起点** | 🚪 | 无特殊效果，可以补充手牌至3张 |
| **道路** | ⬛ | 无特殊效果 |
| **法术点** | 🔮 | 从公共牌库学习1张法术卡 |
| **材料点** | 🌿 | 获得1张随机材料卡 |
| **事件点** | 📜 | 抽取并执行事件卡效果 |
| **黑暗点** | 💀 | 抽取并执行黑暗卡效果，黑暗计量表+1 |
| **宝藏点** | 💎 | 获得1张宝藏卡 |
| **楼层入口** | 🌲/🌟/🏰 | 进入上一层，全体玩家一起移动 |

#### 3. 施展法术（可选）

如果有足够的材料，可以在回合中任意时刻施展法术：
- 每位玩家的法术是**共享知识**，可以讨论最佳使用时机
- 法术效果可以作用于自己或其他玩家

#### 4. 传递资源（可选）

玩家之间可以传递材料卡和法术卡：
- 每次传递需要消耗1个行动点（即少移动1步）
- 鼓励团队合作，合理分配资源

#### 5. 黑暗回合

所有玩家回合结束后，执行黑暗回合：
- 翻开1张黑暗卡（基础难度）
- 如果黑暗计量表达到5，翻开2张黑暗卡
- 如果黑暗计量表达到8，翻开3张黑暗卡

### 胜利条件

**合作胜利**：所有玩家共同努力，达成以下任一条件即获胜：

1. **封印黑暗之神**：至少1名玩家到达塔顶🏰，并且黑暗计量表≤5
2. **完成魔法仪式**：收集全部4种稀有材料（火焰花、月光草、龙鳞、星光尘）并带回起点

**失败条件**：
- 黑暗计量表达到10
- 所有玩家的手牌耗尽且无法补充

### 回合结束

完成所有操作后，将回合交给下一位玩家，并记得执行黑暗回合。

## 📖 完整游戏流程示例

**游戏设置**：3名玩家（火系学徒A、水系学徒B、土系学徒C），初始手牌各3张。

**第1回合（玩家A）**：
1. 掷骰子得到7点，移动7步
2. 停在材料点🌿，获得1张"火焰花"
3. 使用2张"火焰花"施展"火球术"，消除路径上的黑暗点
4. 回合结束，执行黑暗回合（翻开1张黑暗卡"黑暗蔓延"，计量表+1）

**第2回合（玩家B）**：
1. 掷骰子得到4点，移动4步
2. 停在法术点🔮，学习1张"治愈术"
3. 使用"治愈术"恢复自己1点生命值
4. 将手中的"月光草"传递给玩家C（消耗1行动点）
5. 回合结束，执行黑暗回合

**第3回合（玩家C）**：
1. 掷骰子得到9点，移动9步，到达楼层入口🌲
2. 全体玩家一起进入第二层
3. 停在宝藏点💎，获得"魔法圣杯"宝藏卡
4. 宝藏效果：所有法术消耗-1材料，团队收益！
5. 回合结束，执行黑暗回合

**后续回合**：
- 玩家们继续探索第二层、第三层
- 收集稀有材料，互相配合施展高级法术
- 当黑暗计量表接近8时，团队加速向塔顶进发
- 最终，玩家A到达塔顶🏰，此时黑暗计量表=4，团队胜利！

## 💡 新手策略提示

1. **沟通至上**：时刻与队友分享手牌信息，共同规划策略
2. **资源集中**：不要每人都留材料，集中给1-2人施展关键法术
3. **优先清黑暗**：黑暗计量表是倒计时，优先消除黑暗点
4. **合理使用宝藏**：宝藏卡效果强大，在关键时刻使用
5. **留意楼层入口**：尽快到达上层，上层资源更丰富
6. **不要自私**：这是合作游戏，队友获胜=你获胜
7. **预言术很重要**：提前知道黑暗卡内容，可以提前防范

## 🔄 变体规则

### 简单模式（适合新手）

- 黑暗计量表上限改为12
- 每位玩家初始手牌改为5张
- 可以查看黑暗卡堆顶的3张牌

### 困难模式（适合老手）

- 黑暗计量表上限改为7
- 每位玩家初始手牌改为2张
- 黑暗回合翻开的黑暗卡数量+1
- 增加"黑暗之神觉醒"事件：每5回合触发一次，所有玩家后退3步

### 团队对抗模式

- 分成两队（2v2），每队合作对抗另一队
- 先完成任务的一队获胜
- 可以互相干扰（如使用法术让对方后退）

### 限时挑战模式

- 设置45分钟时间限制
- 时间内完成任务则获胜，否则失败
- 增加紧张感和成就感

## ❓ 常见问题

**Q：我可以查看其他玩家的手牌吗？**
A：可以！这是合作游戏，所有信息都可以共享。

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：材料卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用，或者停留在材料点获得新卡。

**Q：可以跳过施展法术直接移动吗？**
A：可以，施展法术是可选操作，根据策略决定。

**Q：多人同时到达塔顶怎么办？**
A：合作游戏没有竞争，只要有人到达且满足胜利条件，全体获胜！

**Q：黑暗计量表达到10后还能继续游戏吗？**
A：不能，黑暗之神完全觉醒，游戏立即失败。

**Q：可以使用手机查询法术组合吗？**
A：建议先自己讨论策略，实在卡关时再求助外部资源。

**Q：宝藏卡可以传递给其他玩家吗？**
A：可以，宝藏卡视为普通卡牌，可以传递。
