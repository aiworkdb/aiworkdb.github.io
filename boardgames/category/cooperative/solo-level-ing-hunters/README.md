# 独行猎人 (Solo Leveling Hunters)

## 🎮 游戏概述

《独行猎人》是一款基于热门动漫"我独自升级"(Solo Leveling) 改编的合作类桌游。所有玩家扮演猎人小队成员，组队进入危险的地下城，通过战斗提升等级、收集稀有装备、完成神秘任务，最终挑战并击败地下城Boss。

这款游戏完美还原了动漫中的升级系统和地下城探险体验，强调团队合作、策略规划和资源管理。玩家需要合理分配角色职责（坦克、输出、辅助），共同克服重重困难。适合动漫粉丝、桌游爱好者和喜欢合作挑战的玩家。

### 地图特点

1. **多层设计**：5层地下城，每层难度递增
2. **资源分布**：宝藏点和休息点均匀分布，鼓励探索
3. **怪物强度**：越深层怪物越强，需要团队合作
4. **通关路径**：从入口到Boss房间需要经历约30-40步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 只能在有道路（⬛）或特殊点的位置停留
- 移动步数由掷骰子决定（2颗20面骰子取较高值）
- 到达楼层出口（🚪）可进入下一层
- 死亡后可在入口层复活，但会损失部分装备

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将地下城地图放在桌面中央
2. **选择角色**：每位玩家选择一个猎人职业（剑士/法师/弓箭手/牧师）
3. **放置棋子**：所有玩家棋子放在第1层入口🚪
4. **初始化资源**：
   - 每位玩家获得起始装备（铁剑/皮甲）
   - 将怪物卡、装备卡、事件卡、消耗品卡分别洗混
   - 设置等级标记为1级，生命值标记为满值
   - 设置回合计数器为1
5. **确定先手**：掷骰子，点数最高者先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 行动阶段

玩家可以选择以下行动（每回合2个行动点）：

| 行动 | 消耗行动点 | 说明 |
| :--- | :--- | :--- |
| **移动** | 1点 | 掷骰子决定移动步数 |
| **攻击** | 1点 | 与相邻怪物战斗 |
| **使用技能** | 1点 | 使用1张技能卡 |
| **使用消耗品** | 1点 | 使用1张消耗品卡 |
| **搜索宝藏** | 1点 | 在宝藏点搜索装备 |
| **休息** | 2点 | 恢复生命值和MP |

#### 2. 移动和探索

- 掷2颗20面骰子，取较高值作为移动步数
- 按照地图上的路径移动棋子
- **重要**：玩家可以商量移动策略，但决定权在自己
- 停留在特殊点（怪物点/事件点/宝藏点）需立即执行相应动作

#### 3. 战斗系统

当玩家停留在怪物点或在怪物相邻格子时，可以发起攻击。

**战斗流程**：
1. 翻开1张怪物卡，显示怪物属性
2. 玩家选择攻击方式：
   - **普通攻击**：掷20面骰子，点数+攻击力 = 伤害
   - **技能攻击**：使用技能卡，按技能效果计算伤害
   - **物品攻击**：使用消耗品，按消耗品效果计算
3. 怪物反击：
   - 怪物攻击力 + 掷10面骰子 = 玩家受到伤害
   - 玩家防御力可以减少伤害（伤害 = 攻击 - 防御，最低0）
4. 战斗持续直到一方生命值归零
5. 胜利奖励：获得经验值、金币，概率获得装备

**伤害计算公式**：
```
玩家伤害 = 攻击力 + 骰子点数 + 武器加成
怪物伤害 = 怪物攻击力 + 骰子点数 - 玩家防御力 - 防具加成
```

#### 4. 事件和执行

根据棋子停留的位置，执行相应动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **入口/出口** | 🚪 | 可以进入下一层（需击败该层所有怪物） |
| **道路** | ⬛ | 无特殊效果 |
| **怪物点** | 👹 | 触发战斗，胜利后可以获得奖励 |
| **事件点** | 🎲 | 抽取并执行事件卡效果 |
| **宝藏点** | 💰 | 获得1张装备卡或消耗品卡 |
| **休息点** | 🌿 | 恢复30点生命值和20点魔法值 |
| **观察点** | 👀 | 查看相邻房间的怪物信息 |
| **Boss房间** | 🐉 | 挑战地下城Boss（需全队一起） |

#### 5. 升级和装备

- **升级**：积累100EXP升1级，属性提升
- **装备**：可以在宝藏点或更换装备时使用
- **交易**：玩家之间可以交易装备和消耗品

#### 6. 怪物阶段

所有玩家回合结束后，执行怪物阶段：
- 未被击败的怪物可以移动（移动步数 = 怪物数量/2）
- 如果怪物移动到玩家相邻格子，触发战斗
- 某些怪物有特殊技能（如远程攻击、群攻等）

#### 7.  event阶段

抽取1张事件卡，所有玩家共同承受事件效果。

### 胜利条件

**合作胜利**：所有玩家共同努力，达成以下所有条件即获胜：

1. **击败Boss**：全队在Boss房间击败地下城Boss
2. **完成主线任务**：完成至少1个主线任务
3. **存活玩家≥1**：至少1名玩家存活

**失败条件**：
- 所有玩家生命值归零（全员死亡）
- 回合计数器达到30（地下城坍塌）
- 放弃任务

### 回合结束

完成所有操作后，将回合交给下一位玩家，并记得执行怪物阶段和事件阶段。

## 📖 完整游戏流程示例

**游戏设置**：3名玩家（剑士A、法师B、弓箭手C），初始装备各1件。

## 💡 新手策略提示

1. **职业搭配**：理想团队是1坦克（剑士）+ 1输出（法师/弓箭手）+ 1辅助（牧师），但2-4人都可以调整
2. **资源集中**：优先给输出职业配备好装备，坦克和辅助可以用次一点装备
3. **升级优先**：尽早击败怪物升级，属性提升非常明显
4. **合理使用技能**：技能消耗MP，不要在弱怪身上浪费，留着打Boss
5. **团队合作**：坦克吸引火力，输出职业全力输出，辅助职业保证生存
6. **宝藏优先**：宝藏点💰可以获得装备，优先探索
7. **休息时机**：生命值低于50%时，优先找休息点🌿恢复
8. **观察点很重要**：使用观察点👀查看相邻怪物，规划路线避免强怪
9. **不要贪心**：如果生命值低，优先休息或返回入口层，不要硬刚
10. **Boss战准备**：挑战Boss前，确保全员生命值满，MP充足，装备最好

## 🔄 变体规则

### 简单模式（适合新手）

- 怪物生命值和攻击力减半
- 玩家初始等级改为3级
- 起始装备改为稀有品质
- 回合上限改为50回合
- 可以查看怪物卡堆顶的3张牌

### 困难模式（适合老手）

- 怪物生命值和攻击力x1.5
- 玩家初始等级改为1级
- 起始装备改为普通品质
- 回合上限改为20回合
- 增加"怪物狂暴"事件：每5回合触发一次，所有怪物攻击力+5

### 单人模式

- 1名玩家控制所有4个职业
- 每回合可以操作所有角色（按顺序）
- 增加角色间的配合技能（如"合击"：2个角色同时攻击，伤害x1.5）

### 限时挑战模式

- 设置60分钟时间限制
- 时间内击败Boss则获胜，否则失败
- 增加紧张感和成就感

### PVP模式（对抗）

- 分成两队（2v2或1v1v1v1）
- 每队独立探索地下城
- 先击败Boss的队伍获胜
- 可以互相干扰（如使用技能让对方怪物强化）

### 随机地下城模式

- 每次游戏随机生成地图
- 使用骰子或卡片随机决定房间类型
- 增加重玩价值

## ❓ 常见问题

**Q：我可以查看其他玩家的手牌和装备吗？**
A：可以！这是合作游戏，所有信息都可以共享。

**Q：如果我没有20面骰子怎么办？**
A：可以使用两个10面骰子（十位和个位），或者使用手机上的骰子应用。

**Q：装备卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用，或者停留在宝藏点获得新装备。

**Q：可以跳过战斗直接移动吗？**
A：可以，但怪物会一直存在，阻碍你的路径。建议优先清理怪物。

**Q：多人同时挑战Boss怎么办？**
A：合作游戏没有竞争，所有玩家一起攻击Boss，共同承担伤害。

**Q：生命值归零后还能继续游戏吗？**
A：可以，在入口层复活，但会损失50%金币和1件随机装备。

**Q：可以使用手机查询攻略吗？**
A：建议先自己讨论策略，实在卡关时再求助外部资源。

**Q：装备可以传递给其他玩家吗？**
A：可以，装备卡视为普通卡牌，可以在休息点传递。

**Q：等级上限是多少？**
A：最高10级，此时属性为：生命值+100，攻击力+20，防御力+10，魔法值+50。

**Q：Boss战有什么技巧？**
A：Boss攻击力很高，需要坦克吸引火力，输出职业保持距离，辅助职业随时治疗。合理使用技能和消耗品。

**Q：如果卡牌不够用怎么办？**
A：可以将已使用的卡牌（如消耗品、技能冷却完毕的技能卡）洗混后重新使用。
