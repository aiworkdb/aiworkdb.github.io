# 🚀 星际餐厅大逃亡 Space Diner Escape

> *"2387年，银河系最著名的太空流浪餐厅'银河小馆'正穿越一片陨石带——引擎故障警报响彻后厨，而订单还在不断涌入。你们能一边端盘子，一边修好飞船逃出去吗？"*

---

## 一、游戏概述

**星际餐厅大逃亡**是一款 1-5 人合作桌游，玩家共同扮演"银河小馆"的厨师与服务员。游戏分两条并行主线：

- 🍳 **经营线**：接待顾客、烹饪菜肴、完成订单，赚取"星际信用点"；
- 🔧 **修复线**：使用信用点购买配件，合力修复飞船的 4 个关键系统，在 6 轮内离开陨石带。

两条主线缺一不可——光顾客人赚不到足够配件，光修船又会被投诉离场。玩家需要分工协作、灵活调度行动点，才能在危机中同时维持餐厅运转与飞船安全。

**适合人群**：家庭聚会、朋友局；喜欢经营类和逃脱类游戏的玩家；10 岁以上均可上手。

## 六、游戏规则

### 准备阶段（10分钟）

1. **铺设游戏板**：将飞船地图放桌中央，将系统损坏条（引擎/氧气/动力/导航）置于对应区域，各标记在 **2格**（初始轻度损坏）。
2. **分配角色**：玩家商量后各取1张角色卡，将棋子放在后厨区。
3. **初始食材**：食材储藏区放入：🥩×4、🌾×4、🐟×3、🥦×3、🧀×3、🌶️×2、🥚×2（合计21份食材）。
4. **洗牌备用**：
   - 将顾客订单卡（30张）洗匀后放在服务台；
   - 将危机事件卡（18张）洗匀后放在单独牌堆；
   - 将配件卡（16张）洗匀后放在商店区；
   - 特殊顾客卡（6张）随机混入顾客订单堆。
5. **初始订单**：在用餐区随机开启 **2 张桌**（从服务台翻2张顾客牌，分配到桌1和桌2）。
6. **初始资金**：全体玩家公用 **10 信用点**（放在公共区域）。
7. **轮次标记**：轮次盘置于 "第1轮"。

### 游戏流程

每轮按以下 4 个阶段进行：

#### 第一阶段：分配行动点
- 每位玩家获得 **4 个行动点**（使用标记物追踪）；
- 拥有特殊技能的角色可能有调整，参见角色卡说明；
- 玩家**商量**本轮分工：谁去烹饪、谁去修船、谁去接单。

#### 第二阶段：行动回合（同时或按序）
玩家可自由顺序使用行动点，每点可做以下**任一操作**：

| 行动 | 消耗 | 说明 |
| :--- | :--- | :--- |
| 🚶 移动 | 1点 | 移动到相邻区域（驾驶舱/控制室需2点） |
| 🍳 烹饪 | 1点 | 在当前灶台推进1步；完成后将成品放在灶台旁 |
| 🍽️ 上菜 | 1点 | 将已完成菜品送至对应餐桌，完成订单领赏金 |
| 🛒 补充食材 | 1点 | 从储藏区取2份同类食材（必须在储藏区） |
| 📋 接单 | 1点 | 从订单堆翻1-2张牌，安排到空餐桌（必须在服务台） |
| 🛒 购买配件 | 1点+信用点 | 在商店购买1张配件卡（必须在商店区） |
| 🔧 修复系统 | 1点+配件 | 在对应区域使用1张配件卡，修复系统损坏格（必须在对应区域） |

> ⚡ **合作提示**：玩家之间可在同一区域"协作行动"——2名玩家同时在同一灶台，每人花1点可完成2步烹饪（效率翻倍）。

#### 第三阶段：检查顾客等待
- 对每桌已等待 **≥等待耐心值** 的顾客，翻出1张顾客投诉牌（从危机牌堆顶取）并立即执行；
- 顾客离席后移走订单牌，餐桌空出可重新接单。

#### 第四阶段：危机事件
1. 从危机事件堆顶**翻出1张**危机牌，立即执行其效果；
2. **检查飞船状态**：若任意系统损坏达 **3格**，本轮额外再翻1张危机牌；
3. **检查失败条件**：若任意系统损坏达 **5格（全满）**，**立即游戏失败**；
4. 轮次盘前进 +1，进入下一轮。

### 胜利条件

在第 **6 轮结束前**（含第6轮），满足以下**全部**条件：

1. ✅ 四个飞船系统各自损坏格 ≤ 1（基本修复）；
2. ✅ 公共信用点余额 ≥ 5（维持餐厅基本运营）；
3. ✅ 当前用餐区无超时顾客（所有等待中的顾客均在耐心范围内）。

当第6轮行动阶段结束时，**统一判定胜负**。

### 失败条件（任意一项即失败）

| 失败条件 | 触发时机 |
| :--- | :--- |
| 任一飞船系统损坏达 5 格（全满） | 危机事件阶段结束时 |
| 6 轮结束后系统未达标 | 第6轮危机阶段结束后 |
| 公共信用点归零且无法维持运营 | 任意阶段内 |

## 七、完整游戏流程示例

### 示例场景（3人游戏，第2轮）

**角色分配**：小张（厨师长）、小李（修理机器人）、小王（章鱼服务员）

**当前状态**：
- 信用点：18 点
- 飞船系统：引擎3格⚠️，其余各1格
- 当前餐桌：桌1（绿星顾客，等第2轮），桌3（黄星顾客贵族鱿鱼人，等第1轮）
- 手头菜品：星云浓汤（桌1用）

**第一阶段：分配行动点**

三人商量：
- 小张（厨师长）负责烹饪，完成桌3的两道菜；
- 小李（修理机器人）去修引擎（引擎已3格，危险！）；
- 小王（章鱼服务员）上菜+接新单。

**第二阶段：行动**

**小张（4点）**：
1. 已在后厨区，直接烹饪"星云浓汤"（桌3第1道）→ 1步完成 ✅（因厨师长技能：消耗½点，算1点）
2. 烹饪"精致星鱼排"：灶台C，步骤1/3（消耗1点）
3. 继续精致星鱼排：步骤2/3（消耗1点，厨师长优惠已用完）
4. 继续精致星鱼排：步骤3/3（消耗1点）→ 精致星鱼排完成 ✅

> 小张本轮共用 1+1+1+1 = 4点，两道菜全部完成！（厨师长技能省了½点）

**小李（4点）**：
1. 移动到商店区（1点），购买引擎备件（花费4信用点，信用点剩14）（1点）
2. 移动到系统控制室（2点移动，但修理机器人技能：移动消耗-1，实际消耗1点）
3. 使用引擎备件修复引擎（1点 + 配件卡，引擎 3格→1格，修理机器人技能修复值+1故-2格）✅

> 小李用了 1+1+1+1 = 4点，引擎从危险3格修回1格！

**小王（4点）**：
1. 拿起桌1的星云浓汤，上菜给桌1（章鱼服务员技能：同时携带2道菜，1点上桌1的浓汤）
2. 拿起两道菜（浓汤已送，拿精致星鱼排），上菜给桌3（1点，章鱼服务员同轮2桌1点）
3. 移动到服务台（1点），接新订单（翻2张，得到桌4绿星、桌5红星暴躁机器人）（1点）

> 小王用了 1+1+1+1 = 4点，桌1桌3收入：绿星2点+黄星6点=8信用点！信用点：14+8=22 🎉

**第三阶段：检查超时**

桌4刚接入，桌5（暴躁机器人耐心=1轮）警告！下轮必须完成。

**第四阶段：危机事件**

翻出危机牌：🔴 "微流星雨" — 所有系统各+1损坏：引擎 1→2，氧气 1→2，动力 1→2，导航 1→2。

**本轮小结**：引擎及时修复，赚到8信用点，但微流星雨让局势吃紧。第3轮目标：先拿下暴躁机器人！

## 八、新手策略提示

### 🔥 分工是核心

- 5人局可以固定分工：2人烹饪（1人普通菜、1人精致菜）、1人上菜/接单、2人负责修船；
- 2-3人局建议1人专攻修船、其余人负责全部餐厅工作。

### ⚠️ 优先修引擎

引擎若到3格，每轮多一张危机牌，会形成恶性循环。宁可少赚点信用点，也要先把引擎稳住。

### 💰 信用点不要贪

配件需要购买，危机有时会扣钱，建议始终保留 **10+ 信用点**的储备。抢高价订单时要计算修船资金。

### 🟡 黄星顾客优先

黄星赏金5-7点，性价比最高，且等待2轮，不用太急。优先接黄星顾客，再用剩余行动点处理其他事。

### ❌ 不要同时接太多单

最多保持 **4张桌同时开放**，超过后厨能力范围只会加速失败。

### 🤝 合作烹饪加速

两人协作同一灶台，每人花1点可完成2步，效率翻倍，对精致料理（需3-4步）特别有用。

## 九、变体规则

### 🟢 轻松模式（适合初次体验）

- 初始系统损坏从2格降至 **1格**；
- 顾客耐心+1轮（绿星4轮，黄星3轮，红星2轮）；
- 初始信用点改为 **15 点**；
- 危机事件堆减少4张（抽掉2张"陨石撞击"和2张"系统干扰"）。

### 🔴 地狱模式（6轮挑战）

- 初始系统损坏从2格提升至 **3格**；
- 每轮结束翻 **2张危机牌**；
- 胜利条件：四系统全部降至 **0格**（全修好）且信用点≥20；
- 去掉"幸运星旅者"特殊顾客卡，混入第二张"美食评论家"。

### 👤 单人模式（厨师长独角戏）

- 只使用"厨师长"角色，拥有 **6 个行动点**（而非4个）；
- 每轮危机牌数量-1；
- 胜利只需4个系统各达 **≤2格**（不要求≤1格）。

### 🌃 夜班模式（加速版，适合20-30分钟）

- 游戏轮数从6轮缩短为 **4轮**；
- 初始信用点调整为 **20点**；
- 初始时开放 **3张餐桌**（而非2张）；
- 删除所有等待耐心1轮的暴躁机器人（红星顾客），仅保留绿星和黄星。

## 十、常见问题

**Q：一道菜必须在同一灶台连续烹饪完成吗？**
A：是的，菜品与灶台绑定，烹饪中途玩家可以离开，但该灶台的菜品继续等待，下一位玩家可以接续烹饪（必须先移动到后厨区）。

**Q：章鱼服务员"同时为2桌上菜"是什么意思？**
A：花费1个行动点，可以把手持的菜品送到2个不同餐桌，相当于1点完成2次上菜。但首先需要手头有对应2道菜。

**Q：修复系统是否可以分多次进行？**
A：可以，比如引擎损坏4格，可以第1轮修2格（用1张引擎备件），第2轮再修2格，不需要一次性修满。

**Q：特殊顾客卡"饥饿宇宙神"翻出后需要立即开2桌吗？**
A：是的，立即从订单堆再翻2张放到空餐桌。若空桌不够，选用已有顾客离席的桌子；若全部满座，跳过"立即多开2桌"的效果，仅赏金翻倍效果保留。

**Q：行动点可以留到下一轮吗？**
A：不可以，每轮开始重新获得4点，未使用的点数会归零。

**Q：信用点不够买配件怎么办？**
A：只能通过完成订单赚取信用点。如果当前信用点不足，优先完成高价值订单（黄星、红星）赚钱，再购买配件。这也是游戏核心张力所在。

**Q：导航模块为什么比其他配件贵，但修复值却更高？**
A：导航系统负责整体航行路线，修复后可减少3格（含精密修复），性价比更高，但需要更多信用点前期积累。建议在经营线稳定后再购买。

**Q：可以同时启动2条修复线吗？**
A：完全可以，2名玩家各拿不同配件，分别前往控制室和驾驶舱同时修复不同系统，是5人局的常见策略。
