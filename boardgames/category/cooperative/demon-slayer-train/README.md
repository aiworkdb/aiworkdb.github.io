# 鬼杀队：无限列车 (Demon Slayer: Infinity Train)

## 🎮 游戏概述

《鬼杀队：无限列车》是一款紧张刺激的合作桌游，改编自超人气动漫《鬼灭之刃》中最经典的"无限列车篇"。

玩家将扮演鬼杀队的剑士，登上被下弦之壹·魇梦控制的神秘列车。在梦中与现实中穿梭，唤醒被控制的乘客，最终与队友合力讨伐恶鬼！

**游戏特色：**
- 🏮 **纯合作模式**：所有玩家共同对抗游戏系统，只有全队胜利才是真正的胜利
- 😴 **梦境机制**：玩家需要在"梦境值"耗尽前唤醒乘客，否则全员陷入永眠
- ⚔️ **呼吸法卡牌**：收集水之呼吸、雷之呼吸等招式卡，释放华丽连击
- 🚂 **列车车厢**：12节车厢逐节探索，每节都有独特事件和危险

适合《鬼灭之刃》粉丝、合作游戏爱好者，以及3-5人的朋友聚会。

3. **梦境值衰减**：每进入一节新车厢，所有玩家失去1点梦境值
4. **最短进程**：全车探索完毕约需12-15回合

## 🎲 游戏规则

### 准备阶段

1. **布置列车**：将12张车厢卡按1-12顺序背面朝上排列成列车（背面表示未探索）
2. **选择角色**：每位玩家选择一个鬼杀队角色棋子，放在第1节车厢（车头）
3. **分发起始卡牌**：
   - 每位玩家获得2张基础呼吸法卡（水之呼吸或雷之呼吸）
   - 每位玩家获得1张"日轮刀"道具卡
   - 每位玩家的梦境值设为10
4. **准备卡组**：将呼吸法卡、道具卡、事件卡分别洗混，放在列车地图旁
5. **设置回合数**：将回合数标记设为1（共15回合，用尽则游戏失败）

### 游戏流程（每个回合）

所有玩家同时参与每个回合，按顺时针顺序依次行动。每回合分为 **三个阶段**：

#### 第一阶段：探索阶段

当前车厢未被探索时，翻开当前车厢卡：
- **普通车厢**：抽取1张事件卡，执行事件效果
- **特殊车厢**：执行车厢特殊效果（见地图详解）
- **鬼出现**：若事件卡或车厢效果指示，放置对应鬼卡到该车厢

#### 第二阶段：行动阶段

每位玩家依次进行 **2点行动力** 的操作，可选择：

| 操作 | 消耗行动力 | 说明 |
| :--- | :--- | :--- |
| **攻击鬼** | 1点 | 使用1张呼吸法卡，掷骰子计算伤害 |
| **移动车厢** | 1点 | 进入下一节车厢（需前节车厢已探索） |
| **使用道具** | 1点 | 使用1张道具卡 |
| **休息** | 1点 | 回复1点梦境值（仅限车头或卧铺车厢） |
| **抽取卡牌** | 1点 | 从呼吸法卡组抽取1张卡 |

#### 第三阶段：鬼的阶段

所有玩家行动完毕后：
1. 每个在场鬼对所在车厢的玩家发动攻击（造成伤害）
2. 所有玩家失去1点梦境值（梦境衰减）
3. 回合标记+1
4. 检查胜利/失败条件

### 胜利条件

**团队胜利（以下任一条件满足）：**
- ✅ 击败第12节车厢的魇梦，且所有乘客已被唤醒
- ✅ 成功探索全部12节车厢，并在最终回合前击败魇梦

**团队失败（以下任一条件发生）：**
- ❌ 任何玩家的梦境值归零（陷入永眠）
- ❌ 回合数达到15回合而未击败魇梦
- ❌ 超过半数玩家陷入永眠

## 📖 完整游戏流程示例

### 示例1：第1回合（游戏开始）

**探索阶段：**
- 回合1开始，第1节车厢（车头）已是安全区，无需翻卡
- 无鬼出现

**玩家A的行动（炎之呼吸使用者）：**
1. 消耗1点行动力：使用"炎之呼吸"卡攻击——但车头没有鬼，行动浪费
2. 消耗1点行动力：选择"休息"，回复1点梦境值（11/10，上限10，实际不变）
3. 玩家A结束行动

**玩家B的行动（水之呼吸使用者）：**
1. 消耗1点行动力：选择"移动车厢"，进入第2节车厢，翻开第2节车厢卡
2. 第2节车厢事件："突然的寒意"——出现1只下级鬼（生命值4）
3. 消耗1点行动力：使用"水之呼吸"卡攻击鬼，掷骰子得3点，总伤害=3+2=5，击败鬼！
4. 乘客被唤醒，玩家B抽取1张道具卡（回复药）
5. 玩家B结束行动

**鬼的阶段：**
- 第2节车厢的鬼已被击败，无攻击
- 所有玩家失去1点梦境值（玩家A：10→9，玩家B：10→9）
- 回合标记变为2

### 示例2：第7回合（中期紧张阶段）

**当前状态：**
- 已探索至第8节车厢
- 玩家A（梦境值：4），玩家B（梦境值：3），玩家C（梦境值：5）
- 第8节车厢有1只"手的鬼"（生命值7）

**玩家C的行动（雷之呼吸使用者）：**
1. 消耗1点行动力：使用"雷之呼吸"卡（攻击力4），掷骰子得5，总伤害9，击败"手的鬼"！
2. 消耗1点行动力：选择"移动车厢"，进入第9节车厢（储物车厢）
3. 储物车厢效果：抽取2张道具卡（获得"鎹鸦信使"和"斑纹觉醒"）
4. 第9节车厢无鬼，乘客已被前回合唤醒

**鬼的阶段：**
- 无在场鬼
- 所有玩家失去1点梦境值（玩家A：4→3，玩家B：3→2，玩家C：5→4）
- 回合标记变为8

**策略提示：** 玩家B的梦境值只剩2点，非常危险！下回合应尽快回到车头休息。

## 💡 新手策略提示

1. **梦境值是生命线**：梦境值归零即淘汰，优先保证自己的梦境值在安全线（≥3）以上
2. **不要急于进攻**：没有鬼的车厢浪费攻击行动，先探索再制定战术
3. **合理分配行动力**：2点行动力很有限，攻击（1点）+ 移动（1点）是最常见组合
4. **车头是安全港**：梦境值低时，及时回到第1节车头休息回复
5. **唤醒乘客有奖励**：尽量完成乘客的唤醒条件，奖励往往能扭转战局
6. **道具卡留关键时刻用**：回复药和斑纹觉醒在最终战前使用效果最佳
7. **团队配合很重要**：炎之呼吸玩家可以吸引火力，为队友创造攻击机会
8. **注意回合数**：第12回合后游戏自动失败，第10回合时若还未到第12节车厢需加快节奏

## 🔄 变体规则

### 简单模式（新手推荐）

- 每位玩家起始梦境值改为12
- 魇梦的生命值减少为8
- 增加1张"隐队支援"道具卡（可让一名玩家满血复活）
- 回合数上限改为18回合

### 困难模式（老手挑战）

- 每位玩家起始梦境值改为8
- 第6节车厢开始，每节车厢出现2只鬼
- 魇梦（觉醒形态）生命值15，每回合自动消灭1名梦境值最低的玩家1点梦境值
- 回合数上限改为12回合
- 增加"血鬼术"机制：部分鬼每回合可发动特殊技能

### 单人模式

- 控制2名角色（共2个棋子）
- 每位角色独立计算梦境值，但共享手牌
-  victory condition不变

### 角色技能变体

为每位玩家分配特殊角色技能（选做）：

| 角色 | 技能效果 |

| **炭治郎** | 水之呼吸卡攻击力+1 |
| **义勇** | 起始手牌+1张 |

## ❓ 常见问题

**Q：梦境值可以减少到负数吗？**
A：不可以。梦境值最低为0，达到0时该玩家立即淘汰。

**Q：如果所有玩家都陷入永眠怎么办？**
A：游戏立即失败，全员阵亡。

**Q：呼吸法卡用完了怎么办？**
A：将弃牌堆洗混后重新组成新的抽牌堆。

**Q：可以在别人的车厢使用道具吗？**
A：可以！道具卡的使用不限制于自己所在的车厢。

**Q：最终战（第12节车厢）可以同时面对多只鬼吗？**
A：困难模式下可以。简单/普通模式只有魇梦一只鬼。

**Q：如果回合数到了15但还没失败，接下来怎么办？**
A：第15回合结束时若未击败魇梦，游戏失败。

**Q：打印卡牌时背景颜色不见了？**
A：打印时务必在打印设置中勾选"背景图形"选项。
