# 怪兽守护者 (Kaiju Defenders)

## 🎮 游戏概述

《怪兽守护者》是一款紧张刺激的合作类桌游，所有玩家共同对抗外星入侵者。每位玩家扮演一只觉醒的巨型怪兽（Kaiju），在沿海城市中抵御外星部队的攻击，保护人类建筑，最终摧毁外星母舰。

这款游戏强调团队合作和战术配合，每只怪兽都有独特的能力。适合朋友聚会、家庭游戏夜，培养团队精神和应急决策能力。

### 地图特点

1. **城市布局**：建筑均匀分布，鼓励玩家保护多个区域
2. **外星威胁**：外星基地分布在城市边缘，逐步向中心推进
3. **母舰位置**：位于地图右下角，是最终目标
4. **最短路径**：从巢穴到母舰需要经过约15-20步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 海洋地形只有特定怪兽可通过
- 移动步数由掷骰子决定（2颗骰子之和）
- 可以与其他怪兽共享同一格

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择怪兽**：每位玩家选择一只守护怪兽（哥斯拉型/魔斯拉型/拉顿型/安基拉斯型）
3. **放置棋子**：
   - 所有怪兽棋子放在中央巢穴🦖
   - 将8个建筑标记放在对应建筑位置
   - 将6个外星棋子放在外星基地👽
4. **初始化资源**：
   - 每位玩家获得3张能量核心卡（随机）
   - 将怪兽能力卡洗混，每只怪兽抽取2张
   - 将外星指令卡放在地图旁
   - 设置城市健康表为16（初始建筑数）
5. **确定先手**：掷骰子，点数最高者先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的路径移动怪兽棋子
- **重要**：玩家可以商量移动策略，但决定权在自己

#### 2. 执行所在位置的事件

根据怪兽停留的位置，执行相应动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **巢穴** | 🦖 | 无特殊效果，可以补充手牌至3张 |
| **道路** | ⬛ | 无特殊效果 |
| **住宅建筑** | 🏠 | 修复1个被破坏的建筑（如有） |
| **商业建筑** | 🏢 | 获得1张随机能量核心卡 |
| **工业建筑** | 🏭 | 获得攻击加成（本回合攻击+1） |
| **外星基地** | 👽 | 与外星形作战（见战斗规则） |
| **母舰港口** | 🚀 | 挑战外星母舰（见胜利条件） |

#### 3. 发动怪兽技能（可选）

如果有足够的能量核心，可以在回合中任意时刻发动技能：
- 每位玩家的技能是**共享知识**，可以讨论最佳使用时机
- 技能效果可以作用于自己或其他怪兽

#### 4. 传递能量核心（可选）

玩家之间可以传递能量核心卡：
- 每次传递需要消耗1个行动点（即少移动1步）
- 鼓励团队合作，合理分配能量

#### 5. 外星回合

所有玩家回合结束后，执行外星回合：
- 翻开1张外星指令卡（基础难度）
- 移动所有外星棋子向城市中心推进
- 如果城市健康表≤8，翻开2张外星指令卡
- 如果城市健康表≤4，翻开3张外星指令卡

### 战斗规则

当怪兽停留在外星基地时，触发战斗：

1. **攻击判定**：掷1颗骰子
   - 点数≥4：成功摧毁外星基地，获得1张能量核心卡
   - 点数≤3：攻击失败，外星基地反击造成1点伤害

2. **使用能量核心**：可以消耗能量核心增加攻击力
   - 每消耗1张能量核心，攻击判定+1

3. **合作攻击**：多个怪兽可以同时攻击同一个基地
   - 每个怪兽独立掷骰
   - 只要有一个成功，基地即被摧毁

### 胜利条件

**合作胜利**：所有玩家共同努力，达成以下任一条件即获胜：

1. **摧毁母舰**：至少1只怪兽到达母舰港口🚀，并通过最终挑战
   - 最终挑战：连续成功3次攻击判定（点数≥4）
   - 可以使用能量核心增加成功率

2. **坚持防御**：坚持10个回合且城市健康表≥5

**失败条件**：
- 城市健康表降至0（所有建筑被摧毁）
- 所有怪兽被击倒（生命值降至0）

### 回合结束

完成所有操作后，将回合交给下一位玩家，并记得执行外星回合。

## 📖 完整游戏流程示例

**游戏设置**：3名玩家（哥斯拉型A、魔斯拉型B、拉顿型C），初始手牌各3张。

**第1回合（玩家A）**：
1. 掷骰子得到8点，移动8步
2. 停在外星基地👽，触发战斗
3. 掷骰子得到3点，攻击失败，受到1点伤害
4. 消耗2张能量核心，重试攻击，得到5点，成功摧毁基地
5. 获得1张能量核心卡作为奖励
6. 回合结束，执行外星回合（翻开1张外星指令卡"能量武器"，摧毁1个建筑，城市健康表-1）

**第2回合（玩家B）**：
1. 掷骰子得到5点，移动5步
2. 停在商业建筑🏢，获得1张"蓝色核心"
3. 发动"飞行冲击"技能，消耗2张能量核心
4. 飞行到另一个外星基地附近
5. 将手中的"红色核心"传递给玩家C（消耗1行动点）
6. 回合结束，执行外星回合

**第3回合（玩家C）**：
1. 掷骰子得到6点，移动6步，到达外星基地
2. 使用"烈焰风暴"技能，本回合移动+2步
3. 攻击外星基地，掷骰子得到4点，成功摧毁
4. 回合结束，执行外星回合

**后续回合**：
- 玩家们继续摧毁外星基地，向母舰港口进发
- 收集能量核心，升级怪兽技能
- 当城市健康表接近4时，团队加速向母舰进发
- 最终，玩家A到达母舰港口🚀，团队合力通过最终挑战，获得胜利！

## 💡 新手策略提示

1. **沟通至上**：时刻与队友分享手牌信息，共同规划策略
2. **能量管理**：不要浪费能量核心，关键时刻使用
3. **优先保护建筑**：城市健康表是倒计时，优先修复被破坏的建筑
4. **合理使用技能**：怪兽技能强大，在关键时刻使用
5. **留意母舰位置**：尽快到达母舰港口，避免城市被完全摧毁
6. **不要自私**：这是合作游戏，队友获胜=你获胜
7. **预判外星行动**：根据城市健康表判断外星回合的难度

## 🔄 变体规则

### 简单模式（适合新手）

- 城市健康表初始改为20
- 每位玩家初始手牌改为5张
- 可以查看外星指令卡堆顶的3张牌
- 外星攻击伤害-1

### 困难模式（适合老手）

- 城市健康表初始改为12
- 每位玩家初始手牌改为2张
- 外星回合翻开的指令卡数量+1
- 增加"母舰护盾"事件：每3回合触发一次，母舰防御+2

### 怪兽对决模式

- 分成两队（2v2），每队合作对抗另一队
- 先摧毁对方母舰的一队获胜
- 可以互相干扰（如使用技能让对方后退）

### 限时挑战模式

- 设置45分钟时间限制
- 时间内完成任务则获胜，否则失败
- 增加紧张感和成就感

## ❓ 常见问题

**Q：我可以查看其他玩家的手牌吗？**
A：可以！这是合作游戏，所有信息都可以共享。

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：能量核心卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用，或者停留在商业建筑获得新卡。

**Q：可以跳过发动技能直接移动吗？**
A：可以，发动技能是可选操作，根据策略决定。

**Q：多人同时到达母舰港口怎么办？**
A：合作游戏没有竞争，只要有人到达且满足胜利条件，全体获胜！

**Q：城市健康表降至0后还能继续游戏吗？**
A：不能，所有建筑被摧毁，游戏立即失败。

**Q：可以使用手机查询怪兽技能组合吗？**
A：建议先自己讨论策略，实在卡关时再求助外部资源。

**Q：能量核心卡可以传递给其他玩家吗？**
A：可以，能量核心卡可以传递，鼓励团队合作。

**Q：如果怪兽被击倒怎么办？**
A：怪兽被击倒后，下回合可以从巢穴重新开始，但手牌清空。
