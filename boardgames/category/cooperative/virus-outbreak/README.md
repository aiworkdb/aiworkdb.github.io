# 病毒危机 (Virus Outbreak)

## 1. 游戏概述

一座繁华都市突发未知病毒疫情！在《病毒危机》中，你和队友们扮演公共卫生团队的精英成员——流行病学专家、疫苗研究员、隔离指挥官和后勤协调员。你们必须通力合作，在病毒全面扩散之前研发出疫苗，拯救这座城市。

游戏融合了**区域控制**、**资源管理**和**危机应对**机制。每一回合，病毒在地图上扩散蔓延，感染人数不断攀升。你们需要权衡有限的时间和资源：是先隔离高危区域，还是加速疫苗研发？是派医疗队救治感染者，还是调配物资保障后勤？

**适合人群**：喜欢合作游戏、紧张决策、团队讨论的玩家。1-5人游玩，可以单人挑战，也可以团队协作。节奏紧张，代入感极强！

## 6. 游戏规则

### 准备阶段

1. **放置地图**：将城市地图放在桌面中央
2. **选择角色**：每位玩家选择1张角色卡，领取对应棋子
3. **放置棋子**：所有玩家棋子放在🏥医院区（起始位置）
4. **设置病毒**：在住宅区🏘️和交通区🚇各放置2个感染标记
5. **洗混卡组**：病毒卡、事件卡、研究卡分别洗混
6. **分发行动卡**：每人抽取对应的行动卡数量
7. **设置进度**：疫苗进度标记放在0位置
8. **设置恐慌**：6个恐慌标记放在一旁备用

### 游戏流程（每回合）

每回合分为3个阶段，所有玩家共同行动：

#### 阶段1：玩家行动阶段
- 按顺时针方向，每位玩家执行行动
- 每位玩家拥有对应行动点数（2-4点）
- 可执行任意行动卡的组合
- 行动点用尽后轮到下一位

#### 阶段2：病毒扩散阶段
- 翻开1张病毒卡，执行效果
- 根据病毒阶段，在对应区域增加感染标记
- 若某区域感染标记≥4，触发「恐慌爆发」：
  - 放置1个恐慌标记
  - 相邻区域感染+1
  - 若6个恐慌标记全部放完 → **游戏失败！**

#### 阶段3：事件阶段
- 翻开1张事件卡，执行效果
- 如果是好消息，所有玩家欢呼
- 如果是坏消息，深呼吸继续战斗

### 胜利条件

**✅ 胜利**：收集5张研究卡，疫苗进度达到5 → 疫苗研发成功，拯救城市！

**❌ 失败**（任一条件）：
- 6个恐慌标记全部放完 → 城市全面崩溃
- 感染标记用完（30个） → 病毒全面扩散
- 病毒卡堆翻完但疫苗未完成 → 时间耗尽

## 7. 完整游戏流程示例

### 示例回合：第3回合

**当前局势**：
- 住宅区🏘️：感染3（高危！）
- 交通区🚇：感染2
- 商业区🏢：感染1
- 其他区域：0
- 疫苗进度：1/5
- 恐慌标记：0/6

**阶段1：玩家行动**
- 玩家A（🩺流行病学专家）：查看病毒卡（提前知道下回合扩散方向），移动到住宅区，执行「治疗感染」（感染3→2）
- 玩家B（🔬疫苗研究员）：在医院区执行「疫苗研究」（进度1→2），获得1张研究卡🧬
- 玩家C（🚔隔离指挥官）：移动到交通区，执行「隔离区域」（交通区停止扩散1回合）

**阶段2：病毒扩散**
- 翻开病毒卡：🦠2 扩散期
- 指定区域：商业区
- 商业区感染1→2，相邻区域各+1
  - 但交通区被隔离，不受影响 ✓
  - 学校区0→1

**阶段3：事件**
- 翻开事件卡：📰✅ 好消息！"社区志愿者加入，物资+2"
- 全队获得2点物资，可用于下回合行动

## 8. 新手策略提示

1. **优先控制住宅区和交通区**：这两个区域感染风险最高，失控后果最严重
2. **分工明确**：让疫苗研究员专注研究，隔离指挥官守住关键区域
3. **不要忽视治疗**：虽然疫苗是终极目标，但如果感染失控就来不及了
4. **利用角色特长**：每个角色都有独特优势，团队配合才能赢
5. **提前看牌**：流行病学专家的看牌能力极其重要，帮助团队提前部署
6. **恐慌标记是倒计时**：6个恐慌标记用完就输了，务必控制高危区域
7. **物资管理**：后勤协调员的物资产出是团队的命脉

## 9. 变体规则

### 变体1：英雄模式（更困难）
- 病毒卡每回合翻开2张
- 恐慌标记只有4个
- 适合经验丰富的团队

### 变体2：竞速模式（更快节奏）
- 疫苗只需3个线索
- 每回合玩家获得额外1行动点
- 30分钟内完成

### 变体3：变异病毒模式
- 游戏中途病毒发生变异
- 原有研究卡作废1张，需重新收集
- 增加不确定性和挑战

### 变体4：单人挑战模式
- 1人控制2个角色
- 行动点+1
- 病毒卡减少至12张

## 10. 常见问题

**Q：如果两个玩家在同一个区域，行动会叠加吗？**
A：不会自动叠加，但某些行动可以协作。例如两人都在住宅区，可以各花1行动点治疗，总共减少2点感染。

**Q：恐慌爆发后还能恢复吗？**
A：恐慌标记一旦放置就不可移除。所以预防恐慌比事后补救重要得多！

**Q：物资有什么用？**
A：物资用于执行某些行动的额外效果。例如「治疗感染」花费1物资可额外减少1点感染。

**Q：研究卡怎么收集？**
A：通过「疫苗研究」行动推进进度，同时满足条件时获得研究卡。每种线索有不同的获取条件。

**Q：单人模式怎么玩？**
A：单人模式下你控制2个角色，每人有4行动点。难度适当降低，但挑战性依然存在。

**Q：游戏太难/太简单怎么办？**
A：调整病毒卡数量和恐慌标记数量。减少病毒卡=简单，减少恐慌标记=困难。
