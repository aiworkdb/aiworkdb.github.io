# 🕵️ 绝密行动

> *灵感来源：碟中谍 / 特工间谍题材*

*打印道具请参考 [printables.html](printables.html) 文件*

**类别**：合作游戏 | **人数**：1-5人 | **时长**：40-60分钟 | **难度**：中等偏难

---

## 📖 游戏背景

全球安全网络遭到神秘组织"幽灵协议"的渗透，多枚核弹头已被秘密部署在世界各地。你们是一支精英特工小队，必须在倒计时归零前完成4项不可能的任务：潜入基地、破解密码、回收核弹、安全撤离。每一步都可能是陷阱，每一秒都至关重要。

---

## 🎯 游戏目标

所有玩家合作，在**时间线耗尽前**完成全部4项任务。任何一项任务失败，游戏即刻结束，全员任务失败。

---

## 📋 四项任务

### 任务一：潜入基地 🔓
- **目标**：将至少2名特工移入敌方基地
- **难点**：基地有3道安全门，需逐一破解
- **时间限制**：6回合
- **警戒触发**：每次破门+2警戒

### 任务二：破解密码 💻
- **目标**：收集并拼合16块密码碎片
- **难点**：碎片分散在不同区域，部分需要破解密码锁
- **时间限制**：8回合
- **警戒触发**：每使用1次暴力破解+3警戒

### 任务三：回收核弹 ☢️
- **目标**：将核弹标记移至安全区域
- **难点**：移动核弹需2名特工同时在场
- **时间限制**：6回合
- **警戒触发**：每移动1格+1警戒

### 任务四：安全撤离 🚁
- **目标**：所有特工到达撤离点
- **难点**：撤离点每2回合变更位置
- **时间限制**：4回合
- **警戒触发**：每被敌人发现1次+2警戒

## 🔄 游戏流程

### 准备阶段
1. 选择特工角色（每人1个，不满5人可1人操控多个）
2. 将时间标记设为24（总回合数）
3. 警戒值归零
4. 洗混危机卡和装备卡
5. 摆放任务一的场景

### 回合流程

#### 1. 简报阶段
- 翻开1张危机卡，执行效果
- 如果当前任务有特殊触发，此时结算

#### 2. 行动阶段（每人3个行动点）

| 行动 | 消耗 | 说明 |
|:---|:---|:---|
| 移动 | 1点 | 移动1格（潜入者免费） |
| 攻击 | 1点 | 投1骰，4+命中，消灭1个敌人 |
| 破解 | 2点 | 投2骰，合计6+成功，解锁1道门 |
| 搜索 | 1点 | 翻开1张装备卡 |
| 治疗 | 1点 | 恢复自己或相邻队友1生命 |
| 配合 | 2点 | 与相邻队友联合行动（效果+1） |

#### 3. 敌人阶段
- 每个敌人在视野内向最近特工移动1格
- 敌人相邻特工时自动攻击（投骰3+命中，造成1伤害）
- 每消灭1个敌人+1警戒

#### 4. 检查阶段
- 检查当前任务目标是否完成
- 若完成，进入下一任务场景
- 减1枚时间标记
- 若警戒值≥20，**任务失败**

## 🏆 游戏结束

### 胜利条件
在时间标记耗尽前完成全部4项任务 → **任务成功！**

### 失败条件
- 时间标记归零仍有未完成任务 → 时间耗尽，任务失败
- 警戒值达到20 → 身份暴露，任务失败
- 所有特工生命归零 → 全员阵亡，任务失败

### 评分
| 完成情况 | 评级 |
|:---|:---|
| 4项任务全部完成，剩余时间≥8 | 🌟🌟🌟🌟🌟 传奇特工 |
| 4项任务全部完成，剩余时间≥4 | 🌟🌟🌟🌟 精英特工 |
| 4项任务全部完成，剩余时间<4 | 🌟🌟🌟 合格特工 |
| 仅完成3项任务 | 🌟🌟 实习特工 |
| 完成不足3项 | 🌟 任务失败 |

## 🎖️ 进阶规则

### 极难模式
- 起始时间减为20
- 敌人命中率变为2+
- 危机卡触发频率加倍

### 幽灵协议模式
- 随机1名玩家秘密成为内鬼
- 内鬼可暗中破坏任务（丢弃装备、误导行动）
- 其他玩家可在行动阶段花2行动点指认内鬼
- 若成功指认，内鬼出局但少1名队友
- 若错误指认，警戒值+5

---

## 💡 策略提示

1. **分工明确**：让每个特工做擅长的事，黑客负责破解，潜入者负责侦察
2. **警戒控制**：警戒值是最容易忽视的失败条件，时刻注意
3. **装备分配**：消音手枪给潜入者，解码器给黑客，合理分配
4. **配合行动**：2人配合比单独行动效率高，优先完成配合条件
5. **留有余量**：不要把时间用尽，每个任务留1-2回合余量应对意外

---

## 📐 变体规则

- **新手模式**：起始时间30，敌人命中率4+，危机卡减半
- **竞速模式**：记录完成时间，下次挑战争取更快
- **单人模式**：操控3名特工，敌人行动简化
