# 轮回地下城 (Loop Dungeon)

## 1. 游戏概述

**轮回地下城**是一款合作类卡牌桌游，灵感来源于当下流行的Roguelike地牢游戏（如《挺进地牢》、《以撒的结合》）。

在游戏中，1-4名玩家扮演被困在时间轮回中的冒险者。每当局势不利（全员阵亡）时，时间就会倒流回地牢入口——但你会保留上一次轮回中收集的「记忆碎片」，用来购买永久升级。目标是在有限的轮回次数内，打通5层地牢，击败最终Boss「轮回守护者」，打破诅咒！

- **适合人群**：喜欢合作游戏、地牢探索、Roguelike元素的玩家；适合家庭聚会或桌游之夜
- **核心体验**：合作决策、风险管理、渐进式成长、一次次"再来一次"的爽感

## 6. 游戏规则

### 准备阶段

1. **选择玩家人数**：1-4人，每人选择一个角色（可用不同颜色棋子区分）
2. **洗牌**：将房间事件卡、战利品卡分别洗匀，面朝下放置
3. **设置地图**：打印地图或使用ASCII地图，将棋子放在1F入口
4. **初始化角色**：
   - 每位玩家：HP 10，金币 2，行动点 2/回合
   - 发2张战利品卡给每位玩家（基础装备）
5. **设置轮回计数器**：初始为1，最多5次轮回

### 游戏流程（每回合）

每回合，所有玩家依次行动，每位玩家有**2个行动点（AP）**，可选择：

| 行动 | 消耗AP | 说明 |
| :--- | :--- | :--- |
| 探索房间 | 1 AP | 抽取1张房间事件卡并执行效果 |
| 移动房间 | 1 AP | 前进到下一个房间（同层或下层） |
| 装备物品 | 1 AP | 装备1件战利品 |
| 使用消耗品 | 1 AP | 使用1张消耗品卡 |
| 使用技能 | 1 AP | 使用1张技能卡 |
| 购买升级 | 1 AP | 在记忆祭坛花费记忆碎片 |
| 休息 | 0 AP | 跳过本回合（恢复1HP） |

#### 战斗规则

当探索到怪物房时，触发战斗：

1. **先攻判定**：玩家骰子 ≥ 4 则先攻；否则怪物先攻
2. **玩家攻击**：掷骰子 + 武器攻击加成 vs 怪物HP
3. **怪物攻击**：掷骰子 + 怪物攻击 vs 玩家HP（减去防具防御）
4. **战斗持续**直到一方HP降为0
5. **胜利奖励**：获得金币和可能的战利品卡

#### 层数推进

- 探索完本层所有5个房间后，可进入下一层
- 第5层第1房为Boss房，击败Boss即获胜

#### 轮回机制（阵亡处理）

当所有玩家HP降为0时：
1. 当前轮回结束
2. 玩家保留**记忆碎片**（用于购买永久升级）
3. 其他所有装备和金币丢失
4. 轮回计数器+1
5. 在新轮回中，玩家从1F入口重新开始，HP恢复为10+升级加成

#### 胜利条件

- **获胜**：击败5F的Boss「轮回守护者」
- **失败**：轮回计数器达到5时仍未击败Boss

### 单人模式规则

- 玩家有**3个行动点/回合**（补偿无队友）
- 可同时控制2个角色（选2个棋子）
- 其他规则相同

## 7. 完整游戏流程示例

### 示例1：第1轮回，第1回合

**场景**：玩家A（HP 10，金币2，铁剑+1攻击，皮甲+1防御）在1F入口

1. **玩家A行动**：消耗1 AP **探索房间** → 抽取到「🟢 金币宝箱」
   - 效果：获得3金币（现在共有5金币）
   - 剩余AP：1

2. **玩家A继续行动**：消耗1 AP **移动房间** → 移动到1F第2房
   - 剩余AP：0，回合结束

3. **玩家B行动**（如果多人）：类似操作...

**第1轮回第2回合**

1. **玩家A行动**：消耗1 AP **探索房间** → 抽取到「⚔️ 史莱姆（HP 3，攻击1）」
   - **战斗开始！**
   - 先攻判定：玩家A掷骰子 = 5（≥4，先攻✅）
   - 玩家A攻击：骰子(5) + 铁剑(1) = 6点伤害 → 史莱姆HP 3-6 = -3（击败！）
   - 战斗胜利！获得3金币（现在共有8金币）

### 示例2：Boss战

**场景**：玩家们到达5F第1房，面对Boss「轮回守护者」（HP 20，攻击3，每回合恢复1HP）

1. **准备阶段**：玩家A（HP 8，火焰剑+2，铁甲+2），玩家B（HP 6，雷电斧+3，皮甲+1）
2. **Boss先攻**（骰子=6 vs 玩家A骰子=3）
3. **Boss攻击玩家A**：骰子(6) + 攻击(3) = 9 vs 玩家A防御(2) → 受到7点伤害
   - 玩家A HP：8-7 = 1 ⚠️
4. **玩家A行动**：使用「治疗药水」恢复3HP → HP变为4
5. **玩家B行动**：攻击Boss → 骰子(4) + 雷电斧(3) = 7点伤害
   - Boss HP：20-7 = 13，然后Boss恢复1HP → 14HP
6. **持续战斗...** 直到Boss被击败或玩家全员阵亡

## 8. 新手策略提示

### 💡 给新玩家的建议

1. **优先探索宝箱房**：积累金币和装备是通关的基础
2. **不要急着下楼**：当前层的装备足够后再进入下一层
3. **合理分配AP**：不要在第一回合用完所有AP，留一些应对突发情况
4. **团队合作**：多人游戏时，让装备好的玩家吸引火力，其他玩家输出
5. **记忆碎片优先升级HP**：更多HP意味着能承受更多失误
6. **保留消耗品**：治疗药水留到Boss战再用
7. **利用陷阱房**：如果有可能，用消耗品解除陷阱，避免受伤

### 🎯 进阶技巧

1. **Build构建**：专注一种武器类型（如火焰流、雷电流）以最大化伤害
2. **记忆碎片投资**：早期投资「额外行动点」性价比极高
3. **Boss战准备**：进入5F前确保每人至少有1件+2武器和+1防具
4. **风险管控**：HP低于3时优先考虑休息或回城（如果规则允许）

## 9. 变体规则

### 🔥 困难模式

- 怪物HP x1.5，攻击+1
- 轮回次数限制改为3次
- 初始金币减少为1

- Boss HP减半
- 初始金币+2

- 房间事件卡不立即揭示，玩家需要主动探索才能知道房间类型
- 增加未知感和紧张感

## 10. 常见问题

### Q1：如果牌堆用完了怎么办？
**A**：将已使用的卡牌洗匀，重新组成牌堆。

### Q2：多人游戏时，玩家之间可以交换装备吗？
**A**：可以！这是合作游戏，鼓励玩家之间交易装备和消耗品。

### Q3：记忆碎片在哪里使用？
**A**：在「记忆祭坛」事件房，或者在轮回之间的休整阶段（家用规则）。

### Q4：Boss战中所有玩家都必须参与吗？
**A**：是的，Boss战是合作战斗，所有存活玩家共同对抗Boss。

### Q5：如果玩家中途阵亡（HP=0）但队伍还没全灭怎么办？
**A**：阵亡玩家跳过回合，直到被队友用「治愈术」或「复活卷轴」复活。如果全员阵亡，则触发轮回。

### Q6：单人模式会不会太难？
**A**：不会！单人模式给了3个AP/回合，而且可以控制2个角色。如果还是太难，可以使用「不死鸟」记忆碎片（第一次阵亡复活）。

### Q7：一局游戏大概多长时间？
**A**：正常难度下，45-60分钟。快速模式约20分钟。熟练后时间会缩短。

### Q8：打印卡牌用什么纸最好？
**A**：推荐200g卡纸，打印后裁剪，可以塑封增加耐用性。如果没有卡纸，普通A4纸也可以，但容易损坏。
