# 星穹传说 (Star Rail Legends)

## 🎮 游戏概述

《星穹传说》是一款灵感来源于热门银河冒险题材的合作策略桌游。玩家将扮演星穹列车的乘员，驾驶列车穿越浩瀚宇宙，在不同的星球上展开冒险，揭开「星核」背后的真相，拯救被星核危机笼罩的世界！

本游戏结合了路线推进、资源管理、回合制战斗和团队配合机制。2-4名玩家必须密切合作，合理分配资源，制定战术策略，才能战胜强大的敌人，完成星际之旅。

适合喜欢合作游戏、科幻题材和策略思考的玩家，游戏时长短、节奏明快，非常适合朋友聚会或家庭娱乐。

## 🎲 游戏规则

### 准备阶段

1. **选择角色**：每位玩家选择一张角色卡，将角色棋子放在起点
2. **布置地图**：将列车轨道地图放在中央，把6张星球卡洗混后放在轨道对应位置
3. **初始资源**：每位玩家获得3点能量、1张随机技能卡、1张物品卡
4. **设置列车**：将列车棋子放在起始星球（雅利洛-VI）
5. **洗混卡组**：将敌人卡、事件卡分别洗混，面朝下放置

### 游戏流程（每个回合）

玩家轮流进行，每回合包含以下步骤：

#### 阶段1：列车移动
- 投掷骰子，列车向前移动1-6步
- 如果抵达新的星球，所有玩家下机探索（进入星球探索阶段）
- 如果停留在已有星球，玩家可以选择继续探索或前往下一站

#### 阶段2：星球探索（可选）
在星球探索地图上，玩家可以：
- **移动**：消耗1点能量，移动1格
- **搜索**：停留在资源点可抽取1张物品卡
- **战斗**：进入战斗区需抽取1张敌人卡并进行战斗
- **任务**：抵达任务点可尝试完成任务（需满足特定条件）

#### 阶段3：战斗（如遇到敌人）
战斗按回合进行：
1. 玩家行动顺序：按角色卡顺序轮流行动
2. 玩家选择：攻击敌人 / 使用技能 / 使用物品 / 防御
3. 敌人行动：敌人攻击后，下一位玩家继续
4. 战斗结束：敌人生命值归零则玩家获胜，获得奖励

#### 阶段4：事件触发（可选）
在星球探索时，玩家可以抽取1张事件卡，触发星球事件。

#### 阶段5：回合结束
- 检查是否收集到足够的星核碎片（5个）
- 如果收集完毕，触发最终战斗
- 击败最终Boss则游戏胜利！

### 战斗规则详解

#### 攻击流程
1. **宣布攻击**：玩家选择1个目标敌人
2. **计算伤害**：角色攻击力 + 武器加成 - 敌人护甲 = 实际伤害
3. **应用伤害**：敌人生命值减少相应数值
4. **反击**：如果敌人未死亡，立即攻击该玩家（敌人攻击力 - 玩家护甲 = 受到的伤害）

#### 合作战斗
- 玩家可以协同攻击同一个敌人
- 玩家可以使用技能帮助队友（治疗、护盾、增益）
- 敌人优先攻击生命值最低的玩家

### 胜利条件

| 胜利条件 | 说明 |
| :--- | :--- |
| **收集星核碎片** | 收集5个星核碎片，触发最终战斗 |
| **击败终焉之兽** | 在最终战斗中击败Boss，游戏胜利！ |
| **全员阵亡** | 所有玩家生命值归零，游戏失败 |
| **回合上限** | 达到20回合未完成目标，游戏失败 |

## 📖 完整游戏流程示例

### 示例1：第一轮探索

**准备**：
- 玩家A选择「开拓者」（生命值10，攻击力2）
- 玩家B选择「姬子」（生命值8，攻击力3）
- 列车抵达雅利洛-VI，两人下机探索

**回合1（玩家A）**：
1. 消耗1点能量，移动至资源点🎒
2. 获得1张「生命药水」物品卡
3. 回合结束

**回合2（玩家B）**：
1. 消耗1点能量，移动至战斗区⚔️（实际雅利洛-VI无战斗区，改为移动至任务点📍）
2. 完成任务：投掷骰子点数≥3，成功！获得1个星核碎片💎
3. 回合结束

### 示例2：战斗流程

**遭遇敌人**：裂界造物（生命值5，攻击力1）

**回合1（玩家A）**：
1. 选择攻击，造成2点伤害（开拓者攻击力2）
2. 裂界造物生命值剩余3
3. 裂界造物反击，造成0点伤害（攻击力1 - 开拓者护盾1 = 0）
4. 玩家A回合结束

**回合2（玩家B）**：
1. 使用技能「炎击」，消耗2点能量，造成4点伤害（攻击力3 + 火焰伤害1）
2. 裂界造物生命值归零，战斗胜利！
3. 奖励：获得1张随机武器卡

### 示例3：合作防御

**遭遇强敌**：星核猎手（生命值12，攻击力3，每回合攻击两次）

**玩家A**（开拓者，生命值10，护甲2）：
1. 使用技能「护盾」，为两人各增加2点护甲
2. 攻击星核猎手，造成2点伤害

**玩家B**（姬子，生命值8，护甲1）：
1. 使用物品「炸弹」💣，造成3点范围伤害
2. 星核猎手生命值剩余7
3. 星核猎手攻击：对玩家A造成1点伤害（攻击力3 - 护甲2 - 护盾2 = 1），对玩家B造成0点伤害（攻击力3 - 护甲1 - 护盾2 = 0）

**下一回合**，两人继续合作，最终击败星核猎手！

## 💡 新手策略提示

1. **合理分配角色**：开拓者适合新手（生存能力强），姬子适合进攻，瓦尔特适合控制，三月七适合辅助
2. **优先收集能量**：能量是使用技能的关键，前期多收集能量标记
3. **团队配合至上**：合作游戏的核心是配合，不要只顾自己输出
4. **护甲很重要**：面对强敌时，先加护甲再进攻
5. **保存技能卡**：关键时刻使用技能卡，不要浪费在低威胁敌人身上
6. **探索优先级**：优先探索任务点📍，收集星核碎片是胜利关键
7. **资源管理**：生命药水留到生命值低于3时使用，能量饮料在需要时使用

## 🔄 变体规则

### 简单模式（新手推荐）
- 敌人生命值减半
- 每位玩家额外获得1张技能卡
- 星核碎片只需收集3个

### 困难模式（老手挑战）
- 敌人攻击力和生命值增加50%
- 每回合结束后随机抽取1张事件卡（可能增加难度）
- 星核碎片需要收集7个

### 竞速模式
- 设置计时器，60分钟内完成游戏
- 在规定时间内胜利则记录分数，分数越高评价越好

### 单人模式
- 1名玩家控制2个角色
- 两个角色共享回合，但可以分别行动
- 适合练习和熟悉游戏

## ❓ 常见问题

**Q：如果没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：卡牌用完了怎么办？**
A：可以将弃牌堆洗混后重新使用。

**Q：玩家可以在自己的回合移动其他玩家吗？**
A：不可以，每位玩家只能移动自己的棋子。但可以使用技能卡帮助队友移动。

**Q：如果所有玩家都阵亡了怎么办？**
A：游戏失败，可以重新开始。建议先玩简单模式熟悉游戏。

**Q：能量点数有上限吗？**
A：有，每位玩家最多持有10点能量。超过上限的能量作废。

**Q：可以同时装备多件武器/防具吗？**
A：每位玩家只能装备1件武器和1件防具。更换时需丢弃原有装备。

**Q：最终Boss战怎么触发？**
A：收集到5个星核碎片后，列车会抵达「量子星」，在那里触发最终战斗。
