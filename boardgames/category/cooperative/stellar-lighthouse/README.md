# 星际灯塔守护者

## 1. 游戏概述

在遥远的未来，星际航行已成为常态。但在未知的星域中，无数飞船因导航失灵、能量耗尽或通讯中断而迷失在茫茫宇宙中。星际灯塔是这些迷失者的唯一希望——它们发射强大的导航信号，指引飞船安全返航。

**玩家角色**：你们是星际灯塔的守护者，负责维护这座古老的太空设施。

**游戏目标**：在所有迷失飞船抵达灯塔前，维护灯塔设施、修复故障、抵御黑暗能量的侵蚀。这是一款**合作游戏**，所有玩家共同对抗宇宙的残酷挑战。

**适合人群**：喜欢团队合作、紧张刺激的玩家；适合家庭聚会或朋友游戏夜。

## 6. 游戏规则

### 准备阶段

1. **放置地图**：将游戏地图平铺在桌面中央。
2. **分配角色**：每位玩家选择一个角色卡，放在灯塔核心区。
3. **放置飞船**：随机抽取6艘飞船（新手局）或12艘（标准局），放在地图外围黑暗区。
4. **设置能量**：将能量标记放在"10"位置（灯塔初始能量）。
5. **洗混卡牌**：
   - 将能量卡、零件卡、星图卡分别洗混，面朝下放置。
   - 将事件卡和黑暗卡洗混，放在地图旁。
6. **决定先手**：投掷骰子，点数最大者先手。

### 游戏流程

**每个回合包含以下步骤**：

#### 步骤1：事件阶段（必须）
- 翻开一张事件卡，按照卡面说明执行。
- 如果是黑暗卡，按照卡面负面效果影响所有玩家。

#### 步骤2：行动阶段（可选）
每位玩家有**2个行动点**，可以：
- **移动**：消耗1点，移动到相邻区域。
- **采集资源**：消耗1点，在当前区域抽取1张对应资源卡。
  - 能量站 → 能量卡
  - 零件库 → 零件卡
  - 星图室 → 星图卡
- **使用资源**：消耗1点，使用手牌中的资源卡。
- **修复故障**：消耗1点，移除一个故障标记（需消耗1张零件卡）。
- **引导飞船**：消耗1点，使用1张星图卡，将一艘飞船向灯塔移动1格。
- **角色能力**：消耗1点，激活角色卡上的特殊能力。

#### 步骤3：检查阶段（必须）
- **检查灯塔能量**：如果能量≤0，游戏失败。
- **检查飞船位置**：如果有飞船抵达灯塔核心区，玩家共同获得奖励。
- **检查故障数量**：如果故障标记≥10个，灯塔崩溃，游戏失败。

#### 步骤4：结束阶段（必须）
- 将能量标记-1（灯塔持续消耗能量）。
- 传递先手令牌给下一位玩家。

### 胜利条件

**共同胜利**：在所有飞船抵达灯塔前，成功引导**至少6艘飞船**抵达灯塔核心区。

**失败条件**（任一触发即失败）：
- 灯塔能量降至0或以下。
- 故障标记达到10个。
- 所有飞船都迷失（超出地图边界）。

## 7. 完整游戏流程示例

### 示例1：单人模式（玩家A控制工程师角色）

**回合1**：
1. **事件阶段**：翻开事件卡"能量波动"，丢失1张能量卡（手牌不足，灯塔能量-1）。
2. **行动阶段**：
   - 行动1：移动到能量站，抽取1张能量卡。
   - 行动2：使用能量卡，为灯塔充能（能量标记+2）。
3. **检查阶段**：灯塔能量=11，安全。
4. **结束阶段**：能量标记-1（=10），传递先手令牌（单人模式忽略）。

**回合2**：
1. **事件阶段**：翻开黑暗卡"系统故障"，跳过下一次行动（在工程师卡上放置一个标记）。
2. **行动阶段**：
   - 因系统故障，失去行动1。
   - 行动2：使用零件卡，修复系统故障（移除标记）。
3. **检查阶段**：故障标记=0，安全。
4. **结束阶段**：能量标记-1（=9）。

### 示例2：多人模式（玩家A-导航员，玩家B-能源师）

**回合3**：
1. **事件阶段**：翻开事件卡"迷失飞船"，新增1艘飞船到地图边缘。
2. **玩家A行动阶段**：
   - 行动1：使用星图卡，引导飞船#3向灯塔移动1格。
   - 行动2：移动到星图室，抽取1张星图卡。
3. **玩家B行动阶段**：
   - 行动1：使用能量卡，为灯塔充能（能量标记+2）。
   - 行动2：移动到能量站，抽取1张能量卡。
4. **检查阶段**：飞船#3距离灯塔只剩2格，灯塔能量=11。
5. **结束阶段**：能量标记-1（=10）。

**回合4**：
1. **事件阶段**：无事件（翻开"平静时刻"）。
2. **玩家A行动阶段**：
   - 行动1：使用星图卡，引导飞船#3抵达灯塔！所有玩家获得1张奖励卡。
   - 行动2：使用角色能力"精准导航"，免费引导飞船#5移动1格。
3. **玩家B行动阶段**：
   - 行动1：修复一个故障标记（消耗1张零件卡）。
   - 行动2：使用角色能力"能量共享"，给玩家A1张能量卡。
4. **检查阶段**：已引导2艘飞船，还需4艘。
5. **结束阶段**：能量标记-1（=9）。

## 8. 新手策略提示

### 优先事项
1. **保持能量≥5**：能量是生命线，低于5会触发警报。
2. **快速修复故障**：故障标记越多，事件卡越负面。
3. **合理分配角色**：
   - 工程师 → 负责修复故障
   - 导航员 → 负责引导飞船
   - 能源师 → 负责采集能量
   - 通讯官 → 负责资源交易
   - 科学家 → 负责升级灯塔

### 资源管理
- **不要贪心**：手牌上限为5张，及时使用。
- **优先使用星图卡**：引导飞船是胜利条件。
- **保留1张能量卡**：应对黑暗卡的突发消耗。

### 团队合作
- **集中在灯塔附近**：减少移动浪费。
- **明确分工**：一人负责采集，一人负责引导。
- **使用角色能力**：每个角色能力每回合可使用一次。

### 避免常见错误
- ❌ 忽略能量消耗（每回合-1）。
- ❌ 手牌过多导致无法抽取新卡。
- ❌ 分散行动，无法互相支援。

## 10. 常见问题

**Q1：如果手牌满了怎么办？**
A：必须丢弃至5张（选择丢弃哪些卡）。

**Q2：可以多个人在同一区域吗？**
A：可以，合作游戏鼓励聚集。

**Q3：事件卡用完了怎么办？**
A：重新洗混丢弃堆，继续游戏。

**Q4：飞船可以移动多格吗？**
A：每使用1张星图卡，移动1格。可以连续使用多张。

**Q5：灯塔能量可以为负数吗？**
A：不可以，能量≤0立即失败。

**Q6：故障标记可以移除吗？**
A：可以，使用1张零件卡并消耗1个行动点。

**Q7：单人模式如何控制多个角色？**
A：每回合可以选择一个角色行动，轮流使用。

**Q8：打印出来的卡牌太小怎么办？**
A：在printables.html中选择"A4纸打印"，调整缩放比例至100%。
