# 末日种子库 (Doomsday Vault)

## 🎮 游戏概述

公元 2048 年，一场全球性的生态连锁崩溃摧毁了地表绝大部分生态系统。人类最后的希望，埋藏在斯瓦尔巴群岛永冻层深处——**全球种子库（Global Seed Vault）**。

玩家扮演种子库的**守护者团队**，必须在生命支持系统逐步衰竭的极限环境下，通过远征探险收集濒危物种种子、维护种子库运转、应对各类危机事件。所有玩家**通力合作**，目标是在系统完全崩溃前，成功保存足够多的植物物种，为人类文明的复兴保留最后的火种。

这是一款融合**资源管理、危机应对、角色分工、手牌管理**的合作策略桌游，适合喜欢紧张合作体验的玩家。

### 移动规则

- 从基地出发，沿连线移动到相邻地点（每次移动消耗 **1 AP**）
- 相邻地点之间可以双向通行
- 玩家同一回合内可以多次移动（只要 AP 够用）
- 在远征地点执行"搜索种子"操作后，回合结束时自动返回基地（不消耗 AP）

### 四、事件卡（命运之轮）

每回合结束后，团队必须从事件牌堆顶端抽取 **1张事件卡** 并立即执行其效果。事件分为四类：

#### 🔴 危机事件（14张）— 系统损害型

| :--- | :--- | :--- |
| **太阳风暴** | ☀️⚡ | 电力系统 -3 |

| **CO₂ 泄漏** | ☠️🫁 | 氧气循环 -2；所有玩家 -1 HP |

## 🎮 游戏规则

### 一、准备阶段（约5分钟）

**步骤 1：布置面板**
- 将种子库主面板放在桌面中央
- 把3条系统轨道（⚡电力 / ❄️温度 / 🫁氧气）的指示物放到初始位置（电力10、温度8、氧氣9）
- 准备好已保存物种库区域（25个空格）

**步骤 2：放置远征地图**
- 将远征地图放在主面板旁边
- 每位玩家将自己的棋子放在🏠基地营地位置

**步骤 3：洗牌与分组**
- 将所有**种子卡**洗混，形成种子牌堆（面朝下放在一旁）
- 将所有**资源卡**按种类分成4堆（⚡🍖💊🔧）
- 将**设备卡**洗混形成设备牌堆
- 将**事件卡**洗混形成事件牌堆（根据人数裁剪，见物料清单表）

**步骤 4：分发初始资源**
- 公共资源库初始配置：⚡×6、🍖×6、💊×4、🔧×4
- 每位玩家个人初始手持：
  - ⚡×2、🍖×2
  - 随机抽取 **3张种子卡**
- 将 **3张设备卡**正面朝上放置（作为可购买/获取的设备展示区），其余面朝下

**步骤 5：选择角色**
- 将5张角色卡正面朝上
- 按顺时针顺序依次选择角色（也可以随机分配）
- 将角色卡放在自己面前，确认 HP 和技能

**步骤 6：确定起始玩家**
- 掷骰子或协商决定第一位玩家
- 按顺时针方向进行回合

---

### 二、游戏流程（每个回合的结构）

每位玩家的回合依次执行以下阶段：

#### 阶段 1：重置行动点
- 回合开始时，玩家获得 **4 个行动点（AP）**（野外探险家因被动技能获得 5 AP）
- 将 AP 指示物摆在面前

#### 阶段 2：执行行动（按任意顺序分配 AP）

可用行动及其 AP 消耗：

| 行动 | 消耗AP | 详细说明 |
| :--- | :--- | :--- |
| **移动** | 1 AP/次 | 移动到相邻地点（探险家仅需0 AP） |
| **远征搜索** | 2 AP | 在当前远征地点搜索，获得 1-3 张种子卡（掷1个D6：1-2=1张，3-4=2张，5-6=3张）。同时触发该地点的**特殊风险** |
| **发电** | 2 AP | 获得能量 ⚡×2（仅在基地可用） |
| **配发口粮** | 1 AP | 获得食物 🍖×1（仅在基地可用） |
| **调配医疗** | 2 AP | 获得医疗包 💊×1（仅在基地可用） |
| **搜索零件** | 2 AP | 获得零件 🔧×1（仅在基地可用） |
| **入库保存** | 1 AP+资源 | 消耗对应资源，将一张手中的种子卡放入已保存物种库 |
| **修复系统** | 1-3 AP | 消耗 🔧×1-2，使任一系统回复对应点数（1AP+🔧1=+1, 2AP+🔧2=+3, 3AP+🔧3=+5）。首席工程师效果+1 |
| **治疗** | 1 AP+💊×1 | 自身或其他玩家恢复 2 HP |
| **传递资源** | 1 AP | 向另一位玩家或公共资源库给出/收取任意数量的同一种类资源 |
| **购买/安装设备** | 变动 | 根据设备卡上的费用支付资源，将设备放入公用区 |
| **休息** | 1 AP | 在基地休息：恢复1 HP，无其他效果 |

#### 阶段 3：回合结束 — 抽取事件卡

- 当前玩家回合结束后，**从事件牌堆顶抽取1张事件卡**
- 大声朗读并执行效果
- 使用过的事件卡放入弃牌堆

#### 阶段 4：检查系统与状态

检查以下条件（如果满足则触发相应效果）：

| 条件 | 效果 |
| :--- | :--- |
| 电力 ≤ 3 | 下一位玩家所有行动 AP 消耗 +1 |
| 温度 ≥ 10（过热）| 已保存物种库中的稀有+种子开始退化风险 |
| 氧气 ≤ 3 | 所有玩家在回合开始时 -1 HP |
| 事件牌堆耗尽 | 进入最终倒计时：只剩3回合 |

---

### 三、胜利条件

**主要胜利**：当已保存物种库中的种子数量 **≥ 25 种** 时，团队立即获胜！

**计算方式**：
- 普通种子每张计为 1 种
- 稀有种子每张计为 1 种
- 史诗种子每张计为 1 种
- 传说种子每张计为 1 种
- 同名种子重复入库只算 **1 次**

> 即你需要收集 **25 个不同名称的物种**，而非25张卡。这鼓励大家去不同的远征地点！

### 四、失败条件

以下任一条件满足时，团队**立即失败**：

1. **双重崩溃**：**2个或更多系统同时归零（值为0）**
2. **全军覆没**：**所有玩家 HP 都归零**（全部进入重伤状态）
3. **物种灭绝**：已保存物种库中的卡片因各种原因被全部清除（极端情况）
4. **时间耗尽**：事件牌堆用完且经过最终3回合后仍未达成25种目标

## 💡 新手策略提示

### 角色配合建议

| 组合 | 策略思路 |
| :--- | :--- |
| 工程师 + 植物学家 | "生产流水线"：一人负责外出采集+修系统，一人专职入库 |
| 后勤官 + 任意角色 | 后勤官是团队的资源引擎，应优先保障队友 |
| 探险家 + 安全主管 | "无畏组合"：几乎无伤跑遍所有远征点 |
| 单人游戏（1人模式） | 推荐**植物学家**或**后勤官**，单人模式下 AP 更珍贵 |

### 通用策略

1. **前 3-5 回合：积累期**
   - 不要急于入库——先积累资源和多样种子
   - 至少安排一名玩家负责维持系统不低于 5
   - 让探险家尽早跑一圈低危险地点（西伯利亚→亚马逊→环礁）

2. **中间回合：平衡期**
   - 开始批量入库普通和稀有种子
   - 关注事件牌堆余量（大约一半用完时要加速）
   - 购买关键永久设备（全地形车优先级最高）

3. **最后 5 回合：冲刺期**
   - 集中所有资源入库
   - 保留专属能力作为最后手段
   - 如果系统危急，宁可不外出也要保住系统

### 系统优先级参考

| 情况 | 最优先保护的系统 | 原因 |
| :--- | :--- | :--- |
| 多人在外远征 | 电力 | 低电力导致所有操作变贵，远征效率骤降 |
| 已有大量稀有+种子 | 温度 | 温度失控会直接丢失高分种子 |
| 全员 HP 较低 | 氧气 | 缺氧导致持续掉血，可能全军覆没 |

### 设备购买优先级

1. **🚙 全地形车**（⚡3+🔧2）— 永久减少移动 AP，整局受益
2. **☀️ 太阳能板**（事件获得）— 每回合自动+1电力，极大缓解压力
3. **🏗️ 种子加速柜**（⚡5+🔧4）— 后期冲刺神器
4. **🤖 医疗机器人**（⚡3+💊2）— 多人游戏中持续回血非常强
5. **📡 无人机侦察**（⚡2+🔧1）— 信息优势，避免白跑一趟

## 🔄 变体规则

### 变体一：困难模式「末日加剧」

- 所有系统初始值 -2（电力8 / 温度6 / 氧气7）
- 事件牌堆中加入全部4张"选择事件"
- 每次远征搜索结果 -1（D6：1-3=1张，4-5=2张，6=3张）
- 胜利条件提高：需保存 **30 种**物种

### 变体二：竞速模式「时间竞赛」

- 设置真实计时器 **50 分钟**
- 时间到时立刻结算（以当前物种数量判断胜负）
- 事件牌堆不再限制回合数
- 适合经验丰富的团队挑战

### 变体三：剧情模式「序章：崩塌前夜」

- 游戏开始时有 **15 个回合的"缓冲期"**：此期间不抽事件卡（模拟灾难初期系统尚稳定）
- 缓冲期过后，事件卡改为**每回合抽 2 张**
- 适合新手教学，让玩家先熟悉行动系统

### 变体四：角色成长模式

- 每当一名玩家成功入库 5 张种子卡，该角色**升级**（获得一个增强版被动技能）
- 首席工程师升级：修复效果+1变为+2
- 植物学家升级：入库资源需求-1变为-2（最低为0）
- 安全主管升级：免疫范围扩大到包括事件卡的 HP 伤害
- 后勤官升级：交换比 2:1 变为 1:1
- 探险家升级：移动和搜索都变为 0 AP 消耗（即免费）

### 变体五：单人规则「孤独守护者」

1. 只操控 1 名角色（推荐**植物学家**或**后勤官**）
2. 初始资源加倍：⚡×4、🍖×4、种子卡×5
3. 初始 HP +2（即 6/7 HP）
4. 每回合获得 **5 AP**（而不是4 AP）
5. 可以在任何时候花费 2 AP 执行"副职行动"（模拟另一个虚拟角色的基础行动）
6. 胜利条件降低：保存 **18 种**物种即可获胜
7. 失败条件不变

## ❓ 常见问题

**Q：种子卡有上限吗？我手里可以攒多少张？**
A：手牌上限为 **8 张种子卡**。如果你在手牌已达8张时又获得了新种子，必须立即选择：将1张旧种子放入弃牌堆 或 立即花 AP 入库腾出空间。

**Q：公共资源库里的资源怎么用？**
A：任何玩家在执行需要消耗资源的行动时，都可以选择从**个人手牌**或**公共资源库**中支付。公共资源库是团队共享的，由传递资源行动来补充。

**Q：如果电力系统归零了会怎样？不是立即失败吗？**
A：单个系统归零**不会**立即失败，但会产生严重负面效果。只有 **2个或以上系统同时归零**才判负。不过电力归零后所有行动+1 AP，基本上等于半瘫痪状态，所以一定要避免！

**Q：入库时资源不够怎么办？**
A：你可以选择**部分支付**（例如入库咖啡需要⚡×1，你只有⚡×0就无法入库）。资源不足时该种子只能留在手牌等待下一回合。你也可以和其他玩家交易资源（后勤官最擅长这个）。

**Q：远征搜索得到的种子是公开的还是保密的？**
A：**公开的**！搜索获得的种子卡必须正面朝上展示给所有玩家看，然后收入手牌。这是合作游戏，信息公开有助于团队决策谁应该入库哪种种子。

**Q：同名种子重复了怎么办？比如拿到了两张水稻？**
A：第二张水稻可以作为**保险**（防止第一张被事件卡破坏），或者用来触发某些设备效果。但计入物种总数时只算 **1 种**。所以拿到重复种子后，优先去寻找新物种！

**Q：事件卡中的"从已保存物种库中移除"怎么办？那个种子就彻底没了？**
A：是的，放入弃牌堆。这就是为什么需要 **25 种** 而不是更少——你必须留有余量来应对这种损失。建议优先入库普通种子（容易补充），把稀有以上的种子留到后期。

**Q：设备的"永久"是什么意思？**
A：永久设备一旦购买/安装就放在公共区域，**所有玩家都可以受益**。它们不会被事件卡破坏（除非特定事件说明）。一次性设备用完后放入弃牌堆。

**Q：1人玩这个游戏体验如何？**
A：请参考"变体五：单人规则"。单人模式下你拥有更多初始资源和 AP，以及降低的胜利条件，体验仍然很有趣！核心乐趣在于 AP 的紧张管理和系统的走钢丝平衡。

**Q：一局大概多少回合？**
A：24张事件卡 = 约 24 回合（不含缓冲期变体）。实际游戏通常在 **15-22 回合**内分出胜负（提前达到25种胜利或系统崩溃失败）。
