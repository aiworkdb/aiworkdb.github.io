# 怪物厨师 (Monster Chef)

## 🎮 游戏概述

《怪物厨师》是一款合作类桌游，灵感来源于热门动漫《迷宫饭》(Dungeon Meshi)。

玩家扮演一群误入地下城的冒险者，随身携带的粮食在战斗中丢失。为了生存并深入地下城营救同伴，你们必须——**狩猎怪物、采集食材、现场烹饪、大快朵颐！**

与传统地下城游戏不同，这款游戏的核心乐趣在于**烹饪系统**：不同的怪物食材组合可以烹饪出不同效果的料理，有的恢复体力，有的赋予特殊能力。玩家需要合理分工、巧妙搭配食材，才能战胜地下城的层层挑战！

适合喜欢合作游戏、烹饪题材，以及《迷宫饭》粉丝的玩家。

## 🗺️ 游戏地图详解

地下城共有 **5层**，每层是一个独立的线性路径地图。玩家需逐层探索，通关第5层 Boss 即为胜利。

### 地图布局概览（第1层示例）

```
🟢 ➡️ 🟡 ➡️ 🟣 ➡️ 🔴 ➡️ 🟠 ➡️ 🔵 ➡️ ⭐
```

各层长度不同，越深层越长：

| 层数 | 节点数 | 说明 |
| :--- | :--- | :--- |
| 第1层 | 7个节点 | 新手层，怪物较弱 |
| 第2层 | 8个节点 | 出现中毒陷阱 |
| 第3层 | 9个节点 | 出现稀有食材 |
| 第4层 | 10个节点 | 强大怪物，需团队合作 |
| 第5层 | Boss战 | 红龙登场！ |

### 地图特点

1. **路径设计**：每层路径为线性，玩家按顺序依次探索节点
2. **资源分布**：食材点均匀分布在各层，鼓励玩家积极采集
3. **怪物强度**：越深层怪物战斗力越强，需要更好的料理Buff支持
4. **合作要素**：玩家可互相赠送食材、协助战斗、分享料理

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将第1层地图放在桌面中央
2. **选择棋子**：每位玩家选择一个颜色的棋子，放在第1层起点🟢
3. **设置体力**：将共享体力标记为 **20点**（写在纸上或用硬币表示）
4. **分发起始物资**：
   - 每位玩家获得 **2张食材卡**（随机）
   - 每位玩家获得 **1张厨具卡**（菜刀🔪）
   - 洗混怪物卡堆、料理卡堆、事件卡堆、陷阱卡堆
5. **决定先手**：掷骰子，点数最大者先手，按顺时针进行

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 移动阶段

- 当前玩家掷1颗骰子，点数=可移动的节点数
- 沿着本层路径向前移动（不能后退）
- 停在目标节点后，执行该节点效果

#### 2. 节点事件执行

根据棋子停留的节点，执行相应动作：

| 节点类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起点** | 🟢 | 可休息：恢复1点体力（每局限1次） |
| **食材点** | 🟡 | 抽取1张随机食材卡加入手牌 |
| **怪物巢穴** | 🔴 | 进行战斗（详见下方战斗规则） |
| **烹饪站** | 🟠 | 消耗手牌食材，烹饪1道料理（详见烹饪规则） |
| **陷阱** | 🟣 | 抽取1张陷阱卡，执行效果 |
| **楼层出口** | 🔵 | 全体玩家到达出口后，进入下一层 |
| **Boss房间** | ⭐ | 触发Boss战（详见Boss战规则） |

#### 3. 战斗规则

当玩家停留在怪物巢穴🔴时，触发战斗：

1. 翻开1张怪物卡，查看怪物战斗力（如：史莱姆战斗力=3）
2. 玩家掷2颗骰子，点数之和 ≥ 怪物战斗力 = **战斗胜利**
3. 若玩家手牌中有料理卡且已食用（获得攻击Buff），可额外加值
4. **胜利**：获得怪物卡上标注的食材卡，加入手牌
5. **失败**：失去2点体力，留在当前节点（下回合可重试）

> 💡 **合作规则**：玩家可邀请其他玩家协助战斗！协助者掷1颗骰子，点数加入主攻玩家的骰子点数中。

#### 4. 烹饪规则

当玩家停留在烹饪站🟠时，可烹饪料理：

1. 展示手牌中的食材卡，选择要烹饪的料理
2. 将所需食材卡放入弃牌堆
3. 掷1颗骰子判定烹饪成功率：
   - 点数 ≥ 4：**烹饪成功**，获得对应料理卡
   - 点数 ≤ 3：**烹饪失败**，食材已消耗，但未获得料理卡（可下次再试）
4. 若玩家拥有厨具卡，可提升成功率（如：拥有平底锅🍳，主菜料理成功率+1）
5. 烹饪成功后，料理卡可立即食用或保留至需要时

#### 5. 食用料理

玩家可在**自己回合的任何时候**食用料理卡：

- 将料理卡放入弃牌堆
- 执行料理卡上的效果（恢复体力/获得Buff等）
- **注意**：攻击/防御Buff仅对下一场战斗有效

#### 6. 回合结束

- 检查共享体力：若体力 ≤ 0，游戏失败
- 检查是否到达楼层出口：若全体玩家均到达出口，进入下一层
- 将回合交给下一位玩家

### Boss战规则（第5层）

当任意玩家到达第5层Boss房间⭐时，触发Boss战：

1. **红龙战斗力** = 10（基础值）
2. **全员参与**：所有玩家围攻Boss，每回合大家依次攻击
3. **攻击方式**：每回合玩家掷2颗骰子，总和与红龙战斗力比较
   - 总和 ≥ 10：对红龙造成1点伤害
   - 若有攻击Buff：额外伤害
4. **红龙反击**：每回合结束后，红龙攻击随机一名玩家，失去2点体力
5. **胜利条件**：红龙受到5点伤害后倒下 → **全员胜利！**
6. **失败条件**：共享体力降至0 → **全员失败...**

### 示例1：第1层普通回合

**玩家A（蓝色棋子）的回合：**

1. **移动阶段**：掷骰子得到4点，向前移动4个节点

## 💡 新手策略提示

1. **优先采集食材**：初期多停留在食材点🟡，积累足够的食材才能烹饪强力料理
2. **料理是核心**：不要舍不得吃料理！体力是_shared_的，及时恢复体力是团队责任
3. **合理分工**：让骰子运气好的玩家负责战斗，运气差的玩家负责采集和烹饪
4. **保留香料**：🌶️香料是稀有食材，留到Boss战前再使用
5. **注意楼层进度**：不要一个人冲太前，确保队友也能跟上进入下一层
6. **复活饭很重要**：如果队友体力耗尽"倒地"，尽快烹饪"复活饭🍚"将其复活
7. **陷阱可规避**：如果骰子点数刚好可以停在陷阱🟣前，优先选择安全路径

## 🔄 变体规则

### 简单模式（新手推荐）

- 共享体力起始值提升至 **30点**
- 红龙战斗力降低至 **8点**
- 烹饪成功率判定降低（点数 ≥ 3 即成功）
- 每位玩家起始多获得1张食材卡

### 困难模式（老手挑战）

- 共享体力起始值降低至 **15点**
- 红龙战斗力提升至 **12点**，且每回合反击伤害提升至 **3点**
- 增加"饥饿机制"：每通过一层，全体失去1点体力
- 怪物巢穴出现频率增加（地图增加🔴节点）

### 竞速模式

- 设置沙漏计时 **45分钟**
- 在规定时间内击败红龙即为胜利
- 超时则游戏失败

### 单人模式

- 控制2个棋子（相当于2人合作）
- 共享体力起始值 **15点**
- 适合练习熟悉游戏机制

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。需要1颗骰子和2颗骰子两种用法。

**Q：食材卡手牌上限是多少？**
A：每位玩家手牌上限为 **5张食材卡**。超过上限时，需选择丢弃至5张。料理卡和厨具卡不计入上限。

**Q：可以跳过战斗吗？**
A：不可以。停在怪物巢穴🔴必须进行战斗，但其他玩家可以协助你战斗。

**Q：烹饪失败后食材会返还吗？**
A：不返还。烹饪是消耗性操作，失败后食材进入弃牌堆。

**Q：多人同时到达楼层出口怎么办？**
A：所有到达出口的玩家同时进入下一层，未到达的玩家需在下回合继续移动至出口。

**Q：共享体力降到0后还能继续吗？**
A：不能。共享体力=0时游戏立即失败。请注意及时烹饪料理恢复体力！

**Q：Boss战中玩家倒地了怎么办？**
A：如果有其他玩家拥有"复活饭🍚"料理卡，可以立即食用将其复活（恢复5点体力）。否则该玩家跳过一个回合。
