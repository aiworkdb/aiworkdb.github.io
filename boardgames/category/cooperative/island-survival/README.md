# 岛屿求生 Island Survival

## 1. 游戏概述

欢迎来到《岛屿求生》——一款紧张刺激的合作生存桌游！玩家将扮演海难幸存者，流落在一座神秘的热带岛屿上。面对恶劣的天气、匮乏的资源、神秘的洞穴和潜伏的危险，你们必须团结协作，利用智慧和勇气建造庇护所、收集食物和淡水、探索未知的区域，最终建造信号塔并点燃求救信号火，等待救援船只的到来。

在这款游戏中，每一天都是一场与自然的搏斗。晴朗的日子适合外出探索，雨天可以收集珍贵的淡水，但暴风雨却会带来毁灭性的破坏。岛屿上分布着不同的区域——阳光明媚的海滩、物种丰富的丛林、神秘莫测的山洞和壮丽的瀑布，每个区域都有独特的资源和挑战。

《岛屿求生》适合1-4名玩家，游戏时长45-75分钟，适合10岁以上的玩家。游戏难度定位为中等，既不会过于简单让人乏味，也不会过于复杂令人望而却步。这是一个考验团队协作、决策能力和危机处理能力的绝佳体验。

### 游戏特色

《岛屿求生》的核心魅力在于其高度的不确定性和策略性。天气系统是整个游戏的灵魂所在——它不仅影响当天的行动效率，更可能在一夜之间改变整个局势。玩家需要在晴天尽可能多地探索和收集资源，在雨天抓住机会收集淡水，在暴风雨来临前做好防护措施。

游戏中的建造系统允许玩家从简陋的庇护所逐步升级为坚固的要塞。通过收集各种材料和图纸，玩家可以建造工具、制作武器、搭建庇护所，最终建造出能发出求救信号的信号塔。这个过程充满了成就感，每一次成功的建造都会让玩家感受到进步的喜悦。

探索机制则为游戏增添了冒险的刺激。岛屿上隐藏着许多秘密——古老的遗迹、神秘的洞穴、珍贵的宝藏，当然也潜伏着各种危险。玩家需要权衡探索的收益与风险，合理分配人员和资源。

### 区域详解

#### 🏖️ 海滩区域（Beach）

**位置**：岛屿东南部，面向大海

**视觉特点**：金色沙滩、贝壳散落、搁浅的小船残骸、漂流木

**可用资源**：
- 椰子（简单）：每采集+2淡水
- 贝壳（简单）：每采集+1布
- 漂流木（简单）：每采集+2木材
- 鲜鱼（中等）：每采集+2食物

**特殊地点**：
- ⛵ 沉船残骸：探索时有几率发现金属和布料
- 🦀 潮汐池：每日可采集少量食物

**危险提示**：涨潮时海滩区域可能被淹没

#### 🌲 丛林区域（Jungle）

**位置**：岛屿中部，覆盖大部分面积

**视觉特点**：茂密植被、藤蔓悬挂、鸟类栖息、蝴蝶飞舞

**可用资源**：
- 香蕉（简单）：每采集+2食物
- 野果（简单）：每采集+1食物
- 木材（简单）：每采集+2木材
- 藤蔓（简单）：每采集+2布
- 珍贵木材（中等）：每采集+3木材

**特殊地点**：
- 🌺 草药园：发现时永久+1治疗资源
- 🦜 鸟巢：有几率发现羽毛和种子

**危险提示**：丛林中可能有野兽出没

#### 🕳️ 山洞区域（Cave）

**位置**：岛屿北部，靠近山顶

**视觉特点**：黑暗深邃、石笋石花、滴水声、神秘回声

**可用资源**：
- 石头（中等）：每采集+2石头
- 金属矿石（困难）：每采集+1金属
- 水晶（困难）：探索奖励，可用于高级建造
- 化石（中等）：收藏价值，可换取援助

**特殊地点**：
- 💎 矿脉：金属的主要来源
- 🖼️ 壁画遗迹：揭示岛屿历史，可能获得特殊奖励

**危险提示**：
- 需要火把照明（消耗1木材）
- 可能有蝙蝠和毒虫

#### 💧 瀑布区域（Waterfall）

**位置**：岛屿西部，临海悬崖

**视觉特点**：水流湍急、水雾弥漫、彩虹常见、鱼类丰富

**可用资源**：
- 淡水（简单）：每采集+3淡水
- 鲜鱼（简单）：每采集+2食物
- 光滑石头（简单）：每采集+1石头
- 瀑布后的秘密（探索）：发现额外宝藏

**特殊地点**：
- 🌈 彩虹下方：探索成功率大幅提升
- 🐟 鱼群聚集区：稳定的食物来源

**危险提示**：瀑布边缘湿滑，可能滑落受伤

## 5. 道具详解

### 5.1 资源卡

资源卡是岛屿生存的基础。玩家需要收集各种资源来建造庇护所、制作工具、获取食物和淡水。

**资源类型详解**：

| 资源图标 | 资源名称 | 主要来源 | 用途 |
|----------|----------|----------|------|
| 🍌 | 食物 | 丛林、海滩 | 维持生命，每回合消耗1 |
| 💧 | 淡水 | 瀑布、海滩 | 维持生命，每回合消耗1 |
| 🪵 | 木材 | 丛林、海滩 | 建造庇护所和信号塔 |
| 🪨 | 石头 | 山洞、瀑布 | 建造坚固结构 |
| 🔩 | 金属 | 山洞 | 高级建造和工具制作 |
| 🧵 | 布料 | 海滩、丛林 | 制作绳索和帆布 |

**资源采集规则**：
- 玩家在每个区域每回合可进行1次采集行动
- 采集成功率受天气影响
- 某些资源需要特殊工具才能高效采集

### 5.2 建造卡

建造系统是游戏的核心机制之一。通过收集材料并使用建造卡，玩家可以逐步升级自己的生存能力。

**建造阶段说明**：
1. **准备材料**：将所需资源放置在建造卡旁边
2. **验证资源**：确认资源数量充足
3. **执行建造**：消耗资源，获得建筑效果
4. **放置标记**：将建造标记放置在岛屿地图上

**建造优先级建议**：
- 初期：优先建造简易庇护所和储水器
- 中期：升级庇护所，制作高效工具
- 后期：集中资源建造信号塔和火堆

### 5.3 探索卡

探索是获取额外资源和发现秘密的主要途径。每个区域都有其独特的探索内容。

**探索流程**：
1. 选择探索区域
2. 抽取探索卡
3. 判定结果（可能需要掷骰）
4. 获得奖励或面对危险

**探索成功条件**：
- 基础成功率：50%
- 晴天加成：+20%
- 拥有地图：+10%
- 拥有向导（经验玩家）：+10%

### 5.4 天气卡

天气系统是《岛屿求生》最具特色的机制之一。天气不仅影响当天的游戏方式，更可能带来机遇或挑战。

**天气效果详解**：

| 天气 | 出现概率 | 正面效果 | 负面影响 | 策略建议 |
|------|----------|----------|----------|----------|
| ☀️ 晴天 | 30% | 探索+2，采集效率提升 | 无 | 全力探索和采集 |
| 🌧️ 小雨 | 25% | 淡水+3 | 无法探索 | 专注于淡水收集 |
| ⛈️ 大雨 | 20% | 淡水+5 | 可能损坏庇护所 | 加固建筑，储备资源 |
| 🌪️ 暴风雨 | 15% | 无 | 无法探索，50%损坏 | 躲避，准备修复 |
| ☁️ 多云 | 10% | 无 | 无 | 平衡发展 |

**天气轮转规则**：
- 每天开始时抽取天气卡
- 天气持续一整天（一个玩家回合）
- 某些建造可以改变天气效果

## 6. 游戏规则

### 6.1 游戏准备

**设置步骤**：

1. **展开地图**：将岛屿地图放置在桌面中央

2. **放置区域标记**：在四个区域各放置对应的区域指示物

3. **资源池**：在地图旁边设立资源池区域，放置所有资源标记

4. **卡牌洗牌**：
   - 将天气卡洗匀，放在指定位置
   - 将资源卡按区域分类洗匀
   - 将探索卡洗匀
   - 将危险卡洗匀

5. **设置进度条**：将求救进度条放在地图边缘，初始为0

6. **玩家就位**：
   - 每位玩家选择一个颜色，获得对应棋子
   - 将所有玩家棋子放置在海滩区域
   - 每人初始获得：食物×3，淡水×3，木材×2

7. **初始设置**：
   - 抽取3张天气卡作为预告（不公开）
   - 在山洞入口放置1个火把需求标记

**准备工作检查清单**：
- [ ] 地图展开平整
- [ ] 所有卡牌洗匀
- [ ] 资源分类摆放
- [ ] 玩家棋子就位
- [ ] 进度条归零
- [ ] 天气卡准备就绪

### 6.2 游戏流程

游戏以"天"为单位进行，每天包含以下阶段：

#### 每日流程

**第一阶段：黎明（天气判定）**
1. 翻开天气卡，宣布今日天气
2. 应用天气效果
3. 检查是否有暴风雨损坏

**第二阶段：行动（玩家回合）**
每位玩家依次进行行动，顺序从最近的幸存者开始：

1. **移动阶段**：玩家可以移动到相邻区域
2. **行动阶段**：选择以下行动之一：
   - 采集资源（根据当前区域和天气）
   - 探索区域（抽取探索卡）
   - 建造建筑（使用建造卡）
   - 休整恢复（恢复1点疲劳）
3. **结束阶段**：汇报资源状态

**第三阶段：黄昏（资源结算）**
1. 所有玩家消耗食物和淡水各1
2. 检查资源耗尽情况
3. 更新求救进度

**第四阶段：夜晚（事件处理）**
1. 可能触发夜间事件
2. 检查是否需要消耗火把
3. 推进救援进度

### 6.3 天气轮转系统

天气是《岛屿求生》的核心机制，其运作方式如下：

**天气卡抽取规则**：
- 每3天为一个天气周期
- 每个周期包含2张晴天、1张多云、2张雨天、1张暴风雨
- 玩家可以通过特殊建筑影响天气

**天气影响表**：

| 天气 | 采集 | 探索 | 淡水 | 特殊效果 |
|------|------|------|------|----------|
| ☀️ 晴天 | +1 | 成功率+20% | 0 | 无 |
| 🌧️ 小雨 | 0 | 禁止 | +2 | 无 |
| ⛈️ 大雨 | -1 | 禁止 | +4 | 庇护所可能受损 |
| 🌪️ 暴风雨 | 禁止 | 禁止 | +1 | 50%概率损坏建筑 |
| ☁️ 多云 | 0 | 0 | 0 | 无 |

### 6.4 建造系统

**建造流程详解**：

1. **查看建造卡**：玩家可以在任意时刻查看可用的建造卡

2. **收集材料**：确保资源池中有足够的建造材料

3. **支付资源**：将所需资源从资源池移至弃牌区

4. **获得建筑**：将建造标记放置在岛屿地图上

5. **激活效果**：立即获得建筑提供的效果

**建造规则**：
- 每个玩家每回合最多建造1个建筑
- 某些建筑需要特定位置
- 建筑被损坏后需要消耗资源修复

### 6.5 胜利条件

**主要胜利条件（必须同时满足）**：
1. ✅ 建造信号塔（消耗：木材×10，金属×5，布料×3）
2. ✅ 点燃求救火堆（消耗：木材×6，布料×2）
3. ✅ 火堆燃烧完整1天（天气不能是暴风雨）
4. ✅ 救援进度达到10

**胜利判定时机**：
- 在夜晚阶段结束时检查胜利条件
- 一旦满足所有条件，游戏立即结束

### 6.6 失败条件

**触发失败的情况**：
1. ❌ 所有玩家同时耗尽食物和淡水
2. ❌ 玩家人数降至0（全员死亡）
3. ❌ 连续3次遭遇暴风雨且所有建筑被摧毁
4. ❌ 第20天结束时仍未满足胜利条件

**失败后处理**：
- 记录失败原因和存活天数
- 整理所有游戏材料
- 可以选择重开新游戏

## 8. 新手策略提示

### 8.1 初期策略（前5天）

**资源优先级**：
1. 食物和淡水永远是第一优先级——没有食物和淡水，游戏直接结束
2. 尽快收集木材——用于建造和火把
3. 探索海滩和丛林——这些区域风险低、收益稳定

**区域选择建议**：
- 第一天全员留在海滩附近采集
- 第二天开始分头行动，覆盖多个区域
- 避免在暴风雨天探索山洞

**建造顺序建议**：
1. 简易庇护所（第2-3天）
2. 储水器（第3-4天）
3. 简易陷阱（第4-5天）
4. 石斧（第3天）

### 8.2 中期策略（第6-12天）

**资源管理**：
- 开始储备应急资源（额外5单位食物和淡水）
- 收集金属用于高级建造
- 建立稳定的资源收入流

**探索策略**：
- 利用晴天深入探索山洞
- 每次探索前确保有足够的资源支撑
- 记录已探索区域和发现

**协作建议**：
- 分配专人负责收集淡水（瀑布玩家）
- 分配专人负责收集食物（丛林玩家）
- 定期召开"生存会议"讨论策略

### 8.3 后期策略（第13天+）

**胜利冲刺**：
- 最后阶段集中所有资源于信号塔和火堆
- 提前计算所需资源缺口
- 确保有足够的缓冲资源应对意外

**风险管理**：
- 避免在暴风雨天进行关键建造
- 准备修复材料应对建筑损坏
- 留有应急资源应对危险事件

### 8.4 通用技巧

**协作技巧**：
- 每天开始时讨论分工
- 共享发现的信息和资源
- 避免单独行动导致风险集中

**风险评估**：
- 始终保持至少3天的食物和淡水储备
- 关注天气预告，提前做好应对
- 危险事件卡要谨慎处理

**效率优化**：
- 利用移动节省行动点数
- 合理规划路线，减少无效移动
- 善用工具提升采集效率

## 10. 常见问题

### Q1：玩家在暴风雨天可以做什么行动？

在暴风雨天，玩家只能在庇护所内进行以下行动：休整恢复（减少疲劳）、使用已有资源进行小型建造（如修理）、消耗食物和淡水。其他户外行动（采集、探索）全部禁止。

### Q2：如果所有食物耗尽了怎么办？

当食物降至0时，所有玩家每回合失去1点生命值。失去全部生命值的玩家将被淘汰，退出游戏。如果所有玩家都被淘汰，游戏失败。建议始终保持至少5单位食物的储备。

### Q3：探索失败会怎样？

探索失败时，玩家不会获得奖励，但通常也不会受到惩罚（除非特定探索卡说明）。某些探索卡在失败时可能触发危险事件，请仔细阅读卡牌说明。

### Q4：建筑被暴风雨损坏后可以修复吗？

可以。修复建筑需要消耗原建造所需资源的50%。例如，简易庇护所被损坏后，修复需要2个木材和1个石头。修复行动需要在庇护所内进行，不受天气影响。

### Q5：救援进度是如何推进的？

救援进度通过以下方式推进：每天自动+0.5（岛屿位置被过往船只发现）、点燃求救火堆后每天+1、特殊事件卡可额外增加进度。当救援进度达到10时，救援船到达，游戏胜利。

### Q6：多个玩家可以同时在同一区域吗？

可以。游戏鼓励玩家协作，所有玩家都可以在同一区域行动。同一区域的玩家可以共享资源、协作探索（提高成功率），也可以分工进行不同行动。

### Q7：天气预告有多准确？

游戏设置中，每3天会预告下一天的天气。但天气系统存在一定的不确定性，某些特殊事件可能导致天气突变。玩家应该根据预告做好规划，同时保留应对突发天气的后备方案。

### Q8：山洞探索需要火把，火把如何获取？

火把是一种消耗品，每次进入山洞需要消耗1个火把。火把可以通过消耗1个木材和1个布制作。玩家也可以选择不进入山洞，在其他区域进行更安全的探索。
