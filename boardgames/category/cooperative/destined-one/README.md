# 天命人 (Destined One)

## 🎮 游戏概述

《天命人》是一款基于《黑神话：悟空》黑暗神话世界观的合作类桌游。玩家扮演天命人，踏上充满危险与神秘的取经路，在黑神话世界中探索未知、挑战强大的妖王、收集神器碎片，最终面对大圣残躯，揭开天命的真相。

这是一款强调团队合作、策略规划和资源管理的中等偏难合作桌游，适合1-4人游玩。玩家必须齐心协力，合理分配资源，制定战术，才能在这个充满敌意的世界中生存并最终获胜。

**适合人群**：喜欢《黑神话：悟空》、《西游记》神话改编、合作类策略游戏的玩家；适合桌游爱好者、动漫游戏粉丝、团队合作挑战爱好者。

## 🗺️ 游戏地图详解

游戏地图是一个 **6x6 的黑神话世界网格**，包含多种地形和事件点，灵感来自《黑神话：悟空》的关卡设计。

### 地图布局概览

```
🏯 ⚔️ 🌲 ⬛ 💀 ⬛
⬛ 📜 ⬛ 🔮 ⬛ 👹
🌲 ⬛ ⬛ ⚔️ ⬛ 💎
⬛ 🔮 ⬛ ⬛ 📜 ⬛
💀 ⬛ 👹 ⬛ 🛡️ ⬛
⬛ 💎 ⬛ 🌲 ⬛ 🐵
```

**图例说明**：
- 🏯 = 起点（天命人出发地）
- ⬛ = 道路（安全区域）
- 🌲 = 森林区域（可获得兵器卡）
- 💀 = 妖气区域（可能触发战斗）
- 🔮 = 神秘区域（可获得法术卡/资源卡）
- 📜 = 事件区域（抽取事件卡）
- ⚔️ = 兵器库（可获得兵器卡）
- 🛡️ = 防具库（可获得防具卡）
- 💎 = 神器碎片点（收集神器碎片）
- 👹 = 妖王区域（挑战妖王）
- 🐵 = 终点（大圣残躯）

### 地图特点

1. **区域分布**：地图分为6个主要区域，每个区域有不同的资源和挑战
2. **资源梯度**：越靠近中心区域，资源越稀有但风险越高
3. **妖王位置**：3个妖王分布在地图的关键位置（右上、中心、右下）
4. **最短路径**：从起点到终点需要探索约15-20个区域
5. **策略选择**：玩家可以选择安全路线（绕路）或风险路线（直行）

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 只能在有地形标识的位置停留
- 移动步数由玩家决定（每回合最多移动3格）
- 可以和其他玩家停留在同一格

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择棋子**：每位玩家选择一个颜色的棋子，放在起点🏯
3. **初始化属性**：
   - 每位玩家生命值 = 10
   - 每位玩家获得：木棒×1、布衣×1、资源卡×3
4. **洗混卡牌**：将妖王卡、事件卡分别洗混，放在地图旁边
5. **放置妖王**：将虎先锋放在右上角妖王区域，亢金龙放在中心妖王区域
6. **决定先手**：掷骰子，点数最高的玩家先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 移动阶段
- 玩家可以移动**最多3格**（上下左右）
- 可以停留在任何有地形标识的位置
- 可以和其他玩家停留在同一格

#### 2. 执行所在位置的事件
根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起点** | 🏯 | 无特殊效果，可以休整（恢复1点生命值） |
| **道路** | ⬛ | 无特殊效果 |
| **森林区域** | 🌲 | 获得1张兵器卡或1张资源卡 |
| **妖气区域** | 💀 | 进行战斗检定：掷骰子，若点数≤3则受到1点伤害 |
| **神秘区域** | 🔮 | 获得1张法术卡或2张资源卡 |
| **事件区域** | 📜 | 抽取并执行事件卡效果 |
| **兵器库** | ⚔️ | 获得1张稀有兵器卡（金箍棒或以上） |
| **防具库** | 🛡️ | 获得1张防具卡 |
| **神器碎片点** | 💎 | 获得1张神器碎片卡 |
| **妖王区域** | 👹 | **挑战妖王**（见下方战斗规则） |
| **终点** | 🐵 | 若已合成神器，则**挑战大圣残躯** |

#### 3. 行动阶段（可选）
玩家可以选择执行以下行动（每回合限1次）：
- **装备兵器/防具**：替换当前装备
- **使用法术卡**：发动法术效果
- **合成神器**：若已收集4个神器碎片
- **传递卡牌**：给相邻玩家传递1张卡牌（合作机制）
- **休整**：恢复1点生命值（限1次/回合）

#### 4. 妖王行动阶段
所有玩家行动完毕后，妖王进行行动：
- 抽取1张妖王卡，执行妖王攻击
- 若玩家在妖王相邻格，则受到妖王攻击力点数的伤害

### 战斗规则（挑战妖王）

当玩家停留在妖王区域时，触发战斗：

1. **攻击阶段**：
   - 玩家掷骰子（1-6）
   - 攻击力 = 兵器攻击力 + 骰子点数
   - 若攻击力 ≥ 妖王防御力（通常为5），则妖王受到1点伤害

2. **妖王反击阶段**：
   - 妖王攻击力 = 基础攻击力 + 1张妖王卡效果
   - 玩家防御力 = 防具防御力
   - 若妖王攻击力 > 玩家防御力，则玩家受到（攻击力-防御力）点伤害

3. **战斗结束条件**：
   - 妖王生命值降为0：玩家获胜，获得奖励（神器碎片+资源卡）
   - 玩家生命值降为0：玩家阵亡，其他玩家可以继续游戏（需复活）

### 胜利条件

**合作胜利**：所有玩家合作，完成以下条件即获胜：
1. 至少1名玩家合成"天命神器"
2. 挑战并击败"大圣残躯"
3. 所有存活玩家到达终点🐵

**失败条件**：
- 所有玩家阵亡
- 回合数超过20回合（可选难度设置）

## 📖 完整游戏流程示例

### 示例1：单人游戏流程（玩家A）

**回合1（探索阶段）：**
1. 玩家A从起点🏯出发，移动2格到达森林区域🌲
2. 获得1张"铁棒"兵器卡（攻击力+2）
3. 行动阶段：装备"铁棒"，替换"木棒"
4. 回合结束，妖王"虎先锋"攻击相邻玩家（无人在相邻格，无效果）

**回合2（战斗阶段）：**
1. 玩家A移动2格到达妖气区域💀
2. 进行战斗检定：掷骰子得到4（≥4安全），未受伤
3. 行动阶段：使用"火眼金睛"法术，查看妖王卡
4. 回合结束

**回合3（挑战妖王）：**
1. 玩家A移动1格到达妖王区域👹（虎先锋）
2. 触发战斗：
   - 玩家A攻击：掷骰子得到5，攻击力=2+5=7 ≥ 5，虎先锋受到1点伤害（14/15）
   - 虎先锋反击：攻击力=2，玩家防御力=1，玩家受到1点伤害（9/10）
3. 玩家A选择继续战斗或撤退
4. 若继续战斗，重复步骤2；若撤退，移动到相邻格

**回合10（获得神器碎片）：**
1. 玩家A经过多次探索和战斗，收集到4个神器碎片
2. 行动阶段：合成"天命神器"，攻击力+5，防御力+3
3. 现在可以挑战大圣残躯了！

**回合15（最终决战）：**
1. 玩家A到达终点🐵，挑战大圣残躯
2. 大圣残躯生命值30，攻击力4
3. 玩家A凭借天命神器，经过5轮战斗，最终击败大圣残躯
4. **游戏胜利！**

### 示例2：多人合作流程（玩家A、B、C）

**回合1：**
- 玩家A：探索森林区域，获得兵器卡

- 玩家A和B同时到达妖王区域，合作挑战虎先锋
- 玩家A主攻（高攻击力），玩家B辅助（使用法术卡"定身法"）

**回合12（神器合成）：**
- 玩家A、B、C各自收集到1-2个神器碎片
- 通过"传递卡牌"机制，玩家A集齐4个碎片
- 玩家A合成天命神器，成为团队核心战斗力

**回合18（最终决战）：**
- 玩家A（天命神器）挑战大圣残躯
- 玩家B和C在旁边辅助（使用法术卡、提供资源）
- 团队合作，最终击败大圣残躯
- **游戏胜利！**

1. **优先探索安全区域**：初期多探索森林🌲和神秘区域🔮，收集装备和资源，避免过早挑战妖王

2. **团队合作至关重要**：及时传递卡牌给需要的队友，集中资源培养1-2名核心玩家

3. **合理分配角色**：
   - 攻击型玩家：专注收集兵器卡，挑战妖王
   - 辅助型玩家：专注收集法术卡，支援队友
   - 探索型玩家：专注收集神器碎片和资源

4. **妖王挑战时机**：建议在攻击力≥4、防御力≥2后再挑战妖王，否则容易阵亡

5. **法术卡节省使用**：法术卡效果强大但资源消耗大，建议在关键时刻使用

6. **神器碎片优先**：神器碎片是胜利的关键，优先探索神器碎片点💎

7. **注意生命值管理**：生命值降为0会阵亡，建议及时休整恢复

8. **妖王区域标记**：记住妖王位置，避免意外进入导致强制战斗

## 🔄 变体规则

### 难度调整

**简单模式（新手推荐）**：
- 玩家初始生命值 = 15（原为10）
- 妖王生命值减半（虎先锋7、亢金龙10、大圣残躯15）
- 增加更多资源卡和事件卡（正面效果）

**困难模式（高手挑战）**：
- 玩家初始生命值 = 8（原为10）
- 妖王攻击力+1，特殊能力增强
- 减少资源卡数量（原为30张，改为20张）
- 添加新妖王："二郎神"（生命值25，攻击力3，特殊能力：可以反击2次）

### 单人模式特殊规则

- 玩家控制2个棋子（双天命人）
- 2个棋子可以互相传递卡牌
- 2个棋子共享胜利条件

### 时间挑战模式

- 设置60分钟时间限制
- 在规定时间内击败大圣残躯的队伍获胜
- 如果时间到未击败，则根据以下优先级判定：
  1. 是否合成神器（是>否）
  2. 击败妖王数量（多>少）
  3. 存活玩家数量（多>少）

### 剧情模式

- 按照《黑神话：悟空》剧情顺序挑战妖王
- 顺序：虎先锋 → 亢金龙 → 大圣残躯
- 每击败一个妖王，解锁新的区域和卡牌

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。也可以使用在线骰子工具。

**Q：卡牌用完了怎么办？**
A：可以将弃牌堆洗混后重新使用。建议打印时多打印几套卡牌备用。

**Q：玩家阵亡后怎么办？**
A：阵亡玩家可以等待队友使用"天命复活"法术卡复活，或者观看其他玩家游戏（观战模式）。

**Q：可以跳过妖王区域吗？**
A：可以，玩家可以选择不进入妖王区域。但击败妖王是获得神器碎片的重要途径。

**Q：多人游戏时，卡牌如何传递？**
A：在行动阶段，玩家可以传递给相邻格的玩家1张卡牌（每回合限1次）。这是合作机制的核心。

**Q：神器碎片可以交易吗？**
A：可以，神器碎片作为卡牌的一种，可以通过"传递卡牌"机制交易。

**Q：妖王会移动吗？**
A：基础规则中妖王不会移动。在困难模式中，妖王每3回合可以移动1格。

**Q：打印时背景颜色不见了怎么办？**
A：请在打印设置中勾选"背景图形"选项。详细请参考 [printables.html](printables.html) 文件的打印提示。

**Q：游戏时间过长怎么办？**
A：可以设置回合数限制（建议15-20回合），或者减少妖王生命值。
