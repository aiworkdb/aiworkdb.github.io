# 虚拟侦探社 (Virtual Detective Agency)

## 🎮 游戏概述

《虚拟侦探社》是一款合作类桌游，所有玩家共同对抗网络犯罪组织。每位玩家扮演一名虚拟侦探，在赛博世界中收集线索、破解密码、分析数据，最终揭露并阻止犯罪分子的阴谋。

这款游戏融合了大热游戏《赛博朋克2077》的科技感元素和经典推理作品的悬疑氛围，强调团队合作与逻辑推理，适合朋友聚会或桌游之夜，培养玩家的信息整合能力和团队配合精神。

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择角色**：每位玩家选择一个虚拟侦探角色（黑客/分析师/追踪者/解密专家/潜入者）
3. **放置棋子**：所有玩家棋子放在起点🎮
4. **初始化资源**：
   - 每位玩家获得2张技能卡（随机）
   - 将线索卡洗混作为牌库
   - 将危机卡放在地图旁
   - 设置犯罪倒计时为10
5. **确定先手**：掷骰子，点数最高者先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的路径移动棋子
- **重要**：玩家可以商量移动策略，但决定权在自己

#### 2. 执行所在位置的事件

根据棋子停留的位置，执行相应动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起点** | 🎮 | 无特殊效果，可以补充手牌至2张 |
| **道路** | ⬛ | 无特殊效果 |
| **线索点** | 🔍 | 搜索并获得1条线索卡 |
| **技能点** | 💻 | 学习1项技能卡 |
| **危机点** | ⚠️ | 抽取并执行危机卡效果，犯罪倒计时-1 |
| **证据点** | 📁 | 获得1份证据卡 |
| **区域入口** | 🏢/🏰/🕳️ | 进入建筑内部，触发特殊事件 |

#### 3. 使用技能（可选）

如果拥有技能卡，可以在回合中任意时刻使用：
- 每位玩家的技能是**共享资源**，可以讨论最佳使用时机
- 技能效果可以作用于自己或其他玩家

#### 4. 传递线索（可选）

玩家之间可以传递线索卡：
- 每次传递需要消耗1个行动点（即少移动1步）
- 鼓励团队合作，合理分配资源

#### 5. 犯罪回合

所有玩家回合结束后，执行犯罪回合：
- 犯罪倒计时-1（基础难度）
- 如果犯罪倒计时≤5，危机卡效果加倍
- 如果犯罪倒计时=0，游戏立即失败

### 胜利条件

**合作胜利**：所有玩家共同努力，达成以下条件即获胜：

1. **揭露真相**：收集至少5条不同类型线索，并成功推理出犯罪计划
2. **阻止犯罪**：在犯罪倒计时归零前，找到并出示"幕后黑手照片"证据卡

**失败条件**：
- 犯罪倒计时归零
- 所有玩家的手牌耗尽且无法补充
- 连续3回合无法获得新线索

### 回合结束

完成所有操作后，将回合交给下一位玩家，并记得执行犯罪回合。

## 📖 完整游戏流程示例

**游戏设置**：3名玩家（黑客A、分析师B、追踪者C），初始手牌各2张。

**第1回合（黑客A）**：
1. 掷骰子得到7点，移动7步
2. 停在线索点🔍，搜索获得1条"IP地址"线索
3. 使用"黑客入侵"技能，查看下一张危机卡内容
4. 回合结束，犯罪倒计时-1（剩余9）

**第2回合（分析师B）**：
1. 掷骰子得到5点，移动5步
2. 停在技能点💻，学习1项"数据分析"技能
3. 使用"数据分析"技能，额外获得1条"交易记录"线索
4. 将"IP地址"线索传递给追踪者C（消耗1行动点）
5. 回合结束，犯罪倒计时-1（剩余8）

**第3回合（追踪者C）**：
1. 掷骰子得到8点，移动8步，到达区域入口🏢
2. 进入"数据中心"建筑内部，触发特殊事件：获得1份"监控录像"线索
3. 建筑内部还有1个证据点📁，再移动2步到达
4. 获得"犯罪计划书"证据卡，可以直接推理真相！
5. 回合结束，犯罪倒计时-1（剩余7）

**后续回合**：
- 玩家们继续探索其他区域，收集剩余线索
- 互相配合使用技能，应对危机卡挑战
- 当犯罪倒计时接近3时，团队加速推理过程
- 最终，分析师B使用"犯罪计划书"证据，成功推理出真相，犯罪被阻止，团队胜利！

## 💡 新手策略提示

1. **沟通至上**：时刻与队友分享手牌信息，共同规划策略
2. **资源集中**：不要每人都留技能卡，集中给1-2人应对关键危机
3. **优先控危机**：犯罪倒计时是倒计时，优先避免或化解危机点
4. **合理使用证据**：证据卡效果强大，在关键时刻使用
5. **留意建筑入口**：进入建筑内部可以获得稀有线索和证据
6. **不要自私**：这是合作游戏，队友获胜=你获胜
7. **数据分析很重要**：提前知道危机卡内容，可以提前防范

## 🔄 变体规则

### 简单模式（适合新手）

- 犯罪倒计时初始值改为12
- 每位玩家初始手牌改为3张
- 可以查看危机卡堆顶的3张牌

### 困难模式（适合老手）

- 犯罪倒计时初始值改为7
- 每位玩家初始手牌改为1张
- 危机回合犯罪倒计时额外-1
- 增加"犯罪升级"事件：每5回合触发一次，所有玩家后退3步

### 团队对抗模式

- 分成两队（2v2或2v3），每队合作对抗另一队
- 先推理出真相的一队获胜
- 可以互相干扰（如使用技能让对方抽到危机卡）

### 限时挑战模式

- 设置45分钟时间限制
- 时间内完成任务则获胜，否则失败
- 增加紧张感和成就感

## ❓ 常见问题

**Q：我可以查看其他玩家的手牌吗？**
A：可以！这是合作游戏，所有信息都可以共享。

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：线索卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用，或者停在线索点获得新卡。

**Q：可以跳过使用技能直接移动吗？**
A：可以，使用技能是可选操作，根据策略决定。

**Q：多人同时获得证据怎么办？**
A：合作游戏没有竞争，证据共享，大家一起受益！

**Q：犯罪倒计时归零后还能继续游戏吗？**
A：不能，犯罪已经发生，游戏立即失败。

**Q：可以使用手机查询推理思路吗？**
A：建议先自己讨论策略，实在卡关时再求助外部资源。

**Q：证据卡可以传递给其他玩家吗？**
A：可以，证据卡视为普通卡牌，可以传递。
