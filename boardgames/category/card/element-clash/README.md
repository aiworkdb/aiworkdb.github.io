# 元素对决 (Element Clash)

## 1. 游戏概述

在《元素对决》中，你是一位掌控五行之力的元素法师。火🔥、水💧、木🌿、金⚔️、土🪨——五大元素相生相克，组成千变万化的对战格局。你需要策略性地出牌，利用元素克制关系击败对手，同时积攒法力释放强大的组合技。

游戏核心是**快速对战**：每局仅5-10分钟，但策略深度远超想象。新手5分钟上手，高手能钻研元素链式反应和法力经济。

**适合人群**：喜欢卡牌对战、策略思考的玩家。2-4人对战，节奏快、变化多，适合午休或旅行时快速来一局。

## 6. 游戏规则

### 准备阶段

1. **洗牌**：将所有元素卡和法术卡分别洗混
2. **发牌**：根据人数发牌（2人15张/3人12张/4人10张）
3. **法力**：每位玩家获得初始法力标记
4. **生命**：每位玩家设定初始生命值
5. **抽法术卡**：每人抽取2张法术卡（手牌上限3张）

### 游戏流程（每回合）

每位玩家轮流执行以下步骤：

#### 步骤1：回复阶段
- 获得1点法力（法力上限5点）
- 如果处于「点燃」状态，受到1点伤害

#### 步骤2：抽牌阶段
- 从元素卡堆抽取1张卡
- 如果元素卡堆耗尽，将弃牌堆洗混后继续

#### 步骤3：行动阶段（选择1项）

**行动A：元素攻击**
- 打出1张元素卡，选择1位对手为目标
- 计算伤害 = 攻击力 × 克制倍率 - 对手防御
- 克制关系：攻击方克制防守方 → 伤害×2
- 对手防御在下回合重置

**行动B：组合技**
- 同时打出2张特定元素卡（须满足相生关系）
- 触发组合技效果（见相生关系表）
- 组合技不计算克制关系

**行动C：施放法术**
- 消耗对应法力，打出1张法术卡
- 执行法术效果
- 法术卡放入弃牌堆

**行动D：蓄力**
- 本回合不行动，获得2点法力
- 手牌上限+1（仅本回合有效）

#### 步骤4：结束阶段
- 手牌超过5张则弃掉多余的
- 防御值重置为0
- 轮到下一位玩家

### 胜利条件

- **2人模式**：将对手生命值降至0即获胜
- **3-4人模式**：最后存活的玩家获胜
- 被淘汰的玩家元素卡弃入弃牌堆，法术卡移出游戏

## 7. 完整游戏流程示例

### 示例回合：玩家A攻击玩家B

**初始状态**：
- 玩家A：生命15，法力3，手牌[🔥火(中), 🌿木(初), ⚔️金(初), 💧水(初)]
- 玩家B：生命12，法力2，手牌[🪨土(中), 🔥火(初)]

**回合行动**：
1. **回复阶段**：玩家A获得1点法力（变为4点）
2. **抽牌阶段**：抽取1张🌿木(初)
3. **行动阶段**：玩家A选择「元素攻击」
   - 打出🔥火(中)，攻击力3，法力消耗2
   - 玩家B手中有🪨土(中)，但火不克制土
   - 伤害 = 3 - 0(防御) = 3点
   - 玩家B生命从12降至9
   - 点燃效果：玩家B下回合受到1点额外伤害
4. **结束阶段**：玩家A手牌[🌿木(初), ⚔️金(初), 💧水(初), 🌿木(初)]，法力2点

### 示例回合：组合技

**玩家A手牌**：[🌿木(初), 🔥火(初), 💧水(中)]

**行动**：打出🌿木+🔥火 → 触发「烈焰风暴」
- 对所有对手（玩家B和C）各造成3点伤害
- 法力消耗：2+2=4点

## 8. 新手策略提示

1. **记住克制关系**：火克木克土克水克火，金克木被火克
2. **保留手牌多样性**：不要全拿一种元素，多样化手牌才能应对各种情况
3. **法力管理**：不要一次性用光法力，留1-2点应急
4. **组合技时机**：组合技威力大但消耗高，关键时刻使用
5. **防御也很重要**：💧水和🪨土的防御效果能让你活得久
6. **观察对手手牌**：注意对手打出过什么元素，推测手牌构成
7. **多人模式别当出头鸟**：不要过早攻击，让别人先消耗

## 9. 变体规则

### 变体1：快速决斗模式（2人）
- 初始生命10点
- 每回合获得2点法力
- 最先获胜2局的玩家赢得比赛

### 变体2：元素大师模式
- 每位玩家选择1种主元素（该元素攻击+1）
- 但主元素的克制元素对你伤害也+1
- 增加角色差异化

### 变体3：团队战模式（4人）
- 2人一组，共享生命值（30点）
- 队友间可以传递1张卡/回合
- 先将对方团队生命降至0者获胜

### 变体4：混沌模式
- 移除法力限制，每回合可打出任意数量卡牌
- 增加随机事件卡
- 节奏极快，适合喜欢混乱的玩家

## 10. 常见问题

**Q：元素卡打出去后还能回收吗？**
A：可以！使用法术卡「🔮元素觉醒」可以从弃牌堆回收2张元素卡。

**Q：防御值什么时候重置？**
A：在你的结束阶段，防御值重置为0。所以💧水和🪨土的防御只持续到你的回合结束。

**Q：被淘汰后能观战吗？**
A：当然可以！观战也是学习策略的好机会，但不要给场上的玩家提示。

**Q：组合技的2张卡必须同时打出吗？**
A：是的，组合技必须同时打出2张满足相生关系的元素卡，且消耗对应法力。

**Q：法力上限是5点，可以超过吗？**
A：正常情况不能超过5点。但法术卡「蓄力」行动可以获得额外法力，超过5点的部分会在回合结束丢弃。

**Q：多人模式被围攻怎么办？**
A：这是多人游戏的策略——尽量低调，不要成为众矢之的。使用防御型元素卡保命。
