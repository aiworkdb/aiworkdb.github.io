# 神经对决 (Neural Clash)

## 1. 游戏概述

**欢迎来到2089年的赛博空间！**

在这个数字化的未来世界，人工智能已经渗透到生活的每一个角落。各大财团和公司都在开发强大的人工智能程序，而独立的黑客们则在赛博空间中展开激烈的对抗，争夺数据控制权、破解防火墙、部署病毒程序，成为传说中的"顶级黑客"。

**《神经对决》** 是一款快节奏的卡牌对战游戏。玩家扮演黑客，通过部署AI程序、释放病毒、构筑防火墙，在赛博空间中与对手展开智力对决。游戏融合了策略规划、资源管理和心理博弈，适合喜欢科幻题材和卡牌游戏的玩家。

**适合人群**：
- 喜欢赛博朋克科幻题材的玩家
- 喜欢卡牌策略游戏的玩家
- 2-4人聚会游戏
- 适合桌面游戏之夜

## 4. 道具详解

### 4.1 数据源卡

数据源卡是游戏的基础资源，代表你可以使用的算力。

| 类型 | 图示 | 算力值 | 获得方式 | 用途 |
| :--- | :--- | :--- | :--- | :--- |
| **低阶数据源** | 💾 | 1-2 | 抽牌阶段 | 支付卡牌费用 |
| **中阶数据源** | 💿 | 3-4 | 抽牌阶段 | 支付卡牌费用 |
| **高阶数据源** | 🌐 | 5-6 | 抽牌阶段 | 支付卡牌费用 |

**使用示例**：
- 如果你想打出一张需要3算力的AI程序卡，你需要打出至少一张提供3算力的数据源卡，或者多张数据源卡合计提供3算力。

## 5. 游戏规则

### 5.1 准备阶段

1. **选择角色**（可选）：每位玩家可以选择一个黑客角色，每个角色有独特能力
   - **攻击者**：攻击型AI程序效果+1
   - **防御者**：防御型AI程序效果+1
   - **辅助者**：辅助型AI程序效果+1
   - **全能者**：所有类型AI程序效果+0.5（向下取整）

2. **构建牌库**：
   - 每位玩家获得20张卡牌（可预先构建或使用随机分配）
   - 建议初始配置：8张AI程序卡 + 5张数据源卡 + 4张病毒卡 + 3张防火墙卡

3. **初始手牌**：
   - 每位玩家抽取5张作为起始手牌
   - 先手玩家抽取4张（平衡先手优势）

4. **初始生命值**：
   - 每位玩家20点生命值
   - 可以用纸笔记录，或使用生命值标记

5. **初始算力**：
   - 每位玩家起始0算力
   - 需要通过打出数据源卡获得算力

### 5.2 游戏流程

游戏按回合进行，每位玩家的回合分为以下几个阶段：

#### 阶段1：算力恢复阶段
- 回收上回合使用的算力代币
- 重置本回合可用的算力为0

#### 阶段2：抽牌阶段
- 抽取2张牌（先手玩家第一回合只抽1张）
- 如果牌库为空，将弃牌库洗入牌库

#### 阶段3：主要阶段（重点！）
玩家可以依次执行以下操作（顺序不限）：
- **打出数据源卡**：获得算力（放置在自己面前）
- **部署AI程序**：消耗算力，打出AI程序卡
- **使用病毒**：消耗算力，对对手使用病毒卡
- **构筑防火墙**：消耗算力，打出防火墙卡
- **使用黑客工具**：使用黑客工具卡的特殊能力

**重要规则**：
- 每回合打出的卡牌数量不限，但受算力限制
- 数据源卡提供的算力只能在本回合使用
- AI程序、防火墙等永久效果持续到被摧毁

#### 阶段4：战斗阶段
- 如果玩家部署了攻击型AI程序，可以宣布攻击
- 攻击目标可以是其他玩家
- 防御方可以使用防火墙或护甲抵消伤害

**战斗流程**：
1. 攻击方宣布攻击目标和使用的攻击型AI程序
2. 防御方选择是否使用防火墙或护甲
3. 计算伤害：攻击力 - 护甲 = 实际伤害
4. 防御方生命值减少相应数值

#### 阶段5：结束阶段
- 玩家手牌上限为10张，超过的部分需要丢弃
- 触发回合结束效果
- 传递给下一位玩家

### 5.3 胜利条件

**主要胜利条件**：
- 成为最后一个生命值大于0的玩家

**附加胜利条件**（可选）：
- 率先对对手造成总计50点伤害
- 收集10张高阶AI程序卡
- 在对手身上植入5张病毒卡

## 6. 完整游戏流程示例

### 示例1：2人对战（完整第一回合）

**玩家A（先手）**：
1. **算力恢复阶段**：无算力可恢复（第一回合）
2. **抽牌阶段**：抽取4张牌（先手惩罚）
   - 获得：数据源卡×2、攻击型AI程序、病毒卡
3. **主要阶段**：
   - 打出1张数据源卡（提供2算力）
   - 消耗2算力，打出"基础攻击程序"（造成2点伤害）
4. **战斗阶段**：
   - 宣布攻击玩家B
   - 玩家B无防火墙，受到2点伤害（生命值：20→18）
5. **结束阶段**：
   - 手牌5张（未超过上限）
   - 结束回合

**玩家B（后手）**：
1. **算力恢复阶段**：无算力可恢复
2. **抽牌阶段**：抽取5张牌（后手优势）
   - 获得：数据源卡×3、防御型AI程序、黑客工具
3. **主要阶段**：
   - 打出2张数据源卡（提供3算力）
   - 消耗2算力，打出"基础防火墙"（获得2点护甲）
   - 剩余1算力无法使用（需要更多数据源）
4. **战斗阶段**：
   - 玩家B选择不攻击（护甲无用武之地）
5. **结束阶段**：
   - 手牌6张
   - 结束回合

### 示例2：多人对战中的策略互动

**3人对战场景**：

玩家A有8点生命值，手中有强力攻击程序。
玩家B有15点生命值，刚构建了高级防火墙。
玩家C有12点生命值，手中有病毒卡。

**玩家C的回合**：
1. 打出数据源卡（获得4算力）
2. 使用"窃取病毒"（消耗3算力）查看玩家A手牌
   - 发现玩家A有"终极攻击程序"
   - 选择丢弃该卡！
3. 用剩余1算力打出1张低阶AI程序（干扰用）
4. 结束回合

**结果**：玩家C成功破坏了玩家A的计划，保护了自己。

## 7. 新手策略提示

### 对于新手玩家

1. **优先构建经济**：
   - 前期多打出数据源卡，积累算力
   - 没有算力，再强的AI程序也无法部署

2. **平衡攻防**：
   - 不要只盯着攻击，适当的防火墙能让你存活更久
   - 建议攻防比例：60%攻击 + 40%防御

3. **合理使用病毒**：
   - 病毒是干扰对手的利器，不要舍不得用
   - 在对手即将打出强力AI程序时使用效果最好

4. **注意手牌管理**：
   - 手牌上限10张，及时使用不需要的卡
   - 不要让对手通过"窃取病毒"看到你的底牌

### 高级策略

1. **预判对手行动**：
   - 观察对手打出了什么数据源卡
   - 推测对手下回合能打出什么AI程序

2. **资源管理**：
   - 高阶数据源卡不要急着用，留到关键时刻
   - 黑客工具卡要用在刀刃上

3. **角色配合**：
   - 如果选择了"攻击者"角色，多构建攻击型AI程序
   - 角色能力是巨大的优势，要充分利用

4. **心理战**：
   - 故意保留算力不让对手知道你的计划
   - 虚张声势，让对手误判你的实力

## 8. 变体规则（可选）

### 8.1 团队模式（4人）

- 2v2组队
- 队友之间可以共享数据源卡
- 可以替队友承受伤害（每次最多3点）
- 胜利条件：摧毁对方队伍所有成员

### 8.2 生存模式

- 所有玩家对抗"系统AI"（由游戏规则控制）
- 系统AI每回合自动打出AI程序
- 玩家需要合作击败系统AI
- 如果系统AI被击败，剩余玩家中生命值最高的获胜

### 8.3 快速模式

- 起始生命值改为10点
- 起始手牌改为7张
- 每回合抽牌改为3张
- 游戏节奏更快，适合短时间娱乐

### 8.4 开放角色模式

- 允许玩家在游戏过程中切换角色
- 切换角色需要消耗5算力
- 增加策略深度和变数

## 9. 常见问题

### Q1：如果我的牌库和弃牌库都为空怎么办？
**A**：如果无法抽牌，跳过抽牌阶段。这是战术的一部分，要注意牌库管理。

### Q2：算力代币和手牌中的数据源卡有什么区别？
**A**：数据源卡在手牌中不能使用，需要打出到桌面上才提供算力。算力代币是打出数据源卡后获得的资源标记。

### Q3：防火墙和护甲有什么区别？
**A**：防火墙是卡牌，可以提供特殊效果；护甲是数值，只能抵挡伤害。有些AI程序可以提供护甲。

### Q4：可以先使用病毒卡，再攻击吗？
**A**：可以！主要阶段中操作顺序不限。建议先使用病毒干扰对手，再发动攻击。

### Q5：黑客工具卡是消耗品吗？
**A**：是的，使用后立即放入弃牌堆。

### Q6：如果我有足够的算力，一回合可以打出多个AI程序吗？
**A**：可以！只要算力足够，每回合出牌数量不限。

### Q7：游戏允许2人玩吗？和多人玩有什么区别？
**A**：允许2人玩。2人对战更考验1v1策略，多人玩更混乱有趣。建议2人时先手玩家只抽4张牌平衡优势。

### Q8：可以自己设计新的AI程序卡吗？
**A**：当然可以！鼓励玩家发挥创意，设计自己的AI程序和黑客工具。只要确保平衡性即可。
