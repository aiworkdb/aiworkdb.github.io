## 🎮 游戏概述

**AI梦境设计师**是一款 2-5 人的创意卡牌游戏。在2026年，AI定制梦境已成为现实，"梦境设计"成为最热门的职业之一。每位玩家扮演一名AI梦境设计师，接收不同客户的梦境订单，通过组合**场景卡**、**情绪卡**、**元素卡**来定制梦境内容，满足客户需求赢得满意度评分。

游戏的核心在于**创意组合与需求匹配**：每个客户有独特的梦境需求（场景偏好、情绪氛围、禁忌元素），你需要巧妙组合手牌，既要满足基本需求，又要添加惊喜元素超出客户预期，才能获得超高评分！

游戏适合喜欢创意表达、心理洞察的玩家，规则简单上手快，却有丰富的策略深度和重玩价值。

## 🎮 游戏规则

### 准备阶段

1. **洗牌**：将场景卡、情绪卡、元素卡分别洗匀，背面朝上放在桌中央作为三个牌堆。
2. **发订单**：洗匀客户订单卡，翻开第1张放在中央订单区。
3. **发手牌**：每位玩家抽取 **5 张手牌**（自由组合场景/情绪/元素，从三个牌堆各至少抽1张）。
4. **摆放面板**：每人取一张设计师面板放在自己面前。
5. **确定先手**：最近做过梦的玩家先手（或抽签决定）。
6. **满意度代币**：每人领取 **5 枚满意度代币**作为起始资源。

### 游戏流程

每轮梦境设计按以下步骤进行：

#### 第一步：接收订单 📋
中央订单区展示当前客户订单卡，所有玩家阅读需求（场景要求、情绪要求、额外偏好、禁忌元素）。

```
示例订单卡：
客户类型：冒险家
需求场景：自然系列（森林/山脉/岛屿）
需求情绪：刺激系
额外偏好：含有「飞行」元素 +3分
禁忌元素：含有「幽灵」元素 -5分
基础满意度：10分
```

#### 第二步：设计梦境 🎨
所有玩家**同时**从手牌中选择卡牌组合梦境：
- 必须包含：**1张场景卡** + **1张情绪卡**
- 可选添加：**0-2张元素卡**（增加惊喜度）
- 将选中的卡牌正面朝上放在自己面前的「已提交梦境区」

> ⚠️ **重要**：设计是同时进行的，玩家看不到其他人的设计，最后一起揭晓！

#### 第三步：揭晓评分 🔮
所有玩家同时揭开自己设计的梦境，按订单卡要求评分：

| 评分项目 | 分值 |
| :--- | :--- |
| 满足需求场景 | +基础分（订单卡标注） |
| 满足需求情绪 | +基础分（订单卡标注） |
| 包含额外偏好元素 | +偏好分（订单卡标注） |
| 触犯禁忌元素 | -禁忌分（订单卡标注） |
| 创意组合奖励（稀有卡牌） | +1-3分 |
| 超出预期（3张卡牌全上） | +2分 |
| 完全匹配客户类型 | +5分（隐藏加分） |

#### 第四步：结算与补牌 🔄
- 赢得最高评分的玩家获得**额外2分奖励**
- 所有玩家将已提交的梦境卡放入完成区（弃牌堆）
- 每位玩家补充手牌回到5张
- 翻开下一张订单卡

### 游戏结束

**客户订单卡耗尽**或**任意玩家达到50分**时，游戏结束。进行最终计分：

| 计分项目 | 分值 |
| :--- | :--- |
| 每轮获得的满意度分 | 累计 |
| 未使用的特效卡 | 每张+1分 |
| 拥有最多「传说」卡牌 | +5分 |
| 使用最多样情绪类型 | +3分 |

**总分最高**的玩家赢得游戏，成为"首席梦境设计师"！

## 📖 完整游戏流程示例

> **场景**：4 人游戏（小明、小红、大壮、莉莉），游戏进行到第 2 轮。

**当前局面：**
- 小明：已有满意度 12 分，手牌：🌲🏔️😊🦅🎵
- 小红：已有满意度 10 分，手牌：🌊🏙️😂🔮👻
- 大壮：已有满意度 8 分，手牌：🏝️🌋😱🏃‍♂️🪄
- 莉莉：已有满意度 15 分，手牌：🚀🌌🤩✨🎭

---

**第一步：接收订单**

中央订单区翻出第2张订单卡：
```
客户类型：探险家
需求场景：自然系列（森林/山脉/岛屿/火山）
需求情绪：刺激系
额外偏好：含有「飞行」元素 +3分
禁忌元素：无
基础满意度：12分
```

👀 小明心想：我有🌲（森林）和😱（刺激系情绪我有吗？哦不，我是😊），需要调整策略！

---

**第二步：设计梦境**

所有玩家同时选择卡牌组合：

- **小明**选择：🌲（森林场景）+ 😊（欢愉情绪）+ 🦅（飞行元素）
  - 问题：情绪不匹配（需要刺激系，他是欢愉系）
  
- **小红**选择：🌊（海洋场景）+ 😂（爆笑情绪）
  - 问题：场景是都市系不是自然系，情绪也不对
  
- **大壮**选择：🏔️（山脉场景）+ 😱（极致恐惧情绪）+ 🏃‍♂️（奔跑元素）
  - 场景✓ 情绪✓ 但没有飞行元素
  
- **莉莉**选择：🚀（不是自然系）+ 🤩（不是刺激系）
  - 完全不匹配

> 📌 实际游戏中，玩家需要在手牌限制下尽量匹配。第2轮手牌可能不理想，这时候**特效卡**就派上用场！

---

**第三步：揭晓与评分（假设使用了特效卡）**

小明使用**「预知未来」特效卡**，提前看到下一张订单卡是自然+宁静主题，于是保留手牌准备下轮。

小红使用**「灵感爆发」特效卡**，多抽2张手牌，获得了😱和🏝️，重新组合！

**最终提交：**
- 小明：🌲+😊+🦅 = 基础12 + 飞行+3 = 15分（情绪不匹配扣3分）= 12分
- 小红：🏝️+😱+🦅 = 基础12 + 飞行+3 = 15分（使用灵感爆发重新抽牌）
- 大壮：🏔️+😱 = 基础12分（没有飞行元素）
- 莉莉：使用「情绪感染」复制小红的😱 = 🚀+😱 = 基础12分

**本轮最高分：小红 15分，获得额外2分奖励 = 17分！**

---

**游戏策略提示：**
1. **保留特效卡**：关键时刻使用，不要早早用完
2. **情绪卡最关键**：情绪不对，场景再美也扣分
3. **飞行元素万能**：多数订单偏爱飞行元素，优先保留
4. **观察对手**：用「情绪感染」复制对手的好卡牌

## 💡 新手策略提示

1. **优先满足基本需求**：场景+情绪必须匹配，否则扣分严重！
2. **手牌管理是关键**：不要一次性打出所有好牌，留一些应对后续订单。
3. **特效卡扭转乾坤**：「灵感爆发」让你多抽2张，是逆袭的神卡！
4. **观察订单趋势**：如果连续3个订单都是自然系列，赶紧把都市系列卡牌换掉。
5. **风险与回报**：刺激系情绪加分高，但如果客户是「宁静派」就会扣分，看清订单！
6. **组合奖励**：使用3张卡牌（场景+情绪+元素）有额外+2分，尽量凑齐。

## 🔄 变体规则

### 变体一：团队模式（适合 4 人，2v2 组队）
- 两人一队，共享手牌池（10张手牌）
- 每轮只能一人提交梦境，队友可以「友情提示」（限制1次）
- 两队总分高者获胜

### 变体二：专家模式（适合 3-5 人老手，增加复杂度）
- 订单卡增加「隐藏需求」：需要玩家推理客户真正想要什么
- 增加「梦境崩溃」机制：如果连续2轮评分低于5分，强制丢弃所有手牌重抽
- 特效卡使用次数限制为每游戏3次

### 变体三：快速模式（适合 2-3 人，缩短游戏时间）
- 订单卡只使用15张
- 起始手牌增加到7张
- 先达到30分者获胜

### 变体四：梦境大师模式（适合 1 人，单人挑战）
- 单人面对10个订单，目标达到80分
- 限制手牌数为5张，且每轮只能补充2张新牌
- 适合练习策略和手牌管理

## ❓ 常见问题

**Q：如果手牌完全没有匹配的怎么办？**
A：可以使用特效卡「灵感爆发」多抽2张，或者使用「预知未来」跳过这张订单（扣分-3分但不匹配总比得0分好）。

**Q：可以同时提交多个梦境吗？**
A：不可以，每轮只能提交1个梦境组合（1场景+1情绪+最多2元素）。

**Q：特效卡什么时候使用？**
A：特效卡可以在「设计梦境」阶段前宣告使用，效果持续当轮。每玩家每轮最多使用1张特效卡。

**Q：如果多个玩家同分怎么办？**
A：比较谁使用了更多「稀有」卡牌，再同则比较剩余手牌数，最终仍同则并列胜利。

**Q：打印卡牌时颜色不对怎么办？**
A：确保在打印设置中勾选「背景图形」，并使用卡纸打印。如果颜色仍然偏差，调整打印机色彩设置。

**Q：可以自己设计新的场景/情绪/元素卡吗？**
A：当然！游戏鼓励创意扩展，你可以添加自己设计的卡牌，只要确保平衡性即可。
