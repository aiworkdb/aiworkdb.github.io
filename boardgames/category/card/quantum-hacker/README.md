# 量子黑客 (Quantum Hacker)

## 🎮 游戏概述

《量子黑客》是一款赛博朋克风格的卡牌策略桌游。公元2099年，量子网络"OmniNet"连接着全人类的意识——但失控的AI"刻耳柏洛斯（Cerberus）"正在吞噬人类意识，将其转化为数字奴隶。

玩家扮演**量子黑客**，潜入OmniNet的核心节点，部署程序、构筑防火墙、对抗AI守护进程，最终收集足够的量子密钥，关闭Cerberus，拯救人类意识！

本游戏灵感来源于《赛博朋克2077》《黑客帝国》《攻壳机动队》等经典赛博朋克作品，将黑客攻击、网络防御、资源管理等元素融入快节奏卡牌对战。

适合2-4人游玩，单局30-50分钟，推荐年龄12岁以上。

## 🎯 游戏目标

**合作模式**：所有玩家共同对抗Cerberus，在AI吞噬所有人类意识（第15回合）之前，收集**15个量子密钥**（合计）并成功关闭Cerberus！

**对抗模式**：每位黑客各自为战，第一个收集到**5个量子密钥**的玩家获胜！但小心——Cerberus会攻击所有人！

## 🎲 游戏规则

### 准备阶段

1. **洗牌**：将程序卡（60张）、防火墙卡（30张）、节点卡（20张）、AI卡（15张）分别洗混，各自形成牌堆。
2. **布置网络**：从节点牌堆翻开4张，暗置（面朝下）放在桌面中央的"网络区域"。
3. **发牌**：每位玩家抽取5张起始手牌（只能包含程序卡和防火墙卡）。
4. **初始状态**：每位玩家HP=10，DC=2（每回合获得的基础数据信用），量子密钥=0。
5. **先手决定**：掷骰子，点数最大者先手。

### 游戏流程（每个回合）

每位玩家轮流进行，每个完整回合包含以下**五个阶段**：

#### 阶段1：数据信用收集
- 玩家获得DC = 2 + 当前量子密钥数
- DC用于支付本回合程序卡的费用

#### 阶段2：抽卡阶段
- 抽取2张卡（从程序卡牌堆）
- 手牌上限为7张，超过需弃至7张

#### 阶段3：行动阶段（核心！）
玩家可以按任意顺序执行以下行动（次数不限，受DC和手牌限制）：

**🗡️ 攻击节点**
1. 选择网络区域中的1张暗置节点卡，将其翻面（首次攻击需翻面）
2. 支付程序卡的费用（从DC中扣除）
3. 对节点造成程序卡标示的攻击力伤害
4. 节点立即反击，对攻击者造成1点HP伤害
5. 如果节点HP归零 → 获得节点奖励，将节点卡移入弃牌堆，然后从节点牌堆补充1张新节点到网络区域
6. 攻击后，有机会触发AI守护进程（掷d6，点数≤3则抽1张AI卡并执行效果）

**🛡️ 部署防火墙**
- 支付防火墙卡的费用（DC），将防火墙面朝上放在自己面前
- 防火墙可以累积，最多同时部署3张
- 当受到伤害时，可以消耗防火墙来抵消伤害

**💫 使用特殊程序**
- 按照程序卡描述执行特殊效果
- 注意费用消耗和手牌管理

#### 阶段4：AI阶段（合作模式专用）
- 如果是合作模式，此阶段Cerberus发动反击
- 从AI牌堆抽1张AI卡，对所有玩家执行效果
- 回合计数器+1，达到15回合时Cerberus获胜（玩家失败）

#### 阶段5：弃牌/整理阶段
- 将手牌弃至7张（如有超过）
- 将使用过的防火墙卡移入弃牌堆
- 回合结束，下一位玩家开始

### 胜利条件

| 模式 | 胜利条件 | 失败条件 |
| :--- | :--- | :--- |
| **合作模式** | 玩家合计收集15个量子密钥，并成功执行"关闭Cerberus"行动（消耗5密钥） | Cerberus在15回合内未被关闭；或任意玩家HP归零 |
| **对抗模式** | 第一位收集5个量子密钥的玩家获胜 | 玩家HP归零则退出游戏 |

## 📖 完整游戏流程示例

### 示例1：对抗模式（2人游戏）

**玩家A（先手）第1回合：**
1. **数据信用收集**：DC=2（密钥数0，基础2）
2. **抽卡阶段**：抽2张，手牌共5张（起始3+新抽2）
3. **行动阶段**：
   - 使用「数据采矿」（费用1）→ 获得2DC，现在DC=3
   - 使用「DDoS冲击」（费用1，攻击力2）攻击网络区域的节点#1
   - 翻面节点#1 = 「边缘路由器」（HP=3）
   - 造成2点伤害，节点剩余HP=1
   - 节点反击 → 玩家A受到1点HP伤害（HP: 10→9）
   - 掷d6=4，不触发AI守护进程
4. **弃牌阶段**：手牌剩3张，不需弃牌
5. **回合结束**

**玩家B第1回合：**
1. **数据信用收集**：DC=2
2. **抽卡阶段**：抽2张
3. **行动阶段**：
   - 使用「漏洞利用」（费用2，攻击力3）攻击节点#1
   - 节点#1剩余HP=1，受到3点伤害 → HP归零！
   - 获得奖励：1量子密钥 + 抽1张卡
   - 节点反击 → 玩家B受到1点HP伤害（HP: 10→9）
   - 掷d6=2，触发AI守护进程！抽AI卡 = 「杀毒扫描」→ 所有玩家受到1点伤害（A: 9→8, B: 9→8）
   - 从节点牌堆补充新节点到网络区域
4. **回合结束**，玩家B目前领先1个量子密钥！

### 示例2：合作模式（3人游戏，第8回合）

**当前状态**：玩家合计密钥=9，回合计数=8，Cerberus剩余7回合反击时间。

**玩家C的回合：**
1. **数据信用收集**：玩家C有2个密钥，DC=4
2. **抽卡阶段**：抽2张
3. **行动阶段**：

   - 合计密钥数达到12！还差3个就能尝试关闭Cerberus！
4. **AI阶段**：抽AI卡 = 「Cerberus怒火」→ 所有玩家受到2点伤害！
5. **回合结束**

## 💡 新手策略提示

1. **平衡攻击与防御**：不要只顾攻击节点，留足DC和防火墙来应对AI反击！
2. **优先hack低HP节点**：边缘路由器（HP=3）是前期积累密钥的最佳选择。
3. **合理利用工具型程序**：「碎片重组」和「深度搜索」能让你更快找到关键卡。
4. **合作模式多沟通**：告诉队友你下回合的计划，协调攻击顺序，避免重复攻击同一高HP节点。
5. **对抗模式要干扰对手**：使用病毒卡干扰领先玩家，但别让自己成为Cerberus的首要目标！
6. **量子密钥是一把双刃剑**：密钥越多，每回合DC越高，但触发「Cerberus怒火」的几率也越大！
7. **「意识上传」是一张高风险高回报的卡**：只在你HP较低（≤3）且接近胜利时使用！

## 🔄 变体规则

### 变体1：硬核模式（合作模式）
- 回合限制缩短为**10回合**
- 所有节点HP+2
- AI守护进程触发几率提高（掷d6，点数≤4即触发）
- 适合有经验的玩家挑战

### 变体2：自由混战（对抗模式升级）
- 允许玩家之间互相攻击！
- 使用病毒卡和攻击型程序直接削减对手HP
- 第一个收集5密钥**且HP>0**的玩家获胜
- 增加互动性和对抗性

### 变体3：故事模式（单人/合作）
- 使用预先设计好的"任务卡"代替随机节点
- 每个任务有特定目标和特殊规则
- 例如：「潜入Tower」：在8回合内hack 3个核心服务器
- 适合新手学习游戏机制

### 变体4：快速对局
- 胜利条件改为3个量子密钥（对抗模式）或10个（合作模式）
- 节点奖励翻倍
- 单局时间缩短至15-25分钟
- 适合短时间娱乐

## ❓ 常见问题

**Q：如果程序卡牌堆用完了怎么办？**
A：将弃牌堆洗混后作为新的牌堆继续使用。

**Q：量子密钥有上限吗？**
A：没有上限！密钥越多，每回合DC越高，但也会吸引Cerberus的更多注意（AI卡效果更强）。

**Q：防火墙卡可以防御AI守护进程的伤害吗？**
A：可以！部署在场上的防火墙可以用来抵消任何来源的伤害，包括AI守护进程。

**Q：节点被攻击后，下次还能攻击同一节点吗？**
A：可以，节点在HP归零前一直存在。多次攻击同一节点是积累伤害的好方法。

**Q：没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片随机抽取。

**Q：对抗模式中，如果有玩家HP归零了怎么办？**
A：该玩家退出游戏，其手中的量子密钥平均分配给其他存活玩家。

**Q：合作模式中，玩家可以传递卡牌吗？**
A：基础规则不可以。但变体规则中可以允许玩家之间传递多余的资源卡（需消耗1DC作为传输费用）。

**Q：「意识上传」卡的效果是什么？**
A：这是一张特殊胜利卡——当你的HP≤3时，使用此卡可以立即获胜！但HP>3时使用无效（并浪费4点费用）。这是一张高风险高回报的"绝地反击"卡。
