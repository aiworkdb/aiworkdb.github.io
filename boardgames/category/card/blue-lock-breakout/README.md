# ⚽ 蓝色禁区：前锋突围（Blue Lock: Striker Breakout）

> 「只有世界第一的前锋，才有存在的意义。」—— Blue Lock 训练营

## 1. 游戏概述

《蓝色禁区：前锋突围》是一款 **2-5 人**的卡牌竞技桌游，灵感来源于现象级足球动漫 **《蓝色禁区》（Blue Lock）** 与 **2026 年世界杯**的热潮。

在 Blue Lock 训练营里，300 名前锋被关在一起，唯一的目标只有一个——**成为世界上最自私、最强大的前锋**。游戏里你就是其中一名"自我主义前锋"：用「自我」资源点燃野心，靠技术卡锤炼射门、盘带与速度，突破后卫与门将的重重拦截，在禁区内完成致命一击。

- 你越强、越自私，就越能碾压对手；
- 率先攻入 **5 球**者，加冕为「世界第一前锋」；
- 足球是团队运动？在这里，不。这里只有你，和你脚下的球。

**适合人群**：家庭聚会、朋友派对、球迷酒局；喜欢轻策略、资源管理与心理博弈的玩家。规则直观，新手 5 分钟即可上手。

## 4. 游戏地图详解

本作采用一条**纵向球场轨道**代表进攻路线：足球从你的球门出发，逐级推进到对方球门。地图不是战斗棋盘，而是一条"进球之路"。

### 地图布局概览（球向上方推进）

```
              ⚽ 球门 (GOAL)         ← 射门成功即进球
            ┌──────────────┐
            │  🧤 门将区    │        ← 射门判定区：对比射门值 vs 门将扑救值
            ├──────────────┤
            │  🔵 禁区      │        ← 球到此处，可发起射门
            ├──────────────┤
            │  🟡 突破区    │        ← 需"速度"≥2 才能进入禁区
            ├──────────────┤
            │  ⚪ 中场      │        ← 中立区，无特殊效果
            ├──────────────┤
            │  🟢 传球区    │        ← 回合开始若球在此，额外抽 1 张
            ├──────────────┤
            │  🥅 己方球门  │        ← 起点；被抢断后回到此处
            └──────────────┘
```

### 地形类型说明表格

| 区域 | 图示 | 效果 |
| :--- | :--- | :--- |
| 己方球门 | 🥅 | 足球的出发点；被对手抢断后，球退回此处 |
| 传球区 | 🟢 | 你的回合开始时若球在此，额外抽 1 张技术卡 |
| 中场 | ⚪ | 中立区，无特殊效果 |
| 突破区 | 🟡 | 想把球送入"禁区"，本回合累计"速度"须 ≥ 2，否则停在突破区 |
| 禁区 | 🔵 | 球到达此处即可发起"射门" |
| 门将区 | 🧤 | 射门判定区：你的射门总值 vs 门将/拦截卡的扑救值 |
| 球门 | ⚽ | 射门成功即进球，自我值 +2，球退回己方球门 |

### 地图特点和资源统计

- 全程 **7 个区域**，需逐格推进，无法直接"瞬移"到球门。
- **自我之源**：传球区（抽牌）与自我卡（加自我）是主要补给点。
- 球的推进由你的**速度值**决定：每点速度让球前进 1 格（受突破区限制）。
- 实战中可用前锋棋子代表"持球前锋"在地图上移动，更直观。

## 6. 游戏规则

### 准备阶段

1. **布置地图**：把球场轨道（见第 4 节）放在桌面中央，足球棋子放在「己方球门」。
2. **分发面板**：每人 1 块前锋面板，初始基础值：**射门 1 / 速度 1 / 技术 1**，进球 0。
3. **发棋子**：每人选 1 枚前锋棋子（5 色），放在地图起点旁代表你自己。
4. **洗牌**：
   - 自我堆（16 张）、技术堆（20 张）洗匀放桌面；
   - 拦截堆（14 张）洗匀，作为"门将牌堆"；
   - 必杀堆（6 张）洗匀，作为隐藏奖励堆。
5. **起手**：每人抽 **4 张技术卡** + **2 张自我卡**（共 6 张手牌）。自我值为 **0**。
6. 随机决定先手（建议掷骰子，点数大者先）。

### 游戏流程（每个回合的操作）

轮到你的回合，按顺序执行：

1. **抽牌阶段**：从技术堆抽 1 张；若球在「传球区」额外抽 1 张。手牌上限 **7 张**（超出须弃至 7）。
2. **自我阶段**：可打出任意数量的自我卡，累加自我值（🔥）。
3. **推进阶段**：打出技术卡累积本回合的**速度**与**射门**；用速度让足球向上推进（每点速度前进 1 格，突破区需速度 ≥2 才进入禁区）。
4. **射门阶段**（可选）：当球到达「禁区」或「门将区」，你可发起射门：
   - 计算 **射门总值** = 基础射门 + 本回合技术加成 + 必杀加成；
   - 从拦截堆翻 **1 张**作为门将扑救值（🧤）；
   - 若 射门总值 > 扑救值 → **进球！** 进球 +1，自我 +2，球退回己方球门；
   - 若 ≤ 扑救值 → 射门失败，球退回「中场」。
5. **结束阶段**：弃掉本回合所有"本回合"临时加成；轮到下一位玩家。

> 提示：你不一定要每回合射门。攒资源、堆技术、找好时机一击致命，是高手玩法。

### 胜利条件

- **率先攻入 5 球**的玩家，立即成为「世界第一前锋」，游戏结束。
- 若某堆牌被抽空且无人达 5 球：游戏结束，**进球最多者胜**；若并列，**自我值最高者胜**；仍并列则加赛 1 球决胜。

## 7. 完整游戏流程示例

> 以下用 **3 人局（玩家 A / B / C）**演示前两回合，让你一看就懂。

### 回合 1 —— 玩家 A

- **抽牌**：技术堆抽 1 张（手牌 6→7，达到上限，不再额外抽）。球在己方球门，不额外抽。
- **自我**：打出「自私突破」🔥，**自我 +2**（现为 2）。
- **推进**：打出「高速盘带」⚽（速度 +2）、「风之步」🌬️（速度 +3）→ 本回合速度 5。足球前进 5 格：己方→传球→中场→突破→**禁区**（突破区速度≥2 满足）。
- **射门**：球在禁区，发起射门。基础射门 1 + 本回合「暴力抽射」🦶（射门+3）= 4。翻拦截堆得「飞身扑挡」🤲（扑救值 3）。4 > 3 → **进球！** 进球 1，自我 +2（现为 4）。球退回己方。
- 结束，临时加成清零。

### 回合 2 —— 玩家 B（试图干扰）

- **抽牌**：抽 1 张技术卡；球不在传球区，不额外抽。
- **自我**：打出「挑衅对手」🗯️，**自我 +1**（现为 1）。
- **推进**：打出「精准传球」🎯（速度+1、射门+1）→ 速度 2，前进 2 格到中场。
- **不射门**：球未到禁区，选择保留自我，下回合再冲。
- 结束。

> 新手注意：玩家 B 看到 A 已进 1 球，于是选择**攒资源**而非冒进——这正是正确策略。先到禁区、堆够射门值，再一次性破门，比每回合勉强起脚更高效。

## 8. 新手策略提示

1. **先到禁区，再谈射门**：速度是命根。没有速度，球永远卡在中场，空有射门值也发不出。
2. **自我是弹药**：没有自我值，技术卡打不出。前期多攒「自我卡」，别急着花光。
3. **假动作是万金油**：几乎能抵消所有拦截惩罚，见到「后卫包夹/钢铁门将」前先铺一张。
4. **别每回合都射**：射门失败球会退回中场，节奏断档。攒到射门总值 ≥ 扑救值 +2 再出手更稳。
5. **必杀卡留到关键球**：「世界波」「流星射门」能无视惩罚必进，留到第 4、5 球时一锤定音。
6. **盯紧对手进球数**：有人到 4 球时，优先用「团队背叛」偷他自我，或用「绝杀」反超。
7. **装备卡（铁靴）越早打越好**：射门永久 +1，滚雪球效应明显。

## 9. 变体规则

- **世界杯狂欢（快速局）**：胜利目标降为 **3 球**，时长压缩到 15 分钟，适合聚会热身。
- **全能前锋（新手友好）**：去掉"突破区需速度 ≥2"限制，球畅通无阻，专注学射门判定。
- **无情淘汰（硬核局）**：被抢断（滑铲）后，自我值减半；率先 7 球胜，拉长博弈。
- **双人死斗**：仅 2 人，各发 5 张起手，拦截堆由双方轮流当"门将"翻牌，心理战更浓。
- **队魂模式（轻合作）**：2 人组队对抗另外 2 人，队友可共享自我卡，但仍只有 1 人能封王（组内先到 5 球者胜）。

## 10. 常见问题

**Q1：自我值上限是多少？**
A：无上限，但手牌上限 7。自我多了就尽快打出技术卡转化为场上优势。

**Q2：球在中场，我能直接射门吗？**
A：不能。射门只在「禁区」或「门将区」允许。先把球推进上去。

**Q3：翻到的拦截卡（门将）扑救值太高打不进怎么办？**
A：用「假动作」减其扑救值，或下回合堆更高的射门总值，或留必杀卡「世界波/流星射门」无视惩罚。

**Q4：技术卡的加成是永久还是仅本回合？**
A：默认**仅本回合**（卡面写"本回合"）。只有标注"装备/永久"的（如铁靴）才永久生效。

**Q5：必杀卡怎么拿到？**
A：开局必杀堆洗匀放一旁；每当你从技术堆抽牌时，有概率（建议：每第 3 次抽牌）改从必杀堆抽 1 张；也可在规则变体中直接每人发 1 张起手。

**Q6：多人局先手太强怎么办？**
A：采用"逆序补偿"——每回合自我阶段，非先手玩家额外 +1 自我（最多 +2），平衡节奏。

**Q7：牌堆抽空了怎么办？**
A：把已弃的对应牌堆洗回继续用；仍无法达成 5 球则按第 6 节"胜利条件"的并列规则结算。
