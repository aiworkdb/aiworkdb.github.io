🏛️ 宫廷药师 · 打印文档

Court Apothecary — 卡牌策略桌游 · 灵感源自《药屋少女的呢喃》
⚠️ 重要打印提示：
  • 使用 A4 纸张 打印，建议设置 最小边距。
  • 在打印对话框中，务必 勾选「背景图形」 选项，否则地图与卡牌的颜色背景将不会打印。
  • 卡牌与棋子建议使用 卡纸（160-200g） 打印，增加耐用性。
  • 游戏地图可用普通纸打印；卡牌打印后沿边框裁剪，建议塑封。
  • 按 README 物料清单的「建议打印数量」决定每种卡牌的打印份数。

📋 使用说明

  • 打印地图：将后宫地图打印于桌面中央，作为行动区域。
  • 裁剪卡牌：沿卡牌边框裁剪，按 README 物料清单的份数打印；建议塑封。
  • 准备棋子：裁剪玩家棋子，或用硬币 / 围棋子代替。
  • 数值记录：每位玩家用纸笔或骰子记录「声望」与「健康（初始 3）」。
  • 开始游戏：所有棋子置于御药房，洗好各牌库、翻开首张毒案，发起始手牌即可开局。
