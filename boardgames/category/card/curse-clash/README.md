## 📖 游戏概述

在咒力横行的现代都市中，你是隶属于咒术高教的咒术师。潜伏的咒灵四处作乱，而你必须与同伴并肩作战，或是在切磋中磨练技艺。

**咒力对决**是一款1v1或2v2的卡牌对战桌游，灵感源自近年最热门的黑暗奇幻动漫。玩家扮演咒术师，构筑自己的术式牌组，运用咒力释放攻击术式、防御术式，甚至展开领域击败对手。

适合人群：动漫爱好者、卡牌游戏爱好者、2-4人的朋友聚会。

```

│  🔵 前卫区   │  🟡 中阵区   │  🔴 深阵区   │
```

**区域说明：**

| 区域 | 效果 |
| :--- | :--- |
| 前卫区 | 近战攻击术式在这里造成+1额外伤害 |
| 中阵区 | 无特殊效果，所有术式均可正常使用 |
| 深阵区 | 在此区域展开领域时，领域效果持续2回合（而非1回合） |

## 🔄 游戏规则

### 准备阶段

1. **决定阵营**：2人时直接1v1；4人时两两组队（队友相对而坐）。
2. **分发起始卡牌**：每位玩家随机抽取5张术式卡作为起始手牌。
3. **设置起始状态**：
   - HP：10点（用生命指示物或纸笔记录）
   - 咒力：起始3点（摆放3个咒力指示物）
4. **决定先手**：掷骰子，点数大者先手。

### 回合流程

每位玩家的回合按以下顺序进行：

#### 1. 咒力恢复阶段
- 玩家获得2点咒力（咒力上限为8，超过部分不累积）
- 如HP≤3，额外获得1点咒力（濒死激发）

#### 2. 抽牌阶段
- 从共用牌堆抽取1张术式卡，加入手牌
- 如牌堆已空，将弃牌堆洗混后作为新牌堆

#### 3. 术式发动阶段（核心阶段）
玩家可按任意顺序发动以下操作（次数不限，只要咒力足够）：
- **发动术式卡**：支付咒力，从手牌打出术式卡
- **装备咒具卡**：支付咒力，从手牌将咒具卡横置于面前（每件咒具限装备1件）
- **展开领域**：支付5-7点咒力，打出领域卡（每玩家限同时展开1个领域）

#### 4. 战斗阶段
- 玩家选择敌方目标，使用攻击术式造成伤害
- 敌方可使用防御术式抵挡（在攻击宣言后、伤害结算前）
- 领域效果在战斗阶段持续生效

#### 5. 结束阶段
- 检查是否有术式效果需要在回合结束时触发
- 将本回合使用过的术式卡（除装备卡外）放入弃牌堆
- 传递回合给下一位玩家

### 胜利条件

- **1v1模式**：将对手HP降至0
- **2v2模式**：将对方队伍所有成员HP降至0（队友可互相治疗）

## 📝 完整游戏流程示例

### 示例1：标准1v1对战（前3回合）

**【回合1 - 玩家A（先手）】**
1. 咒力恢复：起始3点 → 获得2点 → 共5点咒力
2. 抽牌：从牌堆抽1张，手牌现为6张
3. 术式发动：打出「冲击术式」（消耗1咒力，剩余4点），再打出「咒力护盾」（消耗1咒力，剩余3点）
4. 战斗阶段：使用「冲击术式」攻击玩家B，造成2点伤害（玩家B：HP 10→8）
5. 结束阶段：将「冲击术式」放入弃牌堆，「咒力护盾」已消耗也放入弃牌堆

**【回合1 - 玩家B】**
1. 咒力恢复：3点 → 获得2点 → 共5点
2. 抽牌：手牌+1
3. 术式发动：打出「治愈术式」（消耗2咒力），恢复3点HP（HP 8→10→实际回到10上限）
4. 战斗阶段：使用「咒力贯穿」（消耗2咒力）攻击玩家A，造成3点伤害（玩家A：HP 10→7）
5. 结束阶段：弃牌

**【回合2 - 玩家A】**
1. 咒力恢复：4点（上回合剩余3点+本回合恢复2点，但上限8点，实际为7点？→ 重新计算：上回合结束剩余3点，本回合+2 = 5点）
2. 抽牌
3. 术式发动：积攒咒力，打出「游云」（咒具，消耗2点），装备后攻击术式永久+1
4. 战斗阶段：使用「冲击术式」（现造成2+1=3点伤害）攻击玩家B（HP 10→7）
5. 结束阶段

> **提示**：实际游戏中，玩家需根据手牌灵活决策——是积攒咒力准备展开领域，还是主动进攻？

## 💡 新手策略提示

1. **咒力管理是关键**：不要每回合把所有咒力都用完，留1-2点应对突发情况
2. **先手优势要利用**：先手玩家应在前2回合积极进攻，建立HP优势
3. **防御也是进攻**：适时使用「治愈术式」保持HP优势，比单纯攻击更有效
4. **领域是转折点**：攒够5-7点咒力展开领域往往能逆转战局，不要舍不得用
5. **装备咒具要及早**：「游云」等咒具越早装备，整场游戏收益越大
6. **观察对手手牌**：留意对手打出了哪些术式，推测其剩余手牌类型
7. **2v2要配合**：队友之间可以"集火"同一目标，快速减员形成人数优势

## 🎲 变体规则

### 变体1：限制术式模式
每位玩家在游戏开始前，从共用牌堆中**禁止3张卡牌**（放入禁牌堆），增加策略深度。

### 变体2：剧情模式（合作PVE）
1位玩家扮演"特级咒灵"（BOSS），其他玩家合作对抗。BOSS有特殊规则：
- HP为普通玩家的2倍
- 每回合自动恢复3点HP
- 拥有专属BOSS卡牌（可用空白卡自制）

### 变体3：快速对决（15分钟版）
- 起始HP改为6点
- 每回合咒力恢复改为3点
- 起始手牌改为7张
- 先手玩家额外获得1点起始咒力

### 变体4：多领域共存
允许战场上同时存在多个领域（原规则限1个），但每个额外领域需多消耗1点咒力维持。

## ❓ 常见问题

**Q1：咒力上限是8点，如果我本回合结束时有7点咒力，下回合恢复2点后是9点，怎么算？**
> A：咒力上限为8点，超过部分不累积。下回合你将有8点咒力。

**Q2：我可以装备多件咒具吗？**
> A：可以，但同名咒具不能同时装备2件。不同名的咒具可以同时装备。

**Q3：领域卡打出后，什么时候失效？**
> A：领域效果持续到你下个回合结束（即持续2个你的回合）。如果在深阵区展开，则持续3个你的回合。

**Q4：防御术式必须在对手攻击前就打出吗？**
> A：不需要提前打出。在对手"攻击宣言"后、伤害结算前，你都可以打出防御术式回应。

**Q5：牌堆和弃牌堆都空了怎么办？**
> A：游戏进入"咒力枯竭"状态，玩家无法抽牌。此时每回合咒力恢复+1作为补偿。如连续3回合无人获胜，则HP高者获胜。

**Q6：2v2模式中，队友之间可以使用防御/治愈术式互相帮助吗？**
> A：可以！这是2v2模式的核心策略——队友之间可以互相使用治愈术式和防御术式（需声明目标）。

**Q7：打印出来的卡牌大小有要求吗？**
> A：建议使用标准扑克牌尺寸（63mm×88mm）打印，方便手持。也可使用A4纸打印后裁剪。
