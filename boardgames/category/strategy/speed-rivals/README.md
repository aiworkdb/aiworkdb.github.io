# 🏎️ 极速争霸

> *灵感来源：F1赛车 / 赛车竞速*

**类别**：策略游戏 | **人数**：2-5人 | **时长**：40-60分钟 | **难度**：中等

---

## 📖 游戏背景

世界赛车锦标赛拉开帷幕！作为一支车队的首席策略师，你需要管理车手、调配资源、制定进站策略，在变幻莫测的赛道上争夺年度总冠军。速度不是唯一的胜负手——轮胎、燃油、天气、对手的每一步都可能是翻盘的契机。

---

## 🎯 游戏目标

通过参加多场大奖赛积累**冠军积分**，赛季结束时积分最高的车队获胜。

## 🔄 游戏流程

### 赛季结构
一场游戏 = 1个赛季 = 4场大奖赛

每场大奖赛分为3个阶段：

### 一、排位赛

1. 每位玩家秘密选择轮胎配方并扣放
2. 同时翻开，投速度骰+轮胎加值+赛道加值
3. 按结果排定发车位（1-5位）
4. 发车位优势：每领先1位，正赛第1圈速度+1

### 二、正赛（每圈4个阶段）

#### 1. 速度阶段
- 每位玩家投速度骰
- 速度值 = 骰面 + 轮胎加值 + 赛道加值 + 燃油加值
- 燃油加值：满油-1，半油0，轻油+1

#### 2. 超车阶段
- 速度值高于前车的玩家可尝试超车
- 超车判定：投骰4+成功，失败则保持原位
- 弯道中超车需5+（更难）
- 成功超车：交换位置

#### 3. 策略阶段（可选，每圈1次）
可选择执行以下策略之一：

| 策略 | 效果 |
|:---|:---|
| 进站换胎 | 丢失2个位置，更换轮胎配方 |
| 推进引擎 | 本圈速度+3，但燃油消耗+1 |
| 节省燃油 | 本圈速度-1，燃油消耗-1 |
| DRS超车 | 直道上超车判定变为3+ |
| 防守位置 | 本圈不可被超车（除非对方速度高出5+） |

#### 4. 消耗阶段
- 轮胎磨损+1圈计数
- 燃油消耗-1格
- 检查天气是否变化

## 🌦️ 天气系统

| 天气 | 效果 |
|:---|:---|
| ☀️ 晴天 | 无特殊效果 |
| 🌧️ 雨天 | 所有非雨胎速度-2；雨胎正常 |
| ⛅ 变天 | 每圈投骰决定天气（4+变雨，否则转晴） |

## 🏆 游戏结束

4场大奖赛结束后，统计年度总积分。

**额外奖励**：
- 年度冠军车手（单场冠军最多）：+10分
- 最稳定完赛（从未爆胎/退赛）：+5分
- 最佳策略师（进站次数最少且每场前3）：+5分

**积分最高者获得年度总冠军！**

## 🎖️ 进阶规则

### 车手特性
每位车手有独特能力，每场大奖赛可使用1次：

| 车手 | 代号 | 能力 |
|:---|:---|:---|
| 闪电 | ⚡ | 单圈速度+4 |
| 铁壁 | 🛡️ | 本圈绝对不可被超车 |
| 幽灵 | 👻 | 不投骰直接超车1次 |
| 智者 | 🧠 | 查看下一张天气卡 |
| 拼命三郎 | 🔥 | 爆胎不退赛，速度-2继续跑完 |

### 引擎模式
每场比赛可选择引擎模式：
- **耐久模式**：轮胎磨损-1，但最高速度-1
- **冲刺模式**：最高速度+2，但轮胎磨损+1
- **均衡模式**：无加成无减益

---

## 💡 策略提示

1. **轮胎窗口**：软胎冲刺+硬胎长跑是经典策略，但要考虑赛道特性
2. **进站时机**：领先时不要先进站，让对手先进站暴露策略
3. **天气赌博**：变天赛道带雨胎是大胆但可能翻盘的选择
4. **燃油管理**：后半程轻油加速是超车的好时机
5. **积分思维**：不必每场争冠，稳定前3比一场冠军+一场退赛更划算

---

## 📐 变体规则

- **冲刺赛模式**：每场只有2圈，快速决胜负
- **车队对抗**：2人一组，队友互相配合（让车、挡人）
- **传奇模式**：使用真实F1赛道数据，更真实的物理模拟
