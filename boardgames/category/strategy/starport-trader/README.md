# 星际贸易站 (Starport Trader)

## 🎮 游戏概述

《星际贸易站》是一款融合引擎构筑、工人放置和动态市场机制的策略桌游。在2347年的银河系，人类已建立庞大的星际贸易网络。玩家扮演雄心勃勃的星际贸易商，在繁华的太空贸易站上建立自己的商业帝国。

通过建立高效的生产线、雇佣专业工人、把握市场时机、完成贸易合同，你将在激烈的商业竞争中脱颖而出，成为最富有的贸易大亨。

这款游戏完美结合了德式策略游戏的深度和美式游戏的爽快感，每局游戏的市场变化都让玩家面临全新的挑战和决策。

## 🎲 游戏规则

### 准备阶段

1. **布置版图**：将贸易站版图放在桌面中央
2. **设置市场价格**：将4种资源的价格标记放在初始位置（能源3、矿物4、食物2、数据5）
3. **分发玩家面板**：每位玩家获得1个玩家面板，用于建造建筑和存储资源
4. **初始资金**：根据玩家人数分发起始信用点（2人20💰、3人15💰、4人10💰）
5. **初始工人**：每位玩家获得起始工人（2人3个、3人4个、4人5个）
6. **初始建筑**：每位玩家免费建造1个基础生产建筑（自选类型）
7. **初始合同**：每位玩家抽取2张基础合同卡，选择1张保留
8. **设置合同池**：公开5张合同卡作为公共合同池
9. **确定起始玩家**：掷骰子，点数最高者先手

### 游戏流程

游戏进行若干轮，每轮包含以下阶段：

#### 阶段1：生产阶段 🏭

所有玩家同时执行：
- 每个已建造的生产建筑生产对应资源
- 资源放置在玩家面板的仓库中
- 仓库容量限制：每种资源最多存储5个

#### 阶段2：计划阶段 📝

**这是本游戏的核心创新——同时规划！**

1. 所有玩家同时秘密决定工人的放置位置
2. 工人可以放置在贸易站版图的任意功能区域
3. 每个区域有工人数量限制（标注在版图上）
4. 玩家将工人面朝下放在计划区域

#### 阶段3：执行阶段 ⚡

**同时揭示，按顺序执行！**

1. 所有玩家同时揭示工人的放置位置
2. 如果某区域工人超过限制，按以下规则处理：
   - 先放置的玩家优先
   - 超出限制的工人返回玩家手中（本回合无法行动）
3. 按区域依次执行行动：

**区域执行顺序**：

| 顺序 | 区域 | 行动 |
| :--- | :--- | :--- |
| 1 | 招聘中心 | 雇佣临时工人 |
| 2 | 星际银行 | 贷款或储蓄 |
| 3 | 资源市场 | 买卖资源（影响市场价格） |
| 4 | 加工中心 | 加工资源为产品 |
| 5 | 科技实验室 | 研发高科技产品 |
| 6 | 船坞 | 交付合同 |
| 7 | 合同发布 | 获取新合同 |
| 8 | 事件区 | 触发事件 |

#### 阶段4：市场调整 📈

根据本回合的资源交易情况调整市场价格：
- 某资源被购买总量≥3个：价格+1
- 某资源被出售总量≥3个：价格-1
- 价格最低1💰，最高10💰

#### 阶段5：合同刷新 📋

- 已完成的合同从合同池移除
- 补充新合同至5张
- 如果合同池有高级/史诗合同，优先补充基础合同

#### 阶段6：检查结束条件 ✅

游戏在以下情况结束：
- 某位玩家建造了AI研究所（高科技建筑）
- 合同卡池耗尽
- 进行了10轮游戏

### 胜利条件

游戏结束时，计算每位玩家的总分：

**计分项目**：
1. **信用点**：1💰 = 1分
2. **建筑价值**：每个建筑按其建造成本计分
3. **资源价值**：按当前市场价格计算
4. **合同奖励**：未完成合同不得分
5. **成就奖励**：特定成就额外加分

**成就奖励**：
| 成就 | 条件 | 奖励 |
| :--- | :--- | :--- |
| **工业巨头** | 拥有3个以上加工厂 | +10分 |
| **科技先锋** | 拥有2个以上高科技设施 | +15分 |
| **贸易之王** | 完成5个以上合同 | +10分 |
| **资源大亨** | 拥有10个以上资源 | +5分 |

**最高分玩家获胜！**

## 📖 完整游戏流程示例

**游戏设置**：3人游戏（玩家A、B、C）

**初始状态**：
- 每人15💰，4个工人
- 玩家A建造：能源矿场
- 玩家B建造：采矿站
- 玩家C建造：数据中心

**第1轮**：

**生产阶段**：
- 玩家A获得：1⚡
- 玩家B获得：1⛏️
- 玩家C获得：1💾

**计划阶段**（同时秘密决定）：
- 玩家A：2工人→资源市场，1工人→加工中心，1工人→合同发布
- 玩家B：2工人→资源市场，2工人→船坞
- 玩家C：1工人→科技实验室，2工人→星际银行，1工人→事件区

**执行阶段**：

1. **招聘中心**：无人放置，跳过

2. **星际银行**：
   - 玩家C：储蓄5💰，获得1💰利息，共6💰

3. **资源市场**（玩家A、B争夺）：
   - 玩家A（先放置）：出售1⚡，获得3💰（能源价格3）
   - 玩家B（先放置）：购买2⛏️，支付8💰（矿物价格4）
   - 市场价格变化：能源价格-1=2，矿物价格+1=5

4. **加工中心**：
   - 玩家A：使用1⚡+2⛏️（从市场购买），生产1🔩合金

5. **科技实验室**：
   - 玩家C：研发失败（资源不足）

6. **船坞**：
   - 玩家B：无合同可交付

7. **合同发布**：
   - 玩家A：抽取1张合同（能源供应：需3⚡，奖励12💰）

8. **事件区**：
   - 玩家C：触发"能源危机"事件，能源价格+2，现在能源价格=4

**市场调整**：
- 能源：出售1，价格2→4（+2事件）
- 矿物：购买2，价格4→5

**第2-5轮**：
- 玩家们继续建设生产线
- 玩家A建造冶炼厂，开始生产合金
- 玩家B建造货船，增加合同交付能力
- 玩家C建造电子工厂，进入高科技领域

**第6轮**：
- 玩家C完成第一个高级合同，获得25💰
- 玩家A建成完整的合金生产线
- 玩家B开始大规模贸易

**第8轮**：
- 玩家C建造AI研究所，触发游戏结束

**最终计分**：
- 玩家A：信用点35 + 建筑25 + 资源15 = 75分
- 玩家B：信用点42 + 建筑20 + 资源12 = 74分
- 玩家C：信用点38 + 建筑35 + 资源18 + 成就15 = 106分 🏆

**玩家C获胜！**

## 💡 新手策略提示

### 早期策略（1-3轮）

1. **优先建设基础生产**：确保稳定的资源收入
2. **关注市场价格**：低价买入，高价卖出
3. **不要急于加工**：先积累原始资源
4. **选择合适的合同**：确保能完成再接受

### 中期策略（4-7轮）

1. **建立加工链**：从原材料到产品的完整生产线
2. **投资运输能力**：货船可以大幅提升效率
3. **关注对手动向**：避免在热门区域放置工人
4. **利用事件**：积极触发对自己有利的事件

### 晚期策略（8-10轮）

1. **冲刺高科技**：AI核心和量子芯片价值最高
2. **完成史诗合同**：一次获得大量分数
3. **阻止对手**：在关键区域放置工人卡住对手
4. **灵活调整**：根据市场变化快速改变策略

### 通用技巧

1. **引擎构筑思维**：每回合的生产能力比一次性收益更重要
2. **现金流管理**：不要把钱全部投入建设，保留流动资金
3. **市场时机**：在价格低点买入，高点卖出
4. **合同规划**：接受合同前先确保能完成
5. **工人效率**：尽量让每个工人都有行动，不要浪费

## 🔄 变体规则

### 简单模式（适合新手）

- 取消市场价格波动，固定价格
- 取消工人数量限制
- 建筑建造成本-20%
- 合同奖励+20%
- 移除灾难事件

### 困难模式（适合老手）

- 市场价格波动更剧烈（±2）
- 工人数量限制更严格
- 增加"竞争"机制：同一区域的工人互相竞争，只有最强者能执行行动
- 增加"破产"机制：信用点为0时游戏结束
- 灾难事件更频繁

### 团队模式

- 2v2对战，队友共享资源和建筑
- 可以互相交易资源和信用点
- 团队总分高者获胜

###  Solo模式

- 单人挑战AI对手
- AI每轮自动执行固定策略
- 在8轮内达到目标分数获胜

## ❓ 常见问题

**Q：如果仓库满了还能生产吗？**
A：不能，多余资源会浪费。建议及时出售或加工。

**Q：可以建造多个相同的建筑吗？**
A：可以，没有限制。多个相同建筑可以叠加效果。

**Q：如果买不起建筑怎么办？**
A：可以去星际银行贷款，或者先完成合同赚取资金。

**Q：合同可以放弃吗？**
A：可以，但已接受的合同如果未完成，游戏结束时会扣分。

**Q：工人可以重复使用吗？**
A：每轮所有工人都会回到手中，可以重新放置。

**Q：市场价格会降到0吗？**
A：不会，最低价格为1💰。

**Q：可以和其他玩家交易吗？**
A：基础规则不允许，但团队模式中可以。

**Q：如果多人同时想进入已满的区域怎么办？**
A：先放置的玩家优先，后放置的玩家工人返回手中。
