## 🎮 游戏概述

**未来城市建造师**是一款 2-4 人的策略桌游。2026年，全球掀起未来城市建设热潮，每位玩家扮演城市规划师，通过**布局区域**、**建造设施**、**升级科技**来发展自己的未来城市。

游戏的核心在于**引擎构建与区域影响**：每种区域（住宅/商业/工业/公园/交通）都有独特效果，相邻区域会产生协同或冲突。你需要精心规划城市布局，让区域之间产生正向叠加效应。

游戏适合喜欢策略规划、城市建设的玩家，规则中等复杂度，高重玩价值。

## 🎮 游戏规则

### 准备阶段

1. **发放资源**：每位玩家获得 **资金5、电力3、人才2**。
2. **发区域板块**：每位玩家抽取 **5 块区域板块**。
3. **摆放面板**：每人取一张城市面板放在自己面前。
4. **确定先手**：最近乘坐过地铁的玩家先手。

### 游戏流程

每轮按以下步骤进行：

#### 第一步：资源产出 💰
根据已放置的区域，获得对应资源：
- **住宅区**产出人才+1
- **商业区**产出资金+1
- **工业区**产出建材+1
- **公园区**宜居指数+1
- **交通区**所有相邻区域产出+1

#### 第二步：行动阶段 🏗️
每位玩家依次进行 **2 个行动**（可相同）：
- **放置区域**：从手牌中选择1块区域板块放置到城市面板
- **升级科技**：支付成本，对1块已放置区域使用科技升级卡
- **交换板块**：弃掉1块手牌，抽取1块新板块

#### 第三步：区域影响判定 🌟
- 检查相邻区域的影响：
  - **住宅+公园** = 宜居指数额外+2
  - **商业+交通** = 资金额外+2
  - **工业+住宅** = 宜居指数-1（噪音污染）
  - **公园+公园** = 宜居指数额外+3

#### 第四步：事件判定 🌧️
- 翻一张**事件卡**，所有玩家同时受影响。
- 事件卡包括：暴雨（工业停产）、节日（商业+2）、停电（科技失效）等。

### 游戏结束

**10 轮后**或**任意玩家宜居指数达到 30**时，游戏结束。进行最终计分：

| 计分项目 | 分值 |
| :--- | :--- |
| 每块住宅区 | +2 分 |
| 每块商业区 | +1 分 |
| 每块工业区 | +1 分 |
| 每块公园区 | +3 分 |
| 宜居指数 | 每点+1 分 |
| 科技升级次数 | 每次+2 分 |

**总分最高**的玩家赢得游戏，成为"首席城市规划师"！

## 💡 新手策略提示

1. **公园是王道**：公园区宜居指数加成高，前期优先放置！
2. **避免工业挨着住宅**：工业+住宅的负面效果会拖累宜居指数。
3. **交通区是连接器**：交通区让相邻区域产出+1，放在商业和工业之间效果最佳。
4. **科技升级公园**：公园区+智能化升级 = 宜居指数爆表！
5. **关注事件卡**：提前规划应对事件卡，避免被突发事件打乱节奏。
