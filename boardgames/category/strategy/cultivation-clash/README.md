# 修仙争霸 (Cultivation Clash)

## 🎮 游戏概述

《修仙争霸》是一款以中国修仙文化为主题的策略桌游。玩家扮演立志成仙的修真者，在浩瀚的修仙世界中探索秘境、采集灵草、猎杀灵兽、学习功法，最终率先飞升成仙！

游戏融合了热门修仙小说的经典元素：修炼、夺宝、渡劫、飞升，让玩家体验从凡人修炼到登临仙界的完整历程。

适合修仙小说爱好者、策略游戏玩家，以及喜欢中国玄幻文化的朋友。

### 地图特点

1. **修仙之路**：地图上的道路蜿蜒曲折，象征修仙路途艰难
2. **资源分布**：四种资源点均匀分布，鼓励修真者探索不同区域
3. **危险与机遇并存**：天劫区位于关键位置，考验修真者的勇气和智慧
4. **最短路径**：从凡人村到飞升台的最短路径需要经过约15-18步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 只能在有道路（⬜）或特殊点的位置停留
- 移动步数由掷骰子决定（2颗骰子之和）
- 可以自主选择移动路径（在骰子点数范围内）

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择修真者**：每位玩家选择一个颜色的棋子，放在凡人村🧙
3. **初始资源**：每位玩家获得**3张灵石、2张灵草、1张丹药**
4. **洗混卡牌**：将天劫卡洗混后背面朝上放在地图旁；将功法卡和丹药卡洗混后作为牌库

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动
- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的道路移动棋子（只能上下左右移动，不能斜走）
- **可选**：如果已学习"遁术"功法，可额外移动3步

#### 2. 执行所在位置的事件
根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **凡人村** | 🧙 | 无特殊效果，可在此休息恢复 |
| **道路** | ⬜ | 无特殊效果 |
| **灵草地** | 🌿 | 获得1张灵草卡 |
| **灵石矿** | 💎 | 掷1颗骰子，获得对应数量的灵石卡（1-2个） |
| **灵兽林** | 🔮 | 获得1张兽核卡，但需掷骰子≥4，否则失去1张灵草卡 |
| **丹房** | 💊 | 可消耗2张灵草卡换取1张丹药卡（可选） |
| **藏经阁** | 📜 | 可消耗3张灵石卡换取1张功法卡（可选） |
| **天劫区** | ⚡ | 抽取并执行天劫卡效果 |
| **飞升台** | ✨ | 检查飞升条件，满足即可获胜！ |

#### 3. 学习功法/炼制丹药（可选）
如果你有足够的资源，可以在回合结束前：
- **学习功法**：消耗相应资源，从功法牌库抽取1张功法卡
- **炼制丹药**：消耗2张灵草卡，获得1张丹药卡（基础丹药）

#### 4. 使用功法或丹药（可选）
在回合中任意时刻可以使用：
- **功法卡**：使用后立即生效，部分功法可持续多回合
- **丹药卡**：使用后丢弃，效果立即生效

#### 5. 飞升条件检查
当棋子到达飞升台✨时，需满足以下条件才能飞升：
1. 拥有至少**3张功法卡**
2. 掷骰子点数≥**5**（若已学习"天地无极"功法，可掷2次取高）
3. 成功则立即获胜！失败则退回凡人村重新开始

### 回合结束

完成所有操作后，将回合交给下一位修真者。

## 📖 完整游戏流程示例

### 示例1：修真者A的回合（前期资源积累）

1. **修真者A**掷骰子得到**7点**，移动7步
2. 停在**灵草地**🌿，获得1张灵草卡（当前：灵石×3，灵草×3，丹药×1）
3. 继续移动4步，停在**藏经阁**📜
4. 消耗3张灵石卡，学习功法"御剑术"📜⚔️
5. 回合结束，轮到修真者B

### 示例2：修真者B的回合（遭遇天劫）

1. **修真者B**掷骰子得到**4点**，移动4步
2. 不小心停在**天劫区**⚡，抽取天劫卡"雷劫"⚡🌩️
3. 失去2张灵石卡（当前：灵石×1，灵草×2）
4. 幸好之前学习了"护体罡气"📜🛡️，使用后可以抵消此次天劫！
5. 丢弃护体罡气卡，资源保住了！
6. 回合结束，轮到修真者C

### 示例3：修真者C的回合（成功飞升）

1. **修真者C**掷骰子得到**6点**，移动6步
2. 停在**飞升台**✨！检查飞升条件：
   - 拥有功法卡×4（御剑术、遁术、炼丹术、神识探查）✅
   - 掷骰子得到**5点**✅
3. **飞升成功！修真者C率先成仙，获得游戏胜利！** ✨

## 💡 新手策略提示

1. **前期重资源**：游戏初期优先收集灵石和灵草，为学习功法打下基础
2. **功法很重要**："御剑术"和"护体罡气"是生存必备，优先学习
3. **谨慎闯天劫**：天劫区虽然危险，但往往位于通往飞升台的捷径上，需要权衡风险
4. **灵兽林有风险**：停在灵兽林可以获得珍贵的兽核，但需要掷骰子≥4，否则会受伤
5. **藏经阁是关键**：定期去藏经阁学习新功法，功法卡是飞升的必要条件
6. **丹药保命**：当天劫卡带来负面效果时，及时使用丹药卡恢复
7. **观察对手**：如果对手快到飞升台了，可以尝试抢先到达并阻止他们
8. **飞升需谨慎**：到达飞升台后如果掷骰子失败，会被退回起点，确保准备充分再尝试

## 🔄 变体规则

### 组队模式（双修模式）
- 玩家分成两队（2v2），同队修真者共享资源和功法
- 同队玩家可以在自己回合将卡片赠送给队友
- 任意一队成员飞升即全队获胜

### 快速模式（速通修仙）
- 减少地图大小为6x6
- 飞升条件降低为：2张功法卡 + 掷骰子≥4
- 游戏时长缩短至25-35分钟

### 困难模式（天劫降临）
- 增加天劫区数量（每个红色区域放2张天劫卡）
- 天劫卡效果增强（雷劫失去3张灵石卡）
- 初始资源减少（每位修真者只给2张灵石、1张灵草）
- 添加新天劫类型：**万雷天引**（失去所有资源卡，重新开始）

### 多人混战模式（仙魔大战）
- 5-6人游戏时，添加"魔道修真者"角色
- 魔道修真者可以抢夺其他玩家的资源卡
- 增加"正邪对立"机制：正道玩家之间可以互相帮助，魔道玩家必须互相竞争

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：资源卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用，或者返回凡人村重新收集。

**Q：功法卡和丹药卡可以同时拥有多张吗？**
A：可以！没有数量限制，但要注意手牌上限（建议设为10张，超过需丢弃）。

**Q：天劫卡的效果可以叠加吗？**
A：可以。如果连续抽到多张天劫卡，效果会依次生效，所以"神识探查"功法非常重要。

**Q：多人同时到达飞升台怎么办？**
A：按照回合顺序依次检查飞升条件，先满足条件的修真者获胜。

**Q：功法卡使用后就丢弃吗？**
A：大部分功法卡是消耗品，使用后立即丢弃。但"御剑术"等永久性功法可以保留在手中持续使用。

**Q：可以在对手的回合使用功法卡吗？**
A：不可以。功法卡只能在自己回合使用，除非功法说明中明确标注"可随时使用"。
