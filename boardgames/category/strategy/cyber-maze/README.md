# 赛博迷宫 (Cyber Maze)

## 🎮 游戏概述

《赛博迷宫》是一款充满科技感的2-4人策略桌游。玩家扮演神秘的黑客，在虚拟赛博空间中竞速，争夺最高级别的数据核心。游戏融合了路线规划、程序卡构筑与直接对抗三大机制，玩家需要在不断变化的网络迷宫中灵活应对，利用防火墙、加速程序等手段阻挡对手，同时破解数据节点获取分数。

这是一款适合科技爱好者与策略游戏玩家的深度对抗游戏，每一局都是一场脑力与速度的较量。

3. **路径多样性**：同一终点存在多条路径，增加策略深度
4. **动态变化**：每轮部分节点会消失或出现

## 📖 游戏规则

### 🏗️ 准备阶段

**第一步：布置游戏地图**
- 将六边形网格地图展开放在桌面中央
- 将数据核心(⭐)放置在地图中央终点位置

**第二步：放置初始数据节点**
- 将20张数据节点卡随机放置在地图的数据节点位置上
- 每个数据节点放置1张数据卡（面朝上显示分值）
- 确保高价值节点(30分)不超过4个

**第三步：玩家就位**
- 每位玩家选择1个棋子
- 所有玩家棋子放置在起点(🚀)位置
- 每人分发5张程序卡作为起始手牌

**第四步：分发资源**
- 洗混剩余卡牌组成牌堆
- 每位玩家获得10个数据标记作为初始分数（从0开始计）
- 将迷宫变化卡单独洗混放在地图旁

**第五步：设置游戏参数**
- 游戏共进行6轮
- 每轮包含3个阶段：准备阶段 → 行动阶段 → 结算阶段
- 起始玩家由年龄最小的玩家担任

### 🎮 游戏流程

#### 第一阶段：准备阶段

1. **抽取程序卡**：每位玩家从牌堆抽取2张卡
2. **迷宫变化**：翻开迷宫变化卡，根据指示变化地图：
   - 移除1-2个普通节点（显示节点消失动画）
   - 添加1-2个新节点在空白位置
   - 任何玩家在消失节点上需立即移回起点
3. **检查防火墙状态**：所有防火墙保持原位

#### 第二阶段：行动阶段

玩家按照顺时针顺序依次执行行动，每位玩家在回合中可执行以下动作：

**A. 移动（必须）**
- 从当前节点移动到相邻的1个节点
- 移动消耗1个行动点
- 无法移动到有防火墙阻挡的节点
- 无法移动到已消失的节点

**B. 使用程序卡（可选，最多2张）**
- 放置防火墙：消耗1张防火墙卡，放置在任意空节点
- 使用加速：消耗1张加速卡，增加移动步数
- 发动病毒：消耗1张病毒卡，指定对手生效

**C. 破解数据节点（可选）**
- 停留在数据节点上时，可选择破解
- 破解需要消耗1回合准备（停留）
- 破解成功后获得节点上的分数

**D. 移除防火墙（可选）**
- 可移除自己放置的任意防火墙
- 无法移除对手放置的防火墙

#### 第三阶段：结算阶段

1. **检查病毒状态**：被病毒影响的玩家执行相应惩罚
2. **计算本轮得分**：
   - 成功破解数据节点获得对应分数
   - 到达新节点获得5分奖励
   - 到达数据核心获得50分
3. **轮换起始玩家**：本轮起始玩家将指示物交给左手边玩家

### 🌀 迷宫变化机制（核心特色）

每轮准备阶段，根据迷宫变化卡指示进行网络重塑：

**变化类型：**

| 变化类型 | 图示 | 效果 | 频率 |
| :--- | :--- | :--- | :--- |
| 节点消失 | 💨 | 1-2个普通节点消失，经过的玩家移至最近节点 | 每轮30% |
| 节点出现 | ✨ | 1-2个新节点出现在边缘 | 每轮20% |
| 节点升级 | 📈 | 1个节点升级为高价值节点 | 每轮15% |
| 节点降级 | 📉 | 1个高价值节点降级 | 每轮10% |
| 防火墙清除 | 🧹 | 清除场上50%防火墙 | 每轮10% |
| 病毒爆发 | ☣️ | 所有玩家受到1级病毒影响 | 每轮15% |

### 🏆 胜利条件

游戏结束时有以下胜利判定方式：

**主要胜利条件：**
- 6轮结束后，获得最高分的玩家获胜
- 如果平分，则最后到达数据核心的玩家获胜

**额外得分机会：**
- 到达数据核心：+50分
- 破解最高价值数据节点(30分)：+5分额外奖励
- 完全破解某个数据区域：+10分区域奖励
- 每阻止1个对手移动：+3分

## 🎯 完整游戏流程示例

### 示例：玩家A的对局

**游戏设置（4人局）：**
- 玩家A（红色）、玩家B（蓝色）、玩家C（黄色）、玩家D（绿色）
- 玩家A担任第1轮起始玩家
- 每人初始手牌5张

---

**第1轮 - 准备阶段：**

1. 所有玩家各抽2张程序卡（玩家A获得：防火墙卡+加速卡）
2. 翻开迷宫变化卡「节点消失」，1个边缘普通节点消失
3. 玩家A棋子仍在起点

**第1轮 - 玩家A行动阶段：**

1. A移动：从起点→节点N1（普通节点，获得5分）
2. A使用防火墙卡：在节点D2放置防火墙，阻挡C和D
3. A结束回合

**第1轮 - 玩家B行动阶段：**

1. B移动：从起点→节点N1→节点D1（数据节点，10分）
2. B决定破解：支付1回合准备破解
3. B结束回合

**第1轮 - 玩家C行动阶段：**

1. C移动：从起点→起点侧节点（被防火墙阻挡，无法前进）
2. C使用加速卡：使用「短程加速」获得额外移动2步
3. C再次移动：绕过防火墙到达节点N2
4. C结束回合

**第1轮 - 玩家D行动阶段：**

1. D移动：被防火墙阻挡，只能留在起点
2. D使用病毒卡「木马病毒」：指向玩家C
3. D结束回合

**第1轮 - 结算阶段：**

- 玩家A：5分（到达节点N1）
- 玩家B：10分（破解数据节点D1）+ 准备破解中
- 玩家C：-2分（受木马病毒影响后退2步，回到起点附近）
- 玩家D：3分（成功阻止对手）

---

**第3轮 - 决定性时刻：**

经过2轮发展，各玩家位置如下：
- 玩家A：节点N4（准备破解30分核心节点）
- 玩家B：数据核心前节点D3
- 玩家C：起点附近（刚解除病毒状态）
- 玩家D：节点N3

**玩家A的行动：**
1. A完成破解：获得30分 + 5分高价值奖励 = 35分
2. A放置防火墙在D3前，阻挡B
3. A获得总分：当前最高

**玩家B的绝望反击：**
1. B使用「量子跳跃」传送到A附近节点
2. B使用「DDOS攻击」瘫痪A1回合
3. B冲向数据核心...

**最终结果：**
玩家B抢先到达数据核心，获得50分+之前积累的45分=95分获胜！

## 💡 新手策略提示

### 基础策略

1. **早期布局**：第1-2轮优先收集普通数据节点分数，稳扎稳打
2. **防火墙时机**：不要急于放置防火墙，等待对手接近时再阻挡
3. **手牌管理**：始终保持3-4张手牌，避免关键时刻无牌可用
4. **路线备份**：规划多条路线，避免被封锁无路可走

### 进阶技巧

1. **心理博弈**：故意放置防火墙误导对手，真实目标在其他路线
2. **病毒配合**：与防火墙配合使用，效果加倍
3. **加速时机**：在追赶时使用加速卡，快速缩短差距
4. **数据节点优先级**：优先破解20-30分的高价值节点
5. **迷宫利用**：熟悉迷宫变化规律，提前规避风险节点

### 位置策略

| 位置 | 策略建议 |
| :--- | :--- |
| 起点区域 | 快速离开，避免被对手封锁 |
| 中间区域 | 建立防火墙防线，控制节点 |
| 核心区域 | 高价值节点密集，但竞争激烈 |
| 边缘区域 | 相对安全，但距离终点远 |

## ❓ 常见问题

**Q1：棋子被封闭无法移动怎么办？**
A1：如果所有相邻节点都被防火墙阻挡或已消失，该玩家必须使用1张「紧急修复」卡（或从牌堆抽1张）移除最近的防火墙。若无卡可用，则跳过本回合。

**Q2：迷宫变化时在消失节点上怎么处理？**
A2：立即将棋子移至距离最近的未消失节点。如果最近节点有对手，需停在最近的安全节点，并受到1级病毒影响。

**Q3：可以攻击已经在数据核心的玩家吗？**
A3：可以！玩家到达数据核心后仍可被病毒攻击或防火墙阻挡。必须成功防守才能确保胜利。

**Q4：手牌上限是多少？**
A4：每回合结束时，手牌上限为7张。超过上限必须弃牌至7张。

**Q5：破解数据节点失败会怎样？**
A5：破解不会失败，但如果在准备破解期间被病毒攻击，需要重新准备（再停留1回合）。准备期间的节点可以被其他玩家通过。

**Q6：可以移除对手放置的防火墙吗？**
A6：普通方法无法移除。但可以使用「防火墙破解」卡或等待迷宫变化清除。

**Q7：迷宫变化卡的触发顺序是什么？**
A7：如果同时触发多个效果，按照以下顺序处理：节点消失 → 节点出现 → 节点变化 → 防火墙清除 → 病毒爆发。

**Q8：游戏可以提前结束吗？**
A8：可以。如果有玩家累计获得150分，游戏立即结束，该玩家获胜。这鼓励玩家采取激进策略。
