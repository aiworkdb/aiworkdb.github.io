# 丝绸之路 Silk Road

---

## 1. 游戏概述

公元二世纪，一条横跨欧亚大陆的伟大商道连接了东方与西方。从长安出发，穿越茫茫戈壁、翻过雪山高原，经西域绿洲、波斯高原，最终抵达遥远的罗马帝国。无数商队满载丝绸、香料、宝石和茶叶，在这条路上往返奔波，既追逐着惊人的财富，也面对着沙暴、强盗和疾病的致命威胁。

《丝绸之路》是一款2-5人策略桌游，每位玩家扮演一位古代商人，率领自己的商队沿丝绸之路西行。你需要在沿途城市之间规划最优路线，买卖四大类商品赚取差价，同时管理水源、护卫和骆驼等有限资源来应对旅途中的各种风险事件。率先积累到80金币的商人将成为"丝路之王"！

本游戏融合了路线规划、贸易谈判和风险管理三大核心机制，适合喜欢经济策略和资源管理的玩家。

## 6. 游戏规则

### 6.1 准备阶段

1. **展开地图板**：将丝绸之路地图放在桌面中央，8张城市卡放置在对应位置。
2. **分发道具**：根据玩家人数参照物料清单发放金币、骆驼、水源和护卫标记。
3. **放置棋子**：每位玩家将自己的棋子放在长安（起点）。
4. **洗混卡牌**：将商品卡、事件卡、商队卡分别洗混，放在地图旁形成对应牌堆。
5. **补充市场**：每个城市翻开对应数量的商品卡，形成城市市场（长安和罗马各翻5张，其他城市各翻3张）。
6. **决定先手**：掷骰子决定第一位玩家，按顺时针进行。

### 6.2 游戏流程

每位玩家的回合包含以下**三个阶段**，按顺序执行：

#### 阶段一：城市行动（在所在城市执行）

玩家可选择以下**一项或多项**行动（同一行动可重复执行，只要资源足够）：

**🛒 交易**：
- 从所在城市的商品市场中购买商品（支付对应金币，放入手牌）
- 向所在城市出售手牌中的商品（获得对应金币）
- 每回合最多买卖共4张商品（巴格达为6张）
- 买入时可将商品装入骆驼，受运输容量限制

**🏗️ 建设商队**：
- 花费金币从商队牌堆购买商队卡，获得对应能力
- 每回合最多购买1张商队卡

**📜 城市特殊能力**：
- 使用所在城市的特殊能力（每回合限1次）

#### 阶段二：旅途移动（前往下一城市）

- 选择前往相邻城市的一条路线（北路/南路/主路）
- 支付路线所需的水源消耗
- 根据路线距离，移动1-3段距离
- **移动时抽取1张事件卡**，立即结算事件效果
- 如果有武装护卫，可使用护卫来减免负面事件
- 到达目的地后，将棋子移至新城市

#### 阶段三：回合结束

- 补充各城市市场中售出的商品卡（每个城市补到3张，长安/罗马补到5张）
- 如果玩家水源为0且不在城市，则商队陷入困境：下回合无法移动，自动损失1张商品
- 检查是否有玩家达到胜利条件
- 将回合权传递给下一位玩家

### 6.3 贸易机制详解

**买入流程**：
1. 查看所在城市的商品市场
2. 选择要购买的商品卡，支付标价金币
3. 商品卡加入手牌（不算入骆驼运输容量）
4. 城市市场翻开新卡补充

**卖出流程**：
1. 从手牌中选择要卖出的商品卡
2. 查看所在城市的卖出价格
3. 计算最终价格：基础卖出价 + 品质加成（★+0, ★★+1, ★★★+3）+ 城市加成
4. 如果持有"王室使者"事件效果，卖出价格翻倍（仅限一次）
5. 获得金币，卖出商品放入弃牌堆

**驼运机制**：
- 玩家的骆驼容量决定了同时携带的商品数量（初始4件）
- 商品卡分为"手牌"和"运输中"两种状态
- 离开城市时，必须将携带的商品放入运输（受骆驼容量限制）
- 运输中的商品若遭遇"商品变质"事件可能损失

### 6.4 移动机制详解

- 每回合玩家可选择移动或不移动（留在城市执行更多交易）
- 移动必须沿地图上的路线进行，不能跳跃
- 每次移动消耗对应路线的水源
- 移动时必须抽取事件卡并结算
- 到达城市后立即停止，不能继续移动
- 到达罗马（终点）后，可花费5金币返回长安，开启新一轮商旅

### 6.5 胜利条件

游戏采用"金币竞速"机制：

| 条件 | 说明 |
| :--- | :--- |
| 🏆 基本胜利 | 率先积累到**80金币**的玩家获胜 |
| ⏱️ 回合限制 | 如果15回合后无人达到80金币，金币最多者获胜 |
| 🤝 特殊胜利 | 持有3张不同★★★商品卡的玩家直接获胜（大师商人） |

## 7. 完整游戏流程示例

### 示例：3人游戏前三回合

**初始状态**：
- 玩家A（红色）、玩家B（蓝色）、玩家C（绿色），各从15金币、3骆驼、5水源、2护卫开始
- 所有玩家棋子在长安

---

**第1回合 — 玩家A**

**阶段一：城市行动（长安）**
- 玩家A查看长安市场，有丝绸🧣3金、丝绸🧣4金(★★)、茶叶🍵2金、茶叶🍵3金(★★)、宝石💎4金
- 购买2张丝绸（3金+4金）和1张茶叶（2金），花费9金币，剩余6金币
- 3张商品放入运输（骆驼容量4，占用3格）

**阶段二：旅途移动**
- 选择主路：长安→敦煌（距离2，水源消耗2）
- 消耗2水源（剩余3水源）
- 抽取事件卡：🌿 绿洲发现 — 获得3水源！水源恢复为6

**阶段三：回合结束**
- 到达敦煌，市场补充商品
- 回合权交给玩家B

---

**第1回合 — 玩家B**

**阶段一：城市行动（长安）**
- 玩家B决定购买商队升级：花费8金币购买"武装护卫⚔️"卡，获得2名护卫
- 剩余7金币，购买1张丝绸🧣3金，剩余4金币

**阶段二：旅途移动**
- 选择主路前往敦煌，消耗2水源（剩余3）
- 抽取事件卡：🌪️ 沙暴来袭！
- 玩家B有2名护卫，减免为仅损失1水源（剩余2水源）

**阶段三：回合结束**
- 到达敦煌

---

**第2回合 — 玩家A**

**阶段一：城市行动（敦煌）**

- 到达高昌

## 8. 新手策略提示

**🗺️ 路线选择**：
- 新手建议先走北路（敦煌→高昌→撒马尔罕），风险较低且高昌有拒绝重抽的特殊能力
- 南路虽然宝石价格好，但风险高，建议熟悉规则后再尝试
- 初期不必急于赶路，在长安和敦煌多囤货，积累缓冲资金

**💰 贸易策略**：
- 丝绸是性价比最高的商品：长安买入价仅3-4金，罗马可卖15金，利润率超过300%
- 茶叶是新手之友：体积小（每骆驼3件），利润率也不错
- 宝石利润最高但运输效率最低（每骆驼1件），建议到中后期骆驼容量升级后再大量采购
- 香料是最稳定的中间商品，各段路线都有不错的利润

**🛡️ 风险管理**：
- 前3回合务必在敦煌购买护卫，没有护卫的商队在路上寸步难行
- 始终保持至少3点水源的储备，水源耗尽会陷入严重困境
- 不要一次性携带太多贵重商品，分散运输可以降低风险事件造成的损失
- "宝藏地图"商队卡可以让你预知事件，非常值得投资

**🏁 胜利路径**：
- 稳健型：反复走长安→敦煌→撒马尔罕路线，每次运2-3张丝绸，大约4-5趟可达到80金币
- 激进型：直奔罗马，单趟运满高品质丝绸/宝石，配合"王室使者"翻倍效果一波致胜
- 大师型：收集3张★★★商品触发特殊胜利，需要运气和精准的市场判断

## 9. 变体规则

### 9.1 商人竞争模式（高级）

- 在同一城市的玩家可以进行贸易谈判：可以用商品卡交换其他玩家的商品卡
- 每回合限1次交易，双方必须都同意
- 引入"竞价"机制：在罗马等热门城市，卖出价格随竞争玩家数量下降（每多1人-1金币）

### 9.2 皇帝密令模式（合作+竞争）

- 游戏开始时，每位玩家秘密抽取一张"密令卡"，上面写有特定的运货任务（如：将3张丝绸运到罗马）
- 完成密令获得额外10金币奖励
- 如果所有玩家共同完成全部密令，游戏进入"丝路繁荣"结局

### 9.3 快速模式

- 胜利条件降为50金币
- 城市距离全部减1
- 水源消耗全部减1
- 适合时间有限时快速体验

### 9.4 团队模式

- 4人或5人时，可以2v2或2v3组队
- 同队玩家共享金币（公共钱包），但分别行动
- 所有队员金币总和达到120金币的队伍获胜

### 9.5 丝路传说模式（叙事扩展）

- 在事件卡中增加"传说事件"（如：张骞出使、玄奘西行、马可波罗东来）
- 遇到传说事件时需要完成特殊挑战（如：投骰猜数、答题等）
- 完成挑战获得丰厚奖励，失败则遭受严厉惩罚

## 10. 常见问题

**Q1：水源耗尽了怎么办？**

如果水源为0且你不在城市，商队陷入困境——下回合无法移动，并自动损失1张运输中的商品。你需要等到达下一个城市时购买水源补充（每个城市均可花费2金币购买2水源）。建议始终保留至少2点水源作为应急储备。

**Q2：遇到强盗袭击但没有护卫和金币怎么办？**

当强盗袭击且你既没有护卫也支付不起5金币时，你必须随机失去运输中的1张商品卡（由其他玩家帮你随机抽取）。这是游戏中比较严重的惩罚，再次强调购买护卫的重要性。

**Q3：可以一次携带超过骆驼容量的商品吗？**

不可以。离开城市时，你必须将携带的商品分配到骆驼上，严格受容量限制。初始容量为4件，可通过购买商队卡升级。但手牌中的商品（未放入运输）不受骆驼容量限制，留在手中不会丢失但也不能在旅途中卖出。

**Q4：到达罗马后必须返回长安吗？**

不一定。到达罗马后你可以选择：①留在罗马继续交易（罗马卖出价格最高）；②花费5金币返回长安开启新旅程；③选择经原路返回途中城市做中转交易。大多数情况下，建议在罗马卖完货后再返回长安进货。

**Q5：多个玩家同时达到80金币怎么办？**

如果同一回合内有多名玩家达到或超过80金币，按回合行动顺序判定——最先在本回合中达到80金币的玩家获胜。因此行动顺序在游戏后期非常关键。

**Q6：城市市场的商品被买完了怎么办？**

每个城市的商品市场有固定容量（长安/罗马5张，其他城市3张）。每当有商品被买走，回合结束时立即从商品牌堆翻开新卡补充到满。如果商品牌堆耗尽，则将弃牌堆洗混重新使用。不会出现市场空置的情况。

**Q7：事件卡中的"王室使者"翻倍效果可以叠加吗？**

不可以。即使你持有多个"王室使者"效果，每次卖出时也只能触发一次翻倍。多余的效果可以保留到下一次卖出使用。建议在有大量高品质商品时再使用翻倍效果，实现收益最大化。
