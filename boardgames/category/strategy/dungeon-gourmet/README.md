# 迷宫美食家 (Dungeon Gourmet)

## 🎮 游戏概述

《迷宫美食家》是一款融合了地下城探险与美食烹饪的策略桌游，灵感来源于热门动漫《迷宫饭》！

玩家扮演勇敢的冒险家，深入神秘的地下城。在这里，怪物不仅是敌人，更是美味的食材！通过击败怪物收集食材、在篝火处烹饪美味料理、享用料理获得特殊能力，最终第一位烹饪出3道"传奇料理"并逃离地下城的玩家获胜！

这是一个适合朋友聚会和家庭娱乐的策略游戏，规则简单易懂，充满乐趣、策略和美食诱惑。

## 🗺️ 游戏地图详解

游戏地图是一个 **7x8 的地下城网格**，包含多种地形和事件点。

### 地图布局概览

```
🚪 ⬜ 🥩 ⬜ ⚠️ ⬜ 🍲 ⬜
⬜ ⚠️ ⬜ 🎲 ⬜ 📜 ⬜
🐟 ⬜ ⬜ ⚠️ ⬜ ⬜ 🥩
⬜ 🍲 🌿 ⬜ 🎲 ⚠️ ⬜
⚠️ ⬜ ⬜ 🐟 ⬜ 🍲 ⬜
🍄 🎲 ⚠️ ⬜ 🌿 ⬜ ⬜
⬜ ⬜ 🐟 🍲 ⬜ ⚠️ ⬜
🎲 ⬜ ⬜ 📜 ⬜ 🌿 🚀
```

**图例说明**：
- 🚪 = 入口（起点）
- 🚀 = 出口（终点）
- ⬜ = 通道（安全移动）
- 🥩🐟🍄🌿 = 食材点（击败怪物获得食材）
- 🍲 = 篝火（烹饪料理）
- 📜 = 食谱（获得食谱卡）
- ⚠️ = 陷阱（触发陷阱卡）
- 🎲 = 事件（触发事件卡）

### 地形类型说明

| 地形 | 图示 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- |
| **入口** | 🚪 | 绿色 | 所有玩家开始的位置，位于地图左上角 |
| **通道** | ⬜ | 灰色 | 安全的行进路线，没有特殊效果 |
| **食材点** | 🥩🐟🍄🌿 | 彩色 | 停留需击败怪物，获得对应食材卡 |
| **篝火** | 🍲 | 橙色 | 停留可使用食材烹饪料理 |
| **食谱** | 📜 | 黄色 | 停留获得一张食谱卡 |
| **陷阱** | ⚠️ | 红色 | 停留需抽一张陷阱卡并执行 |
| **事件** | 🎲 | 紫色 | 停留需抽一张事件卡并执行 |
| **出口** | 🚀 | 金色 | 终点！拥有3张传奇料理卡即可获胜 |

### 地图特点

1. **路径设计**：地图上的通道形成多条路径，玩家可以选择不同路线
2. **食材分布**：4种食材均匀分布，鼓励玩家探索不同区域
3. **篝火位置**： strategically placed 让玩家需要规划路线来烹饪
4. **最短路径**：从入口到出口的最短路径需要经过约15-18步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 只能在有通道（⬜）或特殊点的位置停留
- 移动步数由掷骰子决定（2颗骰子之和）
- 玩家可以在回合内多次停留在同一格子（如果掷出足够点数）

## 🃏 道具详解

### 食材卡（基础材料）

食材卡是游戏中最基础的道具，所有料理都需要用食材卡来烹饪。击败怪物是获得食材的主要方式！

| 食材 | 图示 | 获得方式 | 用途 |
| :--- | :--- | :--- | :--- |
| **魔兽肉** | 🥩 | 停在🥩食材点，击败魔兽 | 烹饪肉类料理（需要2张） |
| **迷宫鱼** | 🐟 | 停在🐟食材点，击败鱼怪 | 烹饪鱼类料理（需要2张） |
| **魔法蘑菇** | 🍄 | 停在🍄食材点，击败蘑菇怪 | 烹饪蘑菇料理（需要1张） |
| **奇异香草** | 🌿 | 停在🌿食材点，采集香草 | 烹饪香草料理（需要1张） |

**战斗机制**：
当玩家停留在食材点时，需要掷1颗骰子进行"战斗"：
- 点数 ≥ 4：战胜怪物，获得1张对应食材卡
- 点数 ≤ 3：战斗失败，失去1个回合（本回合结束）

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择棋子**：每位玩家选择一个颜色的棋子，放在入口🚪（左上角）
3. **分发初始资源**：
   - 每位玩家获得**2张随机食材卡**
   - 每位玩家获得**1张基础食谱卡**（烤肉或烤鱼）
4. **洗混卡牌**：
   - 将食材卡洗混，放在地图旁
   - 将食谱卡洗混，放在地图旁
   - 将料理卡洗混，放在地图旁
   - 将陷阱卡洗混，放在地图旁
   - 将事件卡洗混，放在地图旁
5. **决定先手**：掷骰子，点数最高者先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 掷2颗骰子，点数相加得到移动步数
- 如果玩家有"烤肉"料理卡效果，额外+2步
- 如果玩家有"速度药剂"料理卡效果，掷两次骰子取大值
- 按照地图上的通道移动棋子（只能上下左右移动，不能斜走）

#### 2. 执行所在位置的事件

根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **入口** | 🚪 | 无特殊效果 |
| **通道** | ⬜ | 无特殊效果 |
| **食材点** | 🥩🐟🍄🌿 | 进行战斗，胜利获得1张食材卡 |
| **篝火** | 🍲 | 如果有食材和对应食谱，可以烹饪料理 |
| **食谱** | 📜 | 获得1张食谱卡 |
| **陷阱** | ⚠️ | 抽取并执行陷阱卡效果 |
| **事件** | 🎲 | 抽取并执行事件卡效果 |
| **出口** | 🚀 | 如果拥有3张传奇料理卡，立即获胜！ |

#### 3. 烹饪料理（可选）

如果你停留在篝火🍲位置，并且有对应的食材和食谱卡，可以烹饪料理：

**烹饪步骤**：
1. 展示你要烹饪的食谱卡
2. 展示所需的食材卡
3. 将食材卡和食谱卡丢弃
4. 从料理卡堆顶抽取对应数量的料理卡
5. 将获得的料理卡加入你的手牌

**示例**：
- 你有"烤肉大餐"食谱卡和2张"魔兽肉"食材卡
- 在篝火处烹饪，丢弃食谱和食材
- 获得1张"烤肉"料理卡

#### 4. 使用料理卡（可选）

在回合中任意时刻可以使用料理卡（除特殊说明外，使用后丢弃）：

| 料理卡 | 使用时机 | 效果 |
| :--- | :--- | :--- |
| **烤肉** | 移动前 | 本次移动+2步 |
| **烤鱼** | 战斗前 | 本次战斗自动胜利 |
| **蘑菇汤** | 移动前 | 查看下一张陷阱卡 |
| **香草沙拉** | 掷骰子前 | 额外掷一次骰子 |
| **速度药剂** | 掷骰子前 | 掷两次骰子取大值 |
| **防御烤肉** | 触发陷阱前 | 免疫本次陷阱效果 |
| **治愈汤** | 任意时刻 | 移除所有负面效果 |
| **豪华全餐** | 获得时 | 传奇料理！获胜条件-1 |

#### 5. 管理手牌

- 手牌上限：**7张**（包括食材卡、食谱卡、料理卡）
- 如果超过7张，必须丢弃多余的卡牌
- 建议保留：食谱卡 > 食材卡 > 料理卡

### 回合结束

完成所有操作后，将回合交给下一位玩家。

## 📖 完整游戏流程示例

### 示例1：正常流程（玩家A的回合）

1. **回合开始**：玩家A是先手
2. **掷骰子**：掷出2+4=6点
3. **移动**：移动6步，停在🥩食材点
4. **战斗**：掷1颗骰子，得到5点（≥4，胜利！）
5. **获得食材**：获得1张"魔兽肉"食材卡
6. **手牌管理**：手牌6张，未超过7张上限
7. **回合结束**：轮到玩家B

### 示例2：烹饪料理（玩家B的回合）

1. **回合开始**：玩家B的回合
2. **掷骰子**：掷出3+3=6点
3. **移动**：移动6步，停在🍲篝火位置
4. **烹饪**：
   - 展示"烤肉大餐"食谱卡
   - 展示2张"魔兽肉"食材卡
   - 丢弃食谱和食材
   - 获得1张"烤肉"料理卡
5. **使用料理**：使用"烤肉"料理卡，下次移动+2步
6. **回合结束**：轮到玩家C

### 示例3：陷阱与应对（玩家C的回合）

1. **回合开始**：玩家C的回合
2. **掷骰子**：掷出4+2=6点
3. **移动**：移动6步，停在⚠️陷阱位置
4. **触发陷阱**：抽取陷阱卡，得到"地洞"（后退3步）
5. **应对**：使用"防御烤肉"料理卡，免疫陷阱效果
6. **继续移动**：使用"香草沙拉"料理卡，额外掷骰子
7. **回合结束**：轮到玩家D

### 示例4：获胜时刻（玩家D的回合）

1. **回合开始**：玩家D已经有2张"豪华全餐"传奇料理
2. **掷骰子**：掷出5+3=8点
3. **移动**：移动8步，到达🚀出口位置
4. **检查获胜条件**：拥有2张传奇料理，还需要1张
5. **未获胜**：运气不好，再努力吧！
6. **回合结束**：轮到玩家A（新一轮）

**注意**：如果玩家D此时拥有3张传奇料理，则立即获胜！游戏结束！

## 💡 新手策略提示

1. **优先收集食材**：初期多收集各种食材，为烹饪料理做准备
2. **食谱很重要**：尽量多收集食谱卡，提供更多烹饪选择
3. **料理是关键**：合理使用料理卡可以事半功倍
   - "烤肉"：适合长距离移动
   - "烤鱼"：适合攻打高难度怪物
   - "蘑菇汤"：适合探索未知区域
4. **规划路线**：观察地图，选择食材丰富、陷阱较少的路径
5. **注意手牌上限**：手牌上限7张，及时使用或丢弃不需要的卡牌
6. **团队合作（多人模式）**：可以和其他玩家交易食材卡（每回合限1次）
7. **篝火时机**：尽量在有足够食材和食谱时再使用篝火
8. **陷阱预防**：如果可能触发陷阱，提前使用"防御烤肉"料理卡

## 🔄 变体规则

### 合作模式

- 所有玩家组成队伍，共同对抗地下城
- 共享食材卡和食谱卡
- 队友之间可以自由传递卡牌（每回合限2张）
- **共同胜利条件**：团队总共烹饪出5道传奇料理

### 时间挑战模式

- 设置45分钟时间限制
- 在规定时间内烹饪出传奇料理最多的玩家获胜
- 如果时间到无人烹饪出传奇料理，则料理卡最多的玩家获胜

### 难度升级模式

- **增加陷阱**：每个陷阱点放2张陷阱卡（同时触发）
- **减少初始资源**：每位玩家只给1张食材卡
- **添加新陷阱类型**：
  - 迷宫怪物（强制战斗，失败失去1张料理卡）
  - 地下河（本回合无法移动，下次移动-2步）
- **食材腐烂**：每3回合，所有玩家随机失去1张食材卡

### 快速模式（适合新手）

- 获胜条件改为：烹饪出2道传奇料理即可
- 初始资源增加：每位玩家获得4张食材卡和2张食谱卡
- 减少陷阱数量：只保留4个陷阱点

### 竞技场模式

- 玩家可以在任意时刻"挑战"其他玩家
- 挑战方式：双方各掷1颗骰子，点数高者胜利
- 胜利奖励：获胜者获得失败者1张随机手牌
- 限制：每玩家每回合只能挑战1次

## ❓ 常见问题

**Q1：如果我没有骰子怎么办？**

A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。甚至可以用围棋棋子（1-6个）来随机抽取！

**Q2：食材卡用完了怎么办？**

A：可以将弃牌堆洗混后重新使用。或者玩家可以互相交易食材卡。

**Q3：可以跳过烹饪直接移动吗？**

A：可以，烹饪是可选操作。但如果你有食材和食谱，建议在篝火处烹饪，因为料理卡非常有用！

**Q4：多人同时到达出口怎么办？**

A：先到达的玩家优先检查获胜条件。如果未满足获胜条件，则按照到达顺序依次检查。

**Q5：手牌超过7张怎么办？**

A：必须在回合结束前丢弃多余的卡牌，直到手牌数为7张或更少。

**Q6：可以查看料理卡堆的内容吗？**

A：不可以。料理卡堆是面向下的，只能随机抽取。

**Q7：如果所有陷阱卡都用完了怎么办？**

A：将陷阱卡弃牌堆洗混后重新使用。

**Q8："豪华全餐"传奇料理只有1张，多人游戏怎么办？**

A：如果"豪华全餐"被其他玩家获得，你可以：
1. 通过交易获得（如果对方同意）
2. 继续烹饪其他料理，积累更多能力令牌
3. 尝试击败更多怪物，运气好的话可能会再次获得食谱

**Q9：可以连续使用多张料理卡吗？**

A：可以！只要在合适的使用时机，你可以连续使用多张料理卡。例如：同时使用"香草沙拉"+ "速度药剂"，可以掷3次骰子！

**Q10：游戏可以进行团队战吗？**

A：可以！2v2或1v1v1v1都可以。团队战的获胜条件是团队总共烹饪出5道传奇料理。

## 🎯 高级策略指南

### 初期策略（前10回合）

- **目标**：积累食材和食谱
- **建议**：多走食材点和食谱点，尽量避免陷阱
- **卡牌优先级**：食谱卡 > 食材卡 > 料理卡

### 中期策略（10-30回合）

- **目标**：开始烹饪料理，获得特殊能力
- **建议**：合理使用篝火，烹饪实用的料理
- **卡牌优先级**：料理卡 > 食材卡 > 食谱卡

### 后期策略（30回合后）

- **目标**：冲刺出口，争取第一个获胜
- **建议**：保留"烤肉"和"速度药剂"料理卡用于最后冲刺
- **卡牌优先级**：传奇料理 > 移动类料理 > 其他

### 卡牌组合推荐

| 组合 | 卡牌 | 效果 |
| :--- | :--- | :--- |
| **快速移动** | 烤肉 + 速度药剂 | 移动步数大幅增加 |
| **战斗无敌** | 烤鱼 + 蘑菇汤 | 战斗100%胜利 + 预知危险 |
| **资源满满** | 全季节汤 + 神秘商人事件 | 获得大量食材卡 |
| **终极冲刺** | 豪华全餐×3 + 烤肉 | 直接获胜！ |
