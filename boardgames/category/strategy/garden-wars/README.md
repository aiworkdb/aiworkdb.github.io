# 🌻 植物大战 Garden Wars

## 🎮 游戏概述

《植物大战 Garden Wars》是一款以花园争霸为主题的2-4人策略桌游。每位玩家扮演一位花园主人，通过种植不同功能的植物来占领花园领地、抵御害虫侵袭、应对天气变化，最终成为花园价值最高的主人。

游戏融合了区域控制、资源管理和卡牌构筑三大核心机制。向日葵产出阳光资源，豌豆射手守护领地，土豆雷设下伏击陷阱，樱桃炸弹清场反击——你需要根据季节变化和对手策略灵活调配植物阵容，在春的播种、夏的生长、秋的收获与冬的蛰伏中，打造属于自己的花园帝国。

适合喜欢策略思考、卡牌博弈和资源规划的玩家，无论是家庭聚会还是朋友竞技，都能享受种花种草的乐趣与紧张刺激的对抗。

### 地图特点

1. **对称平衡**：地图布局保持中心对称，确保每位玩家的起点机会均等
2. **资源分布均匀**：水泉和花圃均匀散布，避免某一区域过强
3. **石地占比较高**：38%的石地增加了策略深度，迫使玩家争夺优质地块
4. **相邻优势**：同种植物相邻种植可触发"共生效应"（详见植物卡说明）

## 📜 游戏规则

### 准备阶段

1. **展开地图**：将花园地图放在桌面中央，确保所有玩家都能看到
2. **选择棋子**：每位玩家选择一个颜色的棋子
3. **确定先手**：掷骰子，点数最大者先手，顺时针进行
4. **初始资源**：每位玩家获得 **3个阳光** 作为起始资源
5. **发放手牌**：
   - 每位玩家从**植物牌堆**抽取 **4张植物卡** 作为起始手牌
   - 将天气卡、害虫卡、肥料卡、特殊卡分别洗混，面朝下放在地图旁
6. **季节指示器**：将季节指示器放在 **🌸春天** 位置
7. **每位玩家初始领地**：分别在地图四角各选择1个沃土地块作为起始据点（2人游戏选对角）

### 季节轮转系统

游戏以**年**为单位循环，每年分为4个季节：

| 季节 | 回合数 | 特殊规则 |
| :--- | :--- | :--- |
| 🌸 **春天** | 每人2回合 | 播种季节：种植费用-1☀️，三叶草可抽2张牌 |
| ☀️ **夏天** | 每人2回合 | 生长季节：所有攻击型植物伤害+1，产量正常 |
| 🍂 **秋天** | 每人2回合 | 收获季节：所有植物产出+1☀️，可额外种植1株 |
| ❄️ **冬天** | 每人1回合 | 蛰伏季节：仙人掌/火焰椒/竹子不可用，向日葵产出-1 |

每个季节结束时，进行**结算阶段**：所有植物产出阳光、害虫效果结算、移除持续1季的效果。

游戏在第2年冬天结束后（共18回合）进入**最终计分**。

### 回合流程

每位玩家的回合按以下步骤执行：

#### 步骤一：季节开始（仅季节的第一位玩家）

- 抽取1张**天气卡**并翻开，效果持续整个季节
- 宣布当前季节及天气效果
- 将季节指示器移动到对应季节

#### 步骤二：收入阶段

- 收集己方所有**植物产出**的阳光（参照植物卡上的产出值）
- 水泉地块上的植物额外 +1☀️
- 天气卡和肥料卡加成叠加计算

#### 步骤三：行动阶段（可执行以下操作，顺序不限）

每位玩家每回合拥有 **3点行动力**，每项操作消耗对应行动力：

| 操作 | 消耗行动力 | 说明 |
| :--- | :--- | :--- |
| 🌱 种植植物 | 1点 | 从手牌中选择1张植物卡，放置在合法地块上，支付费用 |
| 🐛 发动害虫 | 1点 | 从手牌中使用1张害虫卡，选择目标，支付费用 |
| 💩 使用肥料 | 1点 | 从手牌中使用1张肥料卡，选择目标，支付费用 |
| ⚡ 使用特殊卡 | 1点 | 从手牌中使用1张特殊卡，支付费用 |
| 🃏 抽牌 | 1点 | 从任意一个面朝下的牌堆抽取1张卡 |
| ☀️ 购买阳光 | 1点 | 弃置1张手牌，获得2☀️ |

**种植规则**：
- 沃土🌱和花圃🌸：可种植任何植物
- 水泉💧：可种植任何植物，且额外+1☀️产出
- 石地🟫：只能种植仙人掌🌵、土豆雷🥔、蘑菇园🍄
- 每个地块只能有1株植物
- 可以种植在己方已占领的地块（替换植物，旧植物弃置）
- 不可种植在对手已占领的地块（除非旧植物已被摧毁）

#### 步骤四：回合结束

- 检查手牌上限（最多7张），超出须弃置到7张
- 宣布回合结束，轮到下一位玩家

### 占领规则

- 在空地上种植植物即**占领**该地块
- 被害虫摧毁的植物，该地块变为空地（非占领状态）
- 花园价值 = 己方占领地块的植物基础价值之和 + 剩余阳光×0.5（向下取整）

### 胜利条件

游戏在**第2年冬天（第18回合）**结束后进入最终计分：

1. 计算每位玩家的**花园价值**：
   - 每株存活植物价值 = 植物费用×2
   - 占领水泉地块额外 +3分
   - 占领花圃地块额外 +2分
   - 剩余阳光每5个 +1分
2. 花园价值**最高**的玩家获胜
3. 如遇平局，剩余阳光多者获胜；仍平局则占领水泉多者获胜

**玩家**：小绿（先手）、小红

4. **手牌**：🫛豌豆射手（手牌共3张，未超上限）
5. **当前阳光**：3☀️

**回合2 - 小红：**

1. **收入阶段**：小红起始据点无植物，收入 0☀️（当前共有 3☀️）
2. **行动阶段**（3点行动力）：
   - 🌱 **种植**（1点）：在起点沃土(F6)种植 🌻向日葵，花费 0☀️
   - 💩 **使用肥料**（1点）：对向日葵使用 💩普通肥料，花费 0☀️（本季产出+1☀️）

3. **当前阳光**：3☀️

**春季结算**：
- 小绿向日葵：基础产出 2 + 大晴天加成 1 = **3☀️**
- 小红向日葵：基础产出 2 + 大晴天加成 1 + 肥料加成 1 = **4☀️**

**回合3 - 小绿：**

2. **行动**：
   - 🌱 种植 🫛豌豆射手在水泉地块(F1)，花费 1☀️
   - 🃏 抽到 🧱坚果墙

**回合4 - 小红：**

   - 🐛 对小绿的向日葵发动 🐛毛毛虫，花费 1☀️

#### 第1年·夏天·天气：🌧️ 暴风雨

**回合5 - 小绿（季节开始）：**
1. **季节开始**：抽取天气卡 🌧️暴风雨——所有植物防御-1！
2. **收入**：向日葵 2-2(毛毛虫)=0☀️，水泉豌豆射手 0+1(水泉)=1☀️，总计 1☀️
3. **行动**：
   - 🌱 种植 🧱坚果墙在向日葵旁，花费 1-1=0☀️（春天减免已结束，夏天原价，1-1=0是错的，应该是1☀️。修正：花费 1☀️）

   - 🃏 抽到 🍒樱桃炸弹
4. **当前阳光**：5+1-1-2=3☀️

**回合6 - 小红：**
1. **收入**：向日葵 2+1+1=4☀️（毛毛虫仅持续1季已结束），土豆雷 0☀️，仙人掌 0☀️，总计 4☀️

## 💡 新手策略提示

1. **阳光是王道**：前期优先种植向日葵和蘑菇园，确保每季有稳定的高额阳光收入。水泉地块上的向日葵是你的最优选择
2. **不要忽视防御**：1个坚果墙放在关键位置，可以帮你省下大量阳光。豌豆射手配合坚果墙形成坚固防线
3. **善用季节优势**：春天种植费用低，疯狂扩张；夏天攻击力强，主动出击；秋天收获加倍，囤积资源；冬天蛰伏蓄力，用防御型和耐寒植物
4. **石地是隐藏宝地**：虽然石地限制种植种类，但只有仙人掌和土豆雷可以种在那里，意味着对手很难干扰你的石地领地
5. **肥料用在刀刃上**：高级肥料(✨)是永久加成，优先用在向日葵上；黄金肥料(💎)翻倍效果，用在攻击型植物上可以扭转战局
6. **控制手牌数量**：手牌上限7张，不要贪多。多余的牌及时打出或转换成阳光
7. **观察对手布局**：如果对手在水泉集中种植，考虑用樱桃炸弹清场。如果对手依赖石地防御，瓢虫是你的利器
8. **共存效应**：同类型植物相邻种植时，紫罗兰的加成效果非常显著。围绕紫罗兰构建防御阵地是稳妥的策略

## 🔄 变体规则

### 🏴‍☠️ 团队模式（4人）

- 玩家分为2队（2v2），队友坐对面
- 队友共享阳光资源池
- 可以在队友回合中为其使用肥料卡和特殊卡
- 害虫卡只能对对手使用
- 团队花园价值之和最高者获胜

### ⏱️ 闪电战模式

- 每个季节每人只有 **1个回合**（共8回合）
- 初始阳光提升至 **5个**
- 起始手牌提升至 **6张**
- 游戏节奏更快，决策更紧凑

### 🌙 夜晚追加规则

在每个季节结算后增加"夜晚阶段"：
- 夜晚中，所有阳光产出减半
- 害虫攻击力+1（夜间出没）
- 豌豆射手夜间防御翻倍
- 增加灯笼花(🌺)的战略价值——夜间灯笼花相邻地块不受害虫影响

### 🎲 随机花园模式

- 每次游戏前，随机从地块类型袋中抽取确定地图布局
- 可能出现"泉水群"或"石地荒原"等特殊地形
- 增加每局游戏的差异性和重玩价值

### 🧒 亲子简化版

- 移除害虫卡和特殊卡
- 仅保留资源型植物和防御型植物
- 每个季节每人1回合
- 适合6岁以上儿童，专注资源管理和区域控制

## ❓ 常见问题

**Q1：向日葵在冬天产出是多少？**

A：向日葵基础产出2☀️，冬天减益-1☀️，所以冬天产出为1☀️。如果叠加了大晴天天气卡(+1☀️)和普通肥料(+1☀️)，则产出为 2-1+1+1 = 3☀️。所有加成和减益可以叠加计算。

**Q2：坚果墙的"免疫1次害虫攻击"是什么意思？**

A：当害虫卡以坚果墙所在地块为目标时，坚果墙替该地块上的植物承受攻击并进入"破损"状态。破损的坚果墙不能再次免疫。可以使用肥料卡修复坚果墙。

**Q3：樱桃炸弹摧毁的植物包括自己的吗？**

A：是的。樱桃炸弹效果是"摧毁目标3×3区域内所有植物"，不分敌我。使用时需要谨慎选择目标位置，避免误伤自己的植物。春天时范围扩大到4×4，更要小心。

**Q4：土豆雷的伏击效果何时触发？**

A：当任何对手在土豆雷相邻的地块上种植植物时，土豆雷自动触发，摧毁对手刚种植的那株植物，然后土豆雷自身也被消耗移除。注意：大雾天气时土豆雷不触发。

**Q5：如果手牌超过7张怎么办？**

A：回合结束时如果手牌超过7张，必须弃置多余的卡牌直到剩余7张。弃置的卡牌放入对应类型的弃牌堆。被弃置的植物卡不会返回你的花园。

**Q6：可以在已被对手占领的地块上种植吗？**

A：不可以直接种植。必须先用害虫卡（如瓢虫🐞）摧毁对手的植物，使该地块变为空地后，才能在自己的回合种植占领。或者使用移花接木(🔄)特殊卡强行移走对手的植物。

**Q7：水泉地块的额外阳光加成和植物自身产出是什么关系？**

A：水泉加成是在植物基础产出之上额外 +1☀️，与植物卡上标注的产出值相加。例如：向日葵在水泉上产出 = 2(基础) + 1(水泉) = 3☀️。水泉加成不受季节减益影响。
