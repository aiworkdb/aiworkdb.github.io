# 宝藏猎人 (Treasure Hunter)

## 🎮 游戏概述

《宝藏猎人》是一款充满冒险与策略的桌游。玩家扮演勇敢的探险家，在神秘的岛屿上寻找传说中的宝藏。通过收集资源、建造工具、避开陷阱，最终第一个找到宝藏的玩家获胜！

这是一个适合朋友聚会和家庭娱乐的策略游戏，规则简单易懂，充满乐趣和挑战。

| 资源 | 图示 | 获得方式 | 用途 |

## 🗺️ 游戏地图详解

游戏地图是一个 **7x7 的神秘岛屿网格**，包含多种地形和事件点。

### 地图布局概览

```
🚶 ⬛ 🌲 ⬛ ⚠️ ⬛ 🪨
⬛ ⚠️ ⬛ 🍎 ⬛ 🪨 ⬛
🌲 ⬛ ⬛ ⚠️ ⬛ ⬛ 🍎
⬛ 🪨 🌲 ⬛ 🍎 ⚠️ ⬛
⚠️ ⬛ ⬛ 🌲 ⬛ 🪨 ⬛
🍎 🪨 ⚠️ ⬛ 🌲 ⬛ ⬛
⬛ ⬛ 🍎 🪨 ⬛ ⬛ 💎
```

### 地形类型说明

| 地形 | 图示 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- |
| **起点** | 🚶 | 黄色 | 所有玩家开始的位置，位于地图左上角 |
| **道路** | ⬛ | 绿色 | 安全的行进路线，没有特殊效果 |
| **木材资源点** | 🌲 | 绿色 | 停留可获得1张木材卡 |
| **石头资源点** | 🪨 | 蓝色 | 停留可获得1张石头卡 |
| **食物资源点** | 🍎 | 红色 | 停留可获得1张食物卡 |
| **陷阱点** | ⚠️ | 红色 | 停留需抽一张陷阱卡 |
| **宝藏点** | 💎 | 橙色 | 终点！到达即可获胜 |

### 地图特点

1. **路径设计**：地图上的道路形成一条蜿蜒的路径，玩家只能在道路上移动
2. **资源分布**：资源点均匀分布在整个地图上，鼓励玩家探索
3. **陷阱位置**：陷阱点位于关键位置，增加游戏挑战性
4. **最短路径**：从起点到终点的最短路径需要经过约12-15步

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 只能在有道路（⬛）或特殊点的位置停留
- 移动步数由掷骰子决定（2颗骰子之和）

### 地图资源统计

| 资源类型 | 数量 | 位置分布 |
| :--- | :--- | :--- |
| 木材 | 5个 | 分布在地图左上、中、右区域 |
| 石头 | 5个 | 分布在地图右上、中、左下区域 |
| 食物 | 4个 | 分布在地图中上、右下区域 |
| 陷阱 | 6个 | 分布在通往宝藏的主要路径上 |

## 🎲 游戏规则（新手版）

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择棋子**：每位玩家选择一个颜色的棋子，放在起点🚶
3. **分发资源**：每位玩家获得**3张木材、2张石头、2张食物**
4. **放置陷阱和宝藏**：将所有陷阱卡洗混后，随机放在地图上的⚠️标记位置；将宝藏卡放在终点💎位置

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动
- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的路径移动棋子（只能上下左右移动，不能斜走）

#### 2. 执行所在位置的事件
根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起点** | 🚶 | 无特殊效果 |
| **道路** | ⬛ | 无特殊效果 |
| **资源点** | 🌲🪨🍎 | 获得对应资源卡1张 |
| **陷阱点** | ⚠️ | 抽取并执行陷阱卡效果 |
| **宝藏点** | 💎 | 恭喜！你找到了宝藏，游戏胜利！ |

#### 3. 建造工具（可选）
如果你有足够的资源，可以在回合结束前建造工具：
- 铲子 = 木材×2 + 石头×1
- 灯笼 = 木材×1 + 石头×2
- 绳索 = 木材×3

#### 4. 使用工具（可选）
在回合中任意时刻可以使用工具卡：
- **铲子**：选择一个陷阱点，将其移除
- **灯笼**：查看牌堆顶的一张陷阱卡
- **绳索**：直接移动到任意相邻位置

### 回合结束

完成所有操作后，将回合交给下一位玩家。

## 📖 完整游戏流程示例

**玩家A的回合：**
1. 掷骰子得到5点，移动5步
2. 停在木材资源点，获得1张木材卡
3. 用木材×2 + 石头×1建造了铲子
4. 回合结束，轮到玩家B

**玩家B的回合：**
1. 掷骰子得到3点，移动3步
2. 停在陷阱点，抽到"地洞"陷阱，后退3步
3. 使用铲子清除了面前的陷阱
4. 回合结束，轮到玩家C

**玩家C的回合：**
1. 掷骰子得到6点，移动6步
2. 停在宝藏点！游戏胜利！

## 💡 新手策略提示

1. **优先收集资源**：初期多收集木材和石头，为建造工具做准备
2. **工具很重要**：铲子能帮你清除障碍，绳索能让你快速移动
3. **规划路线**：观察地图，选择陷阱少的路径
4. **注意对手**：如果对手快到终点了，可以尝试用陷阱阻碍他
5. **保存食物**：食物在变体规则中会有用处

## 🔄 变体规则

### 组队模式
- 玩家分成两队（2v2）
- 共享资源卡和工具卡
- 同队玩家可以互相传递卡片

### 时间挑战
- 设置30分钟时间限制
- 在规定时间内找到宝藏的玩家获胜
- 如果时间到无人找到宝藏，则资源最多的玩家获胜

### 难度升级
- 增加陷阱数量（每个陷阱点放2张陷阱卡）
- 减少初始资源（每位玩家只给2张木材、1张石头）
- 添加新陷阱类型：流沙（移动步数减半）、幽灵（交换位置）

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：资源卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用。

**Q：可以跳过建造工具直接移动吗？**
A：可以，建造工具是可选操作。

**Q：多人同时到达宝藏点怎么办？**
A：先到达的玩家获胜。
