# 🦕 恐龙领地

> *灵感来源：侏罗纪世界 / 恐龙主题*

**类别**：策略游戏 | **人数**：2-4人 | **时长**：45-70分钟 | **难度**：中等偏难

---

## 📖 游戏背景

一座新的恐龙主题岛屿刚刚落成，各方势力争相入驻。作为恐龙公园的运营总监，你需要围建领地、驯养恐龙、建设设施、吸引游客。在这片史前乐土上，谁能让自己的园区成为最强王国？

---

## 🎯 游戏目标

通过在地图上扩展领地、驯养不同种类的恐龙、建设园区设施来获取**声望点**。游戏结束时，声望点最高的玩家获胜。

## 🔄 游戏流程

### 准备阶段
1. 随机拼合25块地图板块形成岛屿
2. 每位玩家选择1个角落作为起始基地，放置1个围栏标记
3. 起始资源：2肉 + 2植 + 1电
4. 将恐龙模型按种类放在对应巢穴上

### 回合流程

#### 1. 资源阶段
- 每个己方领地产出对应资源
- 基础收入：+1电

#### 2. 行动阶段（3个行动点）
每位玩家每回合有3个行动点，可执行：

| 行动 | 消耗 | 说明 |
|:---|:---|:---|
| 扩展领地 | 1点 | 在相邻空地放置1个围栏标记 |
| 驯养恐龙 | 1点 | 支付费用，从巢穴获取1个恐龙模型 |
| 建设设施 | 1点 | 支付费用，在己方领地放置1块设施 |
| 迁移恐龙 | 1点 | 将1个恐龙移到己方其他领地 |
| 贸易 | 1点 | 以2:1比率交换任意两种资源 |

#### 3. 事件阶段
翻开1张事件卡，所有玩家受影响：

| 事件 | 效果 |
|:---|:---|
| 🌋 火山喷发 | 火山相邻领地失去1个设施 |
| 🌧️ 暴风雨 | 所有草原领地本回合不产资源 |
| 🦖 恐龙暴动 | 每位玩家1个肉食恐龙逃跑（无避难所时） |
| 🧬 基因变异 | 随机1种恐龙能力增强1回合 |
| 📹 游客涌入 | 所有观光台额外+1声望 |
| 🏃 紧急撤离 | 失去1个边缘领地 |

#### 4. 计分阶段
- 每个恐龙：获得对应声望值
- 每个设施观光台：+1声望
- 领地最大连通区域：每个板块+1声望

---

## 🏆 游戏结束

当满足以下任一条件时触发终局：
- 所有恐龙被驯养完毕
- 某位玩家领地达到10个板块
- 事件卡堆耗尽（完成当前回合后结束）

**终局额外计分**：
- 拥有最多种类恐龙：+5声望
- 拥有最大连通领地：+3声望
- 设施最多：+3声望

**声望最高者获胜！**

## 🎖️ 进阶规则

### 恐龙对战
当两位玩家的领地相邻且各有肉食恐龙时，可发起**恐龙对战**：
- 双方各选1只肉食恐龙比较战力（基础声望值）
- 每花费1肉可+1战力
- 胜方占领败方1个领地板块

### 混合基因
拥有恐龙研究所的玩家可尝试**基因混合**：
- 消耗2电+1肉+1植
- 将2种恐龙的能力合成为1只变异恐龙
- 变异恐龙同时拥有两种能力

---

## 💡 策略提示

1. **领地连通**：分散的领地得分低，优先连通扩展
2. **防御投入**：围墙和避难所看似浪费，但暴动事件会让你损失惨重
3. **食性平衡**：纯肉食或纯植食都很难持续，杂食搭配更稳健
4. **资源规划**：电力是瓶颈资源，优先建设电力产出设施
5. **事件预判**：观察事件卡堆剩余量，提前做防御准备

---

## 📐 变体规则

- **快速模式**：4×4地图，2行动点/回合，15声望即结束
- **合作模式**：所有玩家对抗岛屿危机，需在8回合内完成5个目标
- **远古模式**：取消电力资源，增加"部落"系统，更原始的竞争
