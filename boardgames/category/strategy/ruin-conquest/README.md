## 一、游戏概述

**遗址争霸**是一款2-4人竞技策略桌游，发生在一片神秘的古代大陆上。玩家各自扮演一支雄心勃勃的远古探险文明——腓尼基商贾、玛雅祭司、埃及法老的传人、华夏工匠——带领团队深入尘封的遗址区，争相**挖掘文物、掌控地域、建造纪念碑**，最终以**文明声望积分**判定胜负。

游戏融合了**工人派遣**、**区域控制**和**卡牌驱动**三种机制：每个回合玩家须精心安排有限的探险队员，同时借助手中的遗迹卡和事件卡影响战局。资源稀缺、地图动态变化，迫使玩家既要扩张领地，又要防守核心遗址，形成紧张的博弈节奏。

**适合人群**：喜欢中量级策略、欣赏历史文明主题、有桌游基础或愿意花30分钟上手的玩家；也非常适合家庭或朋友聚会中寻求思维挑战的场合。

## 六、游戏规则

### 6.1 准备阶段

1. **铺设地图**：将7×7地图板展开，将49块遗址瓦片随机背面朝上铺满所有格子（参考打印文件，或使用卡纸制作49块标记）。
2. **分发起始资源**：每位玩家从公共池领取：石料×2，木材×2，黄金×1。
3. **分发文明卡**：各玩家选择文明（腓尼基/玛雅/埃及/华夏），领取对应颜色棋子和3张技能卡。
4. **分发遗迹卡**：将24张遗迹卡洗匀，每人发4张手牌，剩余置于牌堆。
5. **放置起始队员**：按照以下位置放置2名队员（含建营）：
   - 玩家1（腓尼基🟡）→ 格 A7 和 B7
   - 玩家2（玛雅🟢）→ 格 G1 和 F1
   - 玩家3（埃及🔵）→ 格 A1 和 B1（3人时）
   - 玩家4（华夏🔴）→ 格 G7 和 F7（4人时）
6. **确定先手**：掷骰点数最高者先行动，顺时针循环。
7. **置放事件牌堆**：将18张事件卡洗匀正面朝下放置，游戏开始。

### 6.2 游戏流程（每回合结构）

游戏共进行 **8回合**，每回合按以下顺序进行：

#### 阶段一：行动阶段（玩家轮流）
每位玩家在自己的行动轮中拥有 **4点行动力**，可以任意组合使用以下操作：

| 操作 | 花费行动力 | 说明 |
| :--- | :---: | :--- |
| 移动队员 | 1-2（见地形） | 将1名队员移动到相邻格（上/下/左/右），森林地形消耗2 |
| 挖掘遗址 | 1 | 翻开队员所在格的瓦片，立即触发效果并获得资源/文物 |
| 建造纪念碑 | 2 | 在队员所在格建造纪念碑（需满足资源和地形要求） |
| 派遣新队员 | 1 | 从己方营地（起始格）出发，放置1名新队员到起始格 |
| 打出遗迹卡 | 0-1（见卡牌） | 打出手中遗迹卡，大部分无需花费行动力 |
| 驱逐对手 | 2 | 进入对手独占格，驱逐其1名队员（被驱逐者回营地） |
| 收购资源 | 1 | 用2黄金换任意3枚资源（石料或木材） |
| 摸遗迹卡 | 1 | 从牌堆摸1张遗迹卡（上限手牌6张） |

> **注意**：每位玩家在自己回合可任意顺序使用行动力，用完即结束（不可转让或储存）。

#### 阶段二：建筑结算
- 统计本回合新建纪念碑数量
- 每座纪念碑立即为建造者提供 **+2声望**（如事件"黄金时代"激活则+4）

#### 阶段三：区域控制结算
- 统计每位玩家**独自占领**的格子数量（有队员或纪念碑，且没有对手队员的格子）
- 每独占2格 = +1声望

#### 阶段四：事件阶段
- 翻开1张事件卡，立即执行效果
- 将翻开的事件卡放入弃牌堆

#### 阶段五：补充与清理
- 每位玩家补充手牌至4张（从遗迹卡牌堆摸）
- 行动力自动重置为4点（等待下一回合）

### 6.3 纪念碑建造条件一览

| 建造地形 | 所需资源 | 声望奖励 | 特殊效果 |
| :--- | :--- | :---: | :--- |
| 文物区（🏺） | 石料×2 | +2 | 该格文物令牌收益+1分 |
| 珍宝区（👑） | 石料×3 + 黄金×1 | +3 | 每回合被动获得1分文物令牌 |
| 圣殿区（🛕） | 石料×4 + 木材×2 | +5 | 控制神殿：每回合结算+2声望 |

### 6.4 胜利条件

8回合结束后，统计各玩家总声望：

| 来源 | 计分方式 |
| :--- | :--- |
| 文物令牌 | 按令牌面值计分（1分/3分/2分） |
| 纪念碑建造 | 建造时即时加分（见上表） |
| 神殿控制 | 游戏结束时，每控制1座神殿+3声望 |
| 区域控制 | 每独占2格+1声望（每回合结算） |
| 文明技能卡 | 部分卡牌有声望增益 |
| 剩余黄金 | 每3枚黄金=1声望（游戏结束时结算） |

**声望最高者获胜！** 平局时：文物令牌总分多者胜；再平则纪念碑数量多者胜；再平则共同获胜。

## 八、新手策略提示

### 🎯 前期重点：快速扩张
- 开局优先移动向地图中心，中心格资源密集，行动效率更高
- 前2回合尽量挖掘5格以上，积累资源用于后期建造
- 不要过早消耗技能卡，留到关键时刻使用

### 🏛️ 中期核心：争夺神殿
- 第4行3座神殿是全图分数最高的格子，每座神殿=5声望+控制加成，必须争取至少1座
- 建造纪念碑时机要算好——对手接近时抢先建造，等对手离开再建造是浪费
- 封印卡非常强力：提前封住未翻神殿可阻止对手获得挖掘先机

### 🃏 手牌管理：卡牌是制胜关键
- 「深度挖掘」与「地图解密」组合使用：先用解密预判高价值格，再用深度挖掘双倍收益
- 保留至少1张驱逐相关卡牌应对对手骚扰
- 「军事干预」是强驱逐，可用来破坏对手的纪念碑建造计划

### ⚖️ 后期节奏：声望换算
- 游戏最后2回合集中结算剩余黄金（3黄金=1声望），别让黄金浪费
- 区域控制每2格1声望不要忽视——分散占领多格往往优于集中防守

### ❌ 常见新手错误
1. **只顾挖掘忘记建造**：文物令牌有上限，纪念碑是被动收入，应及早布局
2. **资源囤积不用**：遗址争霸没有资源上限机制，但胜利靠声望不靠资源，要及时转化
3. **忽视事件牌**：每回合必翻一张，别做可能被事件打断的大计划
4. **过早暴露手牌**：不要在对手面前提前表露你要打什么遗迹卡，出奇制胜

## 九、变体规则

### 9.1 快速模式（45-60分钟）
- 游戏缩短至 **6回合**
- 每位玩家起始资源增加1石料和1木材
- 适合初次游玩或时间有限时使用

### 9.2 大师模式（2人专用）
- 移除所有文明技能卡，改为从遗迹卡牌堆各摸6张
- 地图缩小为5×5（去除第1、7行和A、G列）
- 更注重卡牌博弈，减少运气成分

### 9.3 联盟模式（4人 2vs2）
- 两名玩家结成一队（对角线起点组队）
- 队友可以共享资源（需相邻格时才能传递）
- 胜利条件：队伍合计声望更高的一方获胜

### 9.4 随机事件加强版
- 每回合翻 **2张**事件卡（效果叠加或取其一，由全体投票决定）
- 游戏节奏加快，随机性增强，适合有经验的玩家

## 十、常见问题

**Q：队员可以和对手的队员共处同一格吗？**  
A：可以！两名玩家的队员可以同时在同一格（不会自动驱逐）。只有打出"军事干预"遗迹卡或使用"法老意志"技能才能强制驱逐。区域控制结算时，有对手队员的格子不计为"独占"。

**Q：纪念碑建好后能被摧毁吗？**  
A：普通情况下不能。只有"叛变风波"事件卡在特殊情况下影响队员，不影响已建纪念碑。技能卡"永恒建筑"（埃及文明）可以完全免疫任何摧毁效果。

**Q：遗址瓦片挖掘后就永久翻开吗？**  
A：是的。翻开的瓦片保持正面朝上，其资源已被获取，但地形类型决定该格是否可建纪念碑。

**Q：我的行动力用完了但还想打遗迹卡，可以吗？**  
A：不可以。行动力归零意味着行动阶段结束（但打遗迹卡本身不消耗行动力的那些卡牌，需在行动阶段打出，归零即结束）。建议提前规划好打牌时机。

**Q：黄金可以直接兑换声望吗？**  
A：只在游戏**结束时**结算：每3枚黄金=1声望。游戏进行中黄金只能用于特定卡牌效果或收购资源。

**Q：2人游戏时，起始位置怎么安排？**  
A：玩家1从A7/B7出发，玩家2从G1/F1出发，对角线布局确保公平距离。

**Q：封印卡放置的格子算"控制"吗？**  
A：算的！有封印卡的格子视同该玩家"占领"，区域控制结算时计入独占格数量。

**Q：如果事件牌堆摸完了怎么办？**  
A：将弃牌堆洗匀后重新作为新牌堆使用。
