# 魔法花园 (Enchanted Garden)

## 🎮 游戏概述

《魔法花园》是一款充满魔法与策略的桌游。玩家扮演魔法园丁，在神秘的魔法花园中种植魔法植物、布置魔法装置、吸引魔法生物，最终打造出最美丽的魔法花园！

通过合理搭配植物、巧妙布局花园、收集稀有魔法生物，你将成为一个传奇的魔法园丁。游戏充满了策略思考和资源管理，适合喜欢深度策略游戏的玩家。

## 🗺️ 游戏地图详解

游戏地图是一个 **5x5 的魔法花园网格**，包含多种花园格子和特殊区域。

### 地图布局概览

```
🌻 💧 ☀️ 🏮 🌻
✨ 🌻 💧 ☀️ 🦋
☀️ ✨ 🌻 💧 🏮
💧 ☀️ ✨ 🌻 💧
🦋 🌻 ☀️ ✨ 🏮
```

### 地形类型说明

| 地形 | 图示 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- |
| **起始点** | 🌻 | 绿色 | 所有玩家开始的位置，位于地图四角 |
| **阳光区** | ☀️ | 黄色 | 停留可获得1张阳光卡 |
| **水分区** | 💧 | 蓝色 | 停留可获得1张水分卡 |
| **魔法区** | ✨ | 紫色 | 停留可获得1张魔法能量卡 |
| **装置区** | 🏮 | 橙色 | 可放置魔法装置 |
| **生物区** | 🦋 | 粉色 | 可吸引魔法生物 |
| **花园格** | 🌻 | 绿色 | 可种植魔法植物 |

### 地图特点

1. **网格设计**：5x5的网格让玩家有策略地规划移动路线
2. **资源分布**：三种资源均匀分布在地图上
3. **特殊区域**：装置区和生物区提供额外的策略深度
4. **对称布局**：地图布局对称，保证公平性

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动
- 可以停留在任何花园格子上
- 移动步数由掷骰子决定（2颗骰子之和）
- 可以种植植物、放置装置、吸引生物的格子有特殊标记

### 地图资源统计

| 资源类型 | 数量 | 位置分布 |
| :--- | :--- | :--- |
| 阳光区 | 5个 | 分布在地图四角和中心 |
| 水分区 | 5个 | 分布在地图边缘 |
| 魔法区 | 4个 | 分布在地图对角线 |
| 装置区 | 3个 | 分布在地图中央 |
| 生物区 | 3个 | 分布在地图角落 |

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择棋子**：每位玩家选择一个颜色的棋子，放在地图四角的起始点🌻
3. **分发资源**：每位玩家获得**3张阳光卡、3张水分卡、1张魔法能量卡**
4. **洗混卡牌**：将植物卡、装置卡、生物卡、事件卡分别洗混，面朝下放在地图旁
5. **设置计分板**：将计分板放在桌面一侧，每位玩家分数为0

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的网格移动棋子（只能上下左右移动，不能斜走）
- 可以停留在任何花园格子上

#### 2. 执行所在位置的事件

根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起始点** | 🌻 | 无特殊效果 |
| **阳光区** | ☀️ | 获得1张阳光卡 |
| **水分区** | 💧 | 获得1张水分卡 |
| **魔法区** | ✨ | 获得1张魔法能量卡 |
| **装置区** | 🏮 | 可放置一个魔法装置（需要资源） |
| **生物区** | 🦋 | 可吸引一个魔法生物（需要资源） |
| **花园格** | 🌻 | 可种植一个魔法植物（需要资源） |

#### 3. 种植植物/放置装置/吸引生物（可选）

如果你有足够的资源，可以在回合结束前：

**种植植物**：
- 选择一张植物卡
- 支付相应的资源成本
- 将植物卡放在当前停留的花园格子上
- 立即获得植物的分数

**放置装置**：
- 只能在装置区🏮放置
- 选择一张装置卡
- 支付相应的资源成本
- 将装置卡放在装置区上
- 装置效果立即生效，持续全场

**吸引生物**：
- 只能在生物区🦋吸引
- 选择一张生物卡
- 支付相应的资源成本
- 将生物卡放在生物区上
- 立即获得生物的分数和特殊能力

#### 4. 抽取事件卡（可选）

玩家可以选择在回合结束时抽取一张事件卡：
- 支付1张魔法能量卡
- 抽取一张事件卡并执行效果
- 事件卡执行后放回牌堆底

#### 5. 激活植物/装置/生物能力（可选）

在回合中任意时刻可以激活：

**植物能力**：
- 支付相应的资源激活植物特殊能力
- 每个植物每回合只能激活一次

**装置能力**：
- 装置效果持续全场，无需激活
- 某些装置有主动能力，可以支付资源激活

**生物能力**：
- 支付相应的资源激活生物特殊能力
- 每个生物每回合只能激活一次

### 回合结束

1. 计算本回合获得的分数，更新计分板
2. 检查是否有植物、装置、生物的能力触发
3. 完成所有操作后，将回合交给下一位玩家

### 游戏结束

游戏在以下任一条件满足时结束：

1. **所有花园格子都被植物占满**
2. **牌堆中的植物卡用完**
3. **任意玩家达到20分**（可调整）

游戏结束后，计算每位玩家的最终分数：
- 植物分数总和
- 装置分数总和
- 生物分数总和
- 特殊能力奖励分数

**分数最高的玩家获胜！**

## 📖 完整游戏流程示例

### 示例1：种植植物

**玩家A的回合：**

1. **准备阶段**：玩家A有3张阳光卡、3张水分卡、1张魔法能量卡，分数0分
2. **掷骰子**：掷出2+4=6点，移动6步
3. **停留位置**：停在了一个空的花园格子🌻上
4. **执行操作**：
   - 选择种植一张「太阳花🌻」植物卡
   - 支付2张阳光卡
   - 将太阳花卡放在当前格子上
   - 立即获得2分（太阳花基础分数）
5. **检查相邻格子**：
   - 相邻的格子上没有植物，太阳花的特殊能力「相邻格子+1分」不触发
6. **回合结束**：玩家A分数更新为2分，剩余1张阳光卡、3张水分卡、1张魔法能量卡
7. **轮到玩家B**

### 示例2：放置装置并激活能力

**玩家B的回合：**

1. **准备阶段**：玩家B有2张阳光卡、4张水分卡、2张魔法能量卡，分数1分
2. **掷骰子**：掷出3+3=6点，移动6步
3. **停留位置**：停在了一个装置区🏮上
4. **执行操作**：
   - 选择放置一张「喷泉⛲」装置卡
   - 支付3张水分卡
   - 将喷泉卡放在当前装置区上
   - 喷泉效果生效：所有水分类植物+1分
5. **检查场上植物**：
   - 玩家A的太阳花🌻不是水分类植物，不加分
   - 如果场上有水晶兰🪷（水分类），则+1分
6. **激活能力**（可选）：
   - 喷泉没有主动能力，无需激活
7. **回合结束**：玩家B分数仍为1分，剩余2张阳光卡、1张水分卡、2张魔法能量卡
8. **轮到玩家C**

### 示例3：吸引生物并使用能力

**玩家C的回合：**

1. **准备阶段**：玩家C有4张阳光卡、2张水分卡、3张魔法能量卡，分数3分
2. **掷骰子**：掷出1+5=6点，移动6步
3. **停留位置**：停在了一个生物区🦋上
4. **执行操作**：
   - 选择吸引一张「蝴蝶🦋」生物卡
   - 支付1张阳光卡
   - 将蝴蝶卡放在当前生物区上
   - 立即获得2分（蝴蝶基础分数）
5. **激活能力**（可选）：
   - 支付1张魔法能量卡
   - 激活蝴蝶的特殊能力「相邻生物+1分」
   - 检查相邻生物区，如果有其他生物，则+1分
6. **回合结束**：玩家C分数更新为5分（2+1+2），剩余3张阳光卡、2张水分卡、2张魔法能量卡
7. **轮到玩家D**

### 示例4：使用事件卡

**玩家D的回合：**

1. **准备阶段**：玩家D有1张阳光卡、5张水分卡、4张魔法能量卡，分数2分
2. **掷骰子**：掷出4+2=6点，移动6步
3. **停留位置**：停在了一个魔法区✨上
4. **执行操作**：
   - 获得1张魔法能量卡
   - 现在拥有5张魔法能量卡
5. **抽取事件卡**（可选）：
   - 支付1张魔法能量卡
   - 抽取一张事件卡：「魔法雨🌧️」
   - 执行效果：所有玩家获得1张水分卡
   - 玩家D获得1张水分卡，现在拥有6张水分卡
6. **回合结束**：玩家D分数仍为2分，剩余1张阳光卡、6张水分卡、3张魔法能量卡
7. **轮到玩家A**

## 💡 新手策略提示

1. **优先收集资源**：初期多收集阳光和水分，为种植植物做准备
2. **植物搭配很重要**：注意植物的特殊能力，合理搭配可以产生连锁效果
3. **装置提供持续增益**：尽早放置装置，让效果持续全场
4. **生物能力强大**：吸引生物不仅提供分数，还能提供强大的特殊能力
5. **观察对手**：注意对手的花园布局，选择合适的格子种植植物
6. **合理使用事件卡**：事件卡可以改变局势，但不要盲目使用
7. **平衡进攻与防守**：既要种植高分植物，也要防止对手的战术破坏

## 🔄 变体规则

### 双人模式

- 每位玩家控制两个棋子
- 共享资源卡，但需要合理分配
- 增加策略深度

### 团队合作模式

- 所有玩家合作对抗「枯萎病」
- 共同种植植物，保护花园
- 如果所有植物都被烧毁/冻结，则全员失败

### 快速模式

- 游戏目标分数降低到15分
- 初始资源增加（5张阳光、5张水分、3张魔法能量）
- 加快游戏节奏

### 高级模式

- 增加新的植物、装置、生物卡
- 增加「季节系统」：不同季节影响植物生长
- 增加「魔法灾害」：随机破坏花园格子

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**
A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：资源卡用完了怎么办？**
A：可以将弃牌堆洗混后重新使用。

**Q：可以移除已经种植的植物吗？**
A：一般情况下不能，除非有特殊的植物（如火焰花🔥）或生物能力允许。

**Q：装置和生物可以被破坏吗？**
A：一般情况下不能，除非有特殊的植物（如龙🐉）能力允许。

**Q：多人同时达到20分怎么办？**
A：比较特殊能力奖励分数，分数高的获胜。如果仍相同，则共用冠军。

**Q：可以在同一格子上种植多个植物吗？**
A：不可以，每个格子只能种植一个植物。
