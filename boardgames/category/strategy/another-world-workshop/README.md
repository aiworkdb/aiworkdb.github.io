# 异世界工坊 (Another World Workshop)

## 🎮 游戏概述

《异世界工坊》是一款以"异世界转生"为背景的策略桌游。玩家扮演连接多个异世界的工坊主人，派遣探险队收集稀有材料，制作强大物品，吸引冒险者，最终成为传奇工匠！

结合当下最流行的异世界题材（如《关于我转生变成史莱姆这档事》《无职转生》《异世界食堂》等），这款桌游让你体验经营跨世界工坊的乐趣。

适合喜欢策略规划、资源管理、异世界题材的玩家，规则深度适中，充满 replay 价值！

## 🎲 游戏规则

### 准备阶段

1. **布置板图**：将游戏板图放在桌面中央
2. **选择工坊**：每位玩家选择一个颜色的工坊板和棋子
3. **分发初始资源**：
   - 每位玩家获得5金币、0声望
   - 每位玩家抽取5张材料卡作为初始手牌
   - 每位玩家获得1张随机工具卡
4. **准备卡牌堆**：
   - 将材料卡洗混，放在板图指定位置
   - 将配方卡洗混，放在板图指定位置
   - 将冒险者卡洗混，放在板图指定位置
   - 将事件卡洗混，放在板图指定位置
5. **决定先手**：掷骰子，点数最高的玩家先手

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下**四个阶段**：

#### 阶段1：探险阶段 🗺️

- 移动你的棋子到任意一个异世界区域
- 从该区域的材料堆中抽取**1-2张材料卡**（取决于区域难度）
- 难度简单的区域抽2张，困难的区域抽1张但有几率获得稀有材料
- 如果你的棋子已经在某个区域，可以继续探索，额外抽1张材料卡

#### 阶段2：制作阶段 🔨

- 查看你手中的材料卡和配方卡
- 如果你有足够的材料，可以**制作物品**
- 每回合最多可以制作**2个物品**
- 制作物品需要：
  1. 展示配方卡
  2. 消耗对应的材料卡
  3. 将配方卡放在你的工坊板上
  4. 获得对应的声望点数
- 制作物品后，物品会展示在你的工坊中，吸引冒险者

#### 阶段3：吸引阶段 🧙

- 检查你工坊中展示的物品
- 查看冒险者卡堆顶部的**1-2张卡**
- 如果你的物品满足冒险者的需求，可以**完成订单**
- 完成订单后：
  1. 获得冒险者卡上的金币和声望奖励
  2. 将冒险者卡放在你的得分区
  3. 补充新的冒险者卡到卡堆

#### 阶段4：事件阶段 ⚡

- 抽取**1张事件卡**
- 所有玩家执行事件卡的效果
- 如果是负面事件，可以使用工具卡来规避

### 回合结束

完成所有阶段后，将回合交给下一位玩家。

### 胜利条件

游戏进行**8轮**后，计算所有玩家的声望点数：

- 基础声望：来自制作的物品
- 冒险者奖励：完成订单获得的额外声望
- 金币转换：每5金币可以转换为1声望（四舍五入）

**声望点数最多的玩家获胜！**

如果平局，则金币多的玩家获胜。

**玩家A的第1回合：**

**阶段1：探险阶段**

- 抽取事件卡：「材料丰收」

## 💡 新手策略提示

1. **前期优先探索简单区域**
   - 精灵森林和海洋王国难度低，适合前期积累材料
   - 稳定获得材料比冒险更重要

2. **平衡制作和吸引**
   - 不要只制作物品，要关注冒险者的需求
   - 制作冒险者需要的物品，获得更多奖励

3. **合理使用工具卡**
   - 工具卡是有限资源，不要在前期浪费
   - 「守护符文」留到关键时刻使用

4. **关注事件卡**
   - 事件卡影响所有玩家，要提前规划
   - 负面事件来临时，用手中的工具卡应对

5. **多样化材料收集**
   - 不要只收集一种材料
   - 配方卡需要多种材料，多样化收集更有优势

6. **后期冲击高声望物品**
   - 游戏后期可以冒险去困难区域
   - 传说级物品的声望值很高，是翻盘的关键

## 🔄 变体规则

### 快速模式（适合新手）

- 游戏轮数减少到**6轮**
- 每位玩家初始手牌增加到**7张**
- 简化事件卡效果

### 合作模式

- 所有玩家合作对抗「时钟事件」
- 共同目标是制作「传说级物品」击败最终BOSS
- 事件卡改为「时钟事件」，每回合时钟前进1格
- 时钟到达12时，如果未完成传说物品，全员失败

### 竞技模式（适合老手）

- 玩家可以**偷取**其他玩家的材料卡（每回合1次）
- 可以增加「竞争事件卡」，让玩家之间相互干扰
- 胜利条件改为**10轮**或**率先达到30声望**

### 单人挑战模式

- 玩家独自对抗「时钟」
- 目标是在12回合内制作出「天空披风」「火焰大剑」「传说盾牌」三件物品
- 事件卡改为帮助玩家的「好运事件」和阻碍的「厄运事件」

## ❓ 常见问题

**Q：如果材料卡用完了怎么办？**
A：将所有弃牌堆洗混，重新组成新的材料卡堆。

**Q：可以同时制作多个相同的物品吗？**
A：可以，只要你有足够的材料和对应的配方卡。

**Q：冒险者卡的需求物品，可以用更高等级的物品代替吗？**
A：不可以，必须使用需求中指定的物品。

**Q：如果我的工坊板放不下更多配方卡怎么办？**
A：可以购买「工坊扩建」（花费10金币），增加2个展示槽。

**Q：事件卡的负面效果可以避免吗？**
A：可以，使用「守护符文」工具卡可以免疫一次负面效果。

**Q：游戏结束时，未使用的金币和材料卡怎么处理？**
A：每5金币可以转换为1声望。材料卡和工具卡不再计分。

**Q：如果多名玩家同时完成冒险者订单怎么办？**
A：先完成者获得奖励，后面的玩家可以等待下一张冒险者卡。

**Q：棋子在同一个区域可以多停留几回合吗？**
A：可以，停留可以更深入地探索，但也要注意事件卡的风险。
