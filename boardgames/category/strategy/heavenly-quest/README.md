# 天命取经 (Heavenly Quest)

## 🎮 游戏概述

《天命取经》是一款深度融合中国神话元素的策略桌游，灵感来源于《黑神话：悟空》和《西游记》经典故事。

玩家将扮演天命修行者，在充满危险的取经路上收集神器、战胜妖魔、修行悟道，最终到达雷音寺取得真经。游戏融合了角色扮演、资源管理、战斗策略和合作元素，既有个人修行之路，也可选择合作模式共同取经。

适合喜爱中国神话、策略思考和团队合作的玩家，是一场充满挑战的修行之旅！

## 🗺️ 游戏地图详解

游戏地图是一个 **5x5 的取经路网格**，从东土大唐到雷音寺，途中经过九九八十一难。

### 地图布局概览

```
🏯 💎 🌿 ⚡ 🔥
👹 🏆 📜 💎 🌿
⚡ 🔥 👹 🏆 📜
🌿 💎 ⚡ 🔥 👹
🏆 📜 🌿 💎 🛕
```

### 地形类型说明

| 地形 | 图示 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- |
| **起点（大唐）** | 🏯 | 金色 | 所有玩家开始的位置，位于地图左上角 |
| **灵石矿** | 💎 | 蓝色 | 停留可获得1张灵石卡 |
| **仙草地** | 🌿 | 绿色 | 停留可获得1张仙草卡 |
| **法宝洞** | ⚡ | 紫色 | 停留可获得1张神器卡 |
| **妖魔洞** | 👹 | 红色 | 停留需与妖魔战斗 |
| **仙人台** | 🏆 | 橙色 | 停留可获得1张法宝卡 |
| **真经阁** | 📜 | 白色 | 停留可获得1张真经卡 |
| **终点（雷音寺）** | 🛕 | 金色 | 到达并持有3张真经卡即可获胜 |

### 地图特点

1. **路径设计**：玩家可自由在网格中上下左右移动，探索取经路
2. **资源分布**：灵石和仙草均匀分布，保障基本资源获取
3. **危险分布**：妖魔洞位于关键位置，增加战斗挑战
4. **终极目标**：雷音寺位于右下角，需收集真经才能最终获胜

### 移动规则

- 玩家可**上下左右**移动，不能斜向移动
- 移动步数由掷骰子决定（2颗骰子之和）
- 可停留在任何地形上，触发相应事件

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图放在桌面中央
2. **选择角色**：每位玩家选择一个修行者角色（悟空、八戒、沙僧、白龙马），每个角色有独特能力
   - **悟空**：战斗时骰子+1
   - **八戒**：仙草效果翻倍
   - **沙僧**：灵石消耗-1
   - **白龙马**：移动步数+1
3. **选择棋子**：每位玩家选择一个颜色的棋子，放在起点🏯
4. **分发资源**：每位玩家获得**3张灵石、2张仙草**
5. **洗混卡牌**：将所有卡牌洗混，放在地图旁边

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动

- 掷2颗骰子，点数相加得到移动步数
- 按照地图上的网格移动棋子（只能上下左右移动）

#### 2. 执行所在位置的事件

根据棋子停留的位置，执行相应的动作：

| 位置类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **起点（大唐）** | 🏯 | 无特殊效果，可补充基础资源（各+1） |
| **灵石矿** | 💎 | 获得1张灵石卡 |
| **仙草地** | 🌿 | 获得1张仙草卡 |
| **法宝洞** | ⚡ | 获得1张神器卡（如已有3张神器，可换成新神器） |
| **妖魔洞** | 👹 | 抽取一张妖魔卡，进入战斗 |
| **仙人台** | 🏆 | 获得1张法宝卡 |
| **真经阁** | 📜 | 获得1张真经卡 |
| **终点（雷音寺）** | 🛕 | 如果持有3张真经卡，恭喜！游戏胜利！ |

#### 3. 战斗系统（在妖魔洞触发）

当玩家停留在妖魔洞时，需要战斗：

1. 抽取一张妖魔卡，查看其战斗力（需要骰子点数≥X）
2. 玩家投掷2颗骰子，如果点数之和≥妖魔战斗力，则战斗胜利
3. 可使用神器卡和法术卡增强战斗能力
4. 战斗胜利后，获得妖魔卡上的奖励
5. 战斗失败后，失去1点天命值，回合结束

#### 4. 装备神器（可选）

如果你有足够的资源，可以在回合结束前装备神器：
- 神器卡上有激活所需的资源
- 支付资源后，神器卡放在自己面前，生效
- 每个玩家最多装备3张神器卡

#### 5. 使用法术（可选）

在回合中任意时刻可以使用法术卡：
- 支付相应的资源（如有）
- 执行法术卡上的效果
- 法术卡使用后即放入弃牌堆

### 回合结束

完成所有操作后，将回合交给下一位玩家。

## 📖 完整游戏流程示例

**玩家A（悟空）的回合：**

1. 掷骰子得到7点，移动7步
2. 停在灵石矿💎，获得1张灵石卡
3. 用灵石×2激活"金箍棒"神器（战斗时骰子+1）
4. 回合结束，轮到玩家B

**玩家B（八戒）的回合：**

1. 掷骰子得到4点，移动4步
2. 停在妖魔洞👹，抽取妖魔卡"山妖"（需要≥7）
3. 掷骰子战斗，得到9点（≥7），战斗胜利！
4. 获得奖励：灵石×2
5. 使用"七十二变"法术，额外移动一次
6. 停在仙草地🌿，获得1张仙草卡
7. 回合结束，轮到玩家C

**玩家C（沙僧）的回合：**

1. 掷骰子得到8点，移动8步
2. 停在真经阁📜，获得1张真经卡（已有2张，再得1张就获胜！）
3. 用灵石×1 + 仙草×2激活"筋斗云"神器（移动+2）
4. 回合结束，轮到玩家D

**玩家D（白龙马）的回合：**

1. 掷骰子得到6点，移动6步（神器"筋斗云"效果，+2步，共8步）
2. 停在雷音寺🛕！但只有1张真经卡，未能获胜
3. 回合结束，轮到玩家A

**玩家A（悟空）的回合：**

1. 掷骰子得到5点，移动5步
2. 停在真经阁📜，获得1张真经卡（已有2张）
3. 下回合争取再得1张真经，然后冲向雷音寺！

## 💡 新手策略提示

1. **优先收集资源**：初期多收集灵石和仙草，为装备神器和释放法术做准备
2. **神器很重要**：金箍棒和筋斗云是最实用的神器，优先装备
3. **谨慎战斗**：妖魔战斗力高的话，可以先装备神器或使用法术再战斗
4. **规划路线**：观察地图，选择资源丰富的路径，避免连续遇到妖魔
5. **注意对手**：如果对手快收集到3张真经了，可以尝试抢先到达雷音寺
6. **合理使用法术**：法术卡是一次性的，关键时刻使用效果最佳
7. **角色能力**：充分利用自己角色的独特能力，制定个性化策略

## 🔄 变体规则

### 合作模式

- 所有玩家组成取经团队，共同对抗妖魔
- 共享资源卡和神器卡
- 同队玩家可以互相传递卡片
- 团队共同收集5张真经卡，然后一起到达雷音寺获胜
- 如果任何玩家天命值归零，团队失败

### 时间挑战

- 设置60分钟时间限制
- 在规定时间内取得真经的玩家获胜
- 如果时间到无人获胜，则收集真经卡最多的玩家获胜

### 难度升级

- 增加妖魔战斗力（所有妖魔+2）
- 减少初始资源（每位玩家只给2张灵石、1张仙草）
- 添加新妖魔类型：大圣（需要骰子点数≥13）、佛祖（需要≥15）

### 快速模式

- 只需收集2张真经卡即可获胜
- 地图缩小为4x4
- 游戏时长缩短为30-45分钟

## ❓ 常见问题

**Q：如果我没有骰子怎么办？**

A：可以使用手机上的骰子应用，或者用1-6的数字卡片代替。

**Q：资源卡用完了怎么办？**

A：可以将弃牌堆洗混后重新使用。

**Q：可以跳过装备神器直接移动吗？**

A：可以，装备神器是可选操作。

**Q：多人同时到达雷音寺怎么办？**

A：先到达且持有3张真经卡的玩家获胜。如果都持有3张真经，则比较天命值，高的获胜。

**Q：战斗中可以使用多张法术卡吗？**

A：可以，在战斗阶段可以使用任意多张法术卡。

**Q：神器卡可以替换吗？**

A：可以，如果你已有3张神器，可以放弃一张旧神器，装备一张新神器。

**Q：法宝卡和神器卡有什么区别？**

A：法宝卡效果更强大，但更难获得（只能在仙人台获得）。法宝卡不占用神器卡位置。
