# 星域征服者 (Stellar Conqueror)

## 🎮 游戏概述

在遥远的未来，人类已经掌握了星际旅行的技术。你是一位雄心勃勃的星际指挥官，率领你的探险舰队穿梭于银河系，寻找神秘的外星文明遗迹。通过扫描未知星域、开采稀有资源、发展前沿科技，你将在宇宙中建立属于自己的星际帝国——第一个将旗帜插上传说星球并积累最多胜利点的玩家，将成为真正的星域征服者！

这是一个适合喜欢策略规划和资源管理的玩家的中难度策略桌游。游戏融合了探索、收集、建造和竞争元素，每局游戏都因随机生成的星图而充满新鲜感。

## 🎲 游戏规则

### 准备阶段

**Step 1：布置银河星图**
将游戏地图放在桌面中央。地图为3×3九宫格，玩家都能清楚看到各星域的位置。

**Step 2：选择飞船并放置**
每位玩家选择一种颜色的飞船棋子，全部放在**起点星域（🚀）**。

**Step 3：分发初始资源**
每位玩家获得：
- 能量晶石 × 5
- 星尘 × 2
- 扫描卡 × 2
- 玩家面板（记录胜利点：起始为0）

**Step 4：布置星球卡和事件卡**
- 将普通星球卡、稀有星球卡、传说星球卡分别洗匀，面朝下放在对应星域旁边
- 将事件卡洗匀，放在事件星域旁边作为事件牌堆
- 将科技卡洗匀，放在科技星域旁边作为科技牌堆

**Step 5：确定起始玩家**
年纪最小的玩家先手，之后按顺时针方向轮转。

---

### 游戏流程（每个回合 = 行动阶段 + 结束阶段）

#### 行动阶段

每位玩家在自己的回合中依次执行以下三个步骤：

**步骤1：掷骰子移动（必须）**
- 掷2颗骰子，将点数相加得到移动距离
- 可向上下左右四个方向移动到相邻格子（不能斜走）
- **必须走满全部步数**，中途经过的格子可停留查看（但不触发效果）
- 停在某格子后，触发该星域效果

**步骤2：触发星域效果（必须）**
根据停留的星域类型，执行对应效果：

| 停留星域 | 效果 |
| :--- | :--- |
| 起点 🚀 | 获得1张能量晶石 |
| 探索区 🔭 | 获得1张扫描卡 |
| 普通星域 🌍 | 抽取1张普通星球卡，可选择占领（立即获得对应胜利点）或放弃 |
| 能源星域 ⚡ | 获得2张能量晶石 |
| 传说星域 💫 | 抽取1张传说星球卡，必须消耗2张星尘激活，占领获得5 VP |
| 稀有星域 🌟 | 抽取1张稀有星球卡，必须消耗1张星尘激活，占领获得3 VP |
| 事件星域 ☄️ | 抽取1张事件卡，立即执行效果 |
| 科技星域 🔬 | 抽取1张科技卡，可选择获得卡牌或用3能量兑换1星尘 |
| 珍稀星域 ✨ | 随机获得1张星尘或1张能量晶石（掷骰子：奇数得星尘，偶数得能量） |

**步骤3：执行行动（可选）**

以下行动可以任意组合执行：

| 行动 | 消耗 | 效果 |
| :--- | :--- | :--- |
| 使用扫描卡 | 见扫描卡说明 | 详见扫描卡道具详解 |
| 建造科技卡 | 见科技卡消耗 | 获得对应科技能力（永久或一次性） |
| 激活星球 | 星尘 | 将手牌中的稀有/传说星球激活为已占领 |
| 兑换星尘 | 能量×3 | 在科技星域将3能量兑换为1星尘 |

> ⚠️ **占领限制**：同一星球在同一时间只能被一位玩家占领。占领后，该星球卡被移除游戏。每种星球卡用完即止，需要玩家合理规划战略。

#### 结束阶段

1. 检查胜利条件：如果有玩家达到 **10 胜利点**，游戏立即结束，该玩家获胜
2. 若无人达到10 VP，回合交给左手边玩家，继续游戏

## 📖 完整游戏流程示例

### 示例一：第1回合（玩家A）

**背景**：玩家A拥有5能量、2星尘、2扫描卡，当前在起点星域。

1. **掷骰子**：得到3+4=7点
2. **移动**：从起点向上→右→右→右→右→右→右移动（每步1格，共7步），停在**能源星域（⚡）**
3. **触发效果**：获得2张能量晶石（现有7能量）
4. **行动**：使用扫描卡"远距扫描"（消耗1能量），查看探索区，发现一张"气态巨行星"普通星球卡
5. **回合结束**：玩家A当前持有：能量×7（-1扫描）、星尘×2、胜利点×0

### 示例二：第2回合（玩家B）

**背景**：玩家B拥有5能量、2星尘、2扫描卡、1 VP（已占领1个岩石行星），当前在普通星域。

1. **掷骰子**：得到2+5=7点
2. **移动**：从普通星域向下→向下→右移动（7步），停在**稀有星域（🌟）**
3. **触发效果**：抽取到"钻石星球"稀有卡，消耗1星尘激活，获得3 VP（当前总计4 VP）
4. **行动**：用3能量在科技星域兑换1星尘（当前星尘恢复为1）
5. **回合结束**：玩家B当前持有：能量×4、星尘×1、胜利点×4

### 示例三：关键一回合（玩家C）

**背景**：玩家C拥有8能量、3星尘、8 VP，距离胜利只差2 VP。当前在能源星域。

1. **掷骰子**：得到1+6=7点
2. **移动**：移动7步，停在**传说星域（💫）**
3. **触发效果**：抽取到"永恒星云"传说卡！
4. **激活传说星球**：消耗2星尘，激活传说星球，获得5 VP！
5. **胜利！** 8 + 5 = **13 VP > 10 VP**，玩家C获得胜利！🎉

## 💡 新手策略提示

### 早期策略（前3回合）
1. **优先占领普通星球**：普通星球无需资源即可占领，1-2 VP积累快，是稳健的开局
2. **多去能源星域**：能源是万能资源，有能量才能做任何事，能源星域提供+2能量晶石
3. **不要急着激活稀有星球**：稀有星球虽然+3 VP，但需要消耗星尘，过早消耗会影响传说星球的激活

### 中期策略（第4-8回合）
4. **科技投资回报高**：曲速引擎（+2移动距离）和推进器（+1骰子）能让你走得更远，接触更多资源点
5. **规划传说星球路线**：传说星球5 VP是翻盘利器，建议中期就攒够2星尘，等待机会
6. **注意对手动态**：如果对手接近10 VP，可以考虑占领传说星域阻断其路线

### 后期策略（决胜阶段）
7. **算VP比算资源更重要**：后期每一步决策都要考虑VP差距，必要时可用能量竞标
8. **能量护盾保平安**：在事件星域前建造能量护盾，可以免疫一次负面事件
9. **传说星球是制胜关键**：如果能占领1个传说星球（5 VP），几乎可以锁定胜局

### 通用技巧
- 🎯 **能量优先**：能量可以兑换星尘，可以建造科技，是最重要的资源
- 🗺️ **善用扫描**：在探索区停留抽扫描卡，比直接花钱更划算
- ⚡ **能源区轮转**：能源→科技→普通→能源是经典的三角循环路线

## 🔄 变体规则

### 变体1：双人地图（适合2人）

将地图缩小为5格（起点+4个星域），增加能源星域和科技星域的出现频率，加快游戏节奏。胜利条件降低至**8 VP**。

### 变体2：组队模式（适合4人）

玩家两两组队，共享资源和星球卡。同队玩家可以：
- 自由传递手牌
- 合并能量/星尘池
- 在对手回合提供建议（但不能直接操作）

胜利条件：队伍合计达到 **15 VP** 获胜。

### 变体3：时间挑战

设置45分钟沙漏计时。时间结束时VP最高的玩家获胜。如果出现平局，则比较手牌数量。

### 变体4：极速模式

每回合仅允许执行"移动+触发效果"两个步骤，跳过行动阶段。建造科技和兑换资源改为每回合自动执行基础产出：
- 每回合开始时自动获得1能量
- 每2回合可免费激活1张普通星球卡

### 变体5：星际战争（对抗增强）

添加"引力炸弹"科技卡的使用频率。玩家可以在任意时刻（不限回合）使用引力炸弹攻击他人占领的星球，使其失去1 VP。被攻击的玩家可以用2能量修复星球。

## ❓ 常见问题

**Q：游戏最多进行多少回合？**
A：没有上限。游戏持续到有玩家达到10 VP为止。如果超过20回合仍未有人获胜，建议重新开始一局。

**Q：骰子掉了或找不到怎么办？**
A：使用手机骰子APP，或用1-6的数字牌代替。每次投掷后放回牌堆洗匀再用。

**Q：星球卡用完了怎么办？**
A：将所有已占领星球卡移除游戏，空出的星域变为"荒芜区"（停留无效果）。建议每局游戏前准备备用的普通星球卡。

**Q：多个玩家同时到达同一个星域怎么办？**
A：按到达顺序处理效果（掷骰子时谁先谁后由回合顺序决定）。但占领效果只触发一次——第一个宣布占领的玩家获得胜利点。

**Q：可以用能量购买其他玩家的资源吗？**
A：在基础规则中不可以，只能通过"外星商人"事件卡进行交易。变体规则中可开放玩家间自由交易。

**Q：扫描卡和科技卡有什么区别？**
A：扫描卡用于探索和情报获取，通常一次性消耗；科技卡用于获得永久或一次性强力能力，建造后永久拥有（一次性科技卡使用后弃掉）。

**Q：如果停在传说星域但没有足够的星尘激活怎么办？**
A：可以选择只拿取1能量晶石，放弃占领传说星球。或者用3能量在科技星域兑换1星尘（需额外行动）。

**Q：传说星球被占领后，其他玩家还能去那个格子吗？**
A：可以。占领星球只是获得胜利点，不影响格子通行权。
