# 蒸汽革命 (Steam Revolution)

## 1. 游戏概述

欢迎来到19世纪的工业革命时代！在《蒸汽革命》中，你是一位雄心勃勃的工业家，要在蒸汽与齿轮的世界中建造工厂、铺设铁路、发展城市，最终成为这个时代最伟大的工业巨头。

游戏融合了**引擎构建**、**铁路网络**和**资源转化**三大核心机制。你需要建立一个高效的工业引擎：从矿山获取原料，在工厂加工成品，通过铁路运往城市销售获利，再用利润投资更先进的技术。每一个决策都在塑造你的工业帝国——是专注钢铁帝国，还是发展纺织王国？是垄断铁路运输，还是多元化经营？

**适合人群**：喜欢策略深度、引擎构建、长线规划的玩家。2-5人游玩，节奏沉稳、思考空间大，适合喜欢《丝绸之路》《星际贸易站》风格的玩家。

## 6. 游戏规则

### 准备阶段

1. **放置地图**：将工业地图放在桌面中央
2. **选择颜色**：每位玩家选择1种颜色，领取对应棋子和标记
3. **设置城市**：根据人数放置城市卡（2人4城市，3-5人5-6城市）
4. **放置订单**：在每个城市旁放置1张订单卡
5. **分发初始资源**：根据人数发放初始金币
6. **洗混卡组**：工厂卡、技术卡、事件卡分别洗混
7. **抽取工厂卡**：每人抽取3张工厂卡（选择1张建造为起始工厂）
8. **放置棋子**：所有棋子放在起始城市

### 游戏流程（每回合）

每回合分为4个阶段：

#### 阶段1：收入阶段
- 每座运行中的工厂产出资源
- 每条铁路产生1金通行费（收取者为自己）
- 矿山自动产出对应资源

#### 阶段2：行动阶段（轮流执行）

每位玩家有**3个行动点**，可选择以下行动（可重复）：

**行动A：采集资源**（1行动点）
- 移动到矿山，获得2个对应资源
- 已研发「矿井通风」技术则获得3个

**行动B：工厂加工**（1行动点）
- 在工厂区，消耗原材料生产成品
- 炼钢厂：2煤炭 → 2钢铁
- 纺织厂：2棉花 → 2成品
- 机械厂：1钢铁+1煤炭 → 3成品

**行动C：铺设铁路**（2行动点+3金）
- 在两个相邻区域间铺设铁路
- 拥有铁路后，在这两个区域间移动免费
- 可以收取其他玩家的通行费（1金/次）

**行动D：建造工厂**（2行动点+费用）
- 打出1张工厂卡，支付建造费用
- 将工厂标记放在工厂区
- 每位玩家最多拥有3座工厂

**行动E：完成订单**（1行动点）
- 移动到城市，交付所需资源
- 获得订单奖励金币
- 该城市翻出新的订单卡

**行动F：研发技术**（1行动点+费用）
- 打出1张技术卡，支付研发费用
- 技术效果立即生效，永久持续

**行动G：移动**（1行动点）
- 移动到相邻区域
- 有铁路连接的区域间移动免费

#### 阶段3：事件阶段
- 翻开1张事件卡，执行效果
- 事件影响所有玩家

#### 阶段4：补充阶段
- 翻开新的订单卡补充城市
- 玩家手牌上限检查（工厂卡3张，技术卡2张）

### 胜利条件

游戏进行指定回合数后结束：

**计分方式**：
- 金币：每3金 = 1分
- 工厂：每座工厂 = 2分
- 铁路：每条铁路 = 1分
- 技术：每项技术 = 2分
- 完成订单：每个完成的订单 = 1分
- 城市发展：拥有最多铁路连接的城市 = 3分

**最高分者获胜！成为工业革命时代的霸主！**

## 7. 完整游戏流程示例

### 示例回合：玩家A的第3回合

**初始状态**：
- 金币：8金
- 资源：煤炭×3，钢铁×1
- 工厂：1座炼钢厂
- 技术：无
- 位置：炼钢厂

**阶段1：收入**
- 炼钢厂产出：2钢铁

**阶段2：行动**
1. **行动1**：工厂加工（1行动点）→ 消耗2煤炭+2钢铁，产出3成品
2. **行动2**：移动到伦敦（1行动点）
3. **行动3**：完成订单📦2（1行动点）→ 交付2成品，获得6金

**阶段3：事件**
- 翻开事件卡：📈经济繁荣！所有订单奖励+2金（下回合生效）

**阶段4：补充**
- 伦敦翻出新订单📦3（需要3成品+1钢铁，奖励10金）
- 当前金币：8+6=14金

## 8. 新手策略提示

1. **先建工厂再铺铁路**：工厂是收入来源，铁路是锦上添花
2. **专注1-2种资源链**：不要什么都做，精通一条产业链更有效率
3. **关注订单需求**：根据订单调整生产，不要盲目产出
4. **技术是长期投资**：早期研发1-2项技术，中后期回报丰厚
5. **铁路是双刃剑**：自己用免费，但别人也要付费，考虑铺设位置
6. **金币不是目的**：最终比的是分数，别囤金不花
7. **关注事件**：经济繁荣时多完成订单，萧条时多投资建设

## 9. 变体规则

### 变体1：速战速决模式
- 游戏仅8回合
- 所有费用-2金
- 适合快节奏对局

### 变体2：自由市场模式
- 玩家间可以自由交易资源
- 价格由双方商定
- 增加互动和谈判元素

### 变体3：工业帝国模式（合作）
- 所有玩家共同发展一个工业帝国
- 目标：在15回合内达到100分
- 增加团队协作元素

### 变体4：蒸汽朋克模式
- 增加「发明卡」，可以创造超现实技术
- 例如「蒸汽飞行器」「机械巨兽」
- 增加趣味性和想象力

## 10. 常见问题

**Q：可以拥有超过3座工厂吗？**
A：不可以，每位玩家最多拥有3座工厂。如果你想建新工厂，必须先废弃1座旧工厂。

**Q：铁路可以拆掉吗？**
A：不可以。铁路一旦铺设就永久存在。铺设前请三思！

**Q：资源有上限吗？**
A：资源没有上限，但过多的资源意味着没有及时转化成金币或分数。

**Q：行动点可以存到下回合吗？**
A：不可以。每回合3个行动点，用不完就浪费了。

**Q：多个玩家可以在同一区域吗？**
A：可以。区域没有容量限制，但资源产出是各自的。

**Q：技术卡效果可以叠加吗？**
A：可以！例如同时拥有「蒸汽机改良」和「铁路网络」，铁路效率+1且铺设费用-2。

**Q：游戏时间太长怎么办？**
A：使用速战速决模式，或将回合数减少为8回合。
