# 女巫的花园 (Witch's Garden)

## 🎮 游戏概述

《女巫的花园》是一款充满魔法与田园风情的策略桌游，灵感来源于当下热门的"治愈系"游戏（如《星露谷物语》、《动物森友会》）。

玩家扮演一名初出茅庐的实习女巫，在魔法村郊外开辟自己的魔法花园。通过种植魔法植物、收获神奇原料、酿造魔法药剂，完成村民们的委托订单，最终成为一名受人尊敬的正式女巫！

**适合人群：** 喜欢轻松策略、田园建造、引擎构筑类游戏的玩家；适合家庭聚会和朋友游戏之夜；推荐给想要放松身心、享受慢节奏策略对抗的玩家。

| **龙焰椒种子** | 🌶️ | 商店花3金币购买 / 魔法森林花2金币 | 免费种植 | 收获后得龙焰椒原料×1 |
| **精灵花种子** | 🧚 | 魔法森林花2金币购买 | 免费种植 | 收获后得精灵花粉原料×1 |

**种植规则：**
- 每位玩家在自己的回合可以将手牌中的种子卡种植到花园空格
- 每格仅能种1颗种子
- 种植后该格进入"生长中"状态，需要等待成熟时间

## 🎲 游戏规则

### 准备阶段

1. **布置公共地图**：将公共村庄地图放在桌面中央
2. **分发个人花园板**：每位玩家领取1张花园板（5×5网格）
3. **选择棋子**：每位玩家选择1个颜色的棋子，放在地图起点🏠
4. **初始资源**：
   - 每位玩家获得：胡萝卜种子×3、月光草种子×1、金币×3
   - 将订单卡洗混，面朝下放在"订单公告栏"旁
   - 将事件卡洗混，面朝下放在"事件格"旁
   - 准备声誉标记池（共30个⭐）
5. **决定先手**：掷骰子，点数最大者先手，顺时针进行

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 第一步：移动阶段
- 掷1颗骰子，点数决定移动格数
- 沿公共地图顺时针或逆时针移动（每回合可自由选择方向）
- 停在目标格子，执行该格效果

#### 第二步：执行格子效果

| 格子 | 操作 |
| :--- | :--- |
| **🏪 魔法商店** | 花费金币购买种子卡和工具卡（见价格表） |
| **📋 订单公告栏** | 抽取1张订单卡，或选择1张已公开的订单 |
| **🏛️ 女巫公会** | 提交手牌中匹配的原料卡，获得声誉⭐和金币🪙 |
| **🌾 农田区** | 免费获得1张胡萝卜种子（随机抽取） |
| **⚡ 事件格** | 抽取1张事件卡，立即执行效果 |
| **💰 市场区** | 出售手牌中多余的原料卡，每张得1金币 |
| **🏠 玩家家** | 安全休息，无特殊效果（可整理手牌） |
| **🎪 节日广场** | 所有玩家各获得1张工具卡（随机） |
| **🌳 魔法森林** | 花费2金币获得1张稀有种子（龙焰椒或精灵花） |

#### 第三步：花园行动（核心阶段）

在自己的花园板上进行以下操作（顺序随意，每项限1次）：

1. **种植**：将手牌中的种子卡种入1个空格（放置1颗对应原料标记的棋子/纸片到该格）
2. **收获**：如花园中有成熟植物，收获并获得对应原料卡
3. **使用工具**：使用手牌中的工具卡（魔法水壶/加速肥料/守护精灵）

**成熟判定：** 种植后的植物，每经过自己回合的"花园行动"阶段，成熟计时-1。计时到0时即可收获。

#### 第四步：完成订单（可选）

如果手牌中有能匹配的订单，可立即提交：
- 移动到"女巫公会"（如不在该格）
- 提交所需原料卡
- 获得声誉⭐（放在自己面前）和金币🪙

#### 第五步：回合结束

将所有用过的工具卡和事件卡放入对应弃牌堆。将回合交给下一位玩家。

### 胜利条件

**主要胜利：** 第一位累积 **20点声誉⭐** 的玩家立即获胜！

**次要胜利（时间上限）：** 如所有玩家均完成12回合后无人达到20声誉，则声誉最高的玩家获胜。

## 💡 新手策略提示

1. **平衡种植节奏**：胡萝卜成熟快（2回合）但奖励低；精灵花成熟慢（5回合）但奖励高。建议混搭种植，保证每回合都有收获。

2. **优先完成低等级订单**：青铜订单（🥉）容易完成，可快速积累初期声誉。不要只看高奖励订单，积少成多！

3. **善用工具卡**：魔法水壶💧能让植物立即成熟，是追赶局势的关键。在植物即将成熟时使用是浪费，在还有3-4回合时才用最划算。

4. **控制移动路线**：地图只有12格，规划好路线可以连续访问商店→订单栏→女巫公会，一气呵成完成"购买→接单→提交"的流程。

5. **事件格是双刃剑**：停留在事件格可能获得丰厚奖励，也可能失去植物。建议手上有守护精灵🧚时再主动踩事件格。

6. **市场区是救命稻草**：手牌原料太多又暂时完成不了订单？去市场区💰卖掉多余原料，换金币买更好的种子！

7. **关注对手声誉**：当某位玩家接近20声誉时，可以尝试用事件卡（如"害虫来袭"）干扰对方花园，但注意事件卡可能影响所有人！

## 🔄 变体规则

### 快速模式（适合新手/短时间游戏）

- 胜利条件改为 **12声誉⭐**
- 每位玩家初始金币改为 **5金币**
- 胡萝卜成熟时间改为 **1回合**

### 团队模式（2v2）

- 两名玩家组成一队，共享花园板和手牌
- 共享声誉池，先到30声誉的队伍获胜
- 队友之间可以自由传递手牌（每次传递限2张）

### 困难模式（硬核玩家）

- 每位玩家初始只有 **1张胡萝卜种子**
- 事件卡效果加倍（如"害虫来袭"摧毁2格植物）
- 胜利条件提高到 **30声誉⭐**
- 新增"冬季事件"：每5回合所有未收获植物成熟时间+1

### 单人模式（solitaire）

- 与"女巫考官NPC"对战
- NPC每回合自动：种植1颗随机种子，移动1-3格
- 玩家需要在 **15回合内** 达到20声誉，否则NPC先达到，玩家失败

## ❓ 常见问题

**Q：如果我的花园板所有格子都种满了怎么办？**
> A：你必须收获至少1格植物，腾出空格后才能种植新种子。这也是为什么混搭不同成熟时间的植物很重要！

**Q：可以在同一回合种植和收获同一种植物吗？**
> A：可以！如果你用魔法水壶💧使刚种的植物立即成熟，就可以在同回合收获。

**Q：订单卡可以保留多回合吗？**
> A：可以！订单卡没有期限，但建议尽早完成，因为对手可能比你先达到20声誉。

**Q：如果骰子点数让我移动超出了地图格子怎么办？**
> A：移动是循环的，超出后从地图另一端继续计算。例如：在最后一个格子掷出3点，则移动到第3格。

**Q：事件卡说"所有玩家"，包括触发者自己吗？**
> A：是的！"所有玩家"效果对包括触发者在内的所有人都生效，使用时要三思！

**Q：如果弃牌堆用完了怎么办？**
> A：将对应弃牌堆的卡牌洗混后，重新作为抽牌堆使用。

**Q：可以查看弃牌堆的卡牌吗？**
> A：可以，所有弃牌堆都是公开信息，任何玩家可以随时查看。

**Q：游戏最多可以几人玩？**
> A：官方规则支持2-4人。4人以上会导致等待时间过长，且花园板不够用。如要5-6人玩，需要复印额外的花园板。
