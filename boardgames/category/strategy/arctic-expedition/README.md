# 极地探险 (Arctic Expedition)

## 1. 游戏概述

欢迎来到冰天雪地的北极！在《极地探险》中，你将扮演一名勇敢的极地探险家，带领探险队深入北极未知区域。你要探索六边形网格地图，收集稀缺资源，建立研究站，抵御极端天气，完成科学考察任务。

游戏融合了**区域探索**、**资源管理**和**引擎构建**机制，玩家需要在恶劣的极地环境中做出艰难选择：是冒险深入未知区域获取稀有资源，还是稳妥地在已探索区域建立补给站？

**适合人群**：喜欢策略规划、资源管理、有一定游戏经验的玩家。适合2-4人游玩，喜欢《丝绸之路》《星际贸易站》风格的玩家会特别喜欢这款游戏。

## 6. 游戏规则

### 准备阶段

1. **放置地图**：将游戏地图放在桌面中央
2. **选择角色**：每位玩家选择一个颜色，领取对应棋子
3. **放置起始位置**：所有玩家棋子放在地图中心的「基地🏠」
4. **分发初始资源**：
   - 2人局：每人8张资源卡（自选类型）
   - 3人局：每人6张资源卡
   - 4人局：每人5张资源卡
5. **洗混卡组**：将天气卡、任务卡、探险卡分别洗混，面朝下放置
6. **翻开天气**：翻开第一张天气卡，放置于地图旁
7. **分发任务**：每位玩家抽取2张任务卡，保留1张作为目标
8. **设置计分板**：将计分标记放在0分位置

### 游戏流程（每回合）

每位玩家轮流执行回合，每回合可选择以下行动之一：

#### 行动1：探索移动
- 消耗🍖食物移动（每格1🍖，山脉格需额外1🎒）
- 进入新区域后，翻开该格效果：
  - 苔原🌲：获得1🍖
  - 山脉🏔️：获得1🎒
  - 冰川❄️：无资源，但可抽取1张探险卡
- 若当前天气为❄️🌨️暴风雪，需额外消耗1⛽燃料

#### 行动2：建立站点
- 消耗2🎒装备+1⛽燃料
- 在当前位置建立研究站/气象站
- 建立站点后，该格成为你的安全区（可免费恢复1🍖/回合）

#### 行动3：科研任务
- 消耗2🔬科研点
- 翻开1张任务卡，若满足条件则完成，获得奖励分数
- 每名玩家同时最多持有3个进行中的任务

#### 行动4：休整补给
- 在基地🏠或自己的站点停留
- 恢复2🍖+1⛽
- 抽取1张天气卡（替换当前天气）

#### 行动5：交易交换
- 与另一位玩家交易资源卡（1换1）
- 每次交易后，双方各抽取1张探险卡（可能有正面效果）

### 天气阶段（每回合结束后）

1. 掷1次6面骰子
2. 若点数≤当前天气的「变化值」，则翻开新天气卡
3. 天气变化值：晴朗(1)、下雪(2)、暴风雪(3)、极昼(2)、极夜(4)、冰雾(3)

### 胜利条件

游戏进行**12回合**后结束（每人行动12次），或当某玩家完成**5个任务**时提前结束。

**计分方式**：
- 完成任务：按任务卡标注分数
- 科研点数：每3🔬 = 1分
- 建立站点：每个站点2分
- 探索区域：每探索5个新区域得1分
- 剩余资源：每5张资源卡（任意组合）= 1分

**最高分者获胜！**

### 示例回合：玩家A的第一次行动

**初始状态**：
- 玩家A（红色棋子）位于基地🏠
- 手牌：🍖×3、⛽×2、🎒×1、🔬×0
- 当前天气：☀️晴朗（变化值1）
- 持有任务：「地理测绘」（需探索8个区域）

**回合行动**：选择「探索移动」
1. 消耗1🍖，移动到相邻的苔原格🌲
2. 获得1🍖（苔原格奖励）
3. 手牌变为：🍖×3、⛽×2、🎒×1
4. 探索计数+1（共1/8）

**天气阶段**：
- 掷骰子得4，大于晴朗变化值1，天气不变

**回合结束**：玩家A的第一次行动完成，积累了1个探索区域。

### 示例回合：玩家B建立站点

**初始状态**：
- 玩家B（蓝色棋子）位于苔原格🌲
- 手牌：🍖×2、⛽×3、🎒×3、🔬×1
- 持有任务：「极地科考」（需建立3个站点）

**回合行动**：选择「建立站点」
1. 消耗2🎒+1⛽
2. 在当前苔原格建立气象站🏗️
3. 手牌变为：🍖×2、⛽×2、🎒×1、🔬×1
4. 站点计数+1（共1/3）

## 8. 新手策略提示

1. **优先探索苔原和山脉**：这两个地形提供稳定资源，是生存的基础
2. **不要忽视天气**：暴风雪来临时，最好在基地停留，避免损失资源
3. **任务选择要明智**：选择距离你当前位置近的任务，减少移动成本
4. **平衡资源类型**：不要只囤积食物，燃料和装备在后期同样重要
5. **建立站点要趁早**：站点不仅得分，还能提供持续的资源恢复
6. **关注对手动向**：如果对手即将完成5个任务，要加快节奏
7. **探险卡有风险**： Glacier格的探险卡可能带来惊喜或灾难，量力而行

## 9. 变体规则

### 变体1：极地生存模式（更困难）
- 每回合必须消耗1🍖维持生存，否则失去1点生命（生命=3，归零出局）
- 增加了生存压力，适合经验丰富的玩家

### 变体2：团队探险模式（合作）
- 2-4名玩家组成一支探险队，共同对抗天气和任务
- 资源共享，共同计分
- 胜利条件：全体完成10个任务，且至少2人存活

### 变体3：快速游戏模式（30分钟）
- 游戏回合数减少为8回合
- 初始资源增加50%
- 任务完成分数×1.5倍

### 变体4：极夜马拉松（超级挑战）
- 天气卡中增加「🌑极夜」概率
- 极夜期间无法移动，必须消耗2⛽才能行动
- 适合喜欢高难度挑战的玩家

## 10. 常见问题

**Q：可以同时持有多个任务吗？**
A：可以，每位玩家最多同时持有3个进行中的任务。

**Q：在冰川格❄️行动有什么好处？**
A：冰川格虽然没有资源，但有一个神秘遗迹。到达遗迹格可抽取1张探险卡，有机会获得大量科研点。

**Q：如果食物🍖耗尽怎么办？**
A：如果无法支付移动消耗，你的探险队将进入「饥饿状态」，回合结束时损失1点生命（用分数抵消，1分=1生命）。建议尽快返回基地补给。

**Q：天气卡何时更换？**
A：每回合结束掷骰子，点数≤当前天气变化值时更换。也可以在基地选择「休整补给」行动时主动更换。

**Q：科研点🔬有什么用？**
A：科研点可在「科研任务」行动中完成科研任务，获得大量分数。游戏结束时，每3个科研点也可兑换1分。

**Q：可以攻击其他玩家吗？**
A：本游戏是非对抗性的，玩家之间不可直接攻击。但可以通过「交易交换」互动，有时探险卡的效果也会影响其他玩家。

**Q：游戏时间太长怎么办？**
A：建议使用「快速游戏模式」变体，或将回合数减少为8回合。
