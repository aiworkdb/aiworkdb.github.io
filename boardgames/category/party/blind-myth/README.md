# 盲盒神话 (Blind Box Mythology)

## 🎮 游戏概述

《盲盒神话》是一款充满惊喜与交易的派对桌游。玩家扮演盲盒收藏家，通过购买盲盒、交换重复角色、完成神话套装，最终成为第一个集齐整套神话人物的"盲盒之王"！

游戏融合了当下最火的盲盒文化与中国四大神话体系（西游记、封神榜、山海经、白蛇传），既有运气成分，也需要交易策略，适合朋友聚会、家庭娱乐。

## 🎲 游戏规则

### 准备阶段

1. **洗混卡牌**：将所有人物卡洗混，放入不透明布袋（盲盒袋）
2. **分发金币**：每位玩家获得起始金币（3人局15枚，4-5人局12枚，6人局10枚）
3. **洗混行动卡**：将行动卡洗混，放在桌面中央
4. **决定顺序**：通过猜拳或掷骰子决定起始玩家

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰定价
- 掷盲盒骰（或普通骰子），点数 = 盲盒价格（金币）
- 如果金币不足，可以：
  - 与其他玩家**交易**手牌（免费）
  - 使用**稀有卡**的特殊能力
  - 宣告**破产**，本回合跳过购买

#### 2. 购买盲盒
- 支付金币给"银行"（桌面中央）
- 从盲盒袋中抽取1张人物卡
- **展示给所有玩家看**（抽到重复卡很正常！）

#### 3. 执行卡牌效果
根据抽到的卡牌类型，执行相应操作：

| 卡牌类型 | 操作 |
| :--- | :--- |
| **新角色卡** | 加入自己的收藏，检查是否集齐一套 |
| **重复角色卡** | 可以与其他玩家交易，或放在桌上公开交换 |
| **行动卡** | 立即执行行动卡效果 |
| **稀有卡** | 立即获得特殊能力，可在后续回合使用 |

#### 4. 自由交易时间（可选）
- 玩家可以与其他玩家**自由交易**手牌
- 交易可以是：卡牌换卡牌、卡牌换金币、或免费赠送
- 每次交易需要双方同意
- **交易是游戏的核心策略！**

#### 5. 检查胜利条件
- 如果集齐任意一套神话人物（8张），立即宣告胜利！
- 否则轮到下一玩家

### 胜利条件

**主要胜利条件**：第一个集齐任意一套完整神话人物卡（8张）的玩家获胜！

**次要胜利条件**（如果盲盒袋抽完）：
- 拥有最多**不同套装**的玩家获胜
- 如果平局，拥有最多金币的玩家获胜

## 📖 完整游戏流程示例

### 示例一：新手玩家的第一个回合

**玩家A（小明）的回合：**

1. **掷骰定价**：掷出4点，盲盒价格为4金币
2. **支付金币**：小明有12金币，支付4金币给银行，剩余8金币
3. **抽取盲盒**：从盲盒袋抽出一张卡，展示给所有人：**🐵 孙悟空（西游记）**
4. **获得卡牌**：小明将孙悟空加入自己的收藏
5. **检查套装**：小明目前只有1张西游记卡，还差7张，未获胜
6. **自由交易**：小明问其他玩家："谁有西游记的重复卡？我可以用封神榜卡换！"
   - 玩家C（小红）说："我有重复的🐷猪八戒，我用它换你的⚡哪吒（封神榜）！"
   - 小明同意交易，获得🐷猪八戒
7. **回合结束**：小明有2张卡（🐵孙悟空、🐷猪八戒），轮到玩家B

### 示例二：使用行动卡改变局势

**玩家B（小王）的回合：**

1. **掷骰定价**：掷出2点，盲盒价格2金币（好便宜！）

5. **自由交易**：小王现在有了🐵孙悟空，主动向小明提出："我用🐲麒麟（山海经）换你的🐷猪八戒，咱们各取所需！"
6. **检查胜利条件**：小王有1张西游记卡，未获胜
7. **回合结束**：轮到玩家C

## 💡 新手策略提示

1. **交易是关键**：不要害羞，主动与其他玩家交易！有时候免费赠送一张重复卡，能换来一张你急需的卡。

2. **观察对手**：留意其他玩家在收集哪套神话，如果你们目标相同，要加快速度！

3. **合理使用金币**：不要每次都买盲盒，存一些金币用于抢夺关键卡或应对危机。

4. **稀有卡要省着用**：稀有卡的特殊能力很强，但数量有限，关键时刻再使用。

5. **行动卡可以改变局势**：抽到"强制交易"或"金币抢夺"时，选择对你威胁最大的玩家作为目标。

6. **不要只盯着一个套装**：如果某个套装的角色都被别人抽走了，及时转向其他套装！

7. **利用重复卡**：重复的卡不是废物，它们是交易的重要筹码！

## 🔄 变体规则

### 双人模式（1v1）

- 每人起始金币增加50%
- 盲盒袋中只放入2套神话人物卡（16张）
- 先集齐一套（8张）的玩家获胜
- 增加"战场"规则：玩家可以消耗金币"封锁"对方的一套卡

### 团队模式（2v2 或 3v3）

- 队友之间可以自由交易和共享手牌
- 团队集齐两套神话即可获胜
- 增加"团队行动卡"，可以让队友获得额外回合

### 竞速模式

- 去掉行动卡，只保留人物卡
- 盲盒价格固定为3金币
- 加快游戏节奏，适合短时间娱乐

### 困难模式

- 增加"隐藏角色"（每套1张，共4张），必须集齐9张才能获胜
- 隐藏角色只能通过"幸运再抽"行动卡获得
- 增加"假盲盒"：有10%概率抽到空白卡（什么都没有）

### 赌徒模式

- 玩家可以将金币"押注"在盲盒上
- 如果抽到新角色，获得双倍金币
- 如果抽到重复角色，失去押注的金币
- 高风险高回报，适合刺激玩家

## ❓ 常见问题

**Q：如果盲盒袋里的卡抽完了怎么办？**
A：将弃牌堆（如果有）洗混后重新放入盲盒袋。如果确实没有卡了，游戏提前结束，按照"次要胜利条件"决定赢家。

**Q：可以自由交易吗？有没有限制？**
A：可以自由交易，没有次数限制。但交易必须双方自愿，不能强迫。

**Q：行动卡的效果可以拒绝吗？**
A：取决于行动卡类型。"强制交易"不能拒绝，"金币抢夺"不能拒绝，但"幸运再抽"对自己有利可以选择使用。

**Q：稀有卡可以交易给别人吗？**
A：可以！稀有卡也是手牌的一部分，可以与其他玩家交易。

**Q：如果金币用完了怎么办？**
A：可以通过与其他玩家交易（免费赠送卡牌）来获取需要的卡，或者等待"金币丰收"行动卡。

**Q：可以同时收集多套神话吗？**
A：可以！实际上，有时候多套一起收集更容易获胜，因为抽到重复卡的概率降低了。

**Q：游戏适合几岁的小孩玩？**
A：建议8岁以上，因为需要一定的策略思考和交易谈判能力。但如果小孩能在大人帮助下进行交易，6岁也可以玩。

**Q：一局游戏大概多长时间？**
A：3-4人局约30分钟，5-6人局约50分钟。如果玩家交易频繁，时间可能更长。
