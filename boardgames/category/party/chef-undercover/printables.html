⚠️ 重要打印提示

  • 使用 A4纸 打印，建议设置最小页边距
  • 在打印对话框中，务必勾选"背景图形"（Print Backgrounds）选项，否则彩色背景不会打印
  • 卡牌和棋子建议使用卡纸或照片纸打印，增加耐用性
  • 打印后可用护卡膜塑封，延长使用寿命
  • 建议先打印1页预览，确认颜色正确后再全量打印
  • 点击浏览器菜单 → 打印（Ctrl+P），即可开始打印本页

📌 十二、速查参考卡（每人1张）

每位玩家保留1张，方便随时查阅回合流程和胜利条件。

⚡ 每回合流程

  • ① 烹饪阶段：摸2张 → 打出1-2张
  • ② 事件阶段：翻1张事件卡，全员执行
  • ③ 审问阶段：投票选2人审问，超半数则驱逐
  • ④ 结算阶段：检查菜谱完成 / 破坏值是否达10

🏆 胜利条件

  • 🥘 厨师胜：完成全部3道菜 + 驱逐至少1名卧底
  • 💀 卧底胜：破坏值达到10
  • 💀 卧底胜：厨师冤枉2名以上好厨师
  • 🤝 平局：完成3道菜但卧底未被发现

🃏 卡牌颜色对照

  • 🟢 绿色 = 食材卡（36张）
  • 🔵 蓝色 = 厨具卡（20张）
  • 🔴 红色 = 破坏卡（12张，卧底用）
  • 🟡 黄色 = 线索卡（16张）
  • ⚪ 白色 = 澄清卡（8张）
  • 🌟 金色 = 特技卡（10张，限用1次）
  • 🟣 紫色 = 事件卡（8张）
