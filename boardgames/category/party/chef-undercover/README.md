## 🎮 游戏概述

《卧底大厨》是一款以**美食餐厅**为背景的派对推理桌游。玩家们被分配为**诚实厨师**或隐藏身份的**卧底厨师**，通过一轮轮的"烹饪任务"与"审问环节"，合力找出混在队伍中的破坏者。

- **诚实厨师**：完成3道菜肴，把餐厅成功开业，同时揪出所有卧底。
- **卧底厨师**：暗中破坏菜肴、散布谣言，在被发现之前积累足够的"破坏值"阻止开业。

游戏融合了**角色扮演、社交推理、卡牌出牌**三种机制，每局约30分钟，笑声不断、节奏紧凑，是家庭聚会和朋友局的首选。

## 📜 游戏规则

### 准备阶段

1. **布置场地**：将3块菜谱板放在桌子中央，进度标记全部归零。
2. **放置破坏进度条**：从0格开始，放在所有人可见的位置。
3. **分发身份牌**：
   - 4-5人：1张**卧底牌** + 其余**厨师牌**
   - 6-7人：2张**卧底牌** + 其余**厨师牌**
   - 8人：2-3张**卧底牌**（由主持人决定）
   - 身份牌面朝下秘密查看，不得告知他人。
4. **分发角色牌立卡**：每人随机获得一个厨师职位（主厨/副厨/甜点师/采购员/刷碗工等），职位公开展示但与身份无关，仅增加角色扮演氛围。
5. **洗牌发牌**：将所有卡牌混合洗匀，每人发**5张手牌**，其余作为牌库放中间。

### 游戏流程（每回合）

每轮游戏按照以下4个阶段顺序进行：

#### 阶段一：烹饪阶段（每人1分钟）

按顺序轮流进行，每名玩家：

1. **抽2张牌**（从牌库摸）。
2. **打出1-2张卡**到菜谱（食材卡）或使用特殊效果（厨具卡/线索卡等）。
   - 食材卡须打到与之对应的菜谱板上。
   - 特殊卡可当场生效，向所有玩家宣布效果。
3. **手牌上限为6张**，超出需弃置到弃牌堆。

> 💡 卧底的选择：正常打出食材卡（伪装好厨师），或偷偷打出破坏卡宣布"厨房事故"。

#### 阶段二：事件阶段

所有人出完牌后，翻开**1张随机事件卡**（若有），全员执行事件效果（见下方事件表）。

#### 阶段三：审问阶段（每轮选2人）

所有人投票选出**最可疑的2名玩家**进行审问：

1. 被审问者限时 **60秒** 为自己辩护，其他人可提问。
2. 审问结束后，所有人投票是否驱逐某人：
   - **超过半数**同意 → 驱逐该玩家，翻开其身份牌
     - 若是卧底 → 卧底出局，厨师团队得分+1
     - 若是厨师 → 冤枉了好人，破坏值+2，该厨师含冤出局
   - **未达半数** → 当轮无人被驱逐

> 💡 每轮**只能驱逐1人**（即使2人都被怀疑，也仅驱逐票数更多的那位）。

#### 阶段四：结算阶段

- 检查各菜谱进度是否达标（沙拉需4张、热汤需5张、甜点需4张）
- 已完成的菜谱不再接受破坏
- 破坏值达到10 → 游戏立刻结束（卧底胜利）

### 胜利条件

| 获胜方 | 胜利条件 |
| :--- | :--- |
| 🏆 **诚实厨师获胜** | 完成全部3道菜谱，**且**至少驱逐出1名卧底 |
| 💀 **卧底获胜** | 破坏值达到10，**或**厨师团队冤枉了2名以上好厨师 |
| 🤝 **平局** | 完成3道菜但卧底全程未被发现（厨师惨胜，卧底保全自身） |

## 🎬 完整游戏流程示例

**示例设定**：6人游戏，玩家A-F，其中D是卧底。

### 第1轮

**烹饪阶段：**
- 玩家A（主厨）：抽2张，打出🥬新鲜蔬菜到菜谱①（沙拉进度1/4）
- 玩家B（副厨）：打出🌿香料到菜谱②，热汤进度+2（2/5）
- 玩家C（甜点师）：打出🧁烘焙原料到菜谱③（甜点进度1/4）
- **玩家D（卧底）**：打出🧂"过量盐"破坏卡，宣布"厨房事故"！菜谱①进度-1（回到0/4），破坏值变为1。
  - 玩家E持有🍳平底锅，宣布"取消！"，破坏无效，进度恢复，破坏值不变 ✅
- 玩家E：打出🥕根茎蔬菜到菜谱①（沙拉进度1/4），顺带再摸1张
- 玩家F：打出🔪主厨刀，再打🍖优质肉类到菜谱②（热汤进度+1共4/5）

**审问阶段：**
玩家A发言："D打了破坏卡，虽然被取消了，感觉很可疑！"
玩家D为自己辩解："我以为手牌打错了，那是我摸到的奇怪牌！"
投票：D 2票，B 1票 → 未超半数（3票），无人被驱逐。

### 第2轮

**烹饪阶段：**
- 玩家D（卧底）：改变策略，老实打出🥬蔬菜到菜谱①，继续伪装
- 玩家A：打出🌡️温度计，查看玩家D手牌——发现D还有1张💀破坏卡！
  - A宣布："D手里有破坏卡！"
- 玩家D：立刻打出🛡️澄清卡（出勤记录），声称"这是我新摸的，来不及弃"

**审问阶段：**
激烈讨论，投票：D 4票 → **超半数！驱逐D！**
翻开D的身份牌：**卧底！** 厨师团队得分+1，破坏值-1作为奖励（特技牌效果）。

### 第3轮（卧底出局后）

剩余玩家全力推进菜谱，第3轮完成热汤（5/5✅）和甜点（4/4✅），最终在第4轮完成沙拉（4/4✅）。

**🏆 诚实厨师胜利！餐厅成功开业！**

## 💡 新手策略提示

1. **🔍 关注弃牌堆**：每人打出的牌类型都有迹可循，破坏卡打出后即入弃牌，善用线索卡回溯。

2. **🎯 优先完成热汤**：热汤需要5张食材（最多），且卧底喜欢集中破坏耗时最长的菜谱，多人协作该菜谱效果最佳。

3. **🛡️ 手牌不要超限**：手牌超过6张必须弃置，不要让卧底轻易猜到你的牌型。

4. **📢 主动报牌**：诚实厨师可以主动公告"我要出食材卡到菜谱①"，透明操作更难被怀疑。

5. **⏰ 利用定时器牌**：如果你预感下轮会被怀疑，提前用⏲️厨房定时器牌购买一次免审机会。

6. **🤫 卧底稳住节奏**：新手卧底常常打出太多破坏卡导致暴露。建议前2轮老实干活积累信任，再在第3轮关键时刻出手。

7. **🌟 特技牌不要乱用**：👨‍🍳主厨裁决错误代价极大（破坏值+3），确认度低于70%时不要轻易使用。

8. **🤝 联合嫌疑**：若你是卧底且被怀疑，可以主动指认另一名玩家转移注意力，但小心被反制。

## 🔄 变体规则

### 变体一：黑手党模式（高难度）

卧底人数翻倍（6人3卧底，8人4卧底），诚实厨师的菜谱食材需求+1，但每局开始可打出2张食材卡。适合熟悉基础规则的老玩家。

### 变体二：静默厨房

审问阶段所有玩家**不能说话**，只能通过打出卡牌和手势表达意见。增加观察和解读肢体语言的乐趣，适合习惯社交推理的玩家。

### 变体三：连锁厨房（5人快速版）

只需完成2道菜谱（沙拉+热汤），卧底只有1人。每个烹饪阶段每人只能出1张牌，游戏约15分钟，适合快速休闲局。

### 变体四：美食大师赛（团队模式）

将玩家分为2队（厨师队 vs 卧底队），卧底身份提前公开。改为全程策略博弈——厨师队设障，卧底队公开破坏，看谁能赢得最终对决。

## ❓ 常见问题

**Q1：如果我是诚实厨师但摸到了破坏卡，怎么办？**
> A：在下一个回合开始阶段，将破坏卡明示面朝上弃到弃牌堆。这是证明自己清白的最好方式。注意：若你悄悄弃置而不明示，其他玩家可能会怀疑你藏着破坏卡。

**Q2：破坏卡打出但被平底锅取消后，破坏值会增加吗？**
> A：不会。平底锅"取消"是完全无效化，破坏卡视为没有打出，破坏值维持原状，卡牌进入弃牌堆。

**Q3：卧底被驱逐出局后，游戏还可以继续吗？**
> A：可以。驱逐卧底后，剩余玩家继续推进菜谱直到完成或破坏值达10。若还有第二个卧底，其会继续破坏。

**Q4：澄清卡可以在被投票前打出，提前免疫本轮吗？**
> A：不可以。澄清卡只能在**审问开始后、投票完成前**打出，不能提前储备免疫。

**Q5：8人场有3个卧底时，卧底之间可以互相交流吗？**
> A：不可以！卧底和厨师一样，不知道其他卧底是谁（只知道自己是卧底）。这是游戏核心张力之一——两个卧底可能会互相指认对方！

**Q6：菜谱进度被破坏降到0以下（负数）怎么处理？**
> A：菜谱进度最低为0，不会变成负数。超额破坏效果直接归零，不累积。

**Q7：平局情况下如何判断谁更"赢"？**
> A：平局时，比较各自得分：厨师方得分 = 完成菜谱数；卧底方得分 = 破坏值最终数值÷2。得分高者赢得道德胜利，但可以直接再来一局决胜负！
