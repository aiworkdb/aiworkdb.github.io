# 疯狂厨房 Crazy Kitchen

## 1. 游戏概述

《疯狂厨房》是一款3-8人的派对桌游，玩家将扮演一支忙碌的厨师团队，在时间压力下快速完成各种订单。游戏融合了实时协作、手牌速配和角色扮演机制，营造出既紧张刺激又欢乐搞笑的厨房氛围。

在游戏中，玩家需要根据订单卡快速组合正确的食材卡，同时应对突如其来的厨房事故。每个人扮演不同的角色，拥有独特的技能，需要通力合作才能在时间耗尽前完成足够多的订单。

这是一款适合聚会、家庭聚餐、生日派对的绝佳选择，老少皆宜，笑料百出！

### 四个工位区域详解

**🔥 灶台区 (COOKING STATION)**
- 位置：厨房地图左上方
- 功能：完成热菜类订单的主战场
- 特殊：可使用"大火"技能加速完成

**🥗 备菜台 (PREP STATION)**
- 位置：厨房地图中央上方
- 功能：处理冷食和简单菜品
- 特殊：可进行食材预处理
- 容纳：1-2名玩家同时工作

**🧽 洗碗区 (DISHWASH STATION)**
- 位置：厨房地图右上方

- 特殊：洗碗技能在此区域效果翻倍
- 容纳：1名玩家专职工作

**🚪 出餐口 (SERVING WINDOW)**
- 位置：厨房地图中央下方
- 功能：将完成的菜品交给顾客
- 特殊：必须经过此区域才能计分
- 容纳：1名玩家专职递送

## 5. 道具详解

### 5.1 食材卡 (Ingredient Cards)

每张食材卡包含以下信息：
- **正面**：食材图案 + 中文名称 + Emoji标识
- **背面**：统一的厨房花纹设计

食材卡使用规则：
- 所有食材卡洗匀后放在"食材仓库"区域
- 玩家每回合可从仓库拿取最多2张食材卡
- 持有超过8张食材卡时必须丢弃至手牌上限
- 食材卡可以交易（需在同一工位）

### 5.2 订单卡 (Order Cards)

每张订单卡包含：
- **菜品名称**：中文 + Emoji
- **所需食材列表**：3-4种食材类型
- **完成时限**：普通/紧急标记
- **分值**：1-3分

订单卡使用规则：
- 订单卡展示在"厨房指挥台"区域
- 同时最多展示4张活跃订单
- 完成订单时将对应食材卡放置在订单卡旁
- 验证正确后获得分值并移除订单

## 6. 游戏规则

### 6.1 游戏准备

**步骤1：分发角色**
1. 将所有角色卡洗匀，面朝下放置
2. 从起始玩家开始，顺时针每人抽取1张
3. 查看后保密，角色能力在游戏中可公开声明

**步骤2：设置厨房**
1. 将厨房地图平铺桌面中央
2. 在指挥台区域放置3张订单卡（面朝上）
3. 将所有食材卡洗匀，放在"食材仓库"区域
4. 将事故卡洗匀，放在一旁备用
5. 每个玩家获得3张起始食材卡

**步骤3：放置标记物**
1. 将起始玩家标记放在灶台区
2. 玩家棋子放在各自的起始工位
3. 压力指示物从0开始
4. 计时沙漏放在桌面公共区域

### 6.2 游戏流程

**整体结构：**
游戏分为**若干轮**，每轮包含以下阶段：

```
┌──────────────────────────────────────────────┐
│              每轮游戏流程                        │
├──────────────────────────────────────────────┤
│ 1️⃣ 订单阶段（30秒）                           │
│    - 翻开新订单卡，补充至3张                     │
│    - 宣布本轮特殊事件（如有）                    │
├──────────────────────────────────────────────┤
│ 2️⃣ 行动阶段（每个玩家轮流，共3-5分钟）          │
│    - 翻转沙漏开始计时                           │
│    - 玩家按顺序执行行动                         │
│    - 可执行：拿食材、做菜、传菜、应对事故         │
├──────────────────────────────────────────────┤
│ 3️⃣ 结算阶段                                   │
│    - 检查已完成的订单                          │
│    - 验证食材组合是否正确                        │
│    - 计分并移除完成的订单                       │
│    - 厨房压力+1                               │
├──────────────────────────────────────────────┤
│ 4️⃣ 事故检查                                   │
│    - 压力达到3时，翻开事故卡                    │
│    - 应用事故效果                              │
└──────────────────────────────────────────────┘
```

**计时机制详解：**

| 计时场景 | 工具 | 时长 | 超时惩罚 |
|----------|------|------|----------|
| 每回合行动 | 沙漏 | 30秒/玩家 | 跳过该玩家本回合 |
| 紧急订单 | 沙漏 | 15秒 | 订单自动失败 |
| 事故响应 | 沙漏 | 60秒 | 事故效果加深 |
| 厨房压力 | 整体 | 1分钟/压力点 | 压力继续累积 |

**行动阶段玩家可选动作：**

1. **拿取食材** 🥬
   - 从仓库拿取1-2张食材卡
   - 必须放在自己面前（手牌区）

2. **完成订单** 🍳
   - 将正确组合的食材卡放置在对应订单卡旁
   - 高声喊出"上菜！"并举手示意
   - 其他玩家验证正确性

3. **传递食材** 🤝
   - 将自己的食材卡传递给相邻玩家
   - 传递需获得对方同意
   - 跑堂角色可无需同意直接传递

4. **使用技能** ⚡
   - 消耗金币指示物激活角色技能
   - 大部分技能免费，特殊技能需1-2枚金币

5. **应对事故** 🚒
   - 前往事故发生区域执行清理
   - 完成清理动作后事故解除

### 6.3 胜利条件

**合作胜利：**
- 在游戏时间内（10轮或15分钟），完成**目标分值**的订单
- 目标分值 = 基础分（20分）+（人数×3）
- 例如：4人游戏需要20+12=32分

**个人胜利：**
- 达到合作胜利条件后，计算个人贡献分
- 贡献分最高者获得"最佳厨师"称号
- 个人分 = 完成订单得分 + 技能使用奖励

**失败条件：**
- 厨房压力达到**10级**
- 或者时间结束时未达到目标分值
- 或者发生3次重大事故（烧毁订单）

## 7. 完整游戏流程示例

**场景：5人游戏，第3轮进行中**

### 背景设定
- 当前轮次：第3轮（共10轮）
- 已完成分数：18分
- 厨房压力：3级
- 活跃订单：3张（意面2分、沙拉1分、蛋糕2分）
- 手牌情况：
  - 主厨：牛肉、洋葱、面包
  - 帮厨A：西红柿、奶酪、米饭
  - 洗碗工：空
  - 跑堂：面条、黄瓜
  - 实习生：鸡肉、土豆、冰淇淋

### 订单阶段
**计时开始：30秒沙漏倒计时**

1. 主持人翻开1张新订单卡："披萨拼盘！需要奶酪+番茄+面条，3分！"
2. 补充后当前活跃订单为4张（需移除1张）
3. 主厨宣布："压力不大，没有特殊事件！"

### 行动阶段
**沙漏翻转，第1个玩家（主厨）开始计时**

**主厨（30秒）：**
- 思考：意面订单需要奶酪+番茄+面条，我有洋葱
- 行动：将洋葱传递给帮厨A，换取奶酪
- 结果：成功交换，现在持有奶酪

**帮厨A（30秒）：**
- 思考：沙拉订单需要黄瓜+西红柿，我有西红柿
- 行动：从仓库拿取黄瓜
- 结果：沙拉可以完成！

**洗碗工（30秒）：**
- 思考：目前没有事故，闲置
- 行动：帮跑堂去仓库拿面条

**跑堂（30秒）：**
- 思考：传送带坏了，需要快速传递
- 行动：接收帮厨A的西红柿+自己的黄瓜，喊"上菜！"
- 验证：沙拉订单完成！+1分
- 当前总分：19分

**实习生（30秒）：**
- 思考：意面还差番茄和面条
- 行动：从洗碗工处获得面条
- 结果：还差番茄...

### 结算阶段
1. 沙漏停止，检查所有完成声明
2. 验证沙拉订单：✅ 黄瓜+西红柿 = 1分
3. 移除完成的订单卡
4. 厨房压力+1 → 现在为4级
5. 主厨喊"第4轮开始！"

### 事故检查
- 当前压力4级，未达到触发阈值（每3级触发一次）
- 本轮无事故发生

### 游戏继续
进入第4轮，重复上述流程...

## 8. 新手策略提示

### 8.1 基础策略

**策略1：专注收集 + 快速完成**
- 选择1-2种食材集中收集
- 快速完成简单的1分订单建立优势
- 优点：稳定得分，不依赖他人
- 缺点：无法完成高难度订单

**策略2：信息共享**
- 每个人报告自己持有的食材
- 协调谁去完成哪张订单
- 避免重复收集造成浪费
- 优点：效率最大化
- 缺点：沟通耗时

**策略3：角色技能配合**
- 帮厨优先收集主厨需要的食材
- 跑堂负责在玩家间传递
- 洗碗工专职处理事故
- 优点：发挥团队最大战力
- 缺点：灵活性降低

### 8.2 角色专属技巧

| 角色 | 核心技巧 |
|------|----------|
| 👨‍🍳 主厨 | 先观察全局，再指挥协调；不要亲自动手做菜 |
| 👩‍🍳 帮厨 | 保持手牌4-6张；给主厨喂料效率最高 |
| 🧹 洗碗工 | 前两轮专职收集，第3轮后开始清理事故 |
| 🏃 跑堂 | 站在地图中央位置；保持移动减少等待 |
| 👩‍🍳 调味师 | 手牌多样化；关键时刻替代缺少的食材 |
| 👴 老前辈 | 专注完成5分以上的高分订单 |
| 🎪 实习生 | 保存重投次数用于最后3轮 |

### 8.3 避免常见错误

❌ **不要**：一个人同时尝试完成多个订单
❌ **不要**：收集太多食材导致手牌溢出
❌ **不要**：忽视厨房压力，过度追求速度
❌ **不要**：忘记使用角色技能

✅ **应该**：每轮至少完成1个订单
✅ **应该**：保持手牌在5-7张
✅ **应该**：压力达到6级以上开始谨慎行动
✅ **应该**：提前规划下一轮要完成的订单

## 9. 变体规则

### 变体1：极速模式 ⚡
- 目标：在5分钟内尽可能多得分
- 修改：每轮缩短至45秒，移除事故卡
- 适合：时间有限或想快速开局

### 变体2：竞争模式 🏆
- 目标：个人积分最高
- 修改：每个人独立完成订单，积分归个人
- 特殊：可以"截胡"他人正在完成的订单
- 适合：6人以上，想要对抗性

### 变体3：困难模式 💀
- 修改：每轮翻开2张事故卡
- 目标分数×1.5
- 厨房压力上限降低至8
- 适合：想要挑战的老玩家

### 变体4：亲子模式 👨‍👩‍👧
- 修改：移除所有事故卡
- 目标分数×0.7
- 每轮额外增加15秒
- 适合：与儿童游玩

### 变体5：角色固定模式 🎭
- 修改：指定角色而非抽签
- 可以根据玩家性格选择最适合的角色
- 适合：熟悉游戏后想优化体验

### 变体6：双人模式 👫
- 人数固定为2人
- 每人控制2个角色（棋子公司）
- 目标分数降低至15+人数×2
- 适合：夫妻或朋友二人世界

## 10. 常见问题

### Q1：游戏过程中可以查看其他玩家的手牌吗？
**A：** 不可以。除非玩家主动展示，否则手牌保密。这是游戏的基础规则，确保每个玩家都有信息获取的乐趣。

### Q2：如果需要的食材已经被其他玩家拿走怎么办？
**A：** 可以尝试与其协商交换。如果对方不愿意，可以：
1. 等待其完成订单后弃置该食材
2. 调整策略，尝试完成其他订单
3. 调味师角色可以使用任意食材替代（需消耗技能）

### Q3：事故卡的效果可以叠加吗？
**A：** 可以。如果同时发生多个事故（如打翻+起火），效果同时生效。玩家需要同时应对所有事故，这增加了游戏的挑战性和戏剧性。

### Q4：计时沙漏翻转后，玩家还没完成行动怎么办？
**A：** 该玩家跳过本回合，无法补做。跳过2次以上的玩家获得"摸鱼王"称号（笑称，非正式惩罚）。

### Q5：如果所有人都觉得游戏太简单/太难怎么办？
**A：** 建议调整以下参数：
- 太简单：增加目标分数、使用困难模式、减少计时
- 太难：降低目标分数、使用亲子模式、增加计时

### Q6：订单卡可以拒绝接受吗？
**A：** 不可以。所有翻开的订单必须完成或被其他订单替代。替代时需要将原订单翻至背面，并扣除1分。

### Q7：角色技能可以在一回合内多次使用吗？
**A：** 大部分技能每回合限用1次。特殊技能（如调味师的替代功能）可能需要消耗金币，可根据规则说明多次使用。

### Q8：游戏结束时的分数如何计算？
**A：** 
- 完成订单：按订单分值计分
- 额外奖励：角色技能奖励（如老前辈每单+1）
- 扣分项：订单替代-1分/次、重大事故-2分/次
- 最终分 = 完成订单总分 + 奖励 - 扣分
