# 台词之王 (Line Master)

## 游戏概述

《台词之王》是一款即兴表演+观众投票的派对桌游。每轮一位玩家担任"演员"，从牌堆中随机抽取**场景卡**、**角色卡**和**情绪卡**，在30秒准备后即兴说出一句符合三张卡的"台词"。其余玩家可以打出"抢戏卡"争夺表演权，最后所有玩家用掌声代币投票——谁的表演最精彩，谁就赢得奥斯卡奖杯！

无论你是社牛还是戏精，是段子手还是冷面笑匠，这款游戏都能让全场笑翻。没有标准答案，全靠临场发挥和观众缘！

**适合场景**：朋友聚会、团建破冰、家庭娱乐、即兴表演练习

## 游戏规则

### 准备阶段

1. **布置牌堆**：将场景卡、角色卡、情绪卡分别洗匀，面朝下放在桌面中央，形成三个抽牌堆
2. **分发导演卡**：将导演卡洗匀，每位玩家抽取**2张**作为手牌（剩余放在一旁备用）
3. **分发掌声代币**：每位玩家获得**5枚**掌声代币
4. **确定首轮演员**：最近看过电影或剧集的人担任第一轮演员（也可石头剪刀布决定）
5. **设定回合数**：3-6人每人表演3轮，7-8人每人表演2轮。总回合数 = 玩家人数 × 每人轮数

### 游戏流程

每轮游戏分为以下5个阶段：

#### 第一阶段：抽卡 📥

本轮演员从三个牌堆各抽取1张卡，共获得**场景卡 + 角色卡 + 情绪卡**。演员将三张卡面朝上展示给所有人看。

> 示例：演员抽到「S04第一次约会停电」+「C04程序员」+「E07阴阳怪气」

#### 第二阶段：导演干预 🎥

从演员左手边的玩家开始，顺时针依次，每位玩家可以打出1张导演卡（也可以选择不出）：

- **换角卡**：打出后，演员必须将指定的那张卡放回牌堆底，重新抽取一张
- **加戏卡**：打出后，演员必须额外完成指定条件
- 此阶段其他导演卡暂不可用

#### 第三阶段：筹备 ⏱️

演员有**30秒**时间思考自己的台词。其他玩家在此期间可以小声讨论，但不能给演员建议（除非你打算投他票）。

30秒倒计时由任意非演员玩家计时（建议用手机）。

#### 第四阶段：表演 🎤

演员必须起立（隆重感！），用一句或多句话演绎符合三张卡的**台词**。

**表演规则**：
- 台词不限制长度，推荐1-5句话
- 不能只是念卡上的文字，必须有创造性
- 可以使用肢体语言和表情辅助
- 必须体现出情绪卡的情绪
- 必须符合角色卡的身份口吻
- 必须贴合场景卡的场景设定

> 示例表演（程序员+停电约会+阴阳怪气）：
> *推了推不存在的眼镜* "哎呀，原来你们人类约会还需要灯光的吗？我以为大家都能像我们程序员一样在黑暗中盲打呢——不过也是，不是谁都能在命令行里找到对象的。"

#### 第五阶段：抢戏 🎬

演员表演结束后，其他玩家可以打出**抢戏卡**来争抢表演权：

1. 任意持有抢戏卡的玩家立刻说"我来！"
2. 打出的抢戏卡放入弃牌堆
3. 该玩家用**同样的三张卡**即兴表演自己的版本（无准备时间）
4. 抢戏不限人数，但每人只能抢一次
5. 同一轮最多允许3人抢戏（包括原演员共4个版本）

#### 第六阶段：投票 🗳️

所有表演结束后，进入投票：

1. 所有玩家（包括演员自己）闭眼，然后将想投出的掌声代币握在手中
2. 计时3秒后所有人同时亮出手中的代币
3. 你可以把代币全部投给一个人，也可以分散投给多人
4. **剪辑卡**：此时可以打出，取消指定玩家的投票资格
5. **颁奖卡**：统分前打出，自己额外获得1个掌声

**重要**：掌声代币一旦投出就不再返还。请合理分配你的5枚代币！

#### 第七阶段：颁奖 🏆

- 计算每位表演者获得的掌声代币
- 获得最多的表演者赢得本轮**奥斯卡奖杯** 🏆
- 如果平票，当前持有奥斯卡奖杯最多的玩家从平票者中选出一人
- 表演者收回投出的代币存好；非表演者投出的代币由表演者收下

**本轮示例完整流程**：

```
玩家A是演员，抽到：
  🎬 S04 - 第一次约会停电
  🎭 C04 - 程序员  
  😄 E07 - 阴阳怪气

玩家B打出换角卡，将S04换成了S18（读心术发现老板在想鱼香肉丝）。
玩家C打出加戏卡，要求A必须用说唱方式表演。

A用30秒准备了台词，起立表演了一段阴阳怪气的程序员Rap。
玩家D打出抢戏卡，用同样的卡表演了自己的版本。

投票：A获得3个掌声，D获得2个掌声。
A赢得奥斯卡奖杯！🏆
```

### 胜利条件

游戏进行完所有预定的回合后，**奥斯卡奖杯最多**的玩家获胜！

如果奖杯数相同，则比较：获得的总掌声代币数（包括从别人那里收到的）→ 打出导演卡的数量 → 石头剪刀布决胜。

游戏结束后，拥有最多奥斯卡奖杯的玩家获得 **"全场最佳演员"** 称号！

## 完整游戏流程示例

### 示例一：3人游戏（每人3轮，共9轮）

**参与者**：小明、小红、小刚

#### 准备阶段
- 场景卡、角色卡、情绪卡分别洗匀面朝下放好
- 每人抽2张导演卡：小明（抢戏+换角）、小红（加戏+颁奖）、小刚（抢戏+剪辑）
- 每人分5枚掌声代币
- 小明是首轮演员（昨晚刚看了电影）

#### 第1轮：小明的回合
- **抽卡**：S22（猫说话揭发秘密）+ C28（巫师）+ E11（鬼鬼祟祟）
- **干预**：无人出卡
- **准备**：30秒→思考台词
- **表演**：小明蹲下来压低声音对空气说："咪咪，咱们不是说好了吗，我把你从麻瓜世界救出来，你把我的实验失败记录销毁——你怎么能去找房东告状说我在阳台养龙呢？"
- **投票**：小红投2个，小刚投1个，小明获得3个掌声 → 🏆 ×1

#### 第2轮：小红的回合
- **抽卡**：S09（给暗恋对象发错消息）+ C06（公交车司机）+ E01（深情款款）
- **干预**：小刚打出换角卡，把C06换成了C23（巨人）
- **准备**：30秒思考
- **表演**：小红踮起脚尖，弯下腰，用震耳欲聋的气声说："对不起啊，刚才那条'我暗恋你很久了'本来是发给隔壁写字楼的小张的……但是你既然看到了，那……要不要来我树屋喝杯茶？我家天花板很高的。"
- **抢戏**：小明打出抢戏卡，也表演了一个版本
- **投票**：小红获2票，小明获1票 → 小红 🏆 ×1

（后续7轮略）

**最终结果**：小红4奖杯，小明3奖杯，小刚2奖杯 → 🎉 **小红获得"全场最佳演员"！**

### 示例二：加戏卡+剪辑卡的连锁

**场景**：玩家A表演完，获得不错的反响。但玩家B打出了关键操作：

1. 玩家B打出**剪辑卡**，取消了最大竞争对手玩家C的投票资格
2. 玩家B又打出了**颁奖卡**，自己额外获得1枚掌声
3. 最终，B以1票优势险胜，拿下本轮奖杯！

这个连锁展示了导演卡的策略深度——不仅要会演，还要会玩导演卡！

## 新手策略提示

### 🟢 表演技巧
1. **别追求完美**：重点是搞笑和创意，不是演技！大家的掌声从来不给"最标准"的，而是给"最有趣的"
2. **反差萌**：如果抽到严肃的场景+搞笑角色+悲伤情绪，利用这种反差往往效果拔群
3. **加一句话就够了**：不需要长篇大论，一句精准的吐槽胜过十句废话
4. **利用角色特征**：你是幽灵就要飘，是机器人就要僵硬，是巨人就俯视——这些细节让表演更生动
5. **不怕尴尬**：越放得开越有趣！反正大家都一起丢脸

### 🟡 投票策略
1. **别一上来就全投**：5枚掌声代币用完全部轮次，早期保守、关键时刻加注
2. **投给值得的表演**：即使和某人是好朋友，也要投给真正精彩的表演——公平更容易赢得尊重
3. **分散投资**：如果本轮有多个精彩表演，可以分散投票，不一定押一个人
4. **观察对手代币**：注意谁还有代币，判断他们会在哪轮发力
5. **自己表演时也投票**：你投出的掌声代币会变成你自己的战利品！但要确保值得

### 🔴 导演卡策略
1. **抢戏卡**：留着在关键轮次使用——当对手表演精彩、你确信自己能超越时
2. **换角卡**：在对手抽到"完美组合"时使用，打乱他的节奏
3. **加戏卡**：加戏条件越刁钻越好，但如果对手完成了，你会帮他得分
4. **剪辑卡**：在投票前观察局势，取消最可能投给对手的玩家的资格
5. **颁奖卡**：在你自己表演的那轮使用最划算，相当于稳拿1票

## 变体规则

### 变体一：团队模式（推荐6-8人使用）
- 玩家分为两队，每队2-4人
- 每轮每队各派1人表演（共2人），使用相同的三张卡
- 队内其他成员可以帮队友想台词（30秒准备期间）
- 投票时观众投给"更精彩的队"
- 获胜队伍全队每人获得1座奖杯
- 游戏结束时，奖杯总数多的队伍获胜

### 变体二：极限挑战
- 准备时间缩短为**10秒**
- 演员不能使用任何肢体语言（只靠声音和台词）
- 每张导演卡只能使用一次，用过即弃
- 适合有经验的玩家挑战

### 变体三：主题之夜
- 开始前大家约定一个主题（如"港片""科幻""武侠""言情"）
- 所有场景卡和角色卡的表演必须贴合这个主题
- 表演不符合主题的演员本轮自动0票
- 适合影迷/剧迷聚会

### 变体四：编导模式（竞技性增强）
- 取消投票代币，改用**1-10分打分制**
- 每轮由非演出玩家担任"评委"
- 评委从三个维度打分：**创意性（4分）、表演力（3分）、贴合度（3分）**
- 累计积分最高者获胜
- 适合"我受不了朋友偏心"的严肃玩家

## 常见问题

**Q：如果抽到的三张卡完全矛盾怎么办？**
A：这正是游戏的乐趣所在！最矛盾的组合往往催生最搞笑的台词。比如"深情款款的僵尸在超市排队"——这种反差恰恰是最容易出彩的。如果真的毫无头绪，可以请求更换其中一张（视为自动消耗自己的一次表演权）。

**Q：我社恐怎么办？**
A：完全理解！你可以选择：
- 先用抢戏卡预热，看别人表演两轮建立信心
- 台词不必长，一句话也行
- 或者使用团队模式变体，队友帮你分担压力
- 其实这个游戏最社恐的人往往最出彩——反差感就是笑点！

**Q：有人故意不投给某人怎么办（针对特定玩家）？**
A：这是社交游戏的一部分。但如果真的影响体验，建议启用变体四（编导打分模式），用客观标准取代主观投票。

**Q：导演卡可以用多张吗？**
A：每轮每位玩家只能打出1张导演卡，但不同玩家可以在同一轮打出不同的卡。抢戏环节可以有多人抢戏。

**Q：抢戏卡用完了怎么办？**
A：抢戏卡共5张，用完后放入弃牌堆。每局游戏的导演卡总量有限（15张），用完后不再补充。这也是策略的一部分——早点用还是留到关键时刻。

**Q：演员可以给自己投票吗？**
A：可以！演员投出的掌声代币也会计入自己的得票。但要注意，你的5枚代币是整局游戏共享的——今天投给自己，后面轮次就少一张底牌。

**Q：掌声代币用完了还能投票吗？**
A：不能。一旦5枚代币全部投出，你在后续轮次中无法再投票。所以合理分配非常重要。

**Q：可以要求演员重来吗？**
A：不可以。表演只有一次机会——就像真实片场一样，NG是不存在的（除非被抢戏卡覆盖）。
