# 时间特工 (Time Agents)

## 🎮 游戏概述

《时间特工》是一款充满悬念与推理的派对桌游。在2147年，时间旅行技术已被发明，但历史正被神秘势力篡改。玩家们扮演被派遣到各个历史节点的时间特工，任务是在有限时间内修复错乱的时间线。

然而，队伍中隐藏着**时间破坏者**——他们伪装成特工，暗中篡改历史。特工们必须在时间耗尽前找出破坏者，同时修复时间线；而破坏者则要巧妙隐藏身份，让历史走向崩溃。

这款游戏融合了身份推理、实时决策和因果连锁机制，每局游戏都充满紧张刺激的对抗与心理博弈。

## 🗺️ 时间线版图详解

时间线版图展示了5个关键历史节点，从左到右依次是：

```
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│  古代   │ 中世纪  │ 工业革命│  现代   │  未来   │
│ 🏛️ BC  │ ⚔️ 500  │ ⚙️ 1800 │ 🏙️ 1900 │ 🚀 2100 │
│  事件1  │  事件2  │  事件3  │  事件4  │  事件5  │
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

### 版图特点

1. **因果链设计**：改变古代事件会影响后续所有事件
2. **状态追踪**：每个事件有3种状态（稳定🟢/混乱🟡/崩溃🔴）
3. **连锁反应**：调整一个事件可能导致其他事件自动变化
4. **时间压力**：每轮只有60秒讨论时间

### 事件状态说明

| 状态 | 图示 | 说明 | 影响 |
| :--- | :--- | :--- | :--- |
| **稳定** | 🟢 | 历史正常发展 | 对守护者有利 |
| **混乱** | 🟡 | 历史出现偏差 | 需要修复 |
| **崩溃** | 🔴 | 历史被严重篡改 | 对破坏者有利 |

### 因果链规则

```
如果 古代事件 被调整：
    → 中世纪事件 自动变化
    → 工业革命事件 可能变化
    → 现代事件 可能变化
    → 未来事件 可能变化

如果 中世纪事件 被调整：
    → 工业革命事件 自动变化
    → 现代事件 可能变化
    → 未来事件 可能变化

以此类推...
```

**关键规则**：调整越早的事件，影响范围越大！

## 🎲 游戏规则

### 准备阶段

1. **布置版图**：将时间线版图放在桌面中央
2. **抽取事件**：从每个时代的5张事件卡中随机抽取1张，按顺序放在版图上
3. **设置初始状态**：将所有事件标记为"混乱"状态（🟡）
4. **分发身份**：
   - 洗混身份卡，每位玩家秘密抽取1张
   - 根据玩家人数确定守护者/破坏者比例（见物料清单）
   - **重要**：身份保密，不要告诉任何人！
5. **分发信息碎片**：
   - 根据玩家人数，每人获得4-5张信息碎片卡
   - 玩家只能看自己的碎片，**不能展示给他人**
   - 可以通过语言描述碎片内容，但不能直接展示卡片
6. **放置状态指示物**：在每个事件位置放置"混乱"状态标记
7. **确定起始玩家**：随机选择一位玩家作为时间领航员

### 游戏流程

游戏进行若干轮，每轮包含以下阶段：

#### 阶段1：信息交流（60秒）⏱️

- 启动60秒计时器
- 玩家可以自由交流，分享信息碎片的内容
- **限制**：
  - 不能直接展示卡片
  - 可以说谎（破坏者常用策略）
  - 时间到必须立即停止讨论

#### 阶段2：投票决策（30秒）🗳️

- 玩家同时投票选择要调整的事件（古代/中世纪/工业革命/现代/未来）
- 得票最多的事件将被调整
- 平票时，由时间领航员决定

#### 阶段3：事件调整

根据投票结果，调整选定的事件：

| 当前状态 | 调整后状态 | 效果 |
| :--- | :--- | :--- |
| 混乱🟡 | 稳定🟢 | 守护者目标，触发正向连锁 |
| 混乱🟡 | 崩溃🔴 | 破坏者目标，触发负向连锁 |
| 稳定🟢 | 混乱🟡 | 破坏者可以破坏 |
| 崩溃🔴 | 混乱🟡 | 守护者可以修复 |

**连锁反应**：调整事件后，根据因果链规则，后续事件可能自动变化。

#### 阶段4：身份调查（可选）🔍

- 玩家可以选择花费1个投票标记，调查另一位玩家的身份
- 被调查玩家必须展示身份卡给调查者看
- **注意**：伪装大师会显示守护者身份！

#### 阶段5：轮换领航员

- 时间领航员顺时针传递给下一位玩家
- 开始新一轮

### 胜利条件

#### 守护者胜利条件（满足任一）

1. **完美修复**：所有5个事件都变为"稳定"状态（🟢），且至少找出1名破坏者
2. **揪出内鬼**：通过调查确认所有破坏者身份

#### 破坏者胜利条件（满足任一）

1. **时间崩溃**：3个或更多事件变为"崩溃"状态（🔴）
2. **隐藏成功**：游戏进行10轮后仍未被全部找出

### 游戏结束

当满足任一胜利条件时，游戏立即结束，揭示所有身份，宣布获胜方。

## 📖 完整游戏流程示例

**游戏设置**：5人游戏（4守护者 + 1破坏者）

**初始状态**：
- 时间领航员：玩家A
- 所有事件：混乱状态（🟡）
- 玩家C是破坏者（隐藏身份）

**第1轮**：
1. **信息交流**（60秒）：
   - 玩家A："我的碎片显示古代事件可以影响未来3个事件"
   - 玩家B："我知道中世纪事件的当前状态"
   - 玩家C（破坏者，说谎）："我的碎片显示调整未来事件最安全"
   - 玩家D："我们应该先修复古代事件，影响最大"
   - 玩家E："同意D的观点"

2. **投票决策**：
   - 玩家A、D、E投票：古代事件
   - 玩家B投票：中世纪事件
   - 玩家C投票：未来事件（误导）
   - 结果：古代事件得票最多（3票）

3. **事件调整**：
   - 古代事件从混乱🟡变为稳定🟢
   - **连锁反应**：中世纪、工业革命事件自动变为稳定🟢
   - 守护者阵营大优！

4. **身份调查**：
   - 玩家D调查玩家C
   - 玩家C展示守护者身份（伪装大师能力）
   - 玩家D被误导

5. **轮换领航员**：玩家B成为新领航员

**第2轮**：
1. **信息交流**：
   - 玩家C（破坏者）："我们应该调整现代事件，让它稳定"
   - 实际上玩家C计划让它崩溃

2. **投票决策**：
   - 现代事件得票最多

3. **事件调整**：
   - 玩家C暗中操作，现代事件从混乱🟡变为崩溃🔴
   - **连锁反应**：未来事件自动变为崩溃🔴
   - 现在2个事件崩溃，破坏者接近胜利！

4. **讨论分析**：
   - 玩家们开始怀疑有人故意破坏
   - 玩家D说C是守护者（被误导）
   - 真正的推理开始...

**后续轮次**：
- 守护者们试图修复崩溃的事件
- 破坏者继续伪装和破坏
- 通过信息碎片和投票行为推理身份
- 最终在第7轮，守护者们识破伪装，找出破坏者，修复所有事件，获得胜利！

## 💡 新手策略提示

### 守护者策略

1. **优先修复早期事件**：古代和中世纪事件影响范围最大
2. **交叉验证信息**：多个人描述同一事件的信息，对比是否一致
3. **观察投票模式**：总是投给不利于修复事件的人可能是破坏者
4. **合理使用调查**：投票标记有限，调查关键嫌疑人
5. **注意连锁反应**：调整事件前思考可能的后果

### 破坏者策略

1. **伪装成守护者**：假装积极修复事件，关键时刻破坏
2. **分散注意力**：引导大家讨论不重要的信息
3. **利用时间压力**：在60秒内快速发言，占据主导地位
4. **嫁祸他人**：让其他玩家看起来可疑
5. **把握时机**：前期伪装，后期集中破坏

### 通用技巧

1. **记住所有信息**：尝试记住每个人说的碎片内容
2. **分析因果链**：理解调整某个事件会影响哪些后续事件
3. **注意矛盾**：如果两个人的信息矛盾，至少一个人在说谎
4. **控制讨论节奏**：时间有限，高效沟通很重要
5. **不要过早暴露**：守护者不要过早展示强力身份卡

## 🔄 变体规则

### 简单模式（适合新手）

- 讨论时间延长到90秒
- 破坏者只有1人（无论玩家人数）
- 移除连锁反应机制，每个事件独立
- 增加1轮调查机会

### 困难模式（适合老手）

- 讨论时间缩短到45秒
- 破坏者可以私下交流（用纸条或手机）
- 增加"时间悖论"事件：随机触发，强制改变某个事件
- 隐藏部分信息碎片内容

### 团队对抗模式

- 分成两队（2v2或3v3）
- 每队有1名破坏者（只有队友知道身份）
- 两队竞争修复/破坏时间线
- 可以互相干扰对方的事件调整

### 剧情模式

- 加入主持人角色，负责讲述背景故事
- 每轮有随机剧情事件（如"时间风暴来袭，本轮不能调查"）
- 增加角色扮演元素

## ❓ 常见问题

**Q：破坏者可以主动暴露身份吗？**
A：可以，但通常不利于获胜。暴露后守护者会集中针对你。

**Q：如果60秒内无法达成共识怎么办？**
A：时间到后，由时间领航员直接决定调整哪个事件。

**Q：可以重复调查同一个人吗？**
A：可以，但每次调查都要消耗1个投票标记。

**Q：信息碎片卡用完了怎么办？**

A：守护者需要继续调查，直到找出破坏者才算完全胜利。
