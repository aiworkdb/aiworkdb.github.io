# 谣言工厂 (Rumor Factory)

## 1. 游戏概述

欢迎来到社交聚会的地狱笑话现场！《谣言工厂》是一款充满欢笑和创意的派对游戏。在游戏中，玩家们围坐一圈，通过传递和修改「谣言卡」来创造最离谱、最有趣的故事。

每位玩家轮流作为「谣言发起人」，从牌堆抽取一个荒诞的起始谣言，然后悄悄告诉下一位玩家。每位玩家听到后，可以用「改编卡」进行加工改造，让故事变得更离谱、更搞笑。最后，所有玩家一起投票，选出「最离谱故事」和「最忠于原版」的故事。

**适合人群**：喜欢社交、讲故事、不怕尴尬的玩家。适合聚会、团建、家庭游戏夜。4-8人游玩，轻松愉快，笑到肚子疼！

## 5. 游戏规则

### 准备阶段

1. **围坐一圈**：所有玩家围坐成一个圆圈，确保每个人都能悄悄说话
2. **洗混卡组**：将谣言卡、改编卡分别洗混，面朝下放置
3. **分发卡牌**：每位玩家抽取2张改编卡作为手牌
4. **选择起始玩家**：抽签或自愿担任第一轮「谣言发起人」

### 游戏流程（每轮）

每轮由一位「谣言发起人」开始，按顺时针方向传递：

#### 步骤1：抽取谣言卡
- 发起人从谣言卡堆抽取1张，悄悄阅读（不要让别人看到）
- 发起人用30秒时间记住故事内容

#### 步骤2：传递谣言
- 发起人悄悄将故事告诉右边玩家（只能说一遍，不能重复）
- 接收玩家可以选择：
  - **直接传递**：不改编，原样传给下一位
  - **使用改编卡**：消耗1张改编卡，对故事进行改编后传递
  - **超级改编**：消耗2张改编卡，进行大幅度改编

#### 步骤3：讲述故事
- 故事传到发起人左边玩家后，由该玩家大声讲述最终版本的故事
- 所有玩家一起听，可以笑，可以吐槽

#### 步骤4：投票环节
- 所有玩家用投票卡选出：
  - 「最离谱故事」奖
  - 「最忠于原版」奖
  - 「最搞笑改编」奖
- 发起人揭晓原始谣言卡，大家一起对比

#### 步骤5：计分
- 根据投票结果发放分数
- 发起人补充1张新的改编卡（手牌上限3张）

### 胜利条件

游戏进行**N轮**（N=玩家人数），每轮1位发起人。

**最终计分**：
- 获得「最离谱故事」：发起人+3分，改编者各+1分
- 获得「最忠于原版」：发起人+2分，未改编者+2分
- 获得「最搞笑改编」：该改编者+3分
- 特殊成就卡：触发条件的玩家额外+2分

**最高分者获胜！** 如果平局，大家一起讲一个终极谣言来决定胜负！

## 6. 完整游戏流程示例

### 示例轮次：玩家A作为发起人

**初始状态**：
- 玩家A（发起人）抽取谣言卡：「外星人在菜市场买了一条会说话的鱼」
- 玩家A悄悄告诉右边的玩家B

**传递过程**：
1. **玩家B**：使用「地点迁移」改编卡，改为「外星人在**厕所**买了一条会说话的鱼」
2. **玩家C**：使用「角色替换」改编卡，改为「**奥特曼**在厕所买了一条会说话的鱼」
3. **玩家D**：使用「夸张放大」改编卡，改为「奥特曼在厕所买了**100条**会说话的鱼，鱼开始大合唱」
4. **玩家E**：未改编，直接传递

**最终讲述**：玩家E大声讲述：「奥特曼在厕所买了100条会说话的鱼，鱼开始大合唱」

**投票结果**：
- 「最离谱故事」：全票通过！玩家A（发起人）+3分，玩家B/C/D（改编者）各+1分
- 「最搞笑改编」：玩家D（夸张放大）+3分
- 原始故事揭晓后，大家笑得前仰后合

## 7. 新手策略提示

1. **改编要适度**：太离谱会让后续玩家无法继续，太保守又拿不到高分
2. **观察对手**：了解哪些玩家喜欢夸张改编，哪些喜欢忠于原版
3. **利用改编卡**：改编卡是得分利器，合理使用能让你的改编脱颖而出
4. **记住原版**：如果你是直接传递者，记住原版故事，可能获得「最忠于原版」奖
5. **幽默感第一**：游戏的目的是快乐，不要太计较分数
6. **敢于尝试**：越离谱的故事越容易拿高分，别怕出丑
7. **注意时间**：讲述时间有限，提前想好怎么讲

## 8. 变体规则

### 变体1：限时挑战模式
- 每轮传递时间限制为**15秒**
- 讲述时间限制为**20秒**
- 适合快节奏玩家，笑料更密集

### 变体2：团队对抗模式
- 4-8人分成2队，每队2-4人
- 队内可以商量如何改编
- 团队总分高者获胜

### 变体3：无卡牌模式
- 不使用改编卡，完全靠口才改编
- 每轮每位玩家必须至少改编1个细节
- 适合喜欢纯文字游戏的玩家

### 变体4：主题限制模式
- 每轮设定一个主题（如「古代」、「未来」、「童话」）
- 所有改编必须符合主题
- 增加挑战性和创意空间

### 变体5：反向传递模式
- 故事从发起人开始，逆时针传递
- 增加混乱度和不确定性

## 9. 常见问题

**Q：如果传递时听不清怎么办？**
A：可以礼貌地请求重复一遍，但发起人只能重复**一次**，且每次重复会被扣1分（开玩笑的惩罚）。

**Q：改编卡用完了怎么办？**
A：每轮结束后，所有玩家补充手牌至2张。如果卡堆用完，将弃牌堆洗混后继续使用。

**Q：可以完全不改编吗？**
A：可以！直接传递也是一种策略，有机会获得「最忠于原版」奖。

**Q：投票时可以投给自己吗？**
A：不可以！投票必须投给别人，但你可以游说其他玩家投给你。

**Q：故事传递时可以让别人帮忙提示吗？**
A：不可以！这是单人表演时间，其他人要保持安静，但可以准备爆米花观看。

**Q：如果大家都说「太离谱了」怎么办？**
A：那就对了！这正是游戏的乐趣所在。接受荒诞，享受欢笑。

**Q：游戏时间太短/太长怎么办？**
A：调整轮数即可。建议每人当1次发起人，这样每轮长度适中。
