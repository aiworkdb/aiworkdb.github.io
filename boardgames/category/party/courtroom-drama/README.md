# 法庭风云 (Courtroom Drama)

## 🎭 游戏概述

《法庭风云》是一款充满即兴表演与唇枪舌剑的派对桌游。玩家将化身为律师、证人、法官等法庭角色，围绕一个个离奇搞笑的案件展开激烈辩论。每回合抽取案件卡后，控方律师指控、辩方律师辩护、证人即兴作证，最后由法官和陪审团投票裁决。

这个世界上没有无聊的法庭——只有不够疯的律师。你是舌灿莲花的金牌律师？还是演技浮夸的戏精证人？又或是铁面无私的搞笑法官？在《法庭风云》中，法庭变成了舞台，每一次审判都是一场即兴戏剧。

本游戏融合角色扮演、即兴辩论和投票审判三大机制，无需复杂策略，重在表达能力和临场反应，非常适合朋友聚会、家庭娱乐和破冰社交。

## 🏛️ 法庭布局详解

虽然《法庭风云》不需要传统意义上的地图，但建议玩家按以下法庭布局就座，以营造沉浸感：

```
              ┌───────────────────────┐
              │      法官席 👨‍⚖️        │
              │  （审判席 · 主持庭审）    │
              └───────────┬───────────┘
                          │
         ┌────────────────┼────────────────┐
         │                │                │
    ┌────┴────┐     ┌────┴────┐     ┌────┴────┐
    │ 控方席  │     │ 证人席  │     │ 辩方席  │
    │ ⚖️→    │     │  👤     │     │  ⚖️←   │
    │ 检察官  │     │ 作证区  │     │ 辩护人  │
    └────┬────┘     └────┬────┘     └────┬────┘
         │               │               │
         └───────────────┼───────────────┘
                         │
              ┌──────────┴──────────┐
              │    陪审团席 🏛️       │
              │  （投票裁决区）        │
              └─────────────────────┘
```

### 区域说明

| 区域 | 图示 | 功能说明 | 占用玩家 |
| :--- | :--- | :--- | :--- |
| 法官席 | 👨‍⚖️ | 法官坐镇，负责敲槌、控制时间、引导流程 | 1人（法官角色） |
| 控方席 | ⚖️→ | 控方律师就座，负责提出指控和不利证据 | 1人（控方律师角色） |
| 证人席 | 👤 | 证人轮流坐到此处，即兴作证 | 1-3人（证人角色） |
| 辩方席 | ⚖️← | 辩方律师就座，负责反驳指控和提出有利证据 | 1人（辩方律师角色） |
| 陪审团席 | 🏛️ | 陪审员就座，聆听辩论后投票裁决 | 1-2人（陪审员角色） |

### 布局特点

法庭布局的核心是"对抗结构"——控方和辩方分列两侧，证人居中，法官高高在上，陪审团在最前方聆听。这种布局天然形成了戏剧张力，让每位玩家都有明确的"舞台位置"。实际游戏中，玩家只需围坐在桌边，心中记住各自的角色定位即可，无需严格按照物理位置就座。

| 属性 | 内容 |
| :--- | :--- |

## 📜 游戏规则

### 准备阶段

**第一步：打印和裁剪**
按照 [printables.html](printables.html) 打印所有卡牌和道具。建议使用200g以上卡纸打印，裁剪后塑封以延长使用寿命。裁剪完成后，按照卡牌类型分类放入收纳盒。

**第二步：洗牌和分类**
将所有卡牌按类型分成5堆：角色卡堆、案件卡堆、实物证据卡堆、人证证据卡堆、专家证据卡堆。每堆分别洗匀。再将反转卡单独组成反转卡堆。

**第三步：发放记分板**
每位玩家领取一张记分板（或用纸笔记录），在记分板上写下自己的名字。

**第四步：确定起始法官**
由年龄最大的玩家担任第一回合的法官（或者用"猜拳"决定）。

**第五步：抽取初始手牌**
每位玩家从反转卡堆中抽取2张反转卡，作为自己的初始手牌。反转卡需面朝下持有，不能让其他玩家看到。

### 游戏流程

每回合庭审分为以下7个阶段：

**阶段1：分配角色（约1分钟）**
法官从角色卡堆中随机抽取对应数量的角色卡并秘密发放。每位玩家查看自己的角色卡，不要让其他人知道（2人游戏时除外，2人游戏公开角色）。角色分配原则：控方律师和辩方律师各1人，证人1-3人（按玩家人数），法官1人，剩余玩家担任陪审员。法官自己不参与辩论。

**阶段2：宣读案件（约1分钟）**
法官从案件卡堆中抽取一张案件卡，大声宣读案件名称和案件描述。所有玩家围绕这个案件进入角色。

**阶段3：控方开庭（约2分钟）**
控方律师进行开场陈述（限时60秒），说明指控理由并出示1-2张证据卡。控方律师可以传唤一名证人，对证人提问（限时60秒）。证人需即兴编造证词回答提问。

**阶段4：辩方答辩（约2分钟）**
辩方律师进行反驳陈述（限时60秒），反驳控方的指控并出示1-2张证据卡。辩方律师可以传唤己方证人，对证人提问（限时60秒）。证人同样即兴作证。

**阶段5：自由辩论（约3分钟）**
所有律师和证人进入自由辩论环节。法官控制时间（每人每次发言不超过30秒），控辩双方可交替发言。此阶段可使用反转卡。辩方律师可以使用"反对"权力，法官裁定"反对"是否有效。

**阶段6：总结陈词（约2分钟）**
控方律师和辩方律师各进行60秒总结陈词。先由控方总结，再由辩方总结。此阶段不能使用反转卡和证据卡。

**阶段7：投票裁决（约2分钟）**
法官宣布投票开始。所有陪审员（和法官一起）同时投票：有罪或无罪。投票采用秘密投票方式（用裁决标记正反面表示）。法官统计票数，宣布裁决结果。

### 积分规则

| 结果 | 获得积分的玩家 | 积分 |
| :--- | :--- | :--- |
| 判决有罪 | 控方律师 | +3分 |
| 判决无罪 | 辩方律师 | +3分 |
| 判决与己方立场一致 | 每位陪审员 | +1分 |
| 最佳证人（法官评定） | 被选中的证人 | +2分 |
| 使用反转卡成功影响结果 | 反转卡使用者 | +1分 |
| 出示证据被法官驳回 | 证据出示方 | -1分 |

### 胜利条件

游戏进行5个回合（或根据玩家人数调整：4-5人5回合，6-8人4回合）。所有回合结束后，积分最高的玩家获胜，获得"最佳律师"称号。如果出现平分，则进行一个"加时回合"决出胜负。

### 示例一：日常案件庭审（4人局）

**玩家**：小明（法官）、小红（控方律师）、小刚（辩方律师）、小美（证人）

**阶段1：分配角色**
小明抽取角色卡并秘密发放。小红拿到"控方律师⚖️→"，小刚拿到"辩方律师⚖️←"，小美拿到"证人👤"，小明自己担任法官👨‍⚖️。

**阶段2：宣读案件**
小明翻开一张日常案件卡：**📁-01 吃掉室友蛋糕案**——"被告在凌晨2点偷吃了室友精心准备的生日蛋糕，室友要求赔偿精神损失费。"小明大声宣读后敲下木槌："现在开庭！"

**阶段3：控方开庭**
小红作为控方律师，开始60秒开场陈述："各位陪审员，请想象一下——你花了一整天精心制作的生日蛋糕，上面有奶油花、有巧克力装饰、有蜡烛……结果第二天早上起来一看，只剩下一个空盘子！被告不仅偷吃了蛋糕，还把奶油蹭得满冰箱都是！这不是简单的偷吃，这是对一个人生日最大的不尊重！"

小红随后出示一张实物证据卡——**📱手机短信**（控方有利），内容是："被告在案发当晚发了一条朋友圈：'今晚的蛋糕真好吃😂'。"小红传唤小美作证："请证人描述一下你当时看到了什么？"

小美即兴作证："我当时去厨房倒水，正好看到被告蹲在冰箱前，嘴里全是奶油，旁边就是那个蛋糕盒子！"

**阶段4：辩方答辩**
小刚作为辩方律师，开始60秒反驳："反对！控方的指控完全站不住脚！首先，被告根本没有'偷'——那个蛋糕本来就是室友说'大家分着吃'的！其次，被告的朋友圈写的是'今晚的蛋糕'，并没有说是室友的蛋糕，也许是被告自己买的蛋糕呢？而且我的当事人有严重的夜间梦游症，他根本不知道自己在做什么！"

小刚出示一张辩方实物证据卡——**🏥体检报告**，内容是："被告被诊断为严重梦游症患者。"小刚传唤小美到辩方证人席："证人，你确定看到的是被告在吃蛋糕，而不是在梦游？"

小美即兴编造："嗯……现在仔细想想，被告当时眼睛确实是闭着的！"

**阶段5：自由辩论**
小红和小刚激烈交锋。小红质疑："梦游还能精准定位到冰箱里的蛋糕？这也太巧了吧！"小刚反驳："梦游者的行为本就不可预测，这正是梦游症的特征！"小红使用了反转卡**🌟证据造假**，作废了小刚的体检报告。小刚立刻使用反转卡**💣法庭停电**，所有人闭眼10秒，期间小刚偷偷换了一张手牌。

小明法官维持秩序："反对无效！双方继续辩论！"

**阶段6：总结陈词**
小红总结："被告的朋友圈、奶油满冰箱、证人目击——所有的证据都指向一个结论：被告就是故意偷吃蛋糕的罪魁祸首！请判有罪！"

小刚总结："一个梦游症患者，一段不确定的目击证词，一张被作废的体检报告——控方的证据链充满了疑点！疑罪从无，请判无罪！"

**阶段7：投票裁决**
法官小明和小美（本回合兼陪审员）秘密投票。小明判定：有罪。小美判定：无罪。1票对1票，法官小明作为法官拥有最终裁决权，判定：**有罪**！

**积分结算**：
- 小红（控方律师）：+3分（有罪判决获胜）
- 小美（证人）：+2分（法官评定的最佳证人，因为她的"梦游"翻供制造了全场最佳笑点）
- 小刚（辩方律师）：+0分
- 小明（法官）：获得"最佳法官"称号的潜在加分

## 💡 新手策略提示

**给控方律师的建议**：开场陈述要抓眼球。与其念证据，不如讲一个故事。把案件变成一个让所有人都感同身受的场景，陪审员自然会被你打动。"想象一下你辛苦做的蛋糕被偷吃了"永远比"被告违反了XX条款"更有说服力。

**给辩方律师的建议**：不要正面硬杠控方。最好的辩护策略是"转移注意力"——找到案件中不合理的地方，放大疑点，让陪审员心里埋下"也许不是他干的"的种子。"反对"技能要谨慎使用，关键时刻一次就够了。

**给证人的建议**：即兴作证的核心是"自信"。不管你说什么，只要表情严肃、语气笃定，其他人就会相信。最有杀伤力的证词往往不是最离谱的，而是那些"听起来很合理但仔细一想全是漏洞"的。

**给法官的建议**：法官是全场的灵魂人物。敲槌要有气势，裁定要果断。好的法官既能维持秩序，又能在关键时刻"推波助澜"让全场笑起来。记住，你是导演，不是裁判——让庭审好看比让庭审公正更重要。

**给陪审员的建议**：作为陪审员，你的每一票都举足轻重。不要只听律师怎么说，更要观察证人的"演技"。那个演技最浮夸的证人，要么是真情流露，要么就是在撒谎——就看你怎么判断了。行使"尖锐提问"技能时要挑最致命的角度。

**反转卡使用时机**：不要在开场就把反转卡打光。反转卡最致命的用法是——在对方以为胜券在握的时候，掏出一张反转卡，瞬间翻盘。"证据造假"留给对方最强的一张证据，"法庭停电"留给最胶着的辩论时刻。

## 🔄 变体规则

### 变体一：速审模式（适合赶时间）
每回合省去"自由辩论"阶段，直接从控方开庭到辩方答辩再到投票。每轮发言时间缩短为30秒。整个游戏可以在15-20分钟内完成。

### 变体二：重审模式（适合高玩）
每个案件进行两轮审判。第一轮由一组控辩双方审判，第二轮由另一组控辩双方就同一案件进行"上诉审"。同一案件的两次审判结果可能完全不同，引发更多讨论。

### 变体三：观众投票（适合人多场合）
如果有超过8人想参与，可以让部分玩家担任"旁听观众"。观众虽然没有角色和积分，但在最终投票前拥有一次"观众投票权"——观众投票不计入正式结果，但会影响陪审员的心理。

### 变体四：主题夜（适合聚会活动）
预设一个主题，例如"古装法庭"（所有案件和角色换成古代背景）、"动物法庭"（所有角色变成动物）、"职场法庭"（只使用职场案件卡）。玩家可以配合主题准备简单的道具和装扮。

### 变体五：连续剧模式
多个回合的案件之间建立联系。例如第一回合的"被告"可能在第二回合成为"证人"或"律师"。随着"剧情"推进，人物关系越来越复杂，笑料也越来越多。

## ❓ 常见问题

**Q1：2人游戏怎么玩？**
A：2人游戏时，一名玩家担任控方律师，另一名担任辩方律师。法官职能由两位玩家轮流担任（每人当一回合法官），证人由两位玩家同时扮演（先给对方当证人，再给自己当证人）。虽然没有陪审团，但双方可以邀请在场的朋友甚至路人来当临时"陪审员"。

**Q2：证词可以完全胡编吗？**
A：当然可以！《法庭风云》的核心乐趣就在于即兴编造。唯一的"规则"是证词需要与案件主题相关——审理"蛋糕偷吃案"时，证人不能突然谈论外星人入侵。除此之外，你想说什么都可以，越离谱越好笑。

**Q3：如果律师不知道怎么辩护怎么办？**
A：别担心，这正是游戏的魅力所在！证据卡已经为你提供了弹药——照着证据卡上的内容说就行了。如果实在不知道说什么，可以试试"提问法"：不断向对方证人提出问题，让证人帮你"制造"论点。记住，这里没有"错误答案"，只有"还不够好笑的答案"。

**Q4：反转卡什么时候使用？**
A：反转卡可以在"阶段3（控方开庭）"到"阶段7（投票裁决）"之间的任意时刻使用。但有几个限制：每个玩家每回合最多使用2张反转卡；总结陈词阶段不能使用反转卡；同一张反转卡不能连续使用两次（即两张相同的卡不能在同一回合使用）。

**Q5：法官的权力有多大？**
A：法官拥有流程控制权，包括计时、裁定"反对"是否有效、评定最佳证人、以及在1:1票数时拥有最终裁决权。但法官不能偏袒任何一方（除非使用了"法官偏心"反转卡），也不能在辩论中替任何一方发言。好的法官应该像体育比赛的裁判——保持公平，同时让比赛精彩。

**Q6：积分平局怎么办？**
A：如果游戏结束时积分相同，进入"加时回合"。加时回合使用一张特殊的"终极案件卡"（从案件堆中随机抽取），所有平分玩家轮流担任控方和辩方，其他人投票决定。赢下加时回合的玩家获得额外2分，打破平局。

**Q7：证据卡用完了怎么办？**
A：每回合控方和辩方各抽2张证据卡，最多用4张，所以48张证据卡足够5个回合使用（每回合用4-6张）。如果确实用完了，将已使用的证据卡洗混后重新组成证据堆继续使用。

**Q8：可以自己编写案件卡吗？**
A：非常鼓励！游戏提供了20张案件卡作为基础，但最有趣的案件往往是玩家自己编写的。你可以准备空白卡片，让每位玩家在游戏开始前各编写一张"自创案件卡"，然后混入案件堆中。游戏结束后，大家投票选出"最佳原创案件"。
