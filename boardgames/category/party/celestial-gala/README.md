# 天庭年会 (Celestial Gala)

## 🎮 游戏概述

《天庭年会》是一款充满欢声笑语的派对桌游。玩家扮演各路神仙，参加玉皇大帝举办的天庭年度盛典。席间需献上贡品、拍对马屁、背后捅刀，最终以"面子值"定胜负！

游戏融合了手牌管理、社交谈判和轻度背叛机制，既有《西游记》神话底色，又融入现代职场年会的幽默梗。适合朋友聚会、公司团建、家庭娱乐，4-8人最佳。

## 🎲 游戏规则

### 准备阶段

1. **洗牌发牌**：将贡品卡、马屁卡、阴招卡、神通卡分开洗混，天劫卡单独放一边
2. **分配手牌**：每位玩家抽取4张手牌（贡品卡2张 + 随机2张）
3. **设置面子值**：所有玩家初始面子值为 **3**
4. **确定先手**：掷骰子，点数最高者担任首轮"值日天神"（代理玉帝职权）
5. **放置玉帝令牌**：令牌放在首位玩家面前

### 游戏流程（每个回合）

每回合包含 **5个阶段**，所有玩家依次行动：

#### 阶段1：天劫降临
- 翻出1张天劫卡，立即执行效果
- 天劫效果可能影响正在行动的玩家，也可能波及全场

#### 阶段2：贡品献礼（主要行动）
玩家选择以下行动之一：
- **献上贡品**：打出1张贡品卡，获得对应面子值（若有多人同时献贡，掷贡品骰子决定谁更受宠）
- **拍马屁**：打出1张马屁卡，立即获得面子（可被其他玩家打阴招打断）
- **行使神通**：打出1张神通卡，执行特殊效果

#### 阶段3：阴风阵阵（互动核心）
- 其他玩家可以对当前行动者打出阴招卡
- 阴招可连续打出（最多2张/人/回合）
- 被阴招针对的玩家可用神通卡或灵芝（灵品贡品）抵消

#### 阶段4：玉帝评判
- 值日天神（持令牌者）对争议事件做出仲裁（其实是随机抽签😂）
- 仲裁结果所有人必须接受（包括值日天神自己）

#### 阶段5：回合结束
- 补充手牌至4张
- 令牌传给下一位玩家
- 检查胜利条件

### 胜利条件

**常规模式**：当牌库耗尽或任意玩家面子值达到 **15点**，游戏立即结束，面子值最高的玩家获胜，封为"本年度最受宠神仙"！

**合作模式**：所有玩家共同对抗"天劫"，累计渡过8轮天劫即为全员获胜。

### 特殊规则

- **面子值为0**：不会出局，但本回合无法献贡品（太丢脸了）
- **神品贡品**：打出时需全场确认，其他玩家可联合打阴招阻止
- **平局处理**：面子值相同则持有神品贡品多者胜；再相同则今年资历老者（玩游戏次数多）胜

## 📖 完整游戏流程示例

### 示例1：常规回合（4人局）

**背景**：玩家A（二郎神，面子值5）、玩家B（哪吒，面子值4）、玩家C（嫦娥，面子值6）、玩家D（孙悟空，面子值3）。玩家A持值日令牌。

**阶段1：天劫降临**
- 翻开天劫卡"蟠桃会提前"🍑 → 本回合贡品卡面子值翻倍！

**阶段2：玩家A行动（献贡品）**
- 玩家A打出"定海神针"（仙品，原+4，翻倍后+8！）
- 玩家A面子值变为13，接近胜利！

**阶段3：阴风阵阵**
- 玩家B（哪吒）打出"散播谣言"📢 → 玩家A本回合面子值-2（实际+6）
- 玩家A使用"灵芝"🍄抵消！（灵芝效果：抵消1张阴招）
- 玩家C（嫦娥）打出"天庭热搜"🔥 → 玩家A本回合面子值减半（8→4）
- 玩家A无奈，面子值实际+4，变为9

**阶段4：玉帝评判**
- 值日天神（玩家A自己）仲裁：热搜有效，但自己吃了亏😂

**阶段5：回合结束**
- 玩家A补牌至4张，令牌传给玩家B
- 当前面子榜：C(6) > A(9) > B(4) > D(3)

---

### 示例2：马屁对决（关键时刻）

**背景**：玩家C（嫦娥，面子值13）即将获胜，其他玩家联手阻止。

**玩家C回合**：
1. 打出马屁卡"陛下比西天如来还耀眼！"✨（面子+4，但可被阴招打断）
2. 玩家B立即打出"告御状"📝 → 玩家C面子-2
3. 玩家D打出"神仙打架"💥 → 随机交换两名玩家面子值（C和A交换：C变9，A变13！）
4. 局面逆转！玩家A（二郎神）突然领跑！

## 💡 新手策略提示

1. **贡品要藏好**：神品贡品（混沌灵根/先天至宝）是制胜关键，别太早亮出来
2. **阴招要精准**：别乱打阴招，留着对付快要获胜的玩家
3. **马屁要果断**：马屁卡是快速涨面子的途径，但小心被打断
4. **神通要看准时机**：神通卡效果强大，留到关键时刻再用
5. **结盟与背叛**：派对游戏的核心是社交，可以口头结盟，但也要随时准备背叛😏
6. **天劫是变数**：天劫卡可能帮你可能害你，保持心态平和
7. **面子值管理**：不要太早领跑，容易被集火；也不要太落后，翻盘很难

## 🔄 变体规则

### 职场模式（成人向）
- 卡牌名称替换为职场术语：贡品→年终总结，马屁→拍领导马屁，阴招→背后打小报告
- 增加"加班卡"：本回合无法行动
- 适合公司团建（下班后玩，释放压力）

### 天庭狼人杀（高阶模式）
- 随机1-2名玩家成为"妖怪卧底"，胜利条件不同
- 卧底需暗中破坏其他玩家的关系，不被发现
- 增加"天眼通"阶段：玩家可投票揭发卧底

### 快速模式（15分钟）
- 初始手牌改为3张
- 胜利面子值改为10
- 去掉天劫卡，加快节奏

### 团队模式（4v4）
- 玩家分成"天庭系"和"佛界系"两队
- 队友间可传递手牌（每回合1次）
- 先有一队成员达到15面子的队伍获胜

## ❓ 常见问题

**Q：面子值可以为负数吗？**
A：不可以，面子值最低为0。但你可以继续被打阴招，只是不显示负数。

**Q：神通卡可以连续使用吗？**
A：每回合每位玩家只能使用1张神通卡。

**Q：如果手牌用完了怎么办？**
A：当牌库耗尽时，将弃牌堆洗混后重新使用。

**Q：可以同时打出多张贡品卡吗？**
A：常规不行，但使用神通卡"天神下凡"后可以本回合出2张贡品卡。

**Q：阴招卡可以对自己用吗？**
A：不可以，阴招卡只能对其他玩家使用（虽然有时候自己坑自己也很搞笑😂）。

**Q：如果多人同时达到15面子值怎么办？**
A：本回合结束后再加赛1轮，只许出马屁卡，面子值高者胜。

**Q：小孩子可以玩吗？**
A：可以！10岁以上完全没问题，游戏规则简单，还能锻炼社交能力。

**Q：没有卡纸，用普通纸打印可以吗？**
A：可以，但建议贴在硬纸板上增加手感，或者用透明胶带加固。
