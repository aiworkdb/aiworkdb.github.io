# 魔童闹海 (Nezha's Sea Rumble)

## 🎮 游戏概述

《魔童闹海》是一款灵感源自 2025 年春节档现象级动画电影《哪吒之魔童闹海》（全球影史票房第四、国产动画第一）的**派对类桌游**。

天庭降下"灵珠"与"魔丸"，陈塘关少年哪吒与东海龙族太子敖丙命数相连。一场搅动三界的"闹海"大赛就此展开——各位"神仙 / 龙族英豪"穿过**陈塘关 → 灵脉秘境 → 东海龙宫**，收集天地灵气、炼化神通法宝、避开龙王怒涛与天劫，抢先夺得传说中的**龙珠**与**万龙甲**，成为新一代"闹海魔童"！

这是一款适合朋友聚会、家庭娱乐的轻策略派对游戏：规则简单、回合极短、互动捣蛋拉满，新手也能一眼上手。

- **适合人群**：3-6 人聚会、亲子家庭、动漫粉丝、桌游新手
- **游戏基调**：欢乐、互坑、竞速、国风神话

## 🗺️ 游戏地图详解

游戏地图是一个 **7×7 的「陈塘关 → 东海龙宫」网格**。最左上角是起点**陈塘关 🏯**，最右下角是终点**东海龙宫核心 👑**。

### 地图布局概览

```
🏯  ⬛  🔥  ⬛  🌪️  ⬛  💧
⬛  ⚠️  ⬛  💧  ⬛  🏮  ⬛
🔥  ⬛  ⬛  ⚠️  ⬛  ⬛  🐉
⬛  💧  🔥  ⬛  🏮  ⚠️  ⬛
🌪️  ⬛  ⬛  🔥  ⬛  💧  ⬛
💧  🏮  ⚠️  ⬛  🌪️  ⬛  🐉
⬛  ⬛  💧  ⬛  ⬛  🔥  👑
```

图例：`🏯`起点·陈塘关｜`⬛`海面道路｜`🔥`火灵灵脉｜`💧`水灵灵脉｜`🌪️`风灵灵脉｜`⚠️`险地｜`🏮`法宝铺｜`🐉`龙宫宝物点｜`👑`东海龙宫核心（终点）

### 地形类型说明

| 地形 | 图示 | 颜色 | 说明 |
| :--- | :--- | :--- | :--- |
| **陈塘关·起点** | 🏯 | 黄色 | 所有玩家开始的位置，位于地图左上角 |
| **海面道路** | ⬛ | 浅蓝 | 安全的行进海面，没有特殊效果 |
| **火灵灵脉** | 🔥 | 橙红 | 停留获得 1 张火灵卡 |
| **水灵灵脉** | 💧 | 蓝色 | 停留获得 1 张水灵卡 |
| **风灵灵脉** | 🌪️ | 青色 | 停留获得 1 张风灵卡 |
| **险地** | ⚠️ | 红色 | 停留需抽 1 张陷阱卡执行 |
| **法宝铺** | 🏮 | 紫色 | 停留可炼化或抽取 1 张法宝卡 |
| **龙宫宝物点** | 🐉 | 金色 | 停留获得 1 张宝藏卡（任选） |
| **东海龙宫核心** | 👑 | 金红 | 终点！率先抵达者立即获胜 |

### 地图特点

1. **竞速结构**：从左上角到右下角存在一条蜿蜒主干路，最短路径约 12-14 步。
2. **资源均匀**：三种灵脉散布全图，鼓励玩家沿途收集灵气。
3. **捣蛋点多**：4 个险地分布在通往核心的关键位置，配合法宝可互相拉扯。
4. **法宝铺枢纽**：3 个法宝铺让玩家定期强化，是中后期反超的关键。
5. **资源统计**：火灵灵脉 5 个、水灵灵脉 6 个、风灵灵脉 3 个、险地 4 个、法宝铺 3 个、宝物点 2 个、终点 1 个。

### 移动规则

- 玩家只能**上下左右**移动，不能斜向移动。
- 可以踩在任意格子上，包括其他玩家的棋子所在格（互不阻挡）。
- 每回合移动步数 = 掷 1 颗骰子（1-6）+ 龙珠加成（若有）+ 法宝加成。
- 棋子停在哪一格，就结算哪一格的事件。

## 🃏 道具详解

### 资源卡（天地灵气）

资源卡是游戏最基础的道具，所有法宝都需要消耗灵气来炼化。

| 资源 | 图示 | 获得方式 | 用途 |
| :--- | :--- | :--- | :--- |
| **火灵** | 🔥 | 停在 🔥 火灵灵脉 | 炼化进攻 / 位移法宝（风火轮、火尖枪） |
| **水灵** | 💧 | 停在 💧 水灵灵脉 | 炼化防御 / 恢复法宝（乾坤圈、莲藕化身、七色宝莲） |
| **风灵** | 🌪️ | 停在 🌪️ 风灵灵脉 | 炼化位移法宝（混天绫） |

### 工具卡（神通法宝）

法宝卡可在**法宝铺 🏮**炼化（支付对应灵气）或免费抽取，是抢跑与互坑的核心。

| 法宝 | 图示 | 炼化所需灵气 | 使用效果 |
| :--- | :--- | :--- | :--- |
| **混天绫** | 🎗️ | 风灵×2 | 将任意一名对手棋子拉回 2 格（闹海捣蛋） |
| **风火轮** | 🛞 | 火灵×2 | 本回合额外移动 3 格 |
| **火尖枪** | 🔱 | 火灵×1 + 风灵×1 | 指定一名对手弃掉 2 张灵气卡 |
| **乾坤圈** | ⭕ | 水灵×2 | 获得 1 层护盾，免疫下一次陷阱 |
| **莲藕化身** | 🪷 | 水灵×2 + 火灵×1 | 若本回合被陷阱退回起点，可免疫一次（原地不动） |
| **七色宝莲** | 🪷✨ | 水灵×1 + 火灵×1 + 风灵×1 | 立即恢复，并额外从法宝库抽 1 张法宝卡 |

> 说明：法宝铺默认「免费抽 1 张法宝卡」；若想炼化**指定**法宝，则支付对应灵气并从公共法宝库取该卡。

### 陷阱卡（龙宫险阻）

停在**险地 ⚠️**需抽 1 张陷阱卡并执行；若持有乾坤圈护盾 / 万龙甲则可抵消一次。

| 陷阱类型 | 图示 | 效果 | 应对方法 |
| :--- | :--- | :--- | :--- |
| **龙王怒涛** | 🌊 | 后退 3 格 | 用乾坤圈护盾抵消 |
| **天劫** | ⚡ | 眩晕，跳过本回合 | 用莲藕化身免疫 |
| **龙宫结界** | 🔰 | 本回合无法移动 | 预留风火轮提前冲过 |
| **海妖之歌** | 🎵 | 随机失去 1 张灵气卡 | 用乾坤圈护盾抵消 |
| **业火** | 🔥 | 失去 1 张法宝卡（若无则失 1 灵气卡） | 保留法宝到关键时刻 |
| **迷踪阵** | 🌀 | 随机传送至地图任一格 | 无法预判，靠运气 |

### 宝藏卡（龙宫至宝）

在**龙宫宝物点 🐉**或**东海龙宫核心 👑**可获得，提供永久加成，也是胜利象征。

| 宝藏 | 图示 | 效果 |
| :--- | :--- | :--- |
| **龙珠** | 🐉💎 | 持有者每回合开始额外 +1 移动力（隐形加速器） |
| **万龙甲** | 🛡️ | 免疫 1 次陷阱（永久） |
| **宝莲灯** | 🏮 | 每回合恢复 1 张火灵灵气卡 |
| **混天绫碎片** | 🎗️ | 可免费施放 1 次混天绫 |
| **火尖枪·真** | 🔱 | 火尖枪强化：令对手弃 3 张灵气卡 |
| **风火轮核** | ⚙️ | 风火轮强化：额外移动 +1（共 +4） |

## 🎲 游戏规则

### 准备阶段

1. **布置地图**：将游戏地图平铺桌面中央。
2. **选择棋子**：每位玩家取一个颜色的棋子，全部放在起点 🏯 陈塘关。
3. **分发灵气**：每人获得 **2 张火灵 + 2 张水灵 + 1 张风灵** 作为初始灵气。
4. **放置卡牌**：
   - 陷阱卡洗混后，面朝下分别放在 4 个 ⚠️ 险地处；
   - 法宝卡洗混作公共法宝库，放在地图旁；
   - 宝藏卡洗混，放在 2 个 🐉 宝物点与 👑 核心处。
5. **决定先手**：掷骰子最大者先手，顺时针进行。

### 游戏流程（每个回合）

玩家轮流进行，每个回合包含以下步骤：

#### 1. 掷骰子移动
- 掷 1 颗骰子，点数 = 基础移动步数。
- 若持有**龙珠**，本回合额外 +1 步。
- 若使用**风火轮**，本回合额外 +3 步（强化版 +4）。
- 按上下左右移动棋子，停在目标格。

#### 2. 结算落点事件

| 落点类型 | 图示 | 操作 |
| :--- | :--- | :--- |
| **陈塘关·起点** | 🏯 | 无特殊效果 |
| **海面道路** | ⬛ | 无特殊效果 |
| **灵脉** | 🔥💧🌪️ | 获得对应灵气卡 1 张 |
| **法宝铺** | 🏮 | 免费抽 1 张法宝卡，或支付灵气炼化指定法宝 |
| **险地** | ⚠️ | 抽 1 张陷阱卡并执行（有护盾 / 万龙甲则抵消） |
| **龙宫宝物点** | 🐉 | 获得 1 张宝藏卡（多张时任选 1） |
| **东海龙宫核心** | 👑 | 🎉 立即获胜！ |

#### 3. 使用法宝（可选）
回合内任意时机可使用法宝，但**每件法宝每回合限用 1 次**：
- **混天绫**：拉一名对手后退 2 格
- **风火轮**：本回合 +3（核 +4）移动
- **火尖枪**：令一名对手弃 2（·真 3）张灵气卡
- **乾坤圈**：获得护盾，抵消下次陷阱
- **莲藕化身**：免疫一次被退回起点
- **七色宝莲**：恢复并额外抽 1 张法宝卡

#### 4. 回合结束
完成所有操作后，将回合交给下一位玩家。

### 胜利条件

- **率先抵达 👑 东海龙宫核心**的玩家立即获胜！
- 若多名玩家在同一回合满足（极少见），则**持有宝藏卡最多者**获胜。
- 宝物点的宝藏卡提供永久加成，是竞速的隐形加速器，建议沿途顺手收集。

## 📖 完整游戏流程示例

**【示例一：玩家 A 的正常回合】**
1. 玩家 A 掷骰得到 **5**，无加成 → 移动 5 步。
2. 沿路走到 🏮 **法宝铺**，免费抽 1 张法宝卡 → 拿到「风火轮」。
3. 立即使用风火轮，本回合**再移动 3 格**（共 8 格），停在海面。
4. 回合结束，轮到玩家 B。

**【示例二：吃陷阱 + 法宝反击 + 抢核心获胜】**
1. **玩家 B 回合**：掷骰得 3 → 移动 3 步停在 ⚠️ **险地**，抽陷阱「龙王怒涛」→ **后退 3 格**，被拉回起点附近。
2. **玩家 C 回合**：走到 🏮 法宝铺，支付风灵×2 炼化「混天绫」→ 对玩家 B 使用：将 B **再拉回 2 格**。
3. **玩家 A 回合**：掷骰得 6，因持有 🐉 **龙珠** +1 = 7 步 → 一路冲到 👑 **东海龙宫核心**。
4. 🎉 玩家 A 率先抵达核心，**立即获胜**！

## 💡 新手策略提示

1. **抢位移法宝**：优先拿「风火轮」「混天绫」等位移类法宝，是抢跑与反超的关键。
2. **路过法宝铺别犹豫**：落点结算时优先补强位移 / 干扰类法宝。
3. **攒火灵 + 风灵**：用来炼化「火尖枪」干扰领先对手，把他手里的灵气打空。
4. **留一张护盾**：经过 ⚠️ 险地前，最好留张「乾坤圈」护盾防身。
5. **龙珠是隐形加速器**：优先抢 🐉 宝物点拿龙珠，每回合白赚 1 步。
6. **派对核心是捣蛋**：用混天绫把领先对手拉回来，全场最欢乐的玩法！

## 🔄 变体规则

### 组队模式（2v2 / 3v3）
- 玩家两两分组，同队**共享灵气与法宝**。
- 同队玩家可互相使用法宝支援，先有任意队员抵达核心即全队获胜。

### 快速模式
- 开局每人直接发齐 3 张法宝卡，游戏时长压缩到约 15 分钟，适合茶歇短局。

### 困难模式
- 每个 ⚠️ 险地放 2 张陷阱卡（叠加触发）；
- 初始灵气减半（1 火 + 1 水 + 0 风）；
- 适合熟悉规则后的进阶挑战。

### 单人挑战
- 自我计时，目标在 **12 回合内**抵达东海龙宫核心；挑战成功可记录最少回合数。

## ❓ 常见问题

**Q：如果没有骰子怎么办？**
A：可以使用手机上的骰子 APP，或写 1-6 的纸条抽取代替。

**Q：灵气卡用完了怎么办？**
A：将弃牌堆（被打出的法宝消耗的灵气、被火尖枪打掉的灵气）洗混后重新使用。

**Q：可以斜着走吗？**
A：不可以，只能上下左右四个方向移动。

**Q：多人同时抵达核心怎么办？**
A：极少发生；若同回合多人满足，则持有宝藏卡最多者获胜。

**Q：法宝能无限用吗？**
A：每件法宝每回合限用 1 次；消耗型法宝（如混天绫、火尖枪）用后入弃牌堆，永久型（万龙甲、龙珠等）持续生效。

**Q：棋子能重叠吗？**
A：可以，多个棋子可以站在同一格上，互不阻挡。
