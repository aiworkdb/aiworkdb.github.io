# 桌游工具集

本目录包含用于创建和管理桌游的工具集合。

## 工具列表

### 1. 卡牌设计工具 (card-maker.html) — 专业版

一个功能丰富的网页卡牌设计工具，支持经典桌游卡牌美术风格，适合创建高品质的桌游卡牌。

#### 功能特性

- 🎨 **经典卡牌美术**：金色装饰边框、角落花纹、宝石风格数值标签
- 🃏 **三种卡牌模板**：经典模板（通用）、对战模板（角色卡）、资源模板（道具卡）
- ✦ **四级稀有度系统**：普通（灰）、精良（蓝光）、稀有（金光）、传说（彩虹流光动画）
- 🖼️ **四种背景纹理**：羊皮纸、暗夜魔法、战场铁甲、自然森林
- 🎨 **十二种颜色主题**：紫、红、蓝、绿、橙、粉、青、靛、琥珀、天蓝、黄绿、玫红
- 📊 **数值标签**：费用、攻击、防御、生命值，宝石风格圆形标签
- 😊 **48+ Emoji 图标** + 自定义图片上传
- 📦 **多卡组管理**：创建、切换、删除多个卡组
- 💾 **数据持久化**：localStorage 自动保存，关闭浏览器不丢失
- 📤 **JSON 导入导出**：完整的卡组数据交换
- 🖼️ **PNG 导出**：单张或批量导出高清卡牌图片（html2canvas）
- 🖨️ **打印优化**：带裁切线的 A4 排版打印
- ↩️ **撤销/重做**：Ctrl+Z / Ctrl+Y 支持

#### 使用方法

1. 打开 [card-maker.html](card-maker.html) 文件
2. 在左侧编辑面板设置卡牌属性：
   - **名称** / **类型** / **描述**
   - **模板**（经典 / 对战 / 资源）
   - **稀有度**（普通 / 精良 / 稀有 / 传说）
   - **图标**（48+ Emoji 或自定义图片）
   - **颜色主题**（12 种可选）
   - **背景纹理**（4 种可选）
   - **数值**（费用、攻击、防御、生命值）
3. 中间区域实时预览卡牌效果
4. 点击「+ 添加」将卡牌保存到当前卡组
5. 右侧查看已保存的卡牌列表，支持编辑、复制、删除
6. 顶部栏支持导出 JSON / 导入 JSON / 导出 PNG / 打印

#### 支持的卡牌类型

| 类型 | 说明 |
| :--- | :--- |
| 资源卡 | 游戏中的基础资源（如木材、石头、食物） |
| 工具卡 | 玩家可以使用的工具（如铲子、灯笼、绳索） |
| 陷阱卡 | 阻碍玩家的陷阱或障碍 |
| 宝藏卡 | 游戏的目标物品 |
| 行动卡 | 玩家可以执行的行动 |
| 事件卡 | 随机事件或特殊效果 |
| 角色卡 | 游戏中的角色或单位 |
| 法术卡 | 可施放的魔法或技能 |
| 装备卡 | 可装备的道具 |
| 场地卡 | 场地或环境效果 |

#### 打印建议

- 使用A4纸张打印，设置最小边距
- **务必勾选"背景图形"**选项
- 建议使用卡纸打印以增加耐用性
- 打印后沿裁切线剪裁，可塑封保存

## 技术说明

- 纯前端 HTML/CSS/JavaScript 单文件实现
- CDN 引用 html2canvas（PNG 导出）
- 数据存储在浏览器 localStorage
- 支持所有现代浏览器
