# 桌游工具集

本目录包含用于创建和管理桌游的工具集合。

## 工具列表

### 1. 卡牌生成器 (card-maker.html)

一个简单易用的网页工具，用于创建和打印桌游卡牌。

#### 功能特性

- 🎨 **可视化编辑**：实时预览卡牌效果
- 📝 **丰富选项**：支持自定义名称、图标、类型、描述和颜色
- 💾 **暂存列表**：保存多张卡牌进行批量管理
- 🖨️ **打印输出**：一键打印所有卡牌，适合实体游戏使用

#### 使用方法

1. 打开 [card-maker.html](card-maker.html) 文件
2. 在左侧编辑面板填写卡牌信息：
   - **卡牌名称**：输入卡牌的名称
   - **卡牌图标**：选择适合的图标（支持20+种图标）
   - **卡牌类型**：选择卡牌类型（资源卡、工具卡、陷阱卡、宝藏卡、行动卡、事件卡）
   - **卡牌描述**：输入卡牌的效果或说明
   - **卡牌颜色**：选择卡牌背景色调
3. 点击「添加卡牌」将卡牌加入打印列表
4. 在打印区域预览所有卡牌
5. 点击「打印卡牌」按钮打印

#### 支持的卡牌类型

| 类型 | 说明 |
| :--- | :--- |
| 资源卡 | 游戏中的基础资源（如木材、石头、食物） |
| 工具卡 | 玩家可以使用的工具（如铲子、灯笼、绳索） |
| 陷阱卡 | 阻碍玩家的陷阱或障碍 |
| 宝藏卡 | 游戏的目标物品 |
| 行动卡 | 玩家可以执行的行动 |
| 事件卡 | 随机事件或特殊效果 |

#### 打印建议

- 使用A4纸张打印
- 设置打印边距为最小
- **重要**：在打印对话框中务必**勾选"背景图形"**选项，否则卡牌的颜色背景将不会打印
- 建议使用卡纸打印以增加卡牌耐用性
- 打印后可以裁剪并塑封
